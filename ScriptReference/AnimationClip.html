Description

Toko animasi berbasis keyframe.

AnimasiClip digunakan oleh Animation untuk bermain animasi kembali.

Properties

empty Kembali benar jika klip animasi tidak memiliki kurva dan tidak ada acara.
events Acara Animasi untuk klip animasi ini.
frameRate Frame rate di mana keyframes sampel. Sitemap
hasGenericRootTransform Kembali benar jika Animasi memiliki animasi pada transformasi akar.
hasMotionCurves Kembali benar jika AnimasiClip memiliki kurva gerak akar.
hasMotionFloatCurves Kembali benar jika AnimasiClip memiliki kurva editor untuk gerakan akarnya.
hasRootCurves Kembali benar jika AnimasiClip memiliki kurva akar.
humanMotion Kembali benar jika animasi mengandung kurva yang menggerakkan rig humanoid.
legacy Set untuk benar jika AnimasiClip akan digunakan dengan komponen Animasi Legacy ( alih-alih Animator).
length Panjang animasi dalam detik. Sitemap
localBounds AABB dari Klip Animasi ini di ruang lokal komponen Animasi yang melekat juga.
wrapMode Mengatur mode pembungkus default yang digunakan dalam keadaan animasi.

Constructors

AnimasiClip Membuat klip animasi baru.

Metode publik

AddEvent Menambahkan acara animasi ke klip.
ClearCurves Jelas semua kurva dari klip.
EnsureQuaternionContinuity Kunci quaternion Realigns untuk memastikan jalur interpolasi terpendek.
SampleAnimation Contoh animasi pada waktu tertentu untuk setiap sifat animasi.
SetCurve Menetapkan kurva untuk menghidupkan properti tertentu.
