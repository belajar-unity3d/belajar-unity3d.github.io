Description

Grafik mengendalikan interaksi negara. Setiap state merujuk gerakan.

Properties

anyStatePosition Posisi node AnyState.
anyStateTransitions Daftar transisi AnyState.
behaviours Daftar Perilaku yang ditugaskan untuk mesin negara ini.
defaultState Keadaan bahwa mesin negara akan ada ketika memulai.
entryPosition Posisi node entri.
entryTransitions Daftar transisi masuk di mesin negara.
exitPosition Posisi node keluar.
parentStateMachinePosition Posisi node mesin negara induk. Hanya berlaku ketika dalam mesin negara hirarkis.
stateMachines Daftar mesin sub state.
states Daftar negara.

Metode publik

AddAnyStateTransition Fungsi utilitas untuk menambahkan transisi AnyState ke state atau statemachine yang ditentukan.
AddEntryTransition Fungsi utilitas untuk menambahkan transisi masuk ke pintu keluar dari mesin negara induk itu.
AddState Fungsi utilitas untuk menambahkan state ke mesin negara.
AddStateMachine Fungsi utilitas untuk menambahkan mesin negara ke mesin negara.
AddStateMachineBehaviour Menambahkan kelas perilaku mesin negara tipe T ke AnimatorStateMachine. Perhatikan bahwa tidak ada metode Hapus yang sesuai. Untuk menghilangkan perilaku mesin negara, gunakan objek. Login
AddStateMachineExitTransition Fungsi utilitas untuk menambahkan transisi keluar dari mesin negara sumber ke pintu keluar dari mesin negara induk itu.
AddStateMachineTransition Fungsi utilitas untuk menambahkan transisi keluar dari mesin negara sumber ke tujuan.
GetStateMachineTransitions Dapatkan daftar semua transisi mesin state outgoing dari mesin state tertentu.
MakeUniqueStateMachineName Membuat nama mesin negara yang unik dalam konteks mesin negara induk.
MakeUniqueStateName Membuat nama negara yang unik dalam konteks mesin negara induk.
RemoveAnyStateTransition Fungsi utilitas untuk menghilangkan transisi AnyState dari mesin negara.
RemoveEntryTransition Fungsi utilitas untuk menghilangkan transisi masuk dari mesin negara.
RemoveState Fungsi utilitas untuk menghilangkan keadaan dari mesin negara.
RemoveStateMachine Fungsi utilitas untuk menghapus mesin negara dari mesin negara induknya.
RemoveStateMachineTransition Fungsi utilitas untuk menghilangkan transisi keluar dari mesin negara sumber.
SetStateMachineTransitions Mengatur daftar semua transisi mesin state outgoing dari mesin state tertentu.
