Declaration

publik static string[] GetCoreUnityAssetPackNames();

Returns

Sitemap Mengembalikan array nama paket aset untuk paket aset Unity inti dengan jenis pengiriman cepat-ikuti atau on-demand. Jika Unity tidak membuat paket aset inti untuk aplikasi ini dengan jenis pengiriman yang disebutkan sebelumnya, atau jika plugin PlayCore hilang, ini mengembalikan array kosong.string[] Returns an array of asset pack names for core Unity asset packs with the fast-follow or on-demand delivery type. If Unity did not create any core asset packs for this application with previously mentioned delivery types, or if the PlayCore plugin is missing, this returns an empty array.

Description

Dapatkan nama setiap paket aset Unity inti yang dibangun untuk aplikasi ini yang menggunakan jenis pengiriman cepat-follow atau on-demand.

Paket aset Unity Core adalah paket aset yang dibuat secara otomatis ketika dibangun bundel aplikasi Android. Unity hanya membuat paket aset inti jika Anda mengaktifkan Split Application Binary di Pengaturan Pemain. Metode ini menggunakan API PlayCore, oleh karena itu hanya dapat mengembalikan nama paket aset yang menggunakan jenis pengiriman cepat-follow atau on-demand. Jika metode ini mengembalikan array kosong, untuk membedakan antara dua penyebab potensial, hubungi coreUnityAssetPacksDownloaded. Nama paket aset dikembalikan oleh metode ini dapat dilewatkan ke metode lain seperti QueryAssetPackState atau DownloadAssetPackAsync untuk mendapatkan informasi status atau untuk memulai download. Namun memanggil RemoveAssetPack dengan nama yang dikembalikan oleh metode ini tidak ada. Sitemap Juga: AndroidAssetPacks.coreUnityAssetPaketDownloaded, AndroidAssetPacks.DownloadAssetPackAsync, AndroidAssetPacks.GetAssetPackStateAsync.
