Unity Collaborate

Unity Collaborate adalah bagian dari Tim Unity. Tim Unity memungkinkan tim kecil untuk menyimpan, berbagi, dan menyinkronkan Proyek Unity di lingkungan yang dihosting cloud. Menggunakan Collaborate memungkinkan seluruh tim Anda untuk berkontribusi pada Proyek, terlepas dari lokasi atau peran. Untuk informasi lebih lanjut tentang Tim Unity, lihat situs web Unity3d.

Berkolaborasi memungkinkan Anda untuk mempublikasikan proyek Anda ke cloud untuk penyimpanan. Berkolaborasi mempertahankan sejarah versi yang diterbitkan dari Proyek Anda, memungkinkan Anda untuk mengembalikan file individu atau seluruh Proyek Anda ke negara sebelumnya.

Anda dapat menambahkan anggota tim ke Proyek Anda; memungkinkan Anda dan anggota tim untuk bekerja sama. Berkolaborasi terus memantau perubahan yang dilakukan oleh setiap anggota tim dan menampilkan lencana pada file yang telah berubah, tetapi belum dipublikasikan.

Anda dapat melihat perubahan. Anda kemudian dapat memilih untuk mengubah perubahan pada file Anda atau menerbitkannya dan menangani setiap konflik merge yang terjadi.

Collaborate dirancang untuk mendukung tim multi-disipliner (yaitu, tim dengan beragam pengembang, seniman, ahli audio dan spesialis lainnya) yang bekerja sama pada proyek.

Enabling Collaborate

Untuk mengaktifkan Collaborate, pilih Window > General > Services dari menu top editor untuk membuka jendela Services, kemudian beralih layanan Collaborate on.

Untuk meluncurkan Collaborate saat Anda bekerja, klik tombol Collab yang muncul di bar nav atas dari Editor Unity.

Versions

  • Berkolaborasi 1.3.0 mendukung versi Unity 2020.1 dan lebih tinggi.
  • Berkolaborasi 1.2.16 mendukung versi Unity 2017.1 ke 2019.3.
  • Collaborate adalah beta untuk versi Unity lebih rendah dari 2017.1.

Anda dapat memperoleh Collaborate sebagai paket untuk versi Unity 2018.3 atau lebih tinggi.

Versi kolaborate 3.0 diverifikasi dan dikemas dengan Mesin Unity. Untuk menggunakan Manajer Paket untuk mengubah versi Collaborate di Proyek Anda, ikuti langkah-langkah ini:

  1. Dalam Editor Unity, pilih Window > Package Manager dari menu atas untuk membuka jendela Package Manager.
  2. Pilih Unity Collaborate dari daftar paket.
  3. Di bawah Unity Collaborate, klik See all versions dan pilih versi paket yang diinginkan.
  4. Dalam pane kanan bawah dari jendela Package Manager, klik tombol Update to.

Note: Saat menggunakan paket Unity Collaborate, Anda masih harus mengaktifkan layanan melalui jendela Services untuk menghubungkan Proyek Anda ke layanan.
