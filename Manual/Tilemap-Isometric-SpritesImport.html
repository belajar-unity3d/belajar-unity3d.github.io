Saat mengimpor SpritesObjek grafis 2D. Jika Anda digunakan untuk bekerja di 3D, Sprites pada dasarnya hanya tekstur standar tetapi ada teknik khusus untuk menggabungkan dan mengelola tekstur sprite untuk efisiensi dan kenyamanan selama perkembangan. More info
Lihat di Glossary untuk digunakan di Isometric Tilemap, gunakan pengaturan yang disarankan berikut. Untuk informasi lebih lanjut tentang setiap pengaturan, lihat dokumentasi pada Jenis Tekstur: Sprite (2D dan UI).

  1. Texture Type - Mengatur ini menjadi Sprite (2D and UI). Jenis tekstur lain tidak didukung untuk Tilemaps.
  2. Sprite Mode - Mengatur ini ke Single jika tekstur hanya mengandung Sprite tunggal. Set ke Multiple jika mengandung beberapa tekstur Sprite, misalnya lembaran sprite dengan beberapa tekstur ubin.
  3. Pixels Per Unit (PPU) - Nilai ini adalah jumlah pixelsUnit terkecil dalam gambar komputer. Ukuran piksel tergantung pada resolusi layar Anda. Pencahayaan pixel dihitung pada setiap piksel layar. More info

Ini juga dipengaruhi oleh pengaturan Cell Size dari Grid yang mengandung Tilemap, yang menentukan berapa banyak unit Unity membuat satu Cell. Lihat contoh di bawah ini untuk melihat bagaimana nilai PPU dan pengaturan Grid’s Cell Size berinteraksi.
  4. Mesh Type - Set ke Tight untuk memastikan Tile Meshs mengikuti garis besar Sprites impor, dan ubin ditarik flush bersama di Tilemap. Karena bentuk berlian umum ubin Isometric, pengaturan ini menjadi Full Rect dapat mengakibatkan gambar ruang transparan di sudut ubin Isometric, dan tidak dianjurkan.
  5. Generate Physics Shape - Jika ubin tidak perlu berinteraksi dengan Fisika2D, maka jelaskan opsi ini. Tinggalkan opsi ini diaktifkan untuk menghasilkan Bentuk Fisika berdasarkan bentuk Sprite ubin, untuk digunakan dengan komponen Tilemap Collider. Untuk membuat Bentuk Fisika yang dihasilkan sesuai dengan sel Tilemap, pilih Aset Tile dan atur properti Jenis Collider ke Grid.

Dalam contoh di bawah ini, Sprite impor adalah gambar 256x128, dan Isometric Tilemap memiliki Cell Size dari (XYZ: 1, 0,5, 1) Unit Unity. Untuk membuat Sprite pas persis pada satu Sel dari Tilemap, atur nilai PPU untuk 256. Seluruh lebarnya kemudian sesuai dengan satu unit Unity, yang sama dengan lebar (nilai X: 1) dari satu sel.

Jika Sprite diatur ke nilai PPU 128, maka menjadi 2 (256px / 128) Unity unit lebar. Ini menyebabkan Sprite secara visual muncul untuk menutupi 2 Sel lebar ketika dicat di Tilemap. Namun, posisi Sel asli ubin tetap tidak berubah.

Setelah Sprites diimpor, perbaiki garis besar Sprites dengan membuka Editor Sprite untuk masing-masing dari mereka dan editingtheir outlines. Untuk Sprit di Isometric Tilemap, Anda harus mengatur Pivot Sprite sehingga ‘ground’ relatif terhadap Sprite.

Jika Tekstur diimpor dengan set Sprite Mode ke Multiple dan mengandung beberapa Sprites, kemudian edit garis besar setiap Sprite di Editor Sprite.
