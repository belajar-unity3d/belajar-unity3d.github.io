Paket jendela Manifest

Ketika Anda mengklik file package manifestSetiap paket memiliki manifest, yang menyediakan informasi tentang paket ke Manajer Paket. Manifest mengandung informasi seperti nama paket, versinya, deskripsi untuk pengguna, ketergantungan pada paket lain (jika ada), dan rincian lainnya. More info
Lihat di Glossary dalam tampilan Proyek, jendela Manifest Paket dibuka:

Sitemap Klik tombol (A) untuk memuat paket ini manifest di editor kode default Anda, seperti Visual Studio. Klik tombol Open untuk membuka jendela Paket Manager dan memuat paket ini di View in Package Manager. Jika Anda ingin memilih importir yang berbeda, klik menu drop-down details view dan pilih paket importer yang ingin Anda gunakan.Importer drop-down menu and select the package importer you want to use.

Sitemap Bagian (B) mengandung rincian tentang versi paket khusus ini.Information section contains details about this specific package version.

Sitemap Gunakan kotak teks (C) untuk menentukan teks yang ingin Anda tampilkan di Brief Description dari jendela Manajer Paket. Untuk informasi lebih lanjut, lihat dokumentasi untuk properti details view.description property.

Sitemap Gunakan bagian (D) untuk mengelola daftar paket yang paket ini tergantung.DependenciesDalam konteks Manajer Paket, ketergantungan adalah versi paket khusus yang diperlukan proyek atau paket lain untuk bekerja. Ketergantungan dinyatakan dalam bentuk package_name@package_version. Proyek dan paket menggunakan properti dependencies dalam manifestasi mereka untuk menentukan set paket yang mereka butuhkan. Dependencies

Daftar paket lain yang tergantung pada paket ini. Setiap entri terdiri dari nama paket resmi (misalnya, “com.unity.probuilder”) dan nomor versinya.

Untuk menambahkan ketergantungan baru:

  1. Klik tombol Add . Baris baru muncul dalam daftar.
  2. Masukkan nama paket di sebelah kiri dan versi di sebelah kanan. Untuk informasi lebih lanjut, lihat dokumentasi untuk properti dependencies.

Untuk menghapus ketergantungan:

  1. Klik selector ke kiri paket yang ingin Anda hapus.
  2. Klik tombol Remove . Baris menghilang dari daftar.
