Kanvas Group

Switch to Scripting

Canvas Group dapat digunakan untuk mengontrol aspek-aspek tertentu dari keseluruhan kelompok elemen UI(User Interface) Memungkinkan pengguna untuk berinteraksi dengan aplikasi Anda. Unity saat ini mendukung tiga sistem UI. More info
Lihat di Glossary dari satu tempat tanpa perlu menangani mereka setiap individu. Sifat-sifat Canvas Group mempengaruhi GameObjectObjek mendasar dalam adegan Unity, yang dapat mewakili karakter, props, pemandangan, kamera, waypoints, dan banyak lagi. Fungsi GameObject didefinisikan oleh Komponen yang melekat padanya. More info
Lihat di Glossary itu juga semua anak-anak.

Properties

Property: Function:
Alpha opacity elemen UI dalam kelompok ini. Nilai antara 0 dan 1 di mana 0 sepenuhnya transparan dan 1 sepenuhnya buram. Perhatikan bahwa elemen mempertahankan transparansi mereka sendiri serta, sehingga Canvas Group alpha dan nilai alfa dari elemen UI individu dikalikan satu sama lain.
InteractableProperti komponen UI yang menentukan apakah komponen dapat menerima masukan. More info
Lihat di Glossary		 Tentukan jika komponen ini akan menerima masukan. Ketika diatur untuk interaksi palsu dinonaktifkan.
Block Raycasts Apakah komponen ini bertindak sebagai colliderBentuk tak terlihat yang digunakan untuk menangani tabrakan fisik untuk objek. Seorang sari tidak perlu menjadi bentuk yang sama dengan jala objek - perkiraan kasar sering lebih efisien dan tidak dapat dibedakan dalam gameplay. More info
Lihat di Glossary untuk Raycast? Anda harus memanggil fungsi RayCast pada raycaster grafis melekat pada Kanvas. not berlaku untuk Physics.Raycast.
Ignore Parent Groups Akan kelompok ini juga dipengaruhi oleh pengaturan komponen Canvas Group lebih lanjut di hierarki Game Object, atau akan mengabaikan mereka dan kemudian menimpa mereka?

Details

Khas penggunaan Canvas Group adalah:

  • Mendinginkan atau keluar dari seluruh jendela dengan menambahkan Kelompok Kanvas di GameObject of the Window dan mengontrol properti Alpha.
  • Membuat seluruh set kontrol non-interactable (“terjelajahi”) dengan menambahkan Canvas Group ke parent GameObject dan mengatur properti Interactable untuk palsu.
  • Membuat satu atau lebih elemen UI tidak memblokir acara mouse dengan menempatkan komponen Canvas Group pada elemen atau salah satu orang tua dan mengatur properti Block Raycasts untuk palsu.
