Unggah aset ke paket Anda

Setelah Anda membuat draft paket, Anda dapat menambahkan aset Anda dan mengunggahnya ke draft paket ini. Ada dua cara untuk melakukan ini:

  • Buka Proyek yang berisi aset yang ingin Anda unggah di dalam Editor Unity.
  • Pergi ke halaman Create new package, arahkan ke bagian Paket, dan klik tombol Launch Unity untuk meluncurkan Unity dan buka Proyek Anda.

Untuk mencegah kebingungan selama mengunggah dan mengimpor pengguna Anda, pastikan aset yang ingin Anda unggah berada di dalam folder tingkat atas dengan judul yang sama dengan pengajuan Anda. folder ini berisi semua komponen untuk Anda Proyek:

Note: Jika paket Anda berisi file audio atau video, Anda harus mengunggah sampel aset media Anda di Audio/Video dari bagian Package Detail paket Anda.

Untuk mengunggah aset ke paket Anda:

  1. Unduh paket dari Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
    Lihat di Glossary    .

  2. Buka Proyek Unity yang berisi konten yang ingin Anda kirimkan ke Toko Aset dan impor Alat Toko Aset ke dalamnya.

  3. Dari menu utama dari Editor Unity, pilih Asset Store Tools > Package Upload.

    Kotak dialog Publisher Login muncul, meminta Anda untuk memasukkan kredensial penerbit Anda:

    Paket Unggah alat menampilkan draft dan paket Anda di jendela Package Upload:

  4. Jika Anda tidak melihat draft paket Anda dalam daftar, klik tombol Refresh Packages di sudut kanan atas.

  5. Pilih paket draft yang ingin Anda gunakan dari daftar.

  6. Klik tombol Select untuk memilih folder yang berisi aset yang ingin Anda unggah dari browser file. Anda hanya dapat memilih satu folder, dan jika folder yang dipilih kosong, pesan kesalahan muncul ketika Anda mengklik tombol Validate atau Upload.

  7. Klik tombol Validate untuk menampilkan jendela Validator.

  8. Klik tombol Scan untuk memeriksa paket Anda untuk masalah apa pun. Anda harus memperbaiki masalah ini sebelum mengirimkan paket untuk persetujuan. Ketika Anda selesai, tutup jendela untuk kembali ke jendela Package Upload.

  9. Klik Upload untuk menghubungkan aset Anda ke paket Anda. Status bar di sebelah judul folder yang dipilih menunjukkan kemajuan unggah. Ketika unggah selesai, kotak dialog muncul membiarkan Anda tahu apakah atau tidak mengunggah berhasil.

  10. Untuk menguji file .unitypackage yang baru diciptakan, membuat Proyek Unity baru dan mengimpor paket ke dalamnya.

Anda dapat menutup jendela dan kembali ke halaman Akun Penerbit Anda di mana Anda dapat membuat perubahan akhir ke draft paket Anda dan kirimkan paket Anda untuk persetujuan akhir.
