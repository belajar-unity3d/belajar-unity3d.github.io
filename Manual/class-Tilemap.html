Tilemap

Komponen Tilemap adalah sistem yang menyimpan dan menangani Aset Tile untuk menciptakan tingkat 2D. Ini mentransfer informasi yang diperlukan dari ubin yang ditempatkan di atasnya ke komponen terkait lainnya seperti Tilemap Renderer dan Artikelcode 2D. Paket ini tidak termasuk dalam instalasi default Unity Editor secara default, dan Anda harus mengunduh paket 2D Tilemap Editor melalui .

Ketika Anda create a Tilemap, komponen Grid secara otomatis diperburuk ke Tilemap dan bertindak sebagai panduan ketika Anda meletakkan Ubin ke Tilemap.

Untuk membuat, memodifikasi, dan memilih ubin untuk melukis ke ubinmap, gunakan Palet keramik (menu: Window > 2D > Tile Palette) dan alat-alatnya. Lihat dokumentasi pada palet ubin dan Lukisan Tilemaps untuk informasi lebih lanjut.

Property Function
Animation Frame Rate Tingkat di mana Unity memainkan animasi ubin. Meningkatkan atau mengurangi perubahan kecepatan ini dengan faktor yang setara (misalnya, jika Anda mengatur ini ke 2, Unity memainkan animasi ubin pada kecepatan ganda).
Color Pilih warna untuk menerapkan sebagai kaleng ke ubin di ubin ini. Set ke putih (warna standar) untuk memiliki render Unity ubin tanpa kaleng.
Tile Anchor Masukkan jumlah (dalam sel) di sepanjang sumbu XYZ untuk mengimbangi posisi jangkar ubin pada ubinmap.
Orientation Pilih orientasi ubin di Tilemap. Gunakan ini jika Anda perlu orientate Tiles di sepanjang pesawat tertentu.
    XY Sitemap Ubin di sepanjang pesawat XY.
    XZ Sitemap Ubin di sepanjang pesawat XZ.
    YX Sitemap Ubin di sepanjang pesawat YX.
    YZ Sitemap Ubin di sepanjang pesawat YZ.
    ZX Sitemap Ubin di sepanjang pesawat ZX.
    ZY Sitemap Ubin di sepanjang pesawat ZY.
    Custom Pilih opsi ini untuk mengaktifkan pengaturan orientasi kustom di bawah ini.
Position Mengatur posisi mengimbangi orientasi kustom. Opsi ini dinonaktifkan secara default dan diaktifkan ketika Orientasi Tilemap diatur ke Custom.
Rotation Mengatur rotasi orientasi kustom. Opsi ini dinonaktifkan secara default dan diaktifkan ketika Orientasi Tilemap diatur ke Custom.
Scale Mengatur skala orientasi kustom. Opsi ini dinonaktifkan secara default dan diaktifkan ketika Orientasi Tilemap diatur ke Custom.
Info Memperluas ini untuk menunjukkan Aset yang digunakan di Tilemap.
    Tiles Tampilkan daftar Aset Tile yang digunakan di Tilemap.
    SpritesObjek grafis 2D. Jika Anda digunakan untuk bekerja di 3D, Sprites pada dasarnya hanya tekstur standar tetapi ada teknik khusus untuk menggabungkan dan mengelola tekstur sprite untuk efisiensi dan kenyamanan selama perkembangan. More info
