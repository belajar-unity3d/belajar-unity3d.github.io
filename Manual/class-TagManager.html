Tags

Tags adalah nilai penanda yang dapat Anda gunakan untuk mengidentifikasi objek dalam Proyek Anda (lihat dokumentasi pada Tags untuk rincian lebih lanjut). Untuk menambahkan Tag baru, klik tombol plus (+) di bagian bawah kanan daftar, dan nama Tag baru Anda.

Perhatikan bahwa setelah Anda telah bernama Tag, Anda tidak dapat mengubah nama itu. Untuk menghapus Tag, klik di atasnya dan kemudian klik tombol minus (-) di bagian bawah kanan daftar.

Lapisan Sortasi

Sorting Layers digunakan bersamaan dengan grafis SpriteObjek grafis 2D. Jika Anda digunakan untuk bekerja di 3D, Sprites pada dasarnya hanya tekstur standar tetapi ada teknik khusus untuk menggabungkan dan mengelola tekstur sprite untuk efisiensi dan kenyamanan selama perkembangan. More info
Lihat di Glossary dalam sistem 2D. Sorting mengacu pada urutan overlay dari Sprites yang berbeda.

Untuk menambahkan dan menghapus Layer Sorting, gunakan tombol plus dan minus (+/-) di bagian bawah kanan daftar. Untuk mengubah pesanan mereka, tarik pegangan di sisi kiri item setiap lapisan.

Layers

Gunakan lapisan di seluruh editor Unity sebagai cara untuk membuat kelompok objek yang berbagi karakteristik tertentu (lihat dokumentasi pada Layers untuk rincian lebih lanjut). Menggunakan lapisan terutama untuk membatasi operasi seperti raycasting atau rendering, sehingga mereka hanya diterapkan pada kelompok objek yang relevan. Lapisan ditandai sebagai Builtin Layer adalah lapisan default yang digunakan oleh Unity, yang tidak dapat Anda edit. Anda dapat menyesuaikan lapisan ditandai sebagai User Layer.

Untuk menyesuaikan User Layers, ketik nama kustom ke bidang teks untuk setiap yang ingin Anda gunakan. Perhatikan bahwa Anda tidak dapat menambahkan jumlah Layer tetapi, tidak seperti Tag, Anda dapat mengubah nama Layers.
