Presets

Presets adalah aset yang dapat Anda gunakan untuk menyimpan dan menerapkan pengaturan properti yang identik di beberapa komponen, aset, atau jendela Pengaturan Proyek. Anda juga dapat menggunakan Presets untuk menentukan pengaturan default untuk komponen baru dan default Pengaturan impor untuk aset di Manajer Preset Preset Manager mendukung setiap importir, komponen, atau objek skrip yang Anda tambahkan ke Editor Unity.

Anda hanya dapat menerapkan Presets di Editor. Presets tidak memiliki efek pada runtime. Anda dapat menggunakan scripting ke dukungan preset di Monobehaviour Anda sendiri, ScriptableObject atau kelas ScriptedImporter.

Hemat dan menerapkan Presets

Presets memungkinkan Anda untuk menyimpan konfigurasi properti dari jendela komponen, aset, atau Pengaturan Proyek sebagai aset Preset. Anda kemudian dapat menggunakan aset Preset ini untuk menerapkan pengaturan yang sama ke komponen yang berbeda, aset, atau jendela Pengaturan Proyek.

Misalnya, Anda dapat mengedit sifat komponen RigidbodyKomponen yang memungkinkan GameObject untuk dipengaruhi oleh gravitasi simulasi dan kekuatan lainnya. More info
Lihat di Glossary, menyimpan pengaturan ini ke aset Preset, kemudian menerapkan aset Preset ke komponen Rigidbody dalam GameObjectsObjek mendasar dalam adegan Unity, yang dapat mewakili karakter, props, pemandangan, kamera, waypoints, dan banyak lagi. Fungsi GameObject didefinisikan oleh Komponen yang melekat padanya. More info
Lihat di Glossary lain. Komponen lain dalam GameObjects tidak terpengaruh; Preset hanya menerapkan pengaturannya ke komponen Rigidbody.

Anda dapat menyimpan Presets di folder Assets proyek Anda. Gunakan jendela Project untuk melihat dan pilih Presets untuk mengedit di InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. Menyimpan pengaturan properti ke Preset

Untuk menyimpan pengaturan properti ke aset Preset, ikuti petunjuk di bawah ini. Anda dapat menyimpan pengaturan properti saat dalam mode Edit atau dalam mode Play.

  1. Pilih GameObject, Pengaturan impor aset, atau jendela Pengaturan Proyek dari mana Anda ingin menggunakan kembali pengaturan. Ketika Anda memilihnya, muncul di jendela Inspector.

  2. Di jendela Inspector, mengkonfigurasi sifat yang ingin Anda simpan.

  3. Klik pemilih Preset ( ikon slider) di kanan atas jendela Inspector.

  4. Di jendela Pilih Preset, klik Save current to.

    A Login Simpan dialog muncul.

  5. Pilih lokasi Preset baru Anda, masukkan namanya, dan klik Save.

Menerapkan pengaturan dari Preset

Ada dua cara untuk menerapkan Preset: jendela Select Preset, atau untuk komponen Presets, Anda juga dapat menyeret dan menjatuhkan Preset dari jendela ProjectDalam Unity, Anda menggunakan proyek untuk merancang dan mengembangkan permainan. Proyek menyimpan semua file yang terkait dengan permainan, seperti file aset dan Adegan. More info
Lihat di Glossary ke GameObject yang mengandung komponen tersebut.

Sitemap Menerapkan properti salinan Preset dari Preset ke item. Tidak menghubungkan Preset ke item. Perubahan yang Anda buat pada Preset tidak mempengaruhi item yang sebelumnya telah menerapkan Preset ke.Note: Applying a Preset copies properties from the Preset to the item. It doesn’t link the Preset to the item. Changes you make to the Preset do not affect the items you have previously applied the Preset to.

Untuk menerapkan Preset melalui jendela Select Preset:

  1. Untuk GameObjects atau aset yang Anda inginkan untuk menerapkan Preset ke, pilih mereka sehingga mereka muncul di jendela Inspector. Untuk Pengaturan Proyek yang ingin Anda gunakan Preset ke, buka di jendela Project SettingsPengumpulan yang luas dari pengaturan yang memungkinkan Anda untuk mengkonfigurasi bagaimana Fisika, Audio, Jaringan, Grafik, Input dan banyak bidang lain dari jangkauan proyek Anda. More info
    Lihat di Glossary    .

  2. Pada Inspector, klik pemilih Preset ( ikon slider).

  3. Di jendela Select Preset, cari dan pilih Preset untuk diterapkan.

    Unity menerapkan Preset ini ke komponen, aset, atau jendela Pengaturan Proyek.

  4. Tutup jendela Select Preset.

Jika Anda menerapkan komponen Preset melalui drag-and-drop, perilaku Unity tergantung pada keadaan GameObject Anda:

  • Jika Anda menjatuhkan Preset di jendela Hierarchy yang ada, Unity menambahkan komponen baru dan properti salinan dari Preset.
  • Jika Anda menjatuhkan Preset di area kosong di jendela Hierarchy, Unity menciptakan GameObject baru, kosong dan menambahkan komponen dengan properti yang disalin dari Preset.
  • Jika Anda menjatuhkan Preset di jendela Inspektur ke judul komponen yang ada, sifat salinan Unity dari Preset.
  • Jika Anda menjatuhkan Preset di area kosong di jendela Inspektur, Unity menambahkan komponen baru dan properti salinan dari Preset.

Menerapkan sebagian Login

Anda dapat memilih untuk hanya menerapkan beberapa properti dari Preset dan mengecualikan orang lain. Untuk melakukan ini:

  1. Pilih Preset di jendela Project.

  2. Pada Inspector, klik kanan properti dan pilih Exclude Property. Jendela menampilkan garis horizontal merah di samping tidak termasuk properti.
    %

  3. Menerapkan Preset ke komponen target, aset, atau pengaturan proyek.

Sitemap Untuk memilih semua atau menghapus semua kotak centang di Preset, pilih menu Note: (More items) atau klik kanan nama Preset, dan pilih ⋮ atau Include all properties. Anda masih dapat menyesuaikan properti individu jika Anda perlu.Exclude all properties. You can still adjust individual properties if you need to.

Anda juga dapat menggunakan opsi Exclude untuk Presets yang kemudian Anda set sebagai konfigurasi default untuk komponen dan importir aset. Lihat Manajer Preset untuk rincian lebih lanjut tentang cara melakukan ini.

Mengedit Preset

Untuk mengedit aset Preset, pilih dari jendela Project dan lihat di jendela Inspector.

Sitemap Ketika Anda mengubah properti di Preset, perubahan Anda tidak mempengaruhi item yang telah Anda terapkan Preset ke. Misalnya, jika Anda menerapkan Preset untuk komponen kaku untuk GameObject, dan kemudian mengedit Preset, pengaturan dalam komponen kaku tidak berubah.Note: When you change the properties in a Preset, your changes do not affect items you have already applied the Preset to. For example, if you apply a Preset for a Rigidbody component to a GameObject, and then edit the Preset, the settings in the Rigidbody component do not change.

Aset impor menggunakan Presets oleh folder

Anda dapat menggunakan script untuk menerapkan Preset ke Aset berdasarkan lokasi Aset di jendela Project.

Aset Preset Ekspor

Gunakan Presets untuk merampingkan alur kerja tim Anda. Anda bahkan dapat menggunakan Presets untuk menentukan pengaturan untuk jendela Pengaturan Proyek, termasuk pengaturan Preset sendiri. Gunakan fitur ini untuk mengkonfigurasi proyek kemudian ekspor sebagai asset packageKoleksi file dan data dari proyek Unity, atau elemen proyek, yang dikompresi dan disimpan dalam satu file, mirip dengan file Zip, dengan ekstensi .unitypackage. Paket aset adalah cara praktis berbagi dan menggunakan proyek Unity dan koleksi aset. More info
Lihat di Glossary kustom. Anggota tim Anda dapat import paket aset ini menjadi proyek mereka.

  1. Pada Project windowJendela yang menunjukkan isi folder Assets (Project tab) More info
    Lihat di Glossary    , pilih Presets yang ingin Anda ekspor.
  2. Dalam menu Unity, pergi ke Assets > Export Package, atau klik kanan di dalam jendela Proyek dan pilih Export Package.
    Jendela Exporting package menampilkan item untuk ekspor.
  3. Jika Presets Anda mengandung referensi untuk aset yang ingin Anda masukkan dalam paket, aktifkan Include dependencies.
  4. Klik Export.
  5. Pilih di mana Anda ingin menyimpan paket, masukkan nama file, dan klik Save. Unity menyimpan paket sebagai file .unitypackage.

Lihat di Glossary. Di jendela Inspektur, Anda mengedit pengaturan untuk salah satu transisi di node pertama. Perubahan Anda juga muncul di node lain dan di Preset.
