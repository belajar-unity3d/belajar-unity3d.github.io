Sikat Built-in

Unity dilengkapi dengan koleksi sikat built-in. Mereka berkisar dari lingkaran sederhana untuk desain sketsa yang cepat, untuk bentuk scatter yang lebih acak yang bagus untuk membuat detail dan fitur-fitur tampak alami.

Sikat Kustom

Anda dapat membuat sikat kustom Anda sendiri dengan bentuk unik atau parameter spesifik untuk kebutuhan Anda. Misalnya, gunakan Tekstur heightmapTekstur abu-abu yang menyimpan data tinggi untuk objek. Setiap pixel menyimpan perbedaan tinggi tegak lurus dengan wajah yang pixel mewakili.
Lihat di Glossary dari fitur geologi khusus untuk mendefinisikan Sikat, kemudian gunakan alat Catalog untuk menempatkan fitur di Terrain Anda.

Lihat di Glossary window, and choose Create > Brush to create a new Brush. The default Brush shows a simple circle defined by a white Mask Texture, a Falloff curve, and a Radius Scale of 1. Use the Brush Inspector to change these values, or set a Texture to define the shape of the Brush. You can also use the Remap slider and the Invert Remap Range option to further modify the grayscale values of the Brush Texture.

Pengaturan sikat

Property Function
Mask Texture Tentukan bentuk dan kekuatan Sikat. Pilih Tekstur dalam proyek Anda, dan sistem menciptakan masker berskala abu-abu dari Tekstur. Jika Tekstur yang dipilih memiliki beberapa saluran warna, Sikat menggunakan saluran merah sebagai sumbernya.
Remap Remaps nilai abu-abu dari masker Brush, setelah menerapkan kurva Falloff. Editor remaps nilai hitam di Sikat masker untuk nilai yang Anda pilih menggunakan sisi kiri slider, dan remaps nilai putih di Sikat masker untuk nilai yang Anda pilih menggunakan sisi kanan slider.
Invert Remap Range Mengalihkan sisi kiri dan kanan dari slider Remap, yang pada dasarnya mengalihkan nilai seluruh masker.
Falloff Mendefinisikan kurva yang mempengaruhi kekuatan Sikat dalam mode melingkar. Klik kurva Falloff untuk membuka Unity Editor kurva, di mana Anda dapat mengedit kurva untuk membuat efek mulai dari memudar halus ke tepi tajam.
Radius Scale Memperkuat skala kurva jatuh. Gunakan opsi ini untuk meningkatkan atau mengurangi radius kurva.
