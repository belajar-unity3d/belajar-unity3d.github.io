Pengaturan mode 2D dan 3D

Ketika membuat Proyek baru, Anda dapat menentukan apakah untuk memulai Editor Unity dalam mode 2D atau mode 3D. Namun, Anda juga memiliki pilihan untuk beralih Editor antara mode 2D dan mode 3D setiap saat. Anda dapat membaca lebih lanjut tentang perbedaan antara Proyek 2D dan 3D di sini. Halaman ini memberikan informasi tentang cara beralih mode, dan apa perubahan tepat dalam editor saat Anda melakukannya.

Beralih antara mode 3D dan 2D

Untuk mengubah mode antara mode 2D atau 3D:

  1. Buka pengaturan Editor (top menu: Edit > Project Settings, lalu pilih kategori Editor).
  2. Kemudian set Default Behavior Mode untuk kedua 2D atau 3D.

Pengaturan mode 2D vs 3D

Mode 2D atau 3D menentukan beberapa pengaturan untuk Editor Unity. Ini tercantum di bawah ini.

Dalam mode Proyek 2D:

  • Setiap gambar yang Anda impor diasumsikan menjadi gambar 2D (Sprites) dan set ke mode SpriteObjek grafis 2D. Jika Anda digunakan untuk bekerja di 3D, Sprites pada dasarnya hanya tekstur standar tetapi ada teknik khusus untuk menggabungkan dan mengelola tekstur sprite untuk efisiensi dan kenyamanan selama perkembangan. Dalam mode Proyek 3D:

  • Setiap gambar yang Anda impor tidak diasumsikan menjadi gambar 2D (Sprites).
  • Sprite Packer dinonaktifkan.
  • Scene View diatur ke 3D.
  • Objek game default memiliki waktu nyata, lampu arah.
  • Posisi default kamera adalah pada 0,1,–10. (Ini adalah 0,0,–10. dalam mode 2D.)
  • Kamera diatur menjadi Perspective. (Dalam 2D Mode itu adalah Orthographic.)
  • Di jendela pencahayaan:
    • Skybox adalah bawaan Skybox Material.
    • Ambient Source diatur ke Skybox.
    • Realtime Global Illumination (Enlighten) diatur.
    • Baked Global Illumination diatur untuk.
    • Auto-Building diatur untuk.
