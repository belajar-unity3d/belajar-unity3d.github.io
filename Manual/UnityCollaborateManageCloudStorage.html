Mengelola penyimpanan cloud

Layanan Collaborate menggunakan penyimpanan cloud untuk backup seluruh Proyek Anda di cloud dan membuatnya dapat diakses di mana saja.

Penutup topik ini:

  • Menentukan berapa banyak penyimpanan Anda Proyek menggunakan

  • Penyimpanan data cloud

  • Proyek Pengaruh untuk Collaborate

Menentukan berapa banyak penyimpanan Anda Proyek menggunakan

Penyimpanan cloud dihitung berdasarkan per organisasi. Unity menghitung jumlah penyimpanan awan yang Organisasi Anda gunakan dengan meringkas ukuran versi terbaru dari semua Proyek Kolaborasi Organisasi Anda. Hanya versi terbaru dari jumlah proyek Anda menuju batas penyimpanan Anda. Versi Sejarah (non-head) Proyek Anda tidak dihitung ke penyimpanan cloud.

Untuk melihat penggunaan penyimpanan Anda saat ini:

  1. Sign into the .
  2. Pilih Proyek di Organisasi yang penyimpanan yang ingin Anda lihat.
  3. Di jendela navigasi kiri, klik Settings, kemudian klik Usage.

Pada Project windowJendela yang menunjukkan isi folder Assets (Project tab) More info
Lihat di Glossary utama ada ubin yang menunjukkan:

  • Cloud storage - Berapa banyak penyimpanan Proyek saat ini digunakan dibandingkan dengan sisa Proyek di Organisasi Anda, dan seluruh ruang penyimpanan cloud Anda.
  • Percent of storage per project - Berapa banyak penyimpanan yang digunakan oleh setiap Proyek di Organisasi Anda.
  • Storage used by <Project name> over time - Berapa banyak penyimpanan Proyek saat ini telah digunakan karena diciptakan, sampai saat ini.

Menambahkan lebih banyak penyimpanan

Anda dapat memperoleh lebih banyak penyimpanan cloud dalam dua cara:

  • Upgrade ke Tim Unity Advanced, yang juga menyediakan penyimpanan cloud 25GB.
  • Paket Penyimpanan Pembelian di dashboard ID Unity. Setiap paket penyimpanan menambahkan penyimpanan awan 25GB ke Organisasi.

Upgrading to Unity Teams Advanced

Untuk meningkatkan ke Tim Unity Lanjutan, kunjungi halaman Tim Unity di . Pada halaman Tim Unity Anda dapat mempelajari lebih lanjut tentang fitur Tim Unity Advanced dan kemudian ditingkatkan.

Purchase storage packs

Untuk menambahkan paket penyimpanan:

  1. Masuk ke dasbor.
  2. Di bar navigasi kiri, klik Organizations.
  3. Di jendela Organizations utama, pilih Organisasi.
  4. Di bar navigasi kiri, klik Service Usage.
  5. Di jendela Usage overview, klik tombol Manage storage.
  6. Pada dialog Add cloud storage, pilih jumlah penyimpanan yang Anda butuhkan, lalu klik tombol Continue to purchase. Jika Anda hanya memiliki satu organisasi, lanjutkan ke langkah 6.
  7. Di jendela Checkout, masukkan informasi pembayaran Anda dan kemudian klik Pay now.

Modifying your contracted amount of storage

  1. Masuk ke dasbor.
  2. Di jendela navigasi kiri, klik Organizations.
  3. Di jendela Organizations utama, pilih Organisasi.
  4. Di bar navigasi kiri, klik Service Usage. Jendela Penggunaan Layanan menampilkan periode kontrak Anda dan jumlah paket penyimpanan yang Anda kontrak untuk.
  5. Di jendela Usage overview, klik tombol Manage storage.
  6. Pada tab Add cloud storage, masukkan jumlah paket penyimpanan untuk ditambahkan.
  7. Klik tombol Next.
  8. Di jendela Review order, meninjau informasi pesanan dan kemudian klik tombol Confirm purchase.

Anda dapat mengurangi paket penyimpanan nomor pada tab Manage subscription:

  • Untuk menghapus semua paket penyimpanan terkontrak, matikan Automatic Renewal. Ketika pembaruan otomatis dimatikan, semua paket penyimpanan dihapus pada tanggal pembaruan.
  • Untuk mengurangi jumlah paket penyimpanan yang diperbarui, masukkan jumlah paket penyimpanan untuk memperbarui di bidang Select the number of storage packs you want to renew upon renewal.

Kemudian klik tombol Save.

Penyimpanan data cloud

Jika Anda menggunakan Tim Unity Advanced, Unity tidak menghapus riwayat versi untuk Proyek Anda.

Jika Anda menggunakan Tim Unity Basic, Unity menyimpan data Anda dalam penyimpanan cloud setidaknya 1 tahun. Unity berhak untuk menghapus riwayat Proyek lebih dari 1 tahun. Anda selalu memiliki akses ke versi terbaru dari Proyek Anda, tetapi Anda hanya memiliki akses ke 90 hari terakhir dari sejarah versi Anda. Jika Anda menggunakan Tim Unity Dasar dan ingin akses penuh ke 1 tahun terakhir sejarah Anda, Anda harus meningkatkan ke Tim Unity Lanjutan. Ketika Anda meningkatkan ke Tim Unity Lanjutan, Anda memiliki akses ke sejarah penuh Anda akan maju.

Archiving Projects

Menarsipkan Proyek secara otomatis menyaring penyimpanan cloud Collaborate yang terkait dengan Proyek dan layanan Unity tidak lagi dapat diakses.

Untuk menyaring Proyek:

  1. Sign into the .
  2. Pilih Proyek ke arsip.
  3. Di bar navigasi kiri, klik Settings, lalu klik General.
  4. Di bagian bawah jendela utama, baca catatan penting dan, jika Anda yakin Anda ingin arsip Proyek, klik tombol Archive my Project for ALL Unity services.
  5. Dalam dialog Are you sure you want to archive this Project, masukkan nama Proyek dan kemudian klik tombol Yes, archive it.

Untuk membuka sebuah Proyek:

  1. Sign into the .
  2. Di kanan atas jendela utama, di bawah tombol CREATE NEW PROJECT, klik tautan View Archived Projects.
  3. Temukan Proyek untuk mengaktifkan kembali, klik ellipsi vertikal ke kanan nama Proyek dan kemudian pilih Unarchive Project.

Proyek Pengaruh untuk Collaborate

Untuk mengarsipkan Proyek untuk berkolaborasi hanya:

  1. Sign into the .
  2. Pilih Proyek ke arsip.
  3. Di jendela navigasi kiri, klik CollaborateLayanan cloud-hosted Unity yang menyediakan cara sederhana untuk tim untuk menyimpan, berbagi, dan berkontribusi pada proyek Unity mereka. More info
    Lihat di Glossary    , kemudian klik Storage.
  4. Di jendela utama, klik tombol ARCHIVE.

Untuk membuka arsip Proyek untuk berkolaborasi:

  1. Sign into the .
  2. Select the Project to unarchive.
  3. Di jendela navigasi kiri, klik Collaborate, kemudian klik Storage.
  4. Di jendela utama, klik tombol UNARCHIVE.
