Mendistribusikan permainan Anda dengan UDP

Langkah-langkah di bawah ini menunjukkan proses keseluruhan untuk mempublikasikan permainan ke toko aplikasi melalui UDP.

  1. Membuat permainan di konsol UDP.
    1. Masukkan info game Anda di konsol UDP.
    2. Mengatur bahasa yang didukung.
  2. .
    1. Instal paket.
    2. Tautan proyek Anda ke klien UDP.
    3. Menawarkan SDK UDP.
    4. Menerapkan IAP (jika ada).
    5. Menerapkan Licence Periksa game premium (opsional).
    1. Membangun permainan Anda APKFormat paket Android output oleh Unity. Sebuah APK secara otomatis digunakan untuk perangkat Anda ketika Anda memilih File > Build & Run. More info
      Lihat di Glossary      .
    2. Pastikan semua produk IAP Anda tercantum dalam konsol UDP (jika berlaku).
    3. Uji permainan Anda di Sandbox.
  4. Unggah permainan Anda APK ke konsol UDP.
  5. Finalisasi halaman informasi game di konsol UDP.
    1. Verifikasi pengujian berhasil.
    2. Unggah aplikasi Anda menandatangani kunci pribadi.
    3. Mengatur harga premium (jika berlaku).
    4. Lokalisasi informasi game Anda untuk bahasa tambahan (jika ada).
  6. Rilis permainan Anda di UDP.
  7. Publikasikan permainan Anda ke toko.

Gambar di bawah ini menggambarkan alur kerja keseluruhan.

Membuat permainan di konsol UDP

Untuk membuat permainan baru di konsol UDP:

  1. Di tab My Games, pilih Add Game.
  2. Tambahkan judul untuk permainan Anda dan pilih Create New Game.
    Halaman Info Game dibuka.
  3. Pilih Save.

Ini adalah minimum yang diperlukan untuk membuat permainan baru di konsol UDP. Langkah selanjutnya adalah masukkan informasi permainan Anda.

Ketika Anda membuat permainan baru di konsol UDP itu menghasilkan klien UDP dan ID klien UDP. Anda harus di Editor Unity.

Masukkan informasi permainan di konsol UDP

Untuk mengedit informasi game Anda di konsol UDP:

  1. Di tab game saya, pilih kartu game Anda.
  2. Di halaman Game Info, pilih EDIT INFO untuk memasukkan mode edit.
    1. Memperbarui informasi permainan Anda. Pelajari lebih lanjut tentang persyaratan untuk halaman Game Info. Login Untuk membantu menyelesaikan informasi ini, Anda juga dapat mengimpor metadata dari Google Play.
      To help complete this information, you can also import metadata from Google Play.
  3. Untuk menghemat perubahan pilih SAVE. Untuk membuang perubahan, pilih CANCEL.

Jika Anda ingin memberikan deskripsi untuk permainan Anda dalam beberapa bahasa, tambahkan bahasa yang didukung di konsol UDP. Sitemapmenerapkan UDP dalam proyek Anda.

Menambahkan bahasa yang didukung

Ketika Anda menambahkan bahasa di halaman Game Info, itu menyalin informasi dari versi default. Oleh karena itu, untuk mengurangi duplikasi pekerjaan, menambahkan informasi sebanyak mungkin dalam versi default sebelum Anda menambahkan lebih banyak bahasa. Untuk menambahkan bahasa:

  1. Di halaman Game Info, pilih EDIT INFO.
  2. Dalam bahasa dropdown, pilih Manage languages.
    1. Pilih bahasa yang ingin Anda dukung dan pilih Save.
  3. Di halaman Game Info, pilih Save.

Jika Anda akan menggunakan fitur impor massal untuk mengimpor produk IAP, termasuk deskripsi dalam beberapa bahasa, Anda harus terlebih dahulu menambahkan bahasa di Konsol UDP. Jika tidak, deskripsi IAP tidak akan diimpor untuk bahasa tersebut.
