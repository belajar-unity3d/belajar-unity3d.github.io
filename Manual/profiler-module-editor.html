Profiler Modul Editor

Editor Modul ProfilerJendela yang membantu Anda untuk mengoptimalkan permainan Anda. Ini menunjukkan berapa banyak waktu yang dihabiskan di berbagai bidang permainan Anda. Sebagai contoh, dapat melaporkan persentase waktu yang dihabiskan rendering, aimating, atau dalam logika permainan Anda. More info
Lihat di Glossary adalah alat yang dapat Anda gunakan untuk menambahkan modul kustom Anda sendiri ke jendela Unity Profileer. Anda juga dapat menambahkan counter built-in untuk modul, atau menggunakan API runtime untuk menambahkan counter kustom Anda sendiri ke modul. Untuk informasi tentang cara menerapkan counter Anda sendiri, lihat dokumentasi paket Profiling di .

Ini sangat berguna ketika Anda perlu hone dalam statistik yang ingin Anda analisa lebih dalam, atau untuk menambahkan statistik kustom tentang aplikasi Anda ke jendela Profiler.

Untuk menggunakan Editor Modul Profiler, buka Jendela Profiler (Window > Analysis > Profiler) dan kemudian pilih dropdown Modul Profiler.

Pilih ikon gigi, dan jendela Profiler Module Editor terbuka. Jendela memiliki tiga kolom:

  • Sitemap Daftar ini berisi semua modul yang tersedia yang dapat Anda tambahkan ke jendela Profiler. Modul built-in adalah abu-abu-out dalam daftar, menunjukkan bahwa Anda tidak dapat mengedit konten mereka. Anda dapat menyeret dan menjatuhkan modul untuk mengubah penampilan mereka di jendela Profiler. Ketika Anda membuat Modul Profiler kustom Anda sendiri, itu juga muncul dalam daftar ini.List of Profiler Modules: This list contains all of the available modules you can add to the Profiler Window. Built-in modules are greyed-out in the list, indicating that you cannot edit their contents. You can drag and drop the modules to reorder their appearance in the Profiler window. When you create your own custom Profiler Module, it also appears in this list.
  • Profiler Module information pane (hanya muncul ketika Anda membuat atau memilih modul kustom): Daftar counter yang terkandung dalam modul kustom
  • Available counters (hanya muncul ketika Anda membuat atau memilih modul kustom): Daftar counter yang tersedia Anda dapat menambahkan ke modul kustom.

Membuat modul kustom

Untuk membuat modul kustom, pilih tombol Add di sebelah kiri bawah jendela Editor Modul Profiler. Unity menambahkan modul Profiler baru ke daftar yang disebut New Profiler Module. Untuk mengubah nama modul, klik di bidang teks dan gunakan keyboard Anda untuk mengatur nama.

Menambahkan counter ke modul

Untuk mengumpulkan data dalam modul Profiler kustom, Anda harus menambahkan setidaknya satu counter untuk modul untuk melacak. Anda dapat menambahkan kontra Unity bawaan, atau menggunakan API ProfilerRecorder untuk membuat counter Anda sendiri untuk menambahkan ke modul. Daftar counter yang tersedia muncul di pane Available Counters.

Jika Anda tidak memiliki data yang dimuat ke jendela Profiler, maka setiap counter yang Anda buat tidak muncul di pane Available Counters ketika Anda memuat Editor Modul Profiler. Untuk melihat penghitung kustom Anda, Anda harus menangkap atau memuat beberapa data yang memiliki penghitung yang dipancarkan Anda dengan Profiler, dan membuka kembali Editor Modul Profiler.

Counter Unity yang tersedia dibagi menjadi tiga daerah: Memory, Render, dan Tekstur Virtual. Untuk informasi lebih lanjut tentang counter tertentu yang tersedia di daerah ini, lihat dokumentasi untuk setiap modul.

Jika Anda menggunakan API untuk menambahkan counter Anda sendiri, Anda harus memuat atau mengumpulkan beberapa data ke jendela Profiler terlebih dahulu, dan kemudian counter kustom Anda muncul di pane Available Counters di bawah judul User dari Editor Modul Profiler.

Untuk menambahkan counter ke modul, pilih dalam daftar Available Counters dan kemudian pilih tombol Add Selected. Atau, Anda dapat mengklik dua kali counter untuk menambahkannya ke modul. Unity kemudian menambahkan counter ini ke modul Profiler yang dipilih. Untuk menambahkan beberapa counter ke modul sekaligus, Anda dapat menahan Shift dan klik dua counter untuk memilih semua counter antara dua, atau Anda dapat menahan Ctrl (Command pada macOS) untuk memilih counter satu pada satu waktu, kemudian pilih tombol Add Selected.

Untuk memesan kembali counter, Anda dapat menyeret dan menjatuhkannya. Anda hanya dapat menambahkan maksimum 10 counter ke modul.

Menggunakan modul kustom

Ketika Anda senang dengan modul kustom Anda, pilih tombol Save Changes di sudut kanan bawah jendela Editor Modul Profiler. Unity menutup jendela, dan Anda kemudian dapat melihat modul baru Anda yang tercantum di jendela Profiler.

Untuk secara efektif menggunakan dan membandingkan data Anda terhadap modul bawaan, Anda dapat memesan ulang modul di Jendela Profiler. Untuk melakukan ini, buka Editor Modul Profiler, dan kemudian drag dan drop modul di pane Profiler Modules.

Ini sangat berguna jika Anda ingin memeriksa elemen kustom dari aplikasi Anda terhadap lonjakan dalam data built-in Profiler.
