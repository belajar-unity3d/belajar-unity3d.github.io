Peningkatan ke Unity 2017.1

Halaman ini mencantumkan perubahan pada 2017.1 yang mungkin mempengaruhi proyek yang ada ketika Anda meng-upgrade dari versi Unity sebelumnya.

Contoh:

  • Perubahan dalam format data yang mungkin memerlukan re-baking.

  • Perubahan makna atau perilaku fungsi, parameter atau nilai komponen yang ada.

  • Penerapan fungsi atau fitur apa pun. (Alternatives disarankan.)

UnityWebRequestTexture.GetTexture() nonReadable parameter change

API kenyamanan ini memiliki bug di mana nonReadable bekerja dengan cara yang berlawanan maka harus, yang mengaturnya untuk benar akan mengakibatkan tekstur dan sebaliknya. Sekarang ini telah diperbaiki dan parameter bekerja dengan cara itu didokumentasikan. Catatan, jika Anda membuat DownloadHandlerTexture secara langsung, Anda tidak terpengaruh.

X axis pivots sekarang akurat pada Peregangan Billboard. Anda mungkin perlu mengkonfigurasi pengaturan pivot Anda pada proyek yang terkena.

Y axis pivots juga akurat sekarang dan Unity secara otomatis mengkonfigurasi pivot ini ketika Anda meningkatkan proyek Anda.
