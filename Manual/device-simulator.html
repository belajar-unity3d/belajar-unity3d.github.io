Bagian dokumentasi ini berisi informasi tentang Simulator Perangkat Unity.

Page Description
Pengenalan Simulator perangkat Memperkenalkan Simulator Perangkat dan kemampuannya.
Tampilan Simulator Jelaskan cara menggunakan jendela yang digunakan Simulator Perangkat untuk menampilkan simulasi.
Kelas simulasi Mengandung informasi pada kelas Simulator Perangkat menyediakan sehingga Anda dapat menguji kode yang menanggapi perilaku spesifik perangkat.
Menambahkan perangkat Jelaskan cara menentukan perangkat baru untuk mensimulasikan dan menggunakannya di Simulator Perangkat.
Memperpanjang Simulator Perangkat Jelaskan cara membuat plugin Simulator Perangkat untuk memperluas fungsi dan antarmuka pengguna dari tampilan Simulator.
