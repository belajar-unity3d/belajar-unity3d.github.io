Membuat dan mengelola Lembar Gaya (USS)

Tambahkan gaya Login

Untuk menambahkan Lembar Gaya ke Dokumen UI(User Interface) Memungkinkan pengguna untuk berinteraksi dengan aplikasi Anda. Unity saat ini mendukung tiga sistem UI. More info
Lihat di Glossary saat ini (UXML), gunakan menu dropdown + dan pilih salah satu opsi berikut:

  1. Create New USS: Ini membuka Save File Dialog untuk membuat Lembar Gaya baru di proyek Anda.
  2. Add Existing USS: Ini membuka Open File Dialog untuk menambahkan Lembar Gaya yang ada ke Dokumen UI.

Anda juga dapat klik kanan di mana saja di tampilan pohon pane StyleSheets untuk mengakses menu opsi ini.

Hapus Gaya Login

Untuk menghapus Gaya Lembar dari Dokumen UI aktif, klik kanan pada Lembar Style dan pilih Remove USS. Ini membutuhkan simpanan penuh dari semua perubahan yang belum disadari.

Satu Active StyleSheet selalu muncul berani. Jika Anda membuat Selector USS baru dari bidang Create new selector…, ditambahkan ke Active StyleSheet. Untuk mengubah Active StyleSheet, klik kanan pada StyleSheet dan pilih Set as Active USS.

Ketika mengedit Sub-Documents, Lembar Gaya yang milik dokumen induk muncul diabu. Ini memberikan konteks tetapi tidak dibaca. Meme it
