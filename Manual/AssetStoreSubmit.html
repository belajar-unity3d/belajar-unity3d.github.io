Mengirimkan paket Anda untuk persetujuan

Ketika paket Anda siap dan Anda ingin mempublikasikannya ke Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
Lihat di Glossary, Anda perlu mengirimkannya untuk review sebelum Unity dapat menyetujuinya untuk penerbitan:

  1. Buka portal dan arahkan ke tab Packages.

  2. Pilih Draft dari menu drop-down Status Paket.

  3. Klik tautan ke paket yang ingin Anda kirimkan.

    Halaman Create new package muncul dengan informasi paket Anda.

  4. Anda dapat mengisi bagian Package Detail dan Metadata & Artwork. Jika Anda memerlukan bantuan, ikuti petunjuk di bawah Membuat draft paket baru

  5. Di bawah bagian Package upload, klik Edit untuk mengatur informasi tentang kualitas paket Anda untuk membangun platform, versi Editor, render pipelinesRangkaian operasi yang mengambil isi dari Adegan, dan menampilkannya di layar. Unity memungkinkan Anda memilih dari pipa render yang dibangun sebelumnya, atau menulis sendiri. More info
    Lihat di Glossary     kompatibel, dan ketergantungan Aset lainnya.

    Atau, Anda dapat mengklik Remove untuk menghapus Aset yang diunggah dari draft paket. Dalam hal itu, Anda perlu untuk ulang Aset paket Anda.

  6. Di bawah bagian Review, klik Preview in Asset Store untuk melihat bagaimana paket Anda akan terlihat di Aset Store setelah menerbitkannya. Anda juga dapat melihat pratinjau URL di bawah Public links.

  7. Di bawah bagian Kirim paket untuk persetujuan:

    • Aktifkan Auto publish untuk menunjukkan bahwa Anda ingin paket muncul di Asset Store segera setelah Unity menyetujuinya.

    • Aktifkan kotak cemen I own the rights to sell these assets.

  8. Klik Submit untuk memulai proses persetujuan.

    Popup muncul meminta Anda untuk mengkonfirmasi apakah Anda ingin mengirimkan paket Anda. Tim Asset Store akan mempertimbangkan apakah menerbitkan paket atau tidak. Untuk membantu memindahkan proses keputusan bersama, Anda dapat memberikan informasi tambahan untuk membantu mengevaluasi paket Anda, mengapa harus dipublikasikan, atau apa yang Anda tetapkan (dalam kasus peningkatan).

  9. Masukkan pesan Anda ke tim Asset Store, seperti “Anda perlu mengunduh aset X untuk dapat sepenuhnya menguji aset ini” atau “Anda perlu menggunakan HDRP untuk menguji aset saya”.

  10. Pilih OK untuk menyelesaikan pengajuan atau Cancel untuk menghentikan proses pengiriman.

    Ketika Anda memilih OK, pesan muncul membiarkan Anda tahu paket Anda berhasil (Version submitted.) atau tidak (An error has occurred, dengan daftar masalah untuk memperbaiki).

Sitemap Tim Asset Store akan memakan waktu setidaknya 5 hari kerja untuk menganalisis pengajuan Anda. Namun, jika Anda tidak mendengar kembali setelah satu bulan atau lebih lama, hubungi.Note: The Asset Store team will take at least 5 business days to analyze your submission.
