Paket konteks daftar

Untuk mengubah paket yang muncul dalam daftar, pilih konteks yang Anda inginkan dari menu drop-down PackagesPaket adalah koleksi aset untuk dibagikan dan digunakan kembali dalam Unity. Manajer Paket Unity (UPM) dapat menampilkan, menambahkan, dan menghapus paket dari proyek Anda. Paket-paket ini berasal dari Unity Package Manager dan memberikan metode mendasar untuk memberikan fungsi Unity. Namun, Unity Package Manager juga dapat menampilkan Paket Asset Store yang Anda unduh dari Asset Store. More info
Lihat di Glossary di jendela Manajer Paket.

Anda dapat memilih dari opsi ini:

Context: Description:
Unity Registry Menampilkan semua set fitur dan paket pada Unity paket registry, terlepas dari apakah mereka sudah dipasang di proyek Anda. Ini tidak termasuk paket yang diinstal dari lokasi lain atau dari setiap registry lingkup.

Ini adalah konteks daftar default.
My Registries Menampilkan paket apa pun yang tersedia dari setiap registry lingkup yang dipasang di proyek ini.

Note: Jika Anda menambahkan registry lingkup tetapi Anda tidak melihatnya dalam konteks My Registries atau konteks My Registries tidak tersedia sama sekali, itu mungkin karena server registry paket yang ditambahkan tidak mengimplementasikan titik akhir /-/v1/search atau /-/all, yang berarti tidak kompatibel dengan Manajer Paket Unity.
In Project Menampilkan semua paket feature setsSebuah fitur set adalah koleksi paket terkait yang dapat Anda gunakan untuk mencapai hasil tertentu dalam Editor Unity. Anda dapat mengelola set fitur langsung di Manajer Paket Unity. More info
Lihat di Glossary dan paket saat ini dipasang di proyek Anda, termasuk paket local, git, dan embedded, dan paket yang dipasang dari registry.
My Assets Menampilkan semua Paket Asset Store (.unitypackage) yang telah Anda beli menggunakan ID Unity Anda saat ini masuk.
Built-in Menampilkan paket Unity bawaan, yang mewakili beberapa fitur Unity inti. Anda dapat menggunakan paket ini ke mematikan modul Unity dan mati.

Tip: Anda dapat menemukan lebih banyak tentang paket setiap built-in (module) mengimplementasikan di API documentation. Setiap daftar halaman modul perakitan yang API paket built-in mengimplementasikan.

Ketika Anda memilih konteks baru, pilihan Anda muncul pada menu drop-down Packages dan daftar menampilkan set fitur dan paket yang sesuai dengan pilihan Anda.

Note: Jika Anda memasukkan teks dalam kotak pencarian atau mengatur setiap filters, daftar hanya menampilkan set fitur dan paket yang sesuai konteks, kriteria pencarian, dan filter aktif.
