aktivasi lisensi Offline (manual)

Jika Unity tidak dapat menghubungi server lisensi, Anda tidak dapat mengaktifkan lisensi Anda menggunakan metode online. Alasan umum untuk gagal terhubung ke server lisensi termasuk: Anda tidak memiliki akses ke internet; firewall atau pengaturan proxy Anda memblokir koneksi; atau server lisensi Unity turun. Untuk informasi lebih lanjut, lihat Masalah aktivasi.

Ketika aktivasi online gagal, Anda dapat mencoba mengaktifkan Unity secara manual pada mesin Anda. Anda masih perlu akses ke mesin dengan akses internet, tetapi tidak harus menjadi mesin di mana Anda mencoba mengaktifkan Unity.

Untuk mengaktifkan lisensi Unity Anda, ikuti proses ini:

  1. Hasilkan file aktivasi lisensi (.alf) dari baris perintah. Sebagai contoh, Unity_v2021.1.0a7.alf.
  2. Gunakan file .alf ke meminta file lisensi Unity (.ulf) dari server lisensi Unity.
  3. Gunakan file .ulf untuk mengaktifkan lisensi Anda di Unity dari baris perintah.

Hasilkan file aktivasi lisensi (.alf) dari baris perintah

Di macOS, masukkan berikut ke Terminal (mengganti <editor-installation-location> dengan lokasi instalasi aktual, seperti /Applications/Unity/Unity.app/Contents/MacOS/Unity):

<editor-installation-location> -batchmode -createManualActivationFile -logfile

Pada Windows, masukkan berikut ke Command Prompt (mengganti <editor-installation-location> dengan lokasi instalasi aktual, seperti C:\Program Files\Unity\Editor\Unity.exe):

"<editor-installation-location>" -batchmode -createManualActivationFile -logfile

Sekarang Anda siap untuk meminta file lisensi Unity (.ulf).

Permohonan file lisensi Unity (.ulf)

Setelah Anda menghasilkan file aktivasi lisensi (.alf), langkah berikutnya adalah untuk meminta file lisensi Unity Anda (.ulf) dari server lisensi:

  1. Buka halaman web.

  2. Jika diminta, masuk ke ID Unity Anda. Hal ini diperlukan untuk meminta file lisensi Unity.

    Webpage Manual activation muncul.

  3. Pilih tombol Browse untuk menemukan file aktivasi lisensi (.alf) yang telah Anda hasilkan menggunakan browser file sistem Anda.

    Jika unggah berhasil dan Unity terdeteksi bahwa file yang Anda pilih adalah file aktivasi lisensi, nama file muncul di kotak teks dengan cekmark hijau. Jika Anda melihat X merah, coba lagi.

  4. Pilih tombol Next. Webpage Activate your license muncul.

  5. Pilih jenis lisensi yang Anda aktifkan:

    • Jika Anda memilih opsi Unity Plus or Pro, masukkan Serial Number yang Anda terima dalam email ketika Anda membeli lisensi tunggal. Anda perlu memasukkan kunci dalam format yang tepat yang muncul di email. Sebagai contoh, PS/SC/E3-XXXX-XXXX-XXXX-XXXX.
    • Jika Anda memilih opsi Unity Personal, Anda harus memilih opsi yang terbaik menggambarkan situasi Anda saat ini.

  6. Pilih tombol Next. Webpage Download license file muncul.

  7. Pilih tombol Download license file.

  8. Memungkinkan unduhan untuk halaman web jika sistem Anda meminta Anda.

  9. Buka lokasi unduhan browser Anda dan cari file lisensi Unity baru dengan ekstensi .ulf.

Sekarang Anda siap mengaktifkan lisensi Unity Anda dari baris perintah.

Selesai aktivasi lisensi Unity Anda dari baris perintah

Di macOS, masukkan berikut ke Terminal (mengganti <editor-installation-location> dengan lokasi instalasi aktual, seperti /Applications/Unity/Unity.app/Contents/MacOS/Unity):

<editor-installation-location> -batchmode -manualLicenseFile <yourulffile> -logfile

Pada Windows, masukkan berikut ke Command Prompt (mengganti <editor-installation-location> dengan lokasi instalasi aktual, seperti C:\Program Files\Unity\Editor\Unity.exe):

"<editor-installation-location>" -batchmode -manualLicenseFile <yourulffile> -logfile

Jika Unity menampilkan pesan kesalahan yang aktivasi lisensi tidak berhasil, periksa Masalah aktivasi untuk menemukan solusi untuk masalah Anda.

  • Aktivasi lisensi diperbarui di Unity 2017.2
