Menyiapkan Portal Occlusion di Adegan Anda

  1. Pilih GameObject yang cocok di Adegan Anda untuk bertindak sebagai Portal Occlusion. Calon yang baik untuk Portal Occlusion menengah ke besar GameObjects padat, seperti pintu.
  2. Pastikan GameObject tidak ditandai sebagai Occluder Statis atau Occludee Statis.
  3. Tambahkan komponen Occlusion Portal ke GameObject.
  4. Beri data oklusi untuk Adegan Anda. Lihat Memulai dengan penanaman oklusi untuk instruksi.
  5. Pastikan jendela Culling Occlusion, panel InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info
    Lihat di Glossary     dan Scene viewTampilan interaktif ke dunia yang Anda buat. Anda menggunakan Adegan Lihat untuk memilih dan posisi pemandangan, karakter, kamera, lampu, dan semua jenis lain dari Game Object. More info
    Lihat di Glossary     semuanya terlihat.
  6. Dalam pandangan Adegan, memindahkan CameraKomponen yang menciptakan gambar sudut pandang tertentu di tempat kejadian Anda. Output ditarik ke layar atau ditangkap sebagai tekstur. More info
    Lihat di Glossary     ke posisi di mana itu langsung di depan Portal Occlusion.
  7. Pilih GameObject dengan komponen Portal Occlusion.
  8. Di jendela Inspektur, untuk memindahkan properti Open komponen Portal Occlusion di dan mati. Dalam pandangan Adegan, amati perbedaan dalam penanaman oklusi.

Membuka dan menutup Portal Occlusion di runtime

Gunakan script untuk mengatur properti open Portal Occlusion ke negara yang diinginkan.

void OpenDoor() {
     // Toggle the Occlusion Portal's open state, so that Unity renders the GameObjects behind it
    myOcclusionPortal.open = true;
    
    // Call a function that plays a door opening animation, or otherwise hides the GameObject
    …
}

Referensi komponen Portal Occlusion

Property: Function:
Open Jika diaktifkan, Portal Occlusion terbuka, dan tidak termasuk renderer. Jika dinonaktifkan, Portal Occlusion ditutup, dan occludes Renderers.
Center Mengatur pusat Portal Occlusion. Nilai default adalah 0,0,0.
Size Tentukan ukuran Portal Occlusion.
