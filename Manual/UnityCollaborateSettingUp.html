Menyiapkan Unity Collaborate

Mengaktifkan layanan

Untuk menggunakan Collaborate, Layanan Unity harus diaktifkan pada proyek Anda. Untuk informasi lebih lanjut, lihat Menyiapkan proyek Anda untuk Layanan Unity.

Collaborate didistribusikan sebagai paket Unity pra-instal. Anda dapat memperbarui atau menghapus Collaborate menggunakan Manajer Paket Unity.

Untuk mengaktifkan Collaborate dalam proyek Anda:

  1. Dalam Editor Unity toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
    Lihat di Glossary    , klik tombol kolaborasi.
  2. Klik Start Collab.

Note: Jika tombol kolaborasi tidak tersedia, kemungkinan paket Collaborate tidak dipasang. Gunakan Manajer Paket untuk menginstal ulang.

Menerbitkan perubahan pada proyek Anda

Jika Anda mengaktifkan Collaborate pada proyek yang baru diciptakan, proyek Anda secara otomatis dipublikasikan ke cloud.

Jika Anda menambahkan Collaborate ke proyek yang ada, Anda mungkin perlu mempublikasikan proyek Anda secara manual.

Untuk mempublikasikan perubahan dan bagikan mereka dengan rekan kerja Anda:

  1. Klik tombol kolaborasi.
  2. Pilih file yang ingin Anda publikasikan.
  3. Klik Publish.

Ketika Anda melakukan perubahan pada proyek Anda, tombol kolaborasi menampilkan panah atas untuk menunjukkan ada perubahan yang tidak dipublikasikan.

Mendapatkan perubahan dari server

Ketika anggota tim lain melakukan perubahan pada proyek dan penerbit ke cloud, tombol kolaborasi menampilkan ikon untuk menunjukkan bahwa ada perubahan jarak jauh baru bahwa versi lokal proyek Anda tidak memiliki.

Untuk mendapatkan perubahan terbaru dari server, klik Sync di jendela Collaborate.
