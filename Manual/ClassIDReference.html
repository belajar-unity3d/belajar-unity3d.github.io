ID Kelas YAML Login

Referensi nomor ID kelas umum yang digunakan oleh format file YAML diberikan di bawah ini, baik dalam urutan numerik ID kelas dan urutan alfabet dari nama kelas. Perhatikan bahwa beberapa rentang angka yang dihilangkan secara sengaja dari urutan - ini dapat mewakili kelas yang telah dihapus dari API atau dapat dicadangkan untuk penggunaan di masa depan. Kelas yang didefinisikan dari scriptsSepotong kode yang memungkinkan Anda untuk membuat Komponen Anda sendiri, memicu peristiwa permainan, memodifikasi sifat komponen dari waktu ke waktu dan menanggapi input pengguna dengan cara apa pun yang Anda sukai. More info
