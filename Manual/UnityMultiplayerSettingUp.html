Menyiapkan Multiplayer Unity

Untuk mulai menggunakan Unity Multiplayer, proyek Anda harus mengatur untuk menggunakan Layanan Unity. Setelah Anda melakukan ini, Anda dapat mengaktifkan Layanan Multiplayer.

Untuk melakukan ini, buka jendela ServicesFasilitas Unity yang menyediakan berbagai layanan gratis untuk membantu Anda membuat permainan dan melibatkan, mempertahankan dan memonetisasi penonton. More info
Lihat di Glossary dengan memilih Window > General > Services di bilah menu. Di jendela Layanan, pilih Multiplayer.

Ini membuka jendela Layanan Multiplayer.

Tombol Go To Dashboard membawa Anda ke Dasbor Layanan berbasis web, di mana Anda dapat mengatur konfigurasi Multiplayer untuk proyek Anda.

Jika Anda belum menyiapkan proyek Anda dengan layanan Multiplayer, Anda diminta untuk mengatur New Multiplayer Configuration. Untuk melakukan ini, masukkan jumlah pemain yang Anda inginkan per kamar dan klik Save.

Setelah Anda mengklik Simpan, Dasbor Layanan Multiplayer mencerminkan proyek Anda saat ini.

Anda sekarang siap untuk mengintegrasikan proyek Anda dengan Unity Multiplayer!
