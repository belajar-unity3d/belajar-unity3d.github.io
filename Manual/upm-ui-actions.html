Menambahkan dan menghapus

Anda dapat melakukan berbagai tugas melalui jendela Paket Manager:

  • Instal set fitur
  • Instal paket baru dari registry
  • Instal paket baru dari folder lokal
  • Instal paket baru dari file tarball lokal
  • Instal paket baru menggunakan URL Git
  • Tambahkan paket registry dengan nama
  • Hapus paket yang terpasang
  • Nonaktifkan paket bawaan
  • Mengimpor paket Asset Store ke proyek Anda
  • Beralih ke versi paket lain
  • Memperbarui paket Aset Store
  • Melakukan tindakan pada beberapa paket atau set fitur

Prosedur yang dijelaskan di bagian-bagian ini mengaburkan banyak rincian apa yang Manajer Paket sebenarnya dilakukan di balik scenes. Jendela Paket Manager menyediakan cara yang ramah pengguna untuk menginstal dan menghapus paket, dengan menambahkan dan menghapus paket sebagai dependensi proyek di manifestasi proyek Anda, dan kemudian memilih versi yang benar untuk menginstal, yang tidak selalu cocok dengan versi yang Anda tunjukkan. Untuk informasi lebih lanjut, lihat Ketergantungan dan resolusi.

Mengakses jendela Manajer Paket

Untuk membuka jendela Paket Manager, arahkan ke menu utama Unity dan pergi ke Window > Package Manager.
