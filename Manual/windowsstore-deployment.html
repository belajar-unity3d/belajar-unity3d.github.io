Platform Windows Universal: Penyebaran

Cara menyebarkan Aplikasi Universal Windows

  • Pergi ke Membangun Pengaturan dan membangun Proyek Anda ke folder.

  • Solusi Visual Studio akan dihasilkan.

  • local machine:
    • Pastikan bahwa platform solusi yang dipilih adalah x86 atau x64 dan Debug target diatur ke “Local machine”.
    • Pergi ke Debug -> Start debugging atau tekan F5.
  • remote machine:
    • Jika Anda menggunakan perangkat arsitektur ARM, pastikan bahwa ARM atau ARM64 dipilih sebagai Solution Platform.
    • Mulai Visual Studio Remote debugger pada mesin jarak jauh.
    • Buka Visual Studio project settings -> Debug.
    • Mengatur perangkat target ke “Remote machine”.
    • Temukan mesin jarak jauh dalam daftar dan pilih metode otentikasi "Universal".
    • Pergi ke Debug -> Start debugging atau tekan F5.
    • Jika Anda tidak dapat terhubung ke mesin jarak jauh, pastikan bahwa Firewall tidak menghalanginya.
  • simulator:
    • Set Debug Target untuk “Simulator”.
    • Pergi ke Debug -> Start debugging atau tekan F5.
