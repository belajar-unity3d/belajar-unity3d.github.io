Membuat templat adegan

Anda dapat membuat template sceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary baru dalam salah satu cara berikut:

  • Mulai dari templat kosong.
  • Buat template dari aset adegan yang ada.
  • Buat template dari adegan saat ini.

Setelah Anda membuat template, Anda dapat edit sifatnya atau menciptakan adegan baru dari itu.

Tip:
Sebelum Anda membuat template dari tempat kejadian, buat folder dengan nama sebagai adegan, dan masukkan aset apa pun yang ingin Anda klon di dalamnya. Ketika Anda membuat template, Unity secara otomatis memungkinkan properti Clone untuk aset tersebut. Untuk rincian, lihat Mengedit template adegan.

Membuat template adegan kosong

Anda dapat membuat template adegan kosong dan mengkonfigurasinya nanti. Templat kosong tidak muncul di dialog Adegan Baru sampai Anda edit sifatnya untuk mengasosiasikan aset adegan untuk itu.

Untuk membuat template adegan kosong di folder proyek saat ini:

  • Dari menu, pilih Assets > Create > Scene Template.

Untuk membuat template adegan kosong di folder proyek tertentu:

  1. Lakukan salah satu berikut:

  2. Pada Project windowJendela yang menunjukkan isi folder Assets (Project tab) More info
    Lihat di Glossary    , klik kanan folder untuk membuka menu konteks.

  3. Buka folder di jendela Proyek, dan klik kanan pane aset untuk membuka menu konteks.

  4. Pilih Create > Scene Template.

Membuat template dari aset adegan yang ada

Anda dapat mengubah adegan yang ada ke dalam template adegan. Setelah Anda membuat template dari adegan yang ada, Anda mungkin ingin edit sifatnya untuk menentukan mana dependensinya Unity clones ketika Anda membuat adegan baru dari itu.

Untuk membuat template dari aset adegan yang ada, buka jendela Proyek, dan lakukan salah satu berikut:

  • Klik kanan aset adegan untuk membuka menu konteks. Kemudian pilih Create > Scene Template From Scene.
  • Pilih aset adegan, dan dari menu utama, pilih Assets > Create > Scene Template From Scene.

Membuat template dari adegan saat ini

Untuk membuat template adegan dari adegan saat ini, dari menu, pilih File > Save As Scene Template.

Jika Anda memiliki perubahan yang tidak disadari, Unity meminta Anda untuk menyimpan adegan sebelum menyimpan template.

Setelah Anda membuat template dari adegan saat ini, Anda mungkin ingin edit sifatnya untuk menentukan mana dependensinya Unity clones ketika Anda membuat adegan baru dari itu.

Membuat template dari C# skrip

Lihat di Glossary Anda.

Untuk membuat template adegan kosong, gunakan CreateSceneTemplate.

SceneTemplate.CreateSceneTemplate(string sceneTemplatePath)

Untuk membuat template dari adegan yang ada, gunakan CreateTemplateFromScene. Unity secara otomatis mengasosiasikan adegan dengan template, dan mengekstrak ketergantungan adegan.

SceneTemplate.CreateTemplateFromScene(SceneAsset sourceSceneAsset, string sceneTemplatePath);
