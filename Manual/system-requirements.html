Batasan platform Unity Editor

Apple

Pada macOS, jendela editor sekunder hanya memaksimalkan, dan tidak memasukkan mode layar penuh.

Perangkat silikon apel

Editor silikon Apple memiliki keterbatasan berikut:

  • Download dan menginstal Apple silikon Editor melalui Unity Hub hanya mungkin melalui versi Hub 3.0 Beta 6 atau kemudian, dengan beberapa keterbatasan dalam 3.0 Beta 6. Juni Untuk informasi lebih lanjut, lihat:
    • Untuk beralih ke versi beta terbaru dari Hub, mengubah "channel" di Preferensi > Lanjutan ke “Beta” atau unduh kembali dari .
  • Jika Anda menggunakan Hub 2.4.5 dan ingin menggunakan Apple silikon Editor: uninstall Apple intel versi, download dan menjalankan installer mandiri untuk Unity Editor (Apple silikon) dari, dan menemukannya melalui Hub.
  • Editor silikon Apple asli tidak mendukung lightmapper CPU
  • Editor silikon Apple asli tidak mendukung impor dari SketchUp

Linux

Editor Linux memiliki keterbatasan berikut:

  • Video impor terbatas pada format video VP8
  • Sistem file case sensitif
