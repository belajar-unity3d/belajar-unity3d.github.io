Bekerja dengan Heightmaps

Alat Terrain yang mempengaruhi tinggi, seperti Raise atau Lower Terrain dan Set Tinggi, gunakan tekstur abu-abu yang disebut heightmap. Unity mewakili ketinggian setiap titik pada TerrainPemandangan di tempat kejadian Anda. Terrain GameObject menambahkan pesawat datar besar untuk adegan Anda dan Anda dapat menggunakan jendela Inspektur Terrain untuk menciptakan lanskap rinci. More info
Lihat di Glossary sebagai nilai dalam array persegi panjang. Ini mewakili array ini menggunakan heightmapTekstur abu-abu yang menyimpan data tinggi untuk objek. Setiap pixel menyimpan perbedaan tinggi tegak lurus dengan wajah yang pixel mewakili.
Lihat di Glossary skala abu-abu. Heightmaps dibangun ke Terrain, dan nilai-nilai yang disimpan dalam heightmap mendefinisikan ketinggian setiap titik atau simpul di Terrain.

Mengimpor dan mengekspor peta tinggi

Anda dapat mengimpor dan mengekspor heightmap ke dalam Editor Unity. Ini berguna ketika Anda ingin menggunakan data ketinggian dunia nyata untuk meniru landmark seperti Gunung Everest, atau bekerja pada gambar heightmap dalam editor eksternal seperti Photoshop. Anda juga dapat menggunakan aplikasi model 3D, seperti Houdini dan Mesin Dunia, untuk menghasilkan Terrain, kemudian mengimpor Terrain Anda ke Unity sebagai peta tinggi.

Praktik bagus untuk menyimpan heightmaps sebagai file RAW. File RAW menggunakan format abu-abu 16-bit yang kompatibel dengan sebagian besar editor gambar dan lansekap. Editor Unity memungkinkan Anda untuk mengimpor dan mengekspor file peta tinggi RAW untuk Terrain.

Untuk mengakses pengaturan impor dan ekspor ke dalam Editor, pilih komponen Terrain di InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info
Lihat di Glossary, dan klik tombol Terrain Settings ( ikon gigi di toolbar).

Di bawah Texture Resolutions (On Terrain Data), ada dua tombol yang dilabeli Import Raw dan Export Raw.

  • Import Raw memungkinkan Unity untuk membaca peta tinggi dari format file RAW, dan menghasilkannya di Editor.

  • Export Raw memungkinkan Unity untuk menulis peta tinggi dari Editor ke format file RAW.
