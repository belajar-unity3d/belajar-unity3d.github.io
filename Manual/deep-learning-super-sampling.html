Dalam belajar super sampling

Login Deep Learning Super Sampling (DLSS) adalah teknologi rendering yang menggunakan kecerdasan buatan untuk meningkatkan kinerja grafis dan kualitas. Anda dapat menggunakannya untuk:

  • Jalankan dunia nyata-waktu sinar yang dilacak pada tingkat bingkai dan resolusi tinggi.
  • Memberikan peningkatan kinerja dan kualitas untuk grafis rasterized. Ini sangat berguna untuk aplikasi realitas virtual untuk membantu mereka berjalan pada tingkat bingkai yang lebih tinggi. Ini membantu untuk menghilangkan disorientasi, mual, dan efek negatif lainnya yang terjadi pada tingkat bingkai yang lebih rendah.

Untuk informasi lebih lanjut tentang DLSS, lihat Dokumentasi NVIDIA.

Platform yang didukung

Unity mendukung DLSS pada platform berikut:

  • Windows XP (32/64 bit)
  • Windows XP (32/64 bit)
  • Vulkan pada Windows 64 bit.

Unity tidak mendukung DLSS untuk Metal, Linux, Windows menggunakan arsitektur x86 (Win32), atau platform lainnya. Untuk membangun proyek Anda untuk arsitektur Windows, gunakan arsitektur x86_64 (Win64).

Mode kualitas

Mode kualitas yang tersedia adalah sebagai berikut:

  • Auto: Pilih mode DLSS terbaik untuk resolusi output saat ini.
  • Balance: Kinerja keseimbangan dengan kualitas gambar.
  • Quality: Memperkuat kualitas gambar atas kinerja.
  • Performance: Memperkuat kinerja dan kualitas gambar.
  • Ultra Performance: Menghasilkan kinerja tertinggi dan kualitas gambar terendah.
