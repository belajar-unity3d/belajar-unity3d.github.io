Paket Asset Store

Paket Asset Store adalah koleksi file dan data dari proyek Unity, atau elemen proyek, yang mengompresi Unity dan menyimpan dalam satu file dengan ekstensi .unitypackage. Seperti file zip, paket Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
Lihat di Glossary mempertahankan struktur direktori aslinya ketika tidak dikemas, serta metadata tentang aset (seperti pengaturan impor dan tautan ke aset lain).

Untuk membeli paket Asset Store dari Unity atau penerbit pihak ketiga, mulailah untuk menemukan aset yang memenuhi kebutuhan Anda. Ketika Anda menemukan paket Asset Store yang ingin Anda gunakan, pembelian atau unduhan di Asset Store. Anda juga dapat membuat lebih mudah untuk menemukan paket Aset Store dengan mengatur aset Anda dengan label dan Cek Resi Asset Store yang diuraikan atau tidak digunakan.

Dalam Unity Package Manager, pilih filter Aset Saya untuk melihat Daftar paket Aset Store yang tersedia Anda. Anda dapat Cari berdasarkan nama untuk paket Aset Store. Ketika Anda menemukan paket Asset Store yang ingin Anda gunakan, Anda dapat download dan impor di dalam Paket Manager. Jika paket Aset Store Anda memiliki versi terbaru yang tersedia, Anda juga dapat update langsung di jendela Paket Manager.

Pengguna Unity dapat menjadi penerbit di Toko Aset, dan menjual konten yang mereka buat. Ikuti petunjuk di Penerbitan ke Toko Aset untuk membuat draft paket dan mengunggah aset Anda dari Unity ke dalamnya untuk publikasi di .

Lokasi file aset yang diunduh

Anda dapat menggunakan jendela Paket Manager untuk mengelola paket Aset Store Anda dalam proyek Anda. Namun, jika Anda perlu mengakses file aset Anda langsung, Anda dapat menemukannya di cache paket Asset Store. Cache ini, yang terpisah dari cache global, membuat reusing dan berbagi paket lebih efisien. Anda dapat menemukan cache paket Asset Store di jalur berikut (yang mungkin dalam folder tersembunyi tergantung pada pengaturan komputer Anda):

  • macOS: ~/Library/Unity/Asset Store-5.x
  • Windows: C:\Users\accountName\AppData\Roaming\Unity\Asset Store-5.x
  • Linux: ~/.local/share/unity3d/Asset Store-5.x

folder ini mengandung subfolder yang sesuai dengan vendor Toko Aset tertentu dan tersedia setelah mengunduh dan impor. Manajer Paket menyimpan file aset di dalam subfolder menggunakan struktur yang didefinisikan oleh penerbit paket Asset Store.

Anda dapat menimpa lokasi default dari akar cache paket Asset Store. Untuk informasi, lihat Sesuaikan lokasi cache Aset Store.
