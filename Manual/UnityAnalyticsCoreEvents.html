Acara Inti

Acara Inti berbasis sesi dan acara AnalyticsUnity Analytics
Lihat di Glossary berbasis perangkat. Ketika Anda mengaktifkan Unity AnalyticsSebuah platform data yang menyediakan analitik untuk permainan Unity Anda. More info
Lihat di Glossary untuk proyek Anda, Unity mengirimkan Acara Inti secara otomatis. Login Acara menyediakan dasar untuk beberapa metrik yang dikombinasikan oleh sistem Analytics, termasuk:

  • Pemasangan baru
  • Pengguna aktif harian (DAU)
  • Pengguna aktif bulanan (MAU)
  • Total sessions
  • Sesi per pengguna
  • Waktu yang dihabiskan dalam aplikasi
  • Segmen Pengguna untuk negara dan platform
  • Pendapatan (ketika menggunakan iklan Unity dan/atau IAP Unity)

Lihat Analisis Metrik, Segmen, dan Terminologi untuk definisi metrik ini.

Acara Inti meliputi:

  • AppStart: Dihentikan pada awal sesi baru. Sesi baru dimulai ketika pengguna pertama meluncurkan aplikasi atau membawa aplikasi ke latar belakang setelah lebih dari 30 menit ketidakaktifan.
  • AppRunning: Dihentikan secara berkala sementara aplikasi berjalan.
  • DeviceInfo: Dihentikan pertama kali pengguna meluncurkan aplikasi dan setiap kali perubahan informasi perangkat.

Selain melihat metrik berdasarkan peristiwa inti di Dasbor Analisis, Anda dapat mengunduh acara itu sendiri menggunakan Ekspor Data Raw. Catatan bahwa Ekspor Data Baku memerlukan langganan Unity Pro.

Sitemap Analytics menggunakan Peristiwa Inti lainnya, seperti Note:, AppStop, dan AppUpdate, untuk menghitung metrik, tetapi tidak mengeksposnya melalui AppInstall.Ekspor Data Baku.
