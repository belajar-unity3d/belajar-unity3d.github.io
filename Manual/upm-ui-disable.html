Mengaktifkan paket bawaan

Anda dapat menonaktifkan Built-in packageBuilt-in
Lihat di Glossary jika Anda tidak memerlukan beberapa modul dan Anda ingin menyimpan sumber daya. Namun, ketika Anda menonaktifkan paket bawaan, fungsi Unity yang sesuai tidak lagi tersedia. Hasil ini dalam hal berikut:

  • Jika Anda menggunakan API Scripting yang diimplementasikan oleh paket yang dinonaktifkan, Anda mendapatkan kesalahan kompiler.
  • Komponen yang diimplementasikan oleh paket bawaan yang dinonaktifkan juga dinonaktifkan, yang berarti Anda tidak dapat menambahkannya ke GameObjects. Jika Anda memiliki GameObjectObjek mendasar dalam adegan Unity, yang dapat mewakili karakter, props, pemandangan, kamera, waypoints, dan banyak lagi.   • Saat membangun permainan, Strip Unity semua komponen yang dinonaktifkan. Untuk membangun target yang mendukung pengupasan kode mesin (seperti WebGL, iOS, dan Android), Unity tidak menambahkan kode dari paket bawaan yang dinonaktifkan.

Untuk menonaktifkan paket bawaan:

  1. Buka jendela Paket Manager dan pilih Built-in packages dari menu drop-down PaketPaket adalah koleksi aset untuk dibagikan dan digunakan kembali dalam Unity. Manajer Paket Unity (UPM) dapat menampilkan, menambahkan, dan menghapus paket dari proyek Anda. Paket-paket ini berasal dari Unity Package Manager dan memberikan metode mendasar untuk memberikan fungsi Unity. Namun, Unity Package Manager juga dapat menampilkan Paket Asset Store yang Anda unduh dari Asset Store. More info
    Lihat di Glossary    .

  2. Pilih paket bawaan yang Anda inginkan. Informasinya muncul di details view.

  3. Klik tombol Disable.

    Ketika progress bar selesai, tanda cek tidak lagi muncul di sebelah paket built-in dan tombol Disable menjadi tombol Enable.

  4. Jika Anda ingin mengaktifkan kembali paket built-in yang dinonaktifkan, klik tombol Enable.

Note: Anda dapat menonaktifkan beberapa paket built-in dengan satu klik dengan menggunakan beberapa fitur pilih. Untuk informasi lebih lanjut, lihat Melakukan tindakan pada beberapa paket atau set fitur.
