Konfigurasi untuk Amazon Appstore

Introduction

Panduan ini menjelaskan proses pengaturan Amazon Appstore untuk digunakan dengan sistem pembelian in-app Unity. Ini termasuk menetapkan catatan digital dan hubungan yang diperlukan untuk berinteraksi dengan API Unity IAPUnity In App Purchase
Lihat di Glossary, menyiapkan akun pengembang Amazon, dan menguji dan menerbitkan aplikasi Unity IAP.

Seperti platform lain, toko Amazon memungkinkan pembelian barang virtual dan produk yang dikelola. Produk digital ini diidentifikasi menggunakan pengidentifikasi string dan tipe tambahan untuk mendefinisikan daya tahan, dengan pilihan termasuk berlangganan (disesuaikan berlangganan), habis pakai (dapat dituntut berbelas), dan non-konsumabel (sekap dibeli sekali).

Pelaksanaan lintasan pembelian dalam aplikasi

Ada masalah instalasi lintas toko dengan menerbitkan ke beberapa toko Android IAP (misalnya Amazon dan Google) secara bersamaan dan berbagi pengenal bundel Android. Silakan lihat halaman di Masalah instalasi lintas toko dengan toko pembelian di aplikasi Android untuk mempelajari lebih lanjut.

Amazon Appstore

Getting started

  1. Mengatur akun pengembang Amazon di Portal Login.
  2. Tulis permainan menerapkan API Unity IAP. Untuk referensi, lihat panduan pada Awali IAP Unity dan . Gunakan Amazon Appstore untuk aplikasi tanpa batasan pada item IAP.

Pengaturan perangkat

  1. Untuk perangkat Android, unduh dan instal Amazon Appstore.

  2. Untuk perangkat FireOS, Amazon Appstore harus datang pra-instal.

    Note: Meskipun Anda dapat secara bebas menargetkan perangkat FireOS, FireOS bukan platform yang didukung Unity.

  3. Setelah Anda menginstal Amazon Appstore, pasang Penguji Aplikasi Amazon.

Pengaturan aplikasi Unity

Menyiapkan untuk menggunakan IAP Unity memerlukan beberapa langkah.

  1. Mengimpor Unity IAP plug-inSatu set kode yang dibuat di luar Unity yang menciptakan fungsi dalam Unity. Ada dua jenis plug-ins yang dapat Anda gunakan di Unity: Managed plug-ins (diproduksi. Rakitan NET dibuat dengan alat-alat seperti Studio Visual) dan plug-ins asli (pustaka kode asli yang spesifik platform). More info
    Lihat di Glossary    . Lihat Menyiapkan Unity IAP untuk informasi lebih lanjut (Unity 5.3 atau lebih tinggi).

  2. Mengatur toko target IAP. Anda harus memiliki aplikasi Android yang ditetapkan. Mengatur toko target menggunakan item menu Unity IAP’s Window > Unity IAP > Android > Target Amazon. Ini digunakan untuk beralih antara Google, Amazon, dan toko Android lainnya.

Atau, sebut API:

UnityPurchasingEditor.TargetAndroidStore(AndroidStore.AmazonAppStore)

Amazon Appstore setup

Tidak perlu mengunduh plug-in IAP asli Amazon saat mempersiapkan untuk menggunakan toko Amazon, karena semua fungsi yang disediakan sudah disertakan dalam layanan IAP Unity.

  1. Tambahkan aplikasi Anda. Dari Amazon Developer Portal pilih Add a New App.

  2. Mengatur katalog Anda. Menggunakan deskripsi produk yang disiapkan sebelumnya, tambahkan item ke katalog Amazon menggunakan Amazon Developer Portal. Navigasi ke halaman aplikasi Anda, dan temukan In-App Items section. Gunakan tombol Add a Consumable, Add an Entitlement, atau Add a Subscription untuk mengatur katalog Anda.
