Layar Network Manager HUD ("heads-up") menyediakan fungsi dasar sehingga orang bermain game Anda dapat memulai hosting permainan jaringan, atau menemukan dan bergabung dengan permainan jaringan yang ada. The Network Manager HUD adalah alat yang cepat untuk membantu Anda mulai membangun permainan multiplayer Anda langsung, tanpa terlebih dahulu harus membangun antarmuka pengguna untuk penciptaan permainan / koneksi / bergabung. Hal ini memungkinkan Anda untuk melompat langsung ke pemrograman gameplay Anda, dan berarti Anda dapat membangun versi Anda sendiri dari kontrol ini kemudian dalam jadwal pengembangan Anda.

Namun, dimaksudkan untuk dimasukkan dalam permainan selesai. Idenya adalah bahwa kontrol ini berguna untuk memulai, tetapi Anda harus membuat UI Anda sendiri nanti, untuk memungkinkan pemain Anda untuk menemukan dan bergabung game dengan cara yang sesuai dengan permainan Anda. Misalnya, Anda mungkin ingin stylisasi desain layar, kancing dan daftar game yang tersedia untuk mencocokkan gaya keseluruhan permainan Anda.

Untuk mulai menggunakan Manajer Jaringan HUD, buat GameObject kosong di Adegan Anda (menu: GameObject > Create Empty) dan tambahkan komponen Jaringan HUD ke GameObject baru.

Menggunakan HUD

Manajer Jaringan HUD memiliki dua mode dasar: mode LAN (Local Area Network) dan mode Matchmaker. Mode ini cocok dengan dua jenis permainan multiplayer. Mode LAN adalah untuk membuat atau bergabung game yang dihosting di jaringan daerah lokal (yaitu, beberapa komputer yang terhubung ke jaringan yang sama), dan mode Matchmaker adalah untuk membuat, menemukan dan bergabung permainan di internet (multiple komputer yang terhubung ke jaringan terpisah).

Manajer Jaringan HUD memulai dalam mode LAN, dan menampilkan tombol yang berkaitan dengan hosting dan bergabung dengan permainan multiplayer berbasis LAN. Untuk mengganti HUD ke mode Matchmaker, klik tombol Enable Match Maker (M).

Note: Ingat bahwa fitur Jaringan Manajer HUD adalah bantuan sementara untuk pengembangan. Ini memungkinkan Anda untuk mendapatkan permainan multiplayer Anda berjalan dengan cepat, tetapi Anda harus menggantinya dengan kontrol UI Anda sendiri ketika Anda siap.
