Kompatibilitas platform

Unity mendukung BRG pada:

  • Windows menggunakan DirectX 11
  • Windows XP (32/64 bit)
  • Windows menggunakan Vulkan
  • Windows Server
  • Linux menggunakan Vulkan
  • macOS menggunakan logam
  • iOS
  • Android menggunakan Vulkan
  • WinZip
  • WinRAR 5.40
  • Xbox One
  • Seri Xbox X dan Seri Xbox S
  • Nintendo Switch
  • Google Login

Menyiapkan proyek Anda untuk BatchRenderer Login

Sebelum Anda menggunakan BRG, proyek Anda harus mendukungnya. BRG mengharuskan proyek Anda ke:

  • Gunakan Batcher SRP. Untuk mengaktifkan SRP Batcher, lihat Menggunakan SRP Batcher.

  • Sitemap Untuk melakukan ini, pilih varian warnaSebuah verion dari program naungan yang Unity menghasilkan sesuai dengan kombinasi khusus kata kunci naungan dan status mereka. Objek Shader dapat mengandung beberapa varian naungan. More info
    Lihat di Glossary     > Edit > Project Settings, dan set Graphics ke BatchRendererGroup variants.Keep all.

  • Jika proyek Anda menggunakan URP, praktik terbaik untuk menonaktifkan Strip Unused Variants. Ini membantu untuk menghindari masalah dengan pengupasan Unity yang diperlukan DOTS Instancing varian. Untuk informasi lebih lanjut, lihat Login Peredam naungan. Untuk menemukan pengaturan ini, pilih Edit > Project Settings > URP Global Settings.
