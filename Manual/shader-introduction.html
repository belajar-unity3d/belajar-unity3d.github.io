Terminology

Terminologi di sekitar naungan dapat membingungkan; orang-orang sering menggunakan kata "shader" untuk berarti berbagai hal.

Dalam dokumentasi ini, terminologi adalah sebagai berikut:

  • shader atau shader program - program yang berjalan pada GPU. Kecuali ditentukan, ini berarti program naungan yang merupakan bagian dari pipa grafis.
  • Shader object - contoh kelas Shader. Objek Shader adalah pembungkus untuk program naungan dan informasi lainnya.

  • Shader Graph asset - sebuah file dalam proyek Unity Anda. Ini mendefinisikan objek Shader.
