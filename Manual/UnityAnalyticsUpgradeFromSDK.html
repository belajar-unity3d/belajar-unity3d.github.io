Sitemap Penghapusan SDK Unity Analytics yang ada

Versi 5.2 seterusnya dari Editor Unity hadir dengan built-in Unity Analytics. Kami menyarankan Anda untuk menghapus komponen SDK Unity Analytics yang ada untuk memastikan tidak ada konflik dan untuk membantu Anda mempertahankan proyek yang lebih bersih.

Temukan komponen SDK Unity Analytics yang ada di folder Proyek Anda. Kemungkinan besar mereka akan disimpan ke Aset > Login Be sure to only select analytics-specific assets.

Hapus semua komponen.

Anda dapat mengkonfirmasi bahwa Anda berhasil menghapus SDK Unity Analytics jika Anda menutup dan membuka kembali proyek Anda, dan kemudian tidak melihat pesan berikut prompt. Jika Anda terus melihat prompt mengkonfirmasi semua komponen SDK Unity Analytics telah dihapus.

2. Re-integrasi dari Unity Analytics

Ikuti petunjuk di Menyiapkan Analytics.

Anda dapat menentukan bahwa Anda berhasil menyelesaikan upgrade dengan memeriksa versi SDK di Play untuk Mengvalidasi langkah dan mengkonfirmasi bahwa versi Editor yang benar (mis: u5.2 atau u5.3) ditampilkan dalam Validator.

3. Juni Memperbarui acara Integrasi Lanjutan

Jika Anda melakukan integrasi Advanced yang ada sebelumnya, Anda juga perlu memperbarui namespace dan panggilan untuk menggunakan 5.2 dan up sintaks. Ikuti 5.2 seterusnya Instruksi Integrasi Lanjutan untuk memperbarui.
