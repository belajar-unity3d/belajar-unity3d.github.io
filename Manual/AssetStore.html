Paket Asset Store

Toko Aset Unity

Unity Asset Store mengandung pustaka aset gratis dan komersial yang menyatukan Technologies serta anggota masyarakat menciptakan. Berbagai aset tersedia, termasuk Tekstur, Model, animasi, seluruh contoh proyek, tutorial, dan ekstensi Editor.

Mulai dari Unity 2020.1, jendela Aset Khusus tidak lagi dihosting di dalam Editor Unity. Namun, Anda masih dapat mengakses situs web Asset Store dan Anda masih bisa search untuk Paket Asset Store yang dibeli dan diunduh, serta impor dan unduhan langsung di jendela Paket Manager.

Bagian ini berisi informasi untuk konsumen dan penyedia produk Aset Store:

  • Paket Asset Store berisi informasi tentang cara menemukan, memperoleh, dan mengatur paket Toko Aset Anda langsung di situs Aset Store.
  • Penerbitan ke Toko Aset membantu panduan Produsen paket Asset Store melalui pembuatan, pemasaran, dan mengelola paket Asset Store untuk komunitas Unity.

Situs Web Asset Store

Untuk mengakses Aset Store, buka browser web dan arahkan ke https://assetstore.unity.com/.

Sitemap Menu koleksi membantu Anda mempersempit daftar paket Aset yang cocok. Mereka kelompok terkait paket Aset Store bersama-sama dan menampilkannya dalam daftar. Misalnya, Anda dapat memilih (A) dari menu TerrainThe landscape in your scene. A Terrain GameObject adds a large flat plane to your scene and you can use the Terrain’s Inspector window to create a detailed landscape. More info
See in Glossary untuk melihat paket Asset Store yang mengandung alat-alat bina.Tools menu to see Asset Store packages that contain terrain-building tools.

Sitemap Kotak pencarian memungkinkan Anda mencari paket Asset Store dengan memasukkan kata kunci, seperti nama paket Asset Store, kata atau frasa dalam deskripsinya, atau nama file yang cocok dengan file apa pun dalam paket Asset Store.(B) The search box lets you search for an Asset Store package by entering a keyword, such as the Asset Store package’s name, a word or phrase in its description, or a filename matching any files in the Asset Store package.

Sitemap Tautan cepat memberikan akses ke area berikut:(C) The quick links provide access to the following areas:

  • Daftar aset yang dibeli atau diunduh untuk akun Anda (halaman Aset Saya Anda)
  • Daftar aset yang ditandai sebagai Favorites
  • Shopping Cart%
  • Aset Toko applications, seperti Dashboard, Connect, Answers, Forum, dan Distribute
  • Unity ID (akun), di mana Anda dapat melihat dan memodifikasi pengaturan, seperti daftar kustom, lisensi, pemberitahuan, privasi data, atau menebus voucher dan melihat kredit Toko Aset Anda.

Sitemap Daftar paket Asset Store yang sesuai dengan pencarian, filter, penyortiran, atau kategori yang Anda pilih. Selain itu, Unity menampilkan saran dan daftar khusus paket Asset Store di sini.(D) The list of Asset Store packages that match any searches, filters, sorting, or categories you select. In addition, Unity displays suggestions and special lists of Asset Store packages here.

Sitemap Sidebar filter membantu mempersempit daftar paket Asset Store ditampilkan. Anda dapat menerapkan filter ini untuk setiap pencarian, kategori, pilihan, atau memilah skema yang Anda buat.(E) The filter sidebar helps narrow down the list of Asset Store packages displayed. You can apply these filters to any searches, categories, selections, or sorting schemes you make.
