Jendela Rendering Statistik

Tampilan Game mencakup jendela Rendering Statistik yang menampilkan informasi rendering real-time tentang aplikasi Anda selama mode Play. Untuk membuka jendela ini, klik tombol Stats di sudut kanan atas dari tampilan Game. Unity menampilkan jendela Statistik sebagai overlay di kanan atas pandangan Game. Statistik rendering ditampilkan di jendela bagian Graphics berguna untuk mengoptimalkan kinerja. Pengaturan statistik yang tepat tersedia bervariasi sesuai dengan target build.

Statistics

Bagian grafis dari jendela Statistik berisi informasi berikut:

Statistic Description
