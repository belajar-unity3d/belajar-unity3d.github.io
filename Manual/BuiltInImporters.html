Impor Built-in

Unity mendukung banyak jenis file aset melalui koleksi importir built-in. Sebagian besar importir ini diimplementasikan dalam kode asli Unity Editor, dan diberi nama “pengelasan dasar”. Mereka menyediakan fungsi impor untuk sebagian besar jenis aset dasar Unity, seperti model 3D, Tekstur, dan file audio.

importir skrip built-in

Scripted importers memungkinkan Anda untuk menulis impor kustom Anda sendiri untuk format yang Unity tidak mendukung secara asli. Namun, selain importir asli bawaan yang tercantum di atas, beberapa importir bawaan Unity sendiri diterapkan sebagai importir skrip. Ini karena mereka diterapkan dalam C# dalam paket inti, daripada dalam kode asli Editor itu sendiri. Unity mengimpor aset importir skrip setelah aset importir asli.
