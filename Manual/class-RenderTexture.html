Properties

Render Texture inspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info

Inspektur Tekstur Render menampilkan konten saat ini Tekstur Render secara real-time dan dapat menjadi alat debugging yang tak ternilai untuk efek yang menggunakan tekstur render.

Property: Function:
Dimension Dimensiitas (tipe) dari tekstur render.
2D Tekstur render adalah dua dimensi.
Cube Tekstur render adalah peta kubus.
3D Tekstur render adalah tiga dimensi.
Size Ukuran tekstur render dalam pixelsUnit terkecil dalam gambar komputer. Ukuran piksel tergantung pada resolusi layar Anda. Pencahayaan pixel dihitung pada setiap piksel layar. More info
Lihat di Glossary. Anda hanya dapat memasukkan nilai power-of-dua, seperti 128 dan 256.
Anti-Aliasing Jumlah sampel anti-aliasing. Anda dapat memilih None, 2 samples, 4 samples, atau 8 samples. Jika Anda memilih None, Unity tidak menerapkan anti-aliasing.
Enable Compatible Color Format Aktifkan kotak centang ini untuk membuat Unity menerapkan format yang kompatibel dengan tekstur render jika Color Format didefinisikan tidak didukung oleh platform.
Color Format Format warna tekstur render.
Depth Buffer Format penyangga kedalaman. Anda dapat memilih No depth buffer, At least 16 bits depth (no stencil), atau At least 24 bits depth (with stencil). stencil buffer adalah penyangga tujuan umum yang memungkinkan Anda untuk menyimpan integer 8-bit tambahan (0-255) untuk setiap piksel yang ditarik ke layar.
Enable Mip Maps Periksa kotak ini untuk membuat tekstur render menghasilkan.
Auto generate Mip Maps Periksa kotak ini untuk secara otomatis mengisi mipmaps yang dihasilkan dengan data yang relevan. Jika Anda tidak mengaktifkan ini, Anda harus menggunakan fungsi untuk mengisi mipmaps secara manual. Atau, pilih yang mip untuk render ke ketika Anda memanggil berbagai fungsi SetRenderTarget. Untuk informasi lebih lanjut tentang fungsi SetRenderTarget, lihat dan .
Dynamic Scaling Periksa kotak ini untuk membiarkan skala resolusi dinamis mengubah ukuran tekstur render. Jika Anda tidak mengaktifkan ini, tekstur render mempertahankan ukuran yang sama terlepas dari pengaturan Dynamic ResolutionPengaturan kamera yang memungkinkan Anda untuk menargetkan render individu skala dinamis, untuk mengurangi beban kerja pada GPU. Wrap Mode Kontrol bagaimana tekstur dibungkus:
Repeat Ubin tekstur untuk membuat pola berulang.
Clamp Peregangan tepi tekstur. Ini berguna untuk mencegah artefak pembungkus ketika Anda memetakan gambar ke objek dan Anda tidak ingin tekstur ubin.
Mirror Ubin tekstur untuk membuat pola pengulangan yang mencerminkan tekstur di setiap batas bilangan bulat.
Mirror Once Cermin tekstur sekali, dan kemudian jatuh kembali ke menjepit.
Per-axis Memungkinkan Anda mengatur mode bungkus yang berbeda untuk U axis dan V axis. Pilihan yang tersedia juga Repeat, Clamp, Mirror dan Mirror Once. Misalnya, ketika Anda menggunakan peta lingkungan sepanjang garis untuk reflection probesKomponen rendering yang menangkap pandangan bulat dari sekitarnya di semua arah, seperti kamera. Gambar yang ditangkap kemudian disimpan sebagai kubus yang dapat digunakan oleh benda dengan bahan reflektif. More info
Lihat di Glossary, berguna untuk memiliki Clamp pada koordinat vertikal (V axis), tetapi Repeat pada koordinat horisontal (U axis).
Filter Mode Mengontrol bagaimana pengambilan tekstur menggunakan piksel terdekat. Pilihannya adalah:
Point Menggunakan pixel terdekat. Ini membuat tekstur muncul pixelated.
Bilinear Menggunakan rata-rata berbobot empat texels terdekat. Ini membuat tekstur tampak kabur ketika Anda memperbesarnya.
Trilinear Menggunakan rata-rata berbobot dua mips terdekat, yang disaring bilinearly. Ini menciptakan transisi lunak antara mips, dengan biaya penampilan sedikit lebih kabur.
Aniso LevelTingkat penyaringan anisotropic (AF) tekstur. Memungkinkan Anda untuk meningkatkan kualitas tekstur saat melihat tekstur pada sudut curam. Baik untuk tekstur lantai dan tanah. More info
Lihat di Glossary		 Tingkat penyaringan Anisotropic dari tekstur. Ini meningkatkan kualitas tekstur saat Anda melihat tekstur pada sudut yang curam. Baik untuk lantai, tanah, atau tekstur jalan.

Example

Untuk membuat kamera arena hidup dalam permainan Anda:

  1. Buat aset Tekstur Render baru menggunakan Assets >Create >Render Texture.
  2. Buat Kamera baru menggunakan GameObject > Camera.
  3. Menetapkan Tekstur Render ke Target Texture Kamera baru.
  4. Buat kubus 3D baru menggunakan GameObject > 3D Object > Cube.
  5. Seret Tekstur Render ke kubus untuk menciptakan Bahan yang menggunakan tekstur render.
  6. Masukkan Mode Play, dan amati tekstur kubus diperbarui secara real-time berdasarkan output Kamera baru.
