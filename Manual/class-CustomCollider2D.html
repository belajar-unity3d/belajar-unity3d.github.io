Kustom Collider 2D

Switch to Scripting

Custom ColliderAn invisible shape that is used to handle physical collisions for an object. A collider doesn’t need to be exactly the same shape as the object’s mesh - a rough approximation is often more efficient and indistinguishable in gameplay. More info
See in Glossary 2D adalah Collider yang berinteraksi dengan sistem fisika 2D. Tidak seperti Collider lain, Anda tidak mengkonfigurasi Collider di-Editor, bukan Anda mengkonfigurasi Collider dengan menetapkan geometri FisikaShape2D ke dalamnya melalui API FisikaFormGroup2D. Anda dapat lebih mendefinisikan Collider dengan menambahkan, menghapus dan memodifikasi bentuk PhysicsShape2D. Lihat dokumentasi API FisikaShape2D untuk informasi lebih lanjut. Ini juga berarti bahwa Collider Custom 2D dapat mengandung jumlah tak terbatas PhysicsShape2D tingkat rendah dan mewakili jenis bentuk, atau meniru jenis lain dari Collider 2D.

Ketika menetapkan PhysicsShape2D ke Custom Collider 2D, Anda dapat melakukannya baik selama mode Edit atau mode Play. Ketika memodifikasi Collider Custom 2D dalam mode Edit, Unity menyimpan semua ditugaskan PhysicsShape2D dan vertic terkait di Unity SceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary, dan CustomCollider2D mempertahankan konfigurasinya ketika Adegan dimuat. Ini memungkinkan untuk menggunakan modus Edit yang memberi wewenang scriptsSepotong kode yang memungkinkan Anda untuk membuat Komponen Anda sendiri, memicu peristiwa permainan, memodifikasi sifat komponen dari waktu ke waktu dan menanggapi input pengguna dengan cara apa pun yang Anda sukai. More info
Lihat di Glossary untuk membuat geometri kustom.

Ketika memodifikasi Collider Custom 2D selama mode Play, Collider tidak akan mempertahankan perubahan yang dibuat untuk ditugaskan PhysicsShape2D dan simpul terkait mereka, dan semua perubahan akan hilang ketika keluar mode Play.
