Menggunakan label untuk mengatur “Aset Saya”

Label membantu Anda mengatur daftar paket Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
Lihat di Glossary Anda di Toko Aset dan di jendela Paket Manager. Label seperti kategori yang ditentukan pengguna, atau koleksi yang membantu Anda menemukan paket dengan mudah.

Anda hanya dapat membuat dan menggunakan label di situs Aset Store.

Sitemap Klik (A) untuk beralih ke mode edit di mana Anda dapat menetapkan label ke lebih dari satu paket Asset Store pada waktu.Manage my labels to switch to an edit mode where you can assign labels to more than one Asset Store package at a time.

Sitemap Label yang telah Anda berikan kepada paket Aset Store ini. Jika Anda belum diberi label pada paket Aset Store, (B) muncul.Add label appears instead.

Sitemap Klik (C) untuk menampilkan jendela Edit label di mana Anda dapat membuat dan menetapkan label untuk paket Aset Store ini.Edit Labels window where you can create and assign labels for this Asset Store package.

Sitemap Klik (D) untuk menampilkan jendela di mana Anda dapat membuat dan menetapkan label untuk paket Aset Store ini. Jika Anda telah ditugaskan label ke paket Aset Store, label (s) muncul sebagai gantinya.Add label to display a window where you can create and assign labels for this Asset Store package. If you have already assigned a label to the Asset Store package, the label(s) appear instead.

Sitemap Daftar label Anda muncul di sini. Ketika Anda memilih label dari daftar ini, hanya paket Asset Store yang memiliki label yang ditetapkan dalam daftar. Anda dapat memilih beberapa label sekaligus.(E) Your list of labels appears here. When you select a label from this list, only those Asset Store packages that have that label assigned appear in the list. You can select multiple labels at once.

Membuat label

Jendela Edit Labels memungkinkan Anda membuat dan menerapkan label pada saat yang sama.

Untuk membuat label baru:

  1. Pergi ke Toko Aset Unity dan masuk ke akun Unity Anda.

  2. Pergi ke halaman dan klik tautan Add label.

    Jendela Edit Labels muncul.

  3. Masukkan nama untuk label baru Anda di kotak teks.

  4. Klik Save untuk menyimpan dan menerapkan label ke paket Aset Store ini atau Cancel jika Anda tidak ingin membuat label sama sekali.

    Jendela Edit Label ditutup dan Anda dapat melihat daftar My Assets lagi.

Menambahkan dan menghapus label

Pada halaman My Assets, jika paket Asset Store sudah memiliki label, tautan Edit label muncul di bawah catatan rilis untuk paket Aset Store. Jika tidak memiliki label, tautan Add label muncul. Baik tautan ini membuka jendela Edit Labels, di mana Anda dapat menambahkan atau menghapus label.

Untuk menambahkan (apply) label yang ada ke paket Asset Store:

  1. Klik kedua Add label atau Edit label untuk membuka jendela Edit Labels.
  2. Untuk memilih label dan tambahkan ke paket Asset Store, di jendela Edit Labels, klik di kotak teks. Drop-down menunjukkan label yang belum ditugaskan untuk paket Aset Store ini. Klik pada label yang ingin Anda gunakan. Label sekarang muncul di bawah kotak teks.
  3. Klik x pada label di bawah kotak teks untuk menghapus label apa pun.
  4. Ketika Anda puas dengan label pada paket Toko Aset Anda, klik tombol Save atau Cancel jika Anda tidak ingin melakukan perubahan.

Menambahkan label ke lebih dari satu paket Asset Store sekaligus

  1. Klik Manage my labels untuk menampilkan kotak centang di samping setiap paket Asset Store dalam daftar. Satu set tautan grup juga muncul di bagian atas.

  2. Aktifkan Select all untuk memilih semua paket sekaligus atau klik kotak centang di samping setiap paket Asset Store yang ingin Anda masukkan. Klik Edit label di bagian atas untuk membuka jendela Edit Labels.

  3. Klik di dalam kotak teks untuk memilih label dari daftar untuk menambahkan label itu ke paket Asset Store yang dipilih.

  4. Klik x pada setiap label di bawah kotak teks untuk menghapus label dari paket opsi Asset Store.

  5. Ketika Anda puas dengan label pada paket Asset Store yang dipilih, klik tombol Save atau Cancel jika Anda tidak ingin melakukan perubahan.

  6. Untuk keluar mode edit, klik Done.

Label penghapusan

Label hanya ada ketika mereka digunakan, sehingga menghapus label dari daftar label, menghapus label itu dari semua aset yang menggunakannya:

  1. Di Toko Aset Unity, buka halaman My Assets.

  2. Klik Manage my labels untuk menampilkan kotak centang di samping setiap paket Asset Store dalam daftar. Satu set tautan grup juga muncul di bagian atas.

  3. Pilih setiap paket yang memiliki label yang ingin Anda hapus.

  4. Klik Edit label di bagian atas daftar paket Asset Store.

    Jendela Edit Labels muncul.

  5. Klik x pada label di bawah kotak teks untuk menghapusnya.

  6. Klik tombol Save untuk menutup jendela Edit Labels dan kembali ke daftar My Assets (stilasi dalam mode edit).

  7. Klik Done untuk keluar mode edit.

    Label yang dihapus tidak lagi muncul di bagian Labels dari sidebar filter.
