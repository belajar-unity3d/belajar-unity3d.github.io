Bahan tab

Ketika Unity mengimpor Model SpeedTree tanpa bahan yang ditugaskan, itu menggunakan Bahan diffuse Unity. Jika Model memiliki Bahan, Unity mengimpor mereka sebagai sub-Asset. Anda dapat menggunakan tab Materials untuk mengubah bagaimana penawaran Unity dengan Bahan ketika mengimpor Model SpeedTree Anda.

tab Materials mengandung bagian ini:

Sitemap Pengaturan (A) dan Location memungkinkan Anda untuk mengontrol apakah impor Unity atau mengekstrak Bahan di dalam Aset impor.MaterialsAset yang menentukan bagaimana permukaan harus diberikan. More info
Lihat di Glossary settings allow you to control whether Unity imports or extracts the Materials inside the imported Asset.

Sitemap (B) memungkinkan Anda untuk merevisi Bahan impor untuk Bahan yang ada di Proyek Anda, atau untuk mengekstrak Bahan impor dan membuat Aset terpisah di lokasi pilihan Anda.Bahan Remapped allow you to remap the imported Materials to existing Materials in your Project, or to extract the imported Materials and create separate Assets at a location of your choice.

Sitemap Pane pratinjau menampilkan Model SpeedTree dengan Bahannya.(C) The preview pane displays the SpeedTree Model with its Material.

Location

Properti Location mendefinisikan bagaimana Aset SpeedTree mengakses Bahannya. Sisa properti pada perubahan tab Materi SpeedTree tergantung pada pilihan ini yang Anda pilih.

Value Description
Use Embedded Materials Jika Anda memilih opsi ini, Unity menjaga bahan impor di dalam Aset. Anda dapat secara opsional remap bahan impor ke Materi yang ada di Proyek Anda. Login Ini adalah nilai default dan direkomendasikan.
This is the default and recommended value.
Use External Materials (Legacy) Menunjukkan bahwa Aset ini diimpor dengan versi Unity sebelumnya. Jika Anda memilih opsi ini, Unity secara otomatis mengekstrak Bahan dan menempatkan mereka di folder Bahan di lokasi yang sama sebagai Model.

Materials

Klik tombol Extract Materials untuk mengekstrak Bahan yang tertanam dalam Aset impor Anda. Ini berwarna abu-abu jika tidak ada bahan untuk mengekstrak.

Jika Anda mengimpor Bahan dengan Tekstur default sebagai Normal MapJenis tekstur Peta Bump yang memungkinkan Anda untuk menambahkan detail permukaan seperti benjolan, alur, dan goresan ke model yang menangkap cahaya seolah-olah mereka diwakili oleh geometri nyata.
Lihat di Glossary, pesan berikut muncul:

Karena pengaturan pengimpor Tekstur muncul di jendela InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info
Lihat di Glossary lain, Unity menawarkan untuk melakukan perbaikan ini untuk Anda:

  • Klik tombol Fix now jika Anda ingin Unity untuk secara otomatis mengubah Jenis Tekstur ke Normal Map pada semua file Tekstur yang terkena.
  • Klik tombol Ignore untuk mengimpor file Tekstur tanpa mengubah pengaturannya.

Bahan Remapped

Bagian ini menampilkan daftar Bahan yang dipetakan ke setiap tingkat LODTeknik Tingkat Detail (LOD) adalah optimasi yang mengurangi jumlah segitiga yang dimiliki Unity untuk membuat GameObject ketika jaraknya dari kamera meningkat. More info
Lihat di Glossary untuk Aset SpeedTree ini.

Jika Anda sudah memiliki satu set Bahan yang ingin Anda gunakan untuk Aset ini, memperluas grup On Demand Remap dan klik tombol Search and Remap.

Setelah meremapping Materi apa pun, klik tombol Regenerate Materials untuk mengekspor kembali Bahan eksternal dengan pengaturan impor terbaru.

Sitemap Melindungi kembali bahan eksternal menghilangkan modifikasi manual yang Anda buat ke file Material.Warning: Re-importing the external Materials removes any manual modifications you made to the Material files.
