Paket dan set fitur

Paket berisi fitur sesuai dengan berbagai kebutuhan proyek Anda. Ini dapat mencakup fitur inti Unity yang dipasang bersama dengan Editor, atau paket lain yang dapat Anda instal sesuai kebutuhan.

Untuk informasi lebih lanjut tentang menggunakan paket secara umum, lihat Bekerja dengan Manajer Paket.

Fitur setSebuah fitur set adalah koleksi paket terkait yang dapat Anda gunakan untuk mencapai hasil tertentu dalam Editor Unity. Anda dapat mengelola set fitur langsung di Manajer Paket Unity. More info
Lihat di Glossary adalah bundel paket terkait yang dapat Anda gunakan untuk mencapai tujuan tertentu, seperti lingkungan bangunan atau mengoptimalkan platform tertentu, atau untuk menemukan semua alat pengembangan yang Anda butuhkan di satu tempat.

Jika Anda hanya tertarik untuk menemukan dokumentasi atau informasi tentang paket tertentu, klik tautan di Promo, Paket pra-release, halaman Paket inti dan Paket built-inBuilt-in
Lihat di Glossary. Untuk menemukan paket dengan kata kunci, lihat halaman Paket dengan kata kunci.
