Menyelesaikan konflik file

Konflik terjadi ketika anggota tim lain membuat perubahan aset yang Anda juga telah berubah, dan Anda mencoba untuk memperbarui proyek Anda dengan perubahan tersebut. Berkolaborasi secara otomatis mencoba untuk menggabungkan file dan memberi tahu Anda jika upaya gagal karena konflik. Untuk file teks, Collaborate menggabungkan file menggunakan alat gabungan yang dikonfigurasi saat ini. Untuk scenesAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary dan file lain menggunakan format YAML, berkolaborasi menggunakan alat Smart Merge Unity untuk melakukan merge.

Anda dapat mengurangi terjadinya konflik dengan menerbitkan perubahan kecil sering, dan berkomunikasi dengan rekan tim Anda. Fitur In-Progress dapat membantu Anda mengurangi konflik pada adegan dan prefabsJenis aset yang memungkinkan Anda untuk menyimpan GameObject lengkap dengan komponen dan properti. Prefab bertindak sebagai template dari mana Anda dapat membuat instance objek baru di tempat kejadian. More info
Lihat di Glossary. Untuk informasi lebih lanjut, lihat Pemberitahuan edit In-Progress.

Konfigurasikan alat penggabungan Anda

Anda dapat mengatur alat merge eksternal Anda dalam pengaturan preferensi Editor Unity:

  1. Pada bilah menu Editor, pilih Edit > Preferences.
  2. Klik External Tools.
  3. Pada menu drop-down Revision Control Diff/Merge, pilih alat merge eksternal.

Alat gabungan berikut bekerja dengan baik dengan Collaborate:

  • Mozilla Firefox
  • WinMerge
  • Di luar Bandingkan 4
  • WordPress.org
  • TkDiff

Anda mungkin mengalami masalah menggunakan alat merge eksternal lainnya.

Bekerja dengan konflik

Setelah merging file, jika ada aset dengan konflik, Unity menampilkan ikon peringatan merah pada tombol kolaborasi di Editor toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
Lihat di Glossary. Untuk melihat daftar aset dengan konflik, klik tombol kolaborasi.

Ketika Anda menginjak aset dalam daftar, dialog konflik menunjukkan empat tindakan untuk setiap aset dengan konflik. Kamu bisa:

  • Lihat diff; gunakan alat penggabungan default untuk melihat perbedaan antara file
  • Gunakan alat penggabungan; buka alat penggabungan eksternal untuk membandingkan perbedaan
  • Gunakan perubahan saya; menerbitkan perubahan Anda, menuliskan versi aset yang tersimpan saat ini
  • Gunakan perubahan jarak jauh; hapus perubahan Anda dan gunakan versi aset dari cloud

Gambar di bawah ini menggambarkan ikon yang tersedia pada item yang bertentangan:

Untuk melihat konflik, perselisihan atas file yang bertentangan dan klik ikon mata.

Note: Anda tidak dapat melihat perubahan pada file gambar seperti .pngs.

Anda dapat menggunakan alat merge eksternal untuk menyelesaikan konflik pada file berbasis teks. Untuk menyelesaikan konflik menggunakan alat merge eksternal, klik ikon merge.

Untuk mengganti versi aset di cloud dengan versi aset Anda, klik ikon panah.

Untuk membuang versi aset Anda dan menerapkan versi di cloud ke proyek lokal Anda, klik ikon panah turun.
