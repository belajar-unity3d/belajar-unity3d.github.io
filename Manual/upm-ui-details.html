Labels

Beberapa paket dan label tampilan paket Asset Store di samping nama paket atau nomor versi. Label ini memberikan informasi tentang sumber atau state paket:

  • Jenis label Source menunjukkan di mana paket berasal dari (misalnya, apakah berasal dari folder lokal atau diunduh dari registry paket).
  • Jenis label State menunjukkan tahap paket siklus pengembangan. Misalnya, apakah itu kustom, paket eksperimental, atau di jalur rilis untuk versi Unity tertentu.
  • Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
    Lihat di Glossary     menunjukkan bahwa paket Asset Store berasal dari Asset Store dan juga dapat menunjukkan beberapa status khusus (misalnya, dideprecated).

Beberapa label sumber menyiratkan label negara dan sebaliknya (misalnya, jika paket tertanam dalam proyek Anda, maka Unity secara otomatis menganggapnya adalah paket kustom dalam pengembangan, jadi hanya label custom muncul di details view).

Jendela Paket Manager menampilkan nilai-nilai berikut:

Label Type Meaning
Released state Unity resmi merilis paket ini dan mendukung sepenuhnya. Tim Jaminan Kualitas telah menguji versi paket ini dan menjamin bahwa ia bekerja dengan versi tertentu dari Editor dan semua paket lain yang dirilis untuk versi Editor itu.
Release Candidate state Versi paket ini dilacak sepenuhnya "dilepaskan" dalam tahap TECH berikutnya dari siklus rilis saat ini.
Pre-Release state Versi paket ini adalah pada tahap awal pengembangan, tetapi jaminan Unity untuk melepaskannya pada akhir siklus rilis LTS. Mungkin tidak memiliki dokumentasi lengkap, atau mungkin tidak sepenuhnya divalidasi oleh tim pengembangan atau tim Jaminan Kualitas Unity.
Experimental state Paket ini adalah paket baru atau berisi modifikasi eksperimental. Unity tidak mendukung paket eksperimental karena mereka berada di tahap awal perkembangan.
Custom state
source		 Paket ini adalah embedded dalam proyek Anda. Sebagian besar pengembang paket kustom mulai dengan menanam paket baru di proyek mereka, itulah sebabnya label "Custom".
local source Paket Manager menginstal paket ini dari folder atau file tarball di disk lokal Anda eksternal ke folder proyek Unity Anda.
git source Manajer Paket dipasang paket ini di proyek Anda langsung dari repositori Git.
asset store asset Ini adalah paket Asset Store yang Anda beli atau diunduh dari Asset Store.
deprecated asset Paket ini tidak lagi tersedia di Aset Store kecuali Anda telah mengunduh atau membelinya sebelumnya. Itu tidak lagi dapat ditemukan oleh pelanggan baru.

Note: Label ini hanya berlaku untuk paket Asset Store. Paket unity yang telah diuraikan tidak pernah muncul di Editor Unity.
