Anda dapat mengatur format kompresi tekstur default untuk aplikasi WebGL Anda dari salah satu lokasi berikut:

  • Di Pengaturan WebGL Build dengan pengaturan Texture compression format.

    • Sebagai contoh tentang cara secara bersamaan membuat build untuk browser desktop dan browser seluler dengan format kompresi tekstur yang sesuai, lihat Buat build untuk browser desktop dan seluler dari skrip.

  • Di jendela Pengaturan WebGL > Other Settings > Rendering > pengaturan Texture compression format.

Precedence untuk pengaturan format Kompresi Tekstur

Sebelum Anda mengatur format kompresi tekstur, penting untuk menentukan mana pengaturan ini terjadi sebelumnya. Nilai format kompresi tekstur yang Anda set di Pengaturan Build memiliki prioritas atas nilai yang Anda set di Pengaturan Pemain. Secara default, nilai pengaturan Build diatur ke Use Player Settings.

Anda juga dapat menyesuaikan format kompresi tekstur untuk tekstur individu. Nilai yang Anda set untuk tekstur individual overrides nilai format kompresi tekstur default.
