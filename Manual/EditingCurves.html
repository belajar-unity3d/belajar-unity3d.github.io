Mengedit Kurva

Sementara kontrol ini memiliki perbedaan halus, curves dapat diedit dengan cara yang sama di semua mereka. Halaman ini menjelaskan cara menavigasi dan mengedit kurva di kendali tersebut.

Menambahkan dan memindahkan Kunci pada Kurva

Kunci dapat ditambahkan ke kurva dengan mengklik dua kali pada kurva di titik di mana key harus ditempatkan. Hal ini juga memungkinkan untuk menambahkan key dengan mengklik kanan pada kurva dan pilih Add Key dari menu konteks.

Setelah ditempatkan, keys dapat diseret di sekitar dengan mouse:

  • Klik pada key untuk memilihnya. Seret key yang dipilih dengan mouse.
  • Untuk mengambil key ke grid sambil menyeretnya di sekitar, tahanCommand pada Mac / Control pada Windows sambil menyeret.

Hal ini juga memungkinkan untuk memilih beberapa keys sekaligus:

  • Untuk memilih beberapa keys sekaligus, tahanShift saat mengklik tombol.
  • Untuk memilih key yang dipilih, klik di atasnya kembali sambil menahan Ctrl.
  • Untuk memilih semua keys di area persegi panjang, klik di tempat kosong dan tarik untuk membentuk pilihan persegi panjang.
  • Pemilihan persegi panjang juga dapat ditambahkan ke kunci yang dipilih yang ada dengan menahan Shift.

Keys dapat dihapus dengan memilih dan menekan Delete, atau dengan mengklik kanan pada mereka dan memilih Delete Key dari menu konteks.

Mengedit Kunci

Pengeditan langsung nilai kunci dalam editor kurva adalah fitur baru di Unity 5.1. Gunakan menu Enter/Return atau konteks untuk memulai mengedit tombol yang dipilih, Tab untuk beralih antara bidang, Enter/Return untuk melakukan, dan Escape untuk membatalkan pengeditan.

Menjelajahi Tampilan Kurva

Ketika bekerja dengan Animation View Anda dapat dengan mudah memperbesar rincian kurva yang ingin Anda kerjakan atau memperbesar untuk mendapatkan gambar penuh.

Anda selalu dapat menekan F untuk memilih kurva yang ditunjukkan atau kunci yang dipilih secara keseluruhan.

Zooming

Anda dapat zoomKontrol kamera yang memungkinkan Anda meningkatkan tampilan di layar Anda. Untuk memperbesar kamera di Editor Unity, tekan Alt + klik kanan dan drag. More info

Anda dapat memperbesar hanya sumbu horisontal atau vertikal:

  • zoom sambil menahan Command pada Mac / Control pada Windows untuk memperbesar secara horizontal.
  • zoom sambil menahan Shift untuk memperbesar secara vertikal.

Selanjutnya, Anda dapat menyeret topi akhir dari gulir untuk mengecilkan atau memperluas area yang ditampilkan dalam Tampilan Kurva.

Panning

Anda dapat pan Tampilan Kurva dengan dragging tengah dengan mouse Anda atau dengan menahan Alt sementara dragging kiri dengan mouse Anda.

Mengedit Tangents

Kunci memiliki dua tangents - satu di sebelah kiri untuk lereng masuk dan satu di sebelah kanan untuk lereng keluar. Tanggung mengontrol bentuk kurva antara kunci. Anda dapat memilih dari sejumlah jenis tangen yang berbeda untuk mengontrol bagaimana kurva Anda meninggalkan satu kunci dan tiba di tombol berikutnya. Klik kanan kunci untuk memilih jenis tangen untuk kunci itu.

Untuk nilai-nilai animasi untuk mengubah dengan lancar ketika melewati kunci, tangensi kiri dan kanan harus dilapisi. Jenis tangen berikut memastikan kelancaran:

  • Clamped Auto: Ini adalah mode nyata default. Tanggung secara otomatis diatur untuk membuat kurva melewati kunci. Ketika mengedit posisi kunci atau waktu, tangent menyesuaikan untuk mencegah kurva dari "menerbitkan" nilai target. Jika Anda secara manual menyesuaikan tangents dari kunci dalam mode Auto Clamped, itu beralih ke mode Free Smooth. Dalam contoh di bawah ini, tangent secara otomatis masuk ke lereng dan tingkat keluar sementara kunci sedang dipindahkan:
  • Auto: Ini adalah Modus, dan tetap menjadi pilihan untuk kompatibel kembali dengan proyek-proyek yang lebih tua. Kecuali Anda memiliki alasan tertentu untuk menggunakan mode ini, gunakan default Clamped Auto. Ketika kunci diatur ke mode ini, tangents secara otomatis diatur untuk membuat kurva melewati dengan lancar melalui kunci. Namun, ada dua perbedaan dibandingkan dengan mode Clamped Auto:
    1. Tanggung tidak menyesuaikan secara otomatis ketika posisi atau waktu utama diedit; mereka hanya menyesuaikan ketika awalnya mengatur kunci ke mode ini.
    2. Ketika Unity menghitung tangen, itu tidak memperhitungkan menghindari "meneruskan" dari nilai target dari kunci.
  • Free Smooth: Seret pegangan tangen untuk secara bebas mengatur tangents. Mereka terkunci untuk menjadi co-linear untuk memastikan kelancaran.
  • Flat: Kuis diatur menjadi horisontal (ini adalah kasus khusus Free Smooth).

Kadang-kadang Anda mungkin tidak ingin kurva untuk halus ketika melewati kunci. Untuk membuat perubahan tajam dalam kurva, pilih salah satu mode tangensi Broken.

Saat menggunakan tangents rusak, tangen kiri dan kanan dapat diatur secara individual. Setiap tangen kiri dan kanan dapat diatur ke salah satu jenis berikut:

  • Broken - Free: Seret pegangan tangen untuk secara bebas mengatur tangents.
  • Broken - Linear: Poin tangen terhadap kunci tetangga. Untuk membuat segmen kurva linier, set tangents di kedua ujungnya menjadi Linear. Dalam contoh di bawah ini, semua tiga kunci telah diatur ke Broken - Linear, untuk mencapai garis lurus dari kunci ke kunci.
  • Broken - Constant: Kurva mempertahankan nilai konstan antara dua kunci. Nilai kunci kiri menentukan nilai segmen kurva.
