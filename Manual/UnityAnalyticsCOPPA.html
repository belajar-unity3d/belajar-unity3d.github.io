Kepatuhan COPPA

COPPA berdiri untuk Undang-Undang Perlindungan Privasi Online Anak (dan peraturan terkait). COPPA termasuk kewajiban tertentu pada operator aplikasi “diarahkan ke anak-anak di bawah 13” (“Child Apps”). Jika Anda memiliki pertanyaan tentang apakah aplikasi Anda adalah Aplikasi Anak atau aspek lain dari COPPA, Anda harus merujuk pada materi yang tersedia oleh Komisi Perdagangan Federal AS ((portion of that are available here: COPPA FAQ) dan konsultasikan penasihat hukum Anda sendiri.

Lihat di Glossary memperlakukan data yang dikumpulkan dari aplikasi yang telah Anda desainkan sebagai Aplikasi Anak berbeda dari data yang dikumpulkan dari aplikasi lain. Jika aplikasi Anda adalah Aplikasi Anak, Anda perlu menetapkannya seperti di dalam panel layanan Editor, melalui proses pembuatan Proyek di atau melalui halaman ikhtisar Proyek di .

Untuk memberikan analitik untuk permainan Anda, Unity AnalyticsSebuah platform data yang menyediakan analitik untuk permainan Unity Anda. More info
Lihat di Glossary menghasilkan ID pengguna anonim untuk setiap pengguna dalam permainan Anda. Kami tidak menggunakan ID ini yang dihasilkan dari Aplikasi Anak untuk melacak pengguna di seluruh aplikasi yang dibangun oleh pengembang lain atau untuk memetakan pengguna antara layanan, perangkat, atau browser pada komputer yang sama. Selain ID ini, Unity Analytics juga mengumpulkan informasi pribadi berikut dari pengguna Aplikasi Anak: alamat IP, pengenal untuk pengiklan (IDFA hanya dikumpulkan jika Iklan Unity juga diaktifkan) dan pengenal perangkat (IDFV, ID perangkat Android atau IMEI jika ID perangkat Android tidak tersedia).

Penggunaan unity dari informasi pribadi yang diidentifikasi di atas terbatas untuk menyediakan analisis tingkat aplikasi untuk pengembang aplikasi dan menganalisis dan melaporkan informasi anonim dan tingkat agregat tentang perangkat, aplikasi dan industri game (misalnya, persentase perangkat menggunakan sistem operasi tertentu atau persentase perangkat tertentu oleh daerah). Data tingkat agregat ini tidak mengandung informasi pribadi. Juga, jika Anda telah mengaktifkan iklan Unity di Aplikasi Anak, Unity dapat menggunakan informasi tentang pengguna yang dikumpulkan oleh Unity Analytics dari Aplikasi Anak untuk melayani iklan kontekstual dalam Aplikasi Anak.
