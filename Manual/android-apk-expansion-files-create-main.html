Buat file ekspansi APK utama

Untuk membuat file ekspansi APKFormat paket Android output oleh Unity. Sebuah APK secara otomatis digunakan untuk perangkat Anda ketika Anda memilih File > Build & Run. More info
Lihat di Glossary utama untuk aplikasi Anda, menunjukkan kepada Unity untuk membagi aplikasi. Untuk informasi tentang cara melakukan ini, lihat Membagi aplikasi biner.

Sekarang ketika Anda Membangun aplikasi, Unity menghasilkan file ekspansi APK utama kemudian menyalinnya baik ke direktori output. Unity menggunakan nama aplikasi yang diikuti oleh .main untuk nama file ekspansi APK. Misalnya, jika aplikasi disebut my-app, APK akan menjadi my-app.apk dan file ekspansi APK akan menjadi my-app.main.obb.

Jika Anda memilih Build and Run, Unity menginstal file ekspansi APK dan APK di perangkat Anda. Jika Anda memilih Build dan ingin menginstal aplikasi secara manual menggunakan utilitas ADBAn Android Debug Bridge (ADB). You can use an ADB to deploy an Android package (APK) manually after building. More info
See in Glossary, Anda harus terlebih dahulu menginstal APK dan kemudian menyalin file ekspansi APK ke lokasi yang benar di perangkat Anda. Nama file ekspansi APK harus sesuai dengan format yang Google membutuhkan. Untuk informasi lebih lanjut, lihat file ekspansi.

Jika aplikasi dimulai dan tidak dapat menemukan dan memuat file ekspansi APK, hanya SceneA Scene contains the environments and menus of your game. Think of each unique Scene file as a unique level. In each Scene, you place your environments, obstacles, and decorations, essentially designing and building your game in pieces. More info
See in Glossary pertama tersedia. Dalam kasus ini, Anda harus mencoba mengunduh file ekspansi APK. Untuk informasi lebih lanjut, lihat Download process.
