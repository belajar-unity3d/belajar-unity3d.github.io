Data Explorer

Membangun laporan untuk metrik dan acara Unity AnalyticsUnity Analytics
Lihat di Glossary Anda menggunakan Data Explorer. Laporan Data Explorer menunjukkan bagaimana metrik yang dipilih dan perubahan acara kustom seiring waktu.

Buat laporan baru

Untuk membuat laporan Data Explorer:

  1. Navigasi ke dashboard Analytics untuk proyek Anda.
  2. Klik Data Explorer.
  3. Tambahkan Metrics, Custom Events, dan Ad Metrics ke laporan dengan mengklik tombol + di sebelah jenis data yang diinginkan. (Pilihan Metrik Iklan dinonaktifkan untuk proyek tidak menggunakan iklan Unity.)
  4. Mengatur rentang tanggal dan jenis grafik sesuai keinginan. Lihat Pilihan Laporan untuk informasi lebih lanjut.

Note: Laporan default mencakup metrik tunggal. Anda dapat mengubah jenis dan segmen metrik untuk item ini. (Untuk menghapus item ini, Anda harus terlebih dahulu menambahkan item kedua - Anda tidak dapat menghapus item data terakhir pada laporan.)

Untuk menyimpan laporan untuk melihat masa depan:

  1. Klik tombol New Report.

  2. Pada dialog yang terbuka, masukkan nama untuk laporan dan klik OK.

Untuk melihat laporan yang disimpan di masa depan, klik dropdown Saved Reports dan pilih nama laporan dalam daftar. Anda dapat menyesuaikan rentang tanggal untuk melihat laporan yang sama dengan data terbaru Anda.

Salin laporan

Untuk menyalin laporan Data Explorer yang ada:

  1. Navigasi ke dashboard Analytics Anda untuk proyek Anda.

  2. Klik Data Explorer.

  3. Klik dropdown Saved Reports dan pilih nama laporan dalam daftar.

  4. Sesuaikan rentang tanggal, jenis grafik atau opsi lain yang diinginkan. Lihat Pilihan Laporan untuk informasi lebih lanjut.

  5. Klik tombol New Report.

  6. Pada dialog yang terbuka, masukkan nama untuk salinan laporan dan klik Ok.

Download data laporan

Untuk mengunduh file nilai koma-separated (CSV) yang berisi data dalam laporan:

  1. Navigasi ke dashboard Analytics Anda untuk proyek Anda.
  2. Klik Data Explorer.
  3. Mengisi laporan yang disimpan atau membuat yang baru.
  4. Sesuaikan rentang tanggal atau opsi lain yang diinginkan. Lihat Pilihan Laporan untuk informasi lebih lanjut. Klik tombol CSV untuk mengunduh data untuk laporan yang ditampilkan saat ini.

Sitemap CSV download ekspor tepat apa yang Anda lihat di grafik / stabil. Jika Anda memiliki berlangganan Unity Pro, Anda juga dapat mengunduh semua data Analytics Anda dalam bentuk baku dari halaman Note: atau melalui API REST Ekspor Data Baku.Ekspor Data Baku page or through the Raw Data Export REST API.

  • Custom Event - Acara khusus adalah peristiwa yang Anda kirim dari permainan Anda untuk mengukur perilaku atau faktor lain yang unik untuk permainan Anda. Anda dapat menambahkan hingga dua puluh item data ke satu laporan. Grafik akan menampilkan unit yang tepat untuk skala vertikal selama tidak ada lebih dari dua item data yang menggunakan unit ukuran yang berbeda.

Anda dapat mengatur rentang tanggal untuk laporan dan memilih jenis grafik yang berbeda.

Metrik dan opsi Metrik Iklan

Ketika Anda menambahkan Metric atau Ad Metric ke grafik, Anda harus mengatur nama dan dapat memilih data SegmentSegmen adalah subset dari basis pemain Anda, terpisahkan oleh perbedaan utama. Melihat metrik dan peristiwa oleh segmen dapat mengungkapkan perbedaan perilaku dalam game antara kelompok yang berbeda. More info
Lihat di Glossary:

  • Sitemap Nama metrik
  • Login Segmen data untuk menunjukkan
  • C – Menghapus metrik dari laporan

Untuk setiap item data, Anda dapat menentukan Segment data untuk ditampilkan. A Segment adalah subset data untuk item itu, dipilih sesuai dengan satu set aturan. Misalnya, segmen Android Users mencakup data hanya untuk pemain di platform Android, sedangkan segmen iOS Users mencakup data hanya untuk pemain di platform iOS.

Layanan Analytics mencakup satu set segmen standar. Anda juga dapat membuat Segments kustom Anda sendiri. Gunakan Pembuat Segmen untuk melihat definisi segmen standar dan untuk membangun Segments kustom Anda sendiri. Perhatikan bahwa ketika Anda membuat Segment kustom, itu hanya berlaku untuk metrik dan acara baru dikumpulkan. Data lebih tua tidak diproses untuk dimasukkan dalam Segment baru.

Pilihan acara khusus

Ketika Anda menambahkan Custom Event ke laporan, Anda dapat mengatur opsi tambahan berdasarkan parameter acara.

  • A – Tampilkan grafik pie (hanya untuk item data yang dapat dikategorikan)
  • Login Nama Custom Event
  • Login Segment data untuk menunjukkan
  • Login Acara kustom nama Parameter dan opsi tampilan
  • E - Bagaimana untuk mengumpulkan nilai parameter
  • F – Hapus acara kustom dari laporan

Ketika Custom Event memiliki parameter, Anda dapat memilih bagaimana data untuk acara itu ditampilkan pada grafik. Anda dapat menunjukkan jumlah nilai yang berbeda yang diterima atau, untuk sifat numerik, jumlah atau rata-rata nilai dari semua peristiwa yang diterima:

Simbol yang ditampilkan setelah nama Parameter menunjukkan bagaimana Parameter ditampilkan. Simbol string (A) dan boolean ( Diagram Venn) menunjukkan bahwa nilai Parameter dapat dikategorikan menjadi kelompok yang berbeda. Simbol numerik (#) menunjukkan bahwa nilai Parameter adalah nomor yang dapat rata-rata atau dirangkum selain dihitung. Perhatikan bahwa numerik Parameters muncul dalam daftar dua kali, sekali dengan simbol string dan sekali dengan simbol numerik, memberi Anda pilihan mengobati Parameter numerik sebagai nilai yang dapat dikategorikan atau agregatif.

Ketika menunjukkan nilai Parameter yang dapat dikategorikan, hanya sepuluh nilai yang paling umum rusak menjadi kelompok individu. Sisa nilai-nilai yang dilumasi bersama-sama dalam kelompok Other. Dengan demikian, memperlakukan numerik Parameters sebagai string yang dapat dikategorikan umumnya hanya berguna untuk Parameters dengan set terbatas nilai yang berbeda.

Anda dapat melihat set nilai yang berbeda di bagian bawah grafik. Anda juga dapat melihat grafik pie dari nilai dengan mengklik tombol pie-chart di depan item data. Hanya grafik pai tunggal dapat ditampilkan pada waktu.

Sitemap Untuk mengunduh data dalam grafik pie data yang dapat dikategorikan, klik tombol pie-chart di sebelah Custom Event sebelum mengklik tombol Note:.CSV button.

Jenis grafik

Anda dapat memilih jenis grafik berikut untuk laporan:

  • Spline - Grafik garis halus
  • Column - Grafik bar
  • Stacked - Grafik area stacked
  • Table - tabel nilai
  • Pie - Grafik pai (hanya tersedia untuk individu, dikategorikan parameter Acara Kustom)

Grafik

Anda dapat memperbesar grafik Spline, Column dan Stacked dengan mengklik grafik dan menyeret di bagian yang ingin Anda lihat secara lebih detail.

Ketika Anda diperbesar, grafik menampilkan tombol Reset Zoom, yang mengembalikan grafik untuk menampilkan seluruh rentang tanggal.

Annotations

Anda dapat menggunakan anotasi untuk menandai tanggal faktor eksternal penting yang mungkin mempengaruhi data analitik permainan Anda, seperti pembaruan permainan, kampanye pemasaran, dan promosi. Tambahkan anotasi berbasis tanggal ke laporan Data Explorer menggunakan bagian Annotasi di bawah grafik laporan.

Annotations adalah proyek-wide, tidak spesifik untuk laporan individu. Tampilan annotasi pada semua grafik laporan setiap bagian Annotasi terbuka.

To add an annotation:

  1. Pada laporan Data Explorer, klik tombol + Annotations untuk membuka bagian Annotations (jika belum terbuka).
  2. Klik di kotak tanggal dan atur tanggal untuk anotasi. Anda hanya dapat menambahkan satu catatan anotasi untuk hari tertentu.
  3. Masukkan nama atau teks deskriptif untuk anotasi.
  4. Klik Save.
