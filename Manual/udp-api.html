Referensi UDP
UDP SDK

API

UDP API memungkinkan Anda mengunggah file biner ke konsol UDP. Ini memungkinkan Anda menggunakan sistem CI untuk membangun aplikasi Anda dan mendorong output langsung ke UDP. SitemapMengelola dan menerbitkan permainan Anda di konsol UDP.

Jenis file berikut didukung:

  • APK
  • OBB
  • AAB

API

API terstruktur sebagai berikut: https://distribute.dashboard.unity.com/developer/api/upload/:orgId/:clientId?token=xxx[&obbType=xxxx]

Tabel di bawah ini menjelaskan parameter API.

Parameter Description
orgId ID organisasi dari permainan yang ingin Anda unggah file biner.
clientId ID klien dari permainan yang ingin Anda unggah file biner.
token Token otorisasi yang dihasilkan di konsol UDP, di halaman Developer API.
obbType Jenis file OBB yang ingin Anda unggah. Nilai berikut berlaku:
- mainObb
- patchObb
Anda hanya perlu menentukan parameter query ini ketika Anda mengunggah file OBB.

Kunci bentuk

Anda dapat menggunakan kunci formulir dalam metode POST untuk menentukan nilai ketika mengunggah build Anda ke UDP. Nilai sesuai dengan bagian Biner dari tab Informasi Game.

Tabel di bawah ini menjelaskan kunci bentuk.

Katalog Description Mandatory / Opsional
uploadFile Tentukan path ke file untuk mengunggah. Mandatory
whatsNew Mengatur nilai lapangan What’s New di konsol UDP. Optional
useGoogleService Mengatur nilai lapangan Does your game use Google Play Services? di konsol UDP. Optional

Jika Anda tidak menentukan kunci opsional, UDP menggunakan nilai-nilai dari versi terbaru dari permainan Anda.

Pesan kesalahan umum dan kode kesalahan

Token otentikasi tidak valid

Jika Anda memberikan token otentikasi yang tidak valid, Anda akan mendapatkan pesan kesalahan dengan kode status 401.

{"errorCode":"NotAuthenticated","message":"not authenticated error","target":"","details":null}

Invalid organisasi ID atau ID klien

Jika Anda memberikan ID organisasi yang tidak valid atau ID klien yang tidak valid, Anda akan mendapatkan pesan kesalahan dengan kode status 400.

{"errorCode":"InvalidParameter","message":"The given parameter is missing or invalid","target":"Invalid OrgId or ClientId","details":null}

Jenis file tidak valid

Jika Anda menyediakan jenis file yang tidak valid, Anda akan mendapatkan pesan kesalahan dengan kode status 400.

{"errorCode":"InvalidParameter","message":"The given parameter is missing or invalid: please provide the right binary file(APK, AAB or OBB)","target":"file type","details":null}

Jenis OBB yang tidak valid

Jika Anda memberikan jenis OBB yang tidak valid, Anda akan mendapatkan pesan kesalahan dengan kode status 400.

{"errorCode":"InvalidParameter","message":"The given parameter is missing or invalid","target":"obbType","details":null}

Jenis konten tidak valid

Jika Anda menyediakan jenis konten yang tidak valid, Anda akan mendapatkan pesan kesalahan dengan kode status 400.

{"errorCode":"InvalidParameter","message":"The given parameter is missing or invalid","target":"content type","details":null}
