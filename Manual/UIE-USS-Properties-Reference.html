Referensi properti ASS

Daftar tabel berikut properti AS dalam UI(User Interface) Memungkinkan pengguna untuk berinteraksi dengan aplikasi Anda. Unity saat ini mendukung tiga sistem UI. More info
Lihat di Glossary Toolkit.

Tabel menunjukkan sifat mana yang diwariskan dan animatability mereka. Untuk properti yang mewarisi, elemen mendapat nilai dari unsur induknya jika Anda tidak menentukan nilainya. Contoh berikut menggunakan properti yang mewarisi untuk mengatur font untuk semua elemen.

:root {
    -unity-font: resource("Font/consola.ttf");
}

Klik nama properti untuk melihat informasi rinci tentang properti itu.

Note: Penggunaan properti AS yang sama dengan tautan properti CSS ke dokumentasi pengembang Mozilla.
