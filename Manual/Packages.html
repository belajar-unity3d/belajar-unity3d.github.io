Manajer Paket Unity

“package” adalah wadah yang menyimpan berbagai jenis fitur atau aset, seperti:

  • Alat editor dan perpustakaan, seperti editor teks, pemirsa animasi atau kerangka uji.
  • Alat dan perpustakaan Runtime seperti API Fisika atau pipa Grafik.
  • Koleksi aset, seperti Tekstur atau animasi.
  • Templat proyek untuk berbagi jenis proyek umum dengan orang lain.

Paket memberikan berbagai macam peningkatan ke Unity melalui Manajer Paket. Untuk membantu menemukan dan menggunakan paket-paket ini, jendela Manajer Paket menyediakan koleksi paket yang dapat Anda gunakan bersama-sama, yang disebut "set kucing". Di Editor, Anda dapat mengakses Paket jendela Manager melalui menu ini: Window > Package Manager.

Note: Manajer Paket juga mendukung melihat setiap Paket Asset Store yang telah Anda unduh atau diimpor dari Unity Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info

Cara Kerja Unity dengan paket

Ketika Unity membuka Proyek, Unity Package Manager membaca LookbookSetiap proyek Unity memiliki manifestasi proyek, yang bertindak sebagai titik masuk untuk Manajer Paket. File ini harus tersedia di direktori <project>/Packages. Manajer Paket menggunakannya untuk mengkonfigurasi banyak hal, termasuk daftar dependensi untuk proyek itu, serta repositori paket untuk permintaan. More info
Lihat di Glossary (1) untuk mengetahui paket apa yang dimuat di Proyek. Kemudian mengirim permintaan (2) ke paket server registry (3) untuk setiap paket yang muncul sebagai ketergantungan dalam manifestasi. Paket registry kemudian mengirimkan informasi dan data yang diminta kembali ke Manajer Paket (4), yang kemudian menginstal paket tersebut (5) di Proyek. Setiap Proyek memiliki manifestasi sendiri yang mencantumkan paket untuk memuat sebagai “tergantung” Proyek.

Jika Anda ingin menyertakan paket di Anda Proyek, Anda harus memperbarui Lookbook untuk memasukkannya dalam daftar dependensi. Jika Anda ingin, Anda dapat memodifikasi manifestasi Proyek secara langsung, tetapi lebih aman dan lebih mudah untuk membiarkan Manajer Paket melakukannya. Untuk informasi lebih lanjut tentang menggunakan antarmuka pengguna, lihat dokumentasi untuk Paket jendela Manager.
