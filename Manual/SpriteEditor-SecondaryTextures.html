Editor Sprite: Tekstur sekunder

Modul SpriteObjek grafis 2D. Jika Anda digunakan untuk bekerja di 3D, Sprites pada dasarnya hanya tekstur standar tetapi ada teknik khusus untuk menggabungkan dan mengelola tekstur sprite untuk efisiensi dan kenyamanan selama perkembangan. More info
Lihat di Glossary Secondary Textures Editor dapat mengasosiasikan Tekstur tambahan dengan Sprites yang diedit. Sebuah ShaderProgram yang berjalan di GPU. More info
Lihat di Glossary kemudian dapat mencicipi Tekstur sekunder ini untuk menerapkan efek tambahan pada Sprite, seperti pemetaan normal. Tekstur sekunder saat ini hanya didukung oleh Sprite RenderersKomponen yang memungkinkan Anda menampilkan gambar sebagai Sprites untuk digunakan di kedua adegan 2D dan 3D. More info
Lihat di Glossary dan tidak tersedia untuk komponen Renderer lainnya.

Untuk menggunakan fungsi ini, pilih modul Secondary Textures dari menu drop-down di sebelah kiri atas jendela Sprite Editor.

Menambahkan Tekstur Sekunder

Untuk menambahkan entri Tekstur Sekunder baru, pilih '' +' di kanan bawah panel Secondary Textures. Anda dapat menambahkan maksimum delapan Tekstur Sekunder ke setiap Sprite atau Sprite Sheet. Setiap Tekstur sekunder ditambahkan muncul sebagai entri pada daftar, dengan dua bidang: Nama dan Tekstur.

Anda dapat memasukkan nama kustom untuk Tekstur Sekunder di bidang Nama. Beberapa paket Unity menyarankan nama Tekstur yang dapat digunakan dengan Shaders mereka. panah drop-down ke kanan bidang Nama menampilkan daftar nama yang disarankan. Nama yang disarankan tetap dalam menu drop-down bahkan setelah paket terkait mereka dihapus.

Untuk memilih Tekstur yang digunakan oleh Tekstur Sekunder, tarik Aset Tekstur langsung ke bidang Tekstur, atau buka jendela Object Picker dengan memilih lingkaran ke kanan lapangan.

Tekstur sekunder dicontohkan dengan koordinasi UV yang sama sebagai Tekstur Sprite utama. Align Tekstur Sekunder dengan Tekstur Sprite utama untuk memastikan bahwa efek Tekstur tambahan ditampilkan dengan benar.

Untuk melihat pratinjau Tekstur Sekunder di jendela Sprite Editor dengan Tekstur Sprite utama tersembunyi, pilih entri dalam daftar. Klik di luar daftar Tekstur Sekunder untuk memilih entri, dan Tekstur Sprite utama terlihat lagi.

Menghapus Tekstur Sekunder

Untuk menghapus Tekstur Sekunder dari daftar, pilih dan kemudian pilih tombol '-' di kanan bawah jendela.

Menerapkan Perubahan

Untuk menerapkan perubahan pada Sprites yang diedit, pilih toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
Lihat di Glossary. Entries tanpa Nama atau Tekstur yang dipilih dianggap tidak valid dan secara otomatis dihapus ketika perubahan diterapkan.
