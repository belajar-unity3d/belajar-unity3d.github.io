Opsi lanjutan

Ketika Anda telah mengkonfigurasi proyek Anda di Unity Cloud BuildLayanan integrasi berkelanjutan untuk proyek Unity yang mengotomatiskan proses pembuatan build pada server Unity. More info
Lihat di Glossary, Anda dapat mengatur opsi build canggih untuk setiap target membangun Anda.

Pilihan ini mengakomodasi opsi pemrosesan build yang lebih kompleks yang didukung dalam Editor Unity.

Untuk mengakses opsi canggih target Anda, buka . Pilih proyek Anda, masukkan bagian Unity Cloud Build untuk proyek, lalu pilih tab Config, ditampilkan di bawah ini.

Klik Advanced Options.

Ketika Anda mengklik drop-down Advanced Options, opsi canggih build target ditampilkan.

Klik Edit Advanced Options untuk membawa layar untuk mengkonfigurasi opsi ini.

Opsi lanjutan per membangun target

Semua opsi canggih diatur per target build. Ini berarti bahwa, misalnya, ketika Anda mengklik tautan Advanced Options untuk target iOS, pilihan tersebut hanya untuk target iOS itu. Ketika Anda mengklik Advanced Options untuk target Android, pilihan tersebut hanya untuk target Android itu. Ini memungkinkan Anda menggunakan pre- dan post-methods yang berbeda per platform, per membangun target.
