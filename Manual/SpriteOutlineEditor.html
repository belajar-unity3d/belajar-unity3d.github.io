Lihat di Glossary pada. Editor Outline Kustom memungkinkan Anda untuk menggunakan titik kontrol untuk membuat dan mendefinisikan bentuk garis besar Sprite Mesh.

Secara default, Unity renders setiap Sprite pada persegi panjang Mesh. Mesh ini mungkin termasuk area transparan di luar perbatasan Tekstur, dan render daerah transparan ini dapat mempengaruhi kinerja secara negatif. Ketika Anda menggunakan editor Outline Kustom untuk menentukan garis besar Mesh yang cocok garis besar Tekstur Sprite, Anda mengurangi ukuran area transparan, yang meningkatkan kinerja.

Menggunakan editor Outline Kustom

Editor Outline Kustom memungkinkan Anda untuk membuat atau menghasilkan Mesh Sprite. Ada dua cara untuk membuat garis besar kustom: memiliki Unity secara otomatis generate bentuk, atau membuat dan mengedit secara manual di jendela editor.

Menghasilkan outline

Untuk memiliki Unity secara otomatis menghasilkan garis besar yang mengikuti bentuk tekstur Sprite asli, dan itu juga memperhitungkan area transparan dalam tekstur, klik Generate button. Namun, Anda dapat menyesuaikan bagaimana garis besar yang dihasilkan mengikuti tekstur Sprite dengan menyesuaikan Outline Tolerance slider.

Sesuaikan slider Outline Tolerance untuk memperbaiki garis besar Mesh yang menghasilkan Unity. Meningkatkan nilai meningkatkan seberapa era garis besar mengikuti bentuk tekstur Sprite. Leaving slider di 0 menghasilkan Mesh yang lebih longgar mengikuti tekstur Sprite.

Mengedit garis-garis kustom

Anda dapat membuat bentuk kustom Anda sendiri dan mengeditnya dalam dua cara. Cara pertama adalah dengan secara otomatis generating garis besar, dan kemudian memindahkan titik kontrol garis yang dihasilkan untuk memperbaiki bentuknya lebih lanjut. Atau klik dan tarik di setiap ruang di jendela editor Outline Kustom untuk membuat garis besar persegi panjang dengan empat titik kontrol dan mulai penyulingan garis dengan memindahkan titik kontrol atau menambahkan dan menghapusnya.

Klik dan seret atas tekstur Sprite di jendela editor untuk membuat garis besar dengan empat titik kontrol. Ulangi langkah ini untuk membuat beberapa garis keluar Mesh dalam Sprite. Unity hanya membuat daerah tertutup dalam garis besar Mesh.

Moving control points

Untuk menyesuaikan bentuk garis keluar Mesh, klik dan seret titik kontrolnya. Ketika Anda menggigit titik kontrol, ternyata biru untuk menunjukkan bahwa Anda dapat memilihnya. Klik dan seret titik kontrol ke posisi yang berbeda untuk menyesuaikan bentuk garis besar Mesh. Anda dapat memilih beberapa titik kontrol sekaligus dengan mengklik dan menyeret kursor di daerah untuk memilih semua titik kontrol di daerah itu. Anda kemudian dapat kembali posisi atau remove semua titik kontrol yang dipilih sama.

Menambahkan / Menghapus titik kontrol

Untuk menambahkan titik kontrol, letakkan kursor di tepi garis besar. Pratinjau titik kontrol muncul di sepanjang garis besar. Klik untuk menambahkan titik kontrol baru di lokasi tersebut. Untuk menghapus titik kontrol, pilih dan tekan Del/Command+Del keys.

Tepi bergerak

Untuk memindahkan bagian antara dua titik kontrol, tahan tombol Control/Ctrl untuk memilih tepi garis besar daripada titik kontrolnya. Untuk mengubah bentuk garis besar, klik dan seret tepi yang disorot untuk memindahkannya ke posisi baru.
