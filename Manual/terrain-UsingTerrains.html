Membuat dan mengedit Terrains

Untuk menambahkan Terrain GameObject ke Adegan Anda, pilih GameObject > 3D Object > Terrain dari menu. Ini juga menambahkan Aset Terrain yang sesuai dengan tampilan Proyek. Ketika Anda melakukan ini, lansekap awalnya pesawat besar dan datar. Jendela Inspektur Terrain menyediakan sejumlah alat untuk membuat fitur lansekap rinci.

Alat pengeditan Terrain di Inspector

toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
Lihat di Glossary menyediakan lima pilihan untuk menyesuaikan Terrain Anda:

  • Buat ubin Terrain yang berdekatan.
  • Sculpt dan cat Terrain Anda.
  • Tambahkan pohon.
  • Tambahkan rincian seperti rumput, bunga, dan batu.
  • Mengubah pengaturan umum untuk Terrain yang dipilih.

Untuk informasi lebih lanjut tentang setiap ikon ini, lihat Buat Terrains Tetangga, Alat Terrain, TreesKomponen GameObject dan terkait yang memungkinkan Anda untuk menambahkan aset pohon ke Adegan Anda. Anda dapat menambahkan tingkat cabang dan daun ke pohon di jendela Inspektur Pohon. More info
Lihat di Glossary, Rumput dan rincian lainnya, dan Pengaturan Terrain.

Pilih ikon kuas untuk mengakses alat lukisan, yang memungkinkan Anda untuk memodifikasi Terrain. Gunakan kursor untuk merangkul ketinggian Terrain, atau tekstur cat ke Terrain. Pilih dari beberapa bentuk Brush built-in, atau menentukan Brush Anda sendiri menggunakan tekstur. Anda juga dapat mengubah ukuran dan opacity (kekuatan efek yang diterapkan) dari Sikat. Setelah Anda mendefinisikan properti, kursor Anda mengambil bentuk Sikat yang dipilih. Klik atau seret di Terrain untuk membuat bentuk dan tekstur yang berbeda.

Mirip dengan bagaimana Anda melukis dengan Sikat di Terrain, Anda dapat menambahkan tekstur, pohon, dan detail seperti rumput, bunga, dan batu. Anda juga dapat membuat ubin Terrain yang terhubung tambahan, mengubah ketinggian seluruh ubin, dan bahkan menulis sikat kustom dengan efek kompleks.

Terrain keyboard shortcuts

Terrain InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info
Lihat di Glossary memiliki pintas keyboard berikut diaktifkan secara default.

Keys Function
Comma (, ) dan Periode ( . ) Siklus melalui Sikat yang tersedia
Pergeseran-Koma ( < ) dan Pergeseran-Period ( > ) Siklus melalui objek yang tersedia untuk pohon, tekstur, dan rincian
Buka Bracket ( [ ) dan Tutup Bracket ( ] ) Mengurangi dan meningkatkan Ukuran Sikat
Minus ( - ) dan Equal ( = ) Mengurangi dan meningkatkan Opacity Sikat

Untuk mengatur shortcut kustom Anda sendiri, gunakan Shortcuts Manager.

  • Pada Windows dan Linux, pilih Edit > Shortcuts.
  • Di macOS, pilih Unity > Shortcuts.

Di bawah CategoryKategori Profiler mengidentifikasi data beban kerja untuk subsistem Unity (misalnya, rendering, Scripting dan kategori Animasi). Unity menerapkan pengkodean warna untuk kategori untuk membantu secara visual membedakan jenis data di jendela Profiler. More info
Lihat di Glossary, pilih Terrain untuk menampilkan tugas pintas terkait Terrain. Untuk informasi lebih lanjut tentang pengaturan dan memodifikasi shortcuts, lihat halaman Shortcuts Manager.

Selain itu, keystroke F standar bekerja sedikit berbeda untuk Terrain. Seperti Terrains biasanya sangat besar, ketika Anda menekan tombol F, pandangan SceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary berfokus pada daerah di mana Anda memposisikan kursor. Ini memberikan cara yang sangat cepat dan intuitif untuk melompat ke daerah Terrain yang ingin Anda edit. Jika Anda menekan F sementara kursor tidak di atas ubin Terrain, itu menggunakan perilaku framing standar, mempersembahkan pemilihan di seluruh GameObjectObjek mendasar dalam adegan Unity, yang dapat mewakili karakter, props, pemandangan, kamera, waypoints, dan banyak lagi. Fungsi GameObject didefinisikan oleh Komponen yang melekat padanya. More info
