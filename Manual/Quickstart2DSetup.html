Pengaturan awal untuk game 2D

Note: Untuk panduan ini, Unity merekomendasikan dan mengasumsikan bahwa Anda memilih Pipa Render Universal (URP) dan bukan Pipa Render Built-in.

  1. Instal versi Unity 2019 LTS atau versi kemudian; lihat Instalasi Unity.

  2. Buat proyek baru dengan .

  3. Dalam Paket Manager, instal versi terbaru; lihat.

  4. Mengatur 2D Renderer; lihat langkah-langkah.

  5. Instal paket opsional yang Anda butuhkan; lihat Menginstal paket.

Menginstal paket

Kebanyakan paket yang Anda butuhkan untuk membuat game 2D di Unity disertakan dalam Editor Unity. Daftar tabel berikut paket yang disertakan secara default ketika Anda memilih:

Package Description
Animasi 2D Animasi 2D menyediakan komponen perkakas dan runtime yang diperlukan untuk menerapkan animasi kerangka ke SpritesObjek grafis 2D. Jika Anda digunakan untuk bekerja di 3D, Sprites pada dasarnya hanya tekstur standar tetapi ada teknik khusus untuk menggabungkan dan mengelola tekstur sprite untuk efisiensi dan kenyamanan selama perkembangan. More info
Lihat di Glossary Anda.
Paket 2D PixelUnit terkecil dalam gambar komputer. Ukuran piksel tergantung pada resolusi layar Anda. Pencahayaan pixel dihitung pada setiap piksel layar. More info
Lihat di Glossary Perfect berisi komponen Pixel Perfect CameraKomponen yang menciptakan gambar sudut pandang tertentu di tempat kejadian Anda. Output ditarik ke layar atau ditangkap sebagai tekstur. More info
Lihat di Glossary, yang memastikan seni pixel Anda tetap crisp dan jelas pada resolusi yang berbeda, dan stabil dalam gerakan.
2D PSD Paket impor memungkinkan Anda untuk mengimpor file PSD multilayer dari Photoshop. Anda dapat menggunakan ini untuk Sprites Anda, atau untuk mengatur karakter Anda.
Editor Sprite menyediakan lingkungan dalam-Editor untuk membuat dan mengedit aset Sprite. Editor Sprite memungkinkan Anda menambahkan perilaku khusus untuk mengedit data terkait Sprite.
2D Login Bentuk memungkinkan Anda untuk membuat bentuk organik dan terrainsPemandangan di tempat kejadian Anda. Terrain GameObject menambahkan pesawat datar besar untuk adegan Anda dan Anda dapat menggunakan jendela Inspektur Terrain untuk menciptakan lanskap rinci. More info
Lihat di Glossary, mirip dengan alat gambar vektor. Misalnya, Anda dapat memilih tekstur isi dan Sprites perbatasan.
2D Foto Editor Editor 2D TilemapGameObject yang memungkinkan Anda untuk dengan cepat membuat tingkat 2D menggunakan ubin dan overlay grid. More info
Lihat di Glossary memungkinkan Anda untuk membuat dunia berbasis grid dengan ubin persegi, heksagonal atau isometrik. Tambahkan ubin Anda ke palet ubin, dan cat dan mengisi grid ubin menggunakan pengaturan dan sikat yang berbeda. Alat tambahan memungkinkan Anda menambahkan gambar cerdas, acakisasi atau animasi ke aset ubin.
