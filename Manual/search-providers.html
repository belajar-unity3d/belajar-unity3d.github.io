Penyedia Pencarian

Setiap jenis pencarian memiliki Penyedia Pencarian sendiri. Penyedia Pencarian memungkinkan Anda untuk mencari dan menyaring konten. Setiap Penyedia Pencarian memiliki search token yang unik. Token pencarian adalah string teks yang dapat Anda gunakan di bidang pencarian untuk mencari hanya menggunakan Penyedia Pencarian tertentu.

Secara default, semua pencarian menggunakan Proyek, Hierarchy, dan Pengaturan Penyedia Pencarian kecuali Anda mengecualikan mereka.

Cari memberikan tambahan opt-in Search Providers. Cari mengabaikan penyedia pencarian kecuali Anda secara eksplisit menggunakannya. Penyedia Pencarian Opt-in berbeda dari Penyedia Pencarian default dengan cara berikut:

Anda hanya dapat mengeksekusi mereka menggunakan token pencarian mereka. Anda tidak dapat menggabungkan Penyedia Pencarian opt-in dengan Penyedia Pencarian lainnya.

  • Untuk melakukan pencarian menggunakan semua Penyedia Pencarian default, masukkan istilah pencarian di bidang pencarian. Hasil tampak seperti yang Anda ketik.

  • Untuk melakukan pencarian menggunakan Penyedia Pencarian opt-in, perbaiki istilah pencarian dengan token pencarian Penyedia Pencarian.

    Buka menu More (⋮) untuk melihat daftar Penyedia Pencarian dan perbaikan mereka.

Sertakan Penyedia Pencarian

Anda dapat mengecualikan Penyedia Pencarian spesifik dari pencarian biasa. Untuk mengecualikan penyedia, lakukan salah satu berikut:

  • Mute penyedia dari pane Filter. Ketika Anda menukar penyedia, Cari tidak menggunakannya dalam pencarian, tetapi aktivitas latar belakangnya (seperti bagian pengindeksan Unity) terus berjalan.
  • menonaktifkan penyedia dari preferensi. Ketika Anda menonaktifkan penyedia, Cari tidak menggunakannya sama sekali, dan menghentikan semua kegiatan latar belakangnya.
