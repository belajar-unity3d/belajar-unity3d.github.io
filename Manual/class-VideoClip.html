Video Klip

Foto Klip adalah file video yang diimpor, yang digunakan Web Browser untuk memutar konten video (dan menyertai konten audio, jika video juga memiliki suara). Ekstensi file yang umum untuk file video termasuk .mp4, .mov, .webm, dan .wmv.

Ketika Anda memilih Klip Video, InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info
Lihat di Glossary menunjukkan Video Clip Importer, termasuk opsi, pratinjau, dan detail sumber. Klik tombol Play di kanan atas pratinjau untuk bermain Klip Video, bersama dengan trek audio pertama.

Untuk melihat informasi sumber dari Klip Video, arahkan pane pratinjau di bagian bawah jendela Inspektur, klik nama Klip Video di sudut kiri atas, dan pilih Source Info.

Opsi spesifik platform menyesuaikan proses transcode untuk setiap platform target, memungkinkan pemilihan opsi default untuk setiap platform.

Jika transcode dinonaktifkan, file video digunakan sebagai, berarti kompatibilitas dengan platform target harus diverifikasi secara manual (lihat dokumentasi pada kompatibilitas file video untuk informasi lebih lanjut). Namun, memilih untuk tidak transcode dapat menghemat waktu serta menghindari kehilangan kualitas yang berkaitan dengan transformasi.

Video Klip Impor Properti

Property Function
Keep Alpha Menyimpan saluran alfa dan mengkodekannya selama transcode sehingga dapat digunakan bahkan jika platform target tidak asli mendukung video dengan alfa.

Properti ini hanya terlihat untuk sumber yang memiliki saluran alfa.
Deinterlace Mengontrol bagaimana sumber interlaced didegang selama transcode. Misalnya, Anda dapat memilih untuk mengubah pengaturan interlacing karena bagaimana video dikodekan atau untuk alasan estetika. Video Interlaced memiliki dua sampel waktu dalam setiap bingkai: satu garis yang aneh, dan satu di garis yang bahkan disebutkan.
Off Sumber ini tidak dipernis, dan tidak ada pemrosesan untuk dilakukan (ini adalah pengaturan default).
Even Mengambil garis-garis bahkan dari setiap bingkai dan mencemari mereka untuk membuat konten yang hilang. Garis Odd dijatuhkan.
Odd Mengambil garis-garis aneh dari setiap bingkai dan mencemari mereka untuk membuat konten yang hilang. Bahkan garis dijatuhkan.
Flip Horizontally Jika kotak centang ini diperiksa, konten sumber dibalikkan sepanjang sumbu horisontal selama transcode untuk membuatnya terbalik.
Flip Vertically Jika kotak centang ini diperiksa, konten sumber dibalikkan sepanjang sumbu vertikal selama transcode untuk menukar kiri dan kanan.
Import Audio Jika kotak centang ini diperiksa, trek audio diimpor selama transcode.

Properti ini hanya terlihat untuk sumber yang memiliki trek audio.
Transcode Ketika diaktifkan dengan mencentang kotak centang, sumber ditranscodekan ke dalam format yang kompatibel dengan platform target.

Jika dinonaktifkan, konten asli digunakan, melewati proses transcoding yang berpotensi panjang.

Note: Verifikasi bahwa format sumber kompatibel dengan setiap platform target (lihat dokumentasi pada kompatibilitas file video untuk informasi lebih lanjut).
Dimensions Mengontrol bagaimana konten sumber diubah.
Original Menyimpan ukuran aslinya.
Three Quarter Res Mengembalikan sumber hingga tiga kuartal lebar dan tinggi aslinya.
Half Res Mengembalikan sumber menjadi setengah dari lebar dan tinggi aslinya.
Quarter Res Memungkinkan sumber untuk seperempat lebar dan tinggi aslinya.
Square (1024 X 1024) Memperoleh sumber ke gambar persegi 1024 x 1024. Rasio memeriksa dapat dikontrol.
Square (512 X 512) Mengembalikan sumber ke gambar persegi 512 x 512. Rasio memeriksa dapat dikontrol.
Square (256 X 256) Memperoleh sumber ke 256 x 256 gambar persegi. Rasio memeriksa dapat dikontrol.
Custom Memungkinkan sumber untuk resolusi kustom. Rasio memeriksa dapat dikontrol.
Width Lebar gambar yang dihasilkan.

Properti ini hanya terlihat ketika bidang Dimensi diatur ke Custom.
Height Tinggi gambar yang dihasilkan.

Properti ini hanya terlihat ketika bidang Dimensi diatur ke Custom.
Aspect RatioHubungan dimensi proporsional gambar, seperti lebar dan tingginya.

Properti ini hanya terlihat ketika bidang Dimensi diatur ke opsi selain Asli.
No Scaling Area hitam ditambahkan sesuai kebutuhan untuk menjaga rasio aspek konten asli.
Stretch Peregangan konten asli untuk mengisi resolusi tujuan tanpa meninggalkan daerah hitam.
Codec Codec digunakan untuk mengkodekan trek video.

Untuk informasi tentang dukungan codec, lihat dokumentasi pada Kompatibilitas file video.
Auto Pilih codec video yang paling cocok untuk platform target.
H264 MPEG-4 Advanced Video Coding (AVC) video codec, didukung oleh perangkat keras pada sebagian besar platform.
H265 MPEG-H Bagian 2, atau Efisiensi Tinggi Video Coding (HEVC) kode video, didukung oleh perangkat keras pada beberapa platform.
VP8 Kode video VP8, didukung oleh perangkat lunak pada sebagian besar platform, dan dengan perangkat keras pada beberapa platform seperti Android dan WebGLLogin API yang membuat grafis 2D dan 3D di browser web. Opsi build Unity WebGL memungkinkan Unity untuk mempublikasikan konten sebagai program JavaScript yang menggunakan teknologi HTML5 dan WebGL rendering API untuk menjalankan konten Unity di browser web. Bitrate Mode Rendah, Medium, atau bitrate tinggi, relatif terhadap profil baseline codec yang dipilih.
Spatial Quality Pengaturan ini menentukan apakah gambar video berkurang dalam ukuran selama transcoding, yang berarti mereka mengambil ruang penyimpanan yang kurang. Namun, mengubah gambar juga menghasilkan kekacauan selama pemutaran.
Low Spatial Quality Gambar berkurang secara signifikan dalam ukuran selama transcode (biasanya dengan seperempat dimensi aslinya), dan kemudian diperluas kembali ke ukuran aslinya pada pemutaran. Ini adalah jumlah resizing tertinggi, yang berarti menghemat ruang penyimpanan yang paling tetapi menghasilkan sejumlah besar kekacauan pada pemutaran.
Medium Spatial Quality Gambar sedang berkurang dalam ukuran selama transcode (biasanya setengah dimensi aslinya), dan kemudian diperluas kembali ke ukuran aslinya pada pemutaran. Meskipun ada beberapa resizing, gambar akan kurang kabur daripada yang menggunakan opsi Kualitas Spatial Rendah, dan ada beberapa pengurangan ruang penyimpanan yang diperlukan.
High Spatial Quality Tidak ada perubahan jika opsi ini dipilih. Ini berarti gambar tidak berkurang dalam ukuran selama transcode, dan oleh karena itu tingkat kejernihan visual video dipertahankan.
