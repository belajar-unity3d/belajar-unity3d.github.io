Pengirim dalam batch dengan Mode Chunk

Chunk Mode adalah mode rendering default dari Tilemap Renderer:

Ketika set ke Chunk Mode, Pegangan Renderer Tilemap di sebuah Tilemap dalam batch dan render bersama-sama. Mereka diperlakukan sebagai item semacam tunggal ketika diurutkan dalam Queue Transparan 2D. Ini mengurangi jumlah penarikan panggilan untuk meningkatkan kinerja keseluruhan, namun Renderer lain tidak dapat diberikan di antara bagian mana pun dari Tilemap yang mencegah Sprites render lainnya dapat bercampur dengan Sprites Tilemap.

Menggunakan Atlas Sprite untuk Sorting (Chunk Mode saja)

Pada Chunk Mode, Renderer Tilemap tidak dapat memilah ubin dari beberapa tekstur secara individual dan tidak membuat Sprit ubin secara konsisten (lihat example below).

Pengirim secara individual dengan Mode Individu

Pada Individual Mode, Tilemap Renderer menyortir dan membuat Sprites pada Tilemap dengan pertimbangan Renderer lain di Adegani, seperti Sprite RenderersKomponen yang memungkinkan Anda menampilkan gambar sebagai Sprites untuk digunakan di kedua adegan 2D dan 3D. More info
Lihat di Glossary dan Mesh RenderersKomponen mesh yang mengambil geometri dari Filter Mesh dan render di posisi yang ditentukan oleh komponen Transform objek. More info
Lihat di Glossary. Gunakan mode ini jika Renderers lain berinteraksi dengan Sprites dan objek di Tilemap.

Dalam mode ini, penyortiran Renderer Tilemap berdasarkan posisi mereka di Tilemap dan sifat penyortiran yang ditetapkan dalam Renderer Tilemap. Sebagai contoh, hal ini memungkinkan Sprite karakter untuk pergi ke rintangan Sprites (lihat contoh di bawah).

Custom Axis Sorting Mode untuk Z sebagai Y Tilemap dalam Mode Individu %

Untuk mengurutkan dan render ubin Sprites pada Isometric Z as Y Tilemap, Transparency Sort Axis harus diatur ke Custom Axis. Set pertama Renderer Mode untuk 'Modus Individu' dan pergi ke Edit > Settings > Graphics.

Mengatur Transparency Sort Mode ke Custom Axis, dan atur nilai Y-nya menjadi -0.26. Lihat halaman pada Membuat Isometric Tilemap untuk informasi lebih lanjut tentang pengaturan Transparency Sort Axis.
