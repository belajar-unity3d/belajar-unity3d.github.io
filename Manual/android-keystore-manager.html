Referensi jendela Manajer Keystore

Halaman ini menjelaskan antarmuka jendela Manajer Keystore. Untuk informasi lebih lanjut

Untuk membuka jendela Keystore Manager, buka Pengaturan Penerbitan Android dan pilih Keystore Manager.

Anda dapat menggunakan antarmuka ini untuk:

  • Buat keystore baru
  • Tambahkan kunci ke toko kunci
  • Beban keystore yang ada

Keystore

Sifat di bagian antarmuka ini berkaitan dengan keystore yang jendela Keystore Manager saat ini berhasil.

Property Description
Keystore dropdown Menentukan yang keystore untuk digunakan di jendela Keystore Manager. Anda dapat membuat keystore baru, atau memuat yang ada. Untuk informasi lebih lanjut, lihat:
• Buat keystore baru.
• Beban keystore yang ada.
Password Kata sandi untuk keystore. Jika Anda membuat keystore baru, gunakan ini untuk membuat password untuk keystore. Jika Anda memuat keystore yang ada, gunakan ini untuk memasukkan password untuk keystore yang ada.
Confirm Password Kata sandi yang dikonfirmasi untuk keystore baru. Mengatur nilai properti ini untuk sama dengan Password.

Properti ini hanya muncul jika Anda membuat toko kunci baru.

Kunci yang ada

Bagian antarmuka ini berisi daftar tombol yang membaca-hanya yang berisi toko kunci saat ini.

Nilai Kunci Baru

Sifat di bagian antarmuka ini menjelaskan kunci baru untuk menambahkan ke toko kunci. Untuk informasi lebih lanjut, lihat Tambahkan kunci ke toko kunci.

Note: Aplikasi tidak menampilkan informasi pribadi di bagian ini. Sebagai gantinya, sertifikat yang merupakan bagian dari aplikasi Anda mencakupnya.

Property Description
Alias Nama mengidentifikasi untuk kunci.
Password Sandi untuk digunakan untuk kunci.
Confirm password Kata sandi yang dikonfirmasi untuk kunci. Mengatur nilai properti ini sama dengan Password.
Validity (years) Jumlah tahun bahwa kunci berlaku untuk. Ini harus melebihi jumlah waktu yang Anda harapkan untuk mengelola aplikasi untuk, sehingga Anda dapat menggunakan kunci yang sama untuk menandatangani pembaruan aplikasi. Validitas default adalah 25 tahun.
First and Last Name Nama pertama dan terakhir Anda.
Organizational Unit Divisi organisasi Anda yang merupakan bagian dari.
Organization Organisasi yang mengelola aplikasi Anda. Ini seringkali nama perusahaan Anda.
City or Locality Kota pribadi atau lokalitas Anda.
State or Province Negara atau provinsi pribadi Anda.
Country Code Kode negara pribadi Anda.
