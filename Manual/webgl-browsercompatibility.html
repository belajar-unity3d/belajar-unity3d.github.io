Kompatibilitas browser WebGL

Dukungan WebGLLogin API yang membuat grafis 2D dan 3D di browser web. Opsi build Unity WebGL memungkinkan Unity untuk mempublikasikan konten sebagai program JavaScript yang menggunakan teknologi HTML5 dan WebGL rendering API untuk menjalankan konten Unity di browser web. More info
Lihat di Glossary Unity untuk browser desktop berbeda tergantung pada browser. Ini mendukung browser hanya jika kondisi berikut benar:

  • Browser adalah WebGL 2 mampu. Sitemap Unity tidak lagi memberikan dukungan untuk WebGL 1 dalam membangun dibuat dengan opsi API Auto Graphics. Untuk detail lebih lanjut, lihat Note.WebGL 1 deprecation.
  • Browser adalah HTML 5 standar-kompliant.
  • Browser 64-bit dan mendukung WebAssembly.

Unity WebGL tidak mendukung perangkat seluler. Ini mungkin bekerja pada perangkat high-end, tetapi perangkat saat ini sering tidak kuat dan tidak memiliki cukup memori untuk mendukung konten Unity WebGL.

Unity WebGL mendukung beberapa texture formatsFormat file untuk menangani tekstur selama rendering real-time oleh perangkat keras grafis 3D, seperti kartu grafis atau perangkat seluler. More info
Lihat di Glossary terkompresi. Untuk informasi pada format tekstur terkompresi yang mendukung Unity WebGL, lihat Direkomendasikan, default, dan format kompresi tekstur yang didukung, oleh platform.

WebGL 1 deprecation

Dari Unity 2022.1, pengaturan Auto Graphics API tidak lagi termasuk WebGL 1 Graphics API. Namun, jika Anda masih perlu menambahkan dukungan untuk WebGL 1 di proyek Anda kemudian ikuti langkah-langkah ini:

  1. Buka jendela Player SettingsPengaturan yang memungkinkan Anda mengatur berbagai pilihan khusus pemain untuk permainan akhir yang dibangun oleh Unity. More info
    Lihat di Glossary     dan kembangkan bagian Other Settings.
  2. Tentukan opsi Auto Graphics API dan secara eksplisit tambahkan WebGL 1 (Deprecated) ke daftar API Grafik.
