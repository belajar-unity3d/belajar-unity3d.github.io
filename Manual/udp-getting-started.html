Mulai dengan UDP

Implementasi UDP

Unity merekomendasikan menerapkan UDP dalam siklus pengembangan permainan Anda menuju akhir siklus pengembangan, misalnya ketika Anda telah memutuskan apa produk yang dapat dimurnikan game Anda. Ini memudahkan untuk menerapkan UDP dalam permainan back-catalog Anda untuk memberi mereka sewa baru kehidupan di toko aplikasi baru.

Anda dapat menerapkan UDP dalam permainan Anda dalam salah satu cara berikut.

  • Menggunakan Paket UDP hanya
  • Menggunakan paket UDP dan paket Unity IAPUnity In App Purchase
    Lihat di Glossary     (untuk versi paket Unity IAP 2.0.0+)
  • Menggunakan Unity IAP hanya (untuk versi paket Unity IAP 1.22.0–1.23.5)

Implementasi yang Anda pilih tidak mempengaruhi konsol UDP.

Menggunakan Paket UDP

Implementasi ini mirip dengan implementasi Google Play In-App Billing. Jika Anda sebelumnya telah mengkonfigurasi permainan Anda untuk Google Play maka Unity merekomendasikan menggunakan paket UDP.

Paket UDP tersedia dari Manajer Paket Unity atau dari Toko Aset Unity.

Untuk instalasi paket UDP yang berdiri sendiri, lihat.

Menggunakan paket UDP dan Unity IAP

Versi paket Unity IAP 2.0.0 dan di atas tidak mengandung UDP DLL. Ini membutuhkan versi paket UDP 2.0.0 dan di atas. Dari versi ini, dan instal paket Unity IAP dari Asset Store.

Menggunakan Unity IAP

Jika permainan Anda sudah menggunakan Unity IAP, Anda dapat terus menggunakan paket Unity IAP.

Note: Unity merekomendasikan menggunakan paket UDP bersama dengan versi paket Unity IAP 2.0.0+, tersedia dari Sebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
Lihat di Glossary.

UDP disertakan dalam Unity IAP dari versi 1.22.0 - 1.23.5. Jika Anda menggunakan paket Unity IAP (1,22.0 - 1.23.5) tidak menginstal paket UDP secara terpisah. Untuk memeriksa versi Unity IAP diinstal, pergi ke Services > Unity IAP > IAP Updates. Ikuti panduan implementasi umum sebelum Anda menerapkan item IAP Anda dengan Unity IAP.

Jika menggunakan paket Unity IAP, Anda dapat menerapkan item IAP dalam permainan Anda menggunakan kode, atau implementasi kode. Untuk integrasi yang lebih baik dengan UDP, menerapkan IAP menggunakan kode.

Persyaratan sistem

UDP didukung dalam Unity 5.6.1 atau lebih tinggi. Unity merekomendasikan untuk menggunakan 2018.4 atau di atas.

Dari versi paket berikut dan di atas, Anda dapat menggunakan paket UDP dan Unity IAP bersama:

  • WinRAR 5.60 beta
  • Unity IAP - 2.0.0 (Versi TokoAsset)

Mengakses konsol UDP

Untuk mengakses konsol UDP, buka Portal Distribusi dan pilih.

Untuk mengakses konsol UDP dari Editor Unity, pilih Services > Unity Distribution Portal > Configure. Dalam Pengaturan Portal Distribusi Unity, pilih Go to UDP console.
