Profiler

Unity Profiler adalah alat yang dapat Anda gunakan untuk mendapatkan informasi kinerja tentang aplikasi Anda. Anda dapat menghubungkannya ke perangkat di jaringan atau perangkat Anda terhubung ke mesin Anda untuk menguji bagaimana aplikasi Anda berjalan di platform rilis yang dimaksudkan. Anda juga dapat menjalankannya di Editor untuk mendapatkan ikhtisar alokasi sumber daya saat Anda mengembangkan aplikasi Anda.

Profiler mengumpulkan dan menampilkan data pada kinerja aplikasi Anda di daerah seperti CPU, memori, renderer, dan audio. Ini adalah alat yang berguna untuk mengidentifikasi daerah untuk peningkatan kinerja dalam aplikasi Anda, dan bergetar di daerah-daerah tersebut. Anda dapat menentukan hal-hal seperti bagaimana kode, aset, pengaturan sceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info

Lihat di Glossary, dan membangun pengaturan mempengaruhi kinerja aplikasi Anda. Ini menampilkan hasil dalam serangkaian grafik, sehingga Anda dapat memvisualisasikan di mana lonjakan dalam kinerja aplikasi Anda terjadi.

Selain menggunakan built-in Unity Profileer, Anda dapat menggunakan rendah tingkat asli plug-in Profiler API untuk mengekspor data profil ke alat profil pihak ketiga, dan untuk menyesuaikan analisis profil Anda. Anda juga dapat menambahkan alat profil yang kuat seperti, dan proyek Anda untuk menganalisis data kinerja secara lebih detail.

Untuk mengakses jendela Profiler pergi ke menu: Window > Analysis > Profiler. Untuk gambaran rinci tentang jendela, lihat dokumentasi WordPress.org.
