Analisis Unity dan Peraturan Perlindungan Data Umum UE (GDPR)

Peraturan Perlindungan Data Umum (GDPR) adalah hukum Uni Eropa yang mengatur privasi data warga Uni Eropa.

  • Untuk informasi lebih lanjut tentang EU GDPR, lihat url https://www.eugdpr.org

Mempertahankan kepatuhan dengan GDPR ketika Anda menggunakan Unity AnalyticsUnity Analytics
Lihat di Glossary adalah tanggung jawab bersama. Unity mengumpulkan data untuk membantu Anda meningkatkan pengalaman pemain dengan iklan dan gameplay. Beberapa data tersebut mencakup informasi identitas pribadi (PII) yang diatur di bawah GDPR. Unity menyediakan alat untuk memungkinkan pemain untuk memilih koleksi PII, dan untuk mengelola data pribadi yang mengumpulkan Unity tentang mereka, seperti yang diperlukan oleh GDPR. tanggung jawab Anda termasuk menambahkan tombol opt-out ke aplikasi Anda, dan menyediakan tautan ke kebijakan privasi Unity dari kebijakan privasi Anda sendiri.

Ketika aplikasi Anda menggunakan iklan Unity, Unity menampilkan pemberitahuan kepada pemain pertama kali iklan ditampilkan di telepon mereka memberi mereka pilihan untuk memilih atau keluar dari pengumpulan informasi identitas pribadi. Iklan subsequent juga menampilkan tombol yang dapat digunakan pengguna untuk mengelola opsi privasi data mereka. Untuk informasi lebih lanjut tentang GDPR dan SDK Iklan Unity, lihat .

Jika Anda menggunakan iklan Unity dan Analytics, mekanisme opt-out Unity Ads berlaku untuk kedua layanan.

Pilihan di atas mencakup setiap data yang dikumpulkan oleh Unity untuk menyesuaikan iklan dan layanan pemain. Namun, jika Anda mengumpulkan data informasi identitas pribadi Anda sendiri, Anda bertanggung jawab untuk melindungi dan mengelola data tersebut.

The Unity Analytics Database

Jika Anda tidak menggunakan iklan Unity, maka Anda harus menggunakan plug-in Privasi Data Unity Analytics untuk memberikan kontrol pemain Anda atas pengumpulan data Unity Analytics.

Data Privacy plug-in adalah bagian dari paket Unity Analytics Library. Anda dapat menggunakan Unity Package Manager (menu: Window > Package Manager) untuk memeriksa ulang dua paket diaktifkan dalam proyek.

Plug-in melakukan not mendukung platform berikut:

  • Linux
  • Windows Server
  • Tizen
  • Apple TV
  • Blackberry

Layanan Unity Analytics menghapus informasi identitas pribadi yang dikirim dari permainan yang berjalan pada platform tersebut secara otomatis. Hubungi jika Anda memiliki pertanyaan.

Jika permainan Anda menampilkan iklan dari jaringan Iklan Unity, SDK Iklan Unity telah menampilkan pilihan opt-out pengumpulan data ke pemain, dan mengkonfigurasi Analisis Unity berdasarkan pilihan privasi data pemain. Anda hanya perlu menggunakan plug-in Privasi Data Unity Analytics ketika Anda tidak menggunakan layanan iklan Unity.

Method 2: Using you own UI

Untuk menggunakan antarmuka pengguna Anda sendiri untuk tombol, Anda dapat meminta URL halaman opt-out data pengguna dan kemudian buka URL di browser atau tampilan web:

  1. Buat kontrol UI Anda sendiri yang memberi tahu pemain kemampuan mereka untuk memilih keluar dari pengumpulan data.

    Sitemap Data Privacy plug-in mencakup ikon di folder Note:. Unity mendorong Anda untuk menggunakan ikon ini pada tombol privasi data Anda (atau kontrol serupa) untuk memberikan kue grafis yang konsisten untuk pemain yang menghadapi kontrol privasi data dalam permainan Unity.Paket / Perpustakaan Analog / DataPrivacy folder. Unity encourages you to use this icon on your data privacy button (or similar control) to provide a consistent graphical cue to players encountering data privacy controls in Unity games.

  2. Dalam menanggapi klik atau interaksi pemain dengan kontrol ini, sebut fungsi DataPrivacy.FetchPrivacyUrl(). Fungsi ini mengambil objek Action<string> yang membentang ketika permintaan jaringan selesai. Anda dapat secara opsional melewati fungsi Action<string> kedua untuk menangani kasus di mana permintaan jaringan gagal.

  3. Dalam handler Anda untuk permintaan FetchPrivacyUrl(), gunakan Application.OpenURL() untuk membuka URL yang diterima di browser.

Sebagai contoh, script berikut membuka halaman Privasi Data Pemain dalam menanggapi klik GameObject:

using System;
using UnityEngine;
using UnityEngine.Analytics;
    
public class OptOutHandler : MonoBehaviour {

    static void OnFailure(string reason)
    {
        Debug.LogWarning(String.Format("Failed to get data privacy page URL: {0}", reason));
    }

    void OnURLReceived(string url)
    {
        Application.OpenURL(url);
    }

    public void OpenDataURL()
    {
        DataPrivacy.FetchPrivacyUrl(OnURLReceived, OnFailure);
    }
    
    void OnMouseOver(){
        if(Input.GetMouseButtonUp(0)){
            OpenDataURL();
        }
    }
}
