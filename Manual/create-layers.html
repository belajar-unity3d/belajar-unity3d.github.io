Tambahkan lapisan baru

Ada dua cara Anda dapat membuat lapisan baru:

  • Pilih Tags dan jendela Layers (menu utama: Edit > Project Settings > Tags and Layers > Layers).
  • Pilih GameObject, pilih dropdown Layer di jendela InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info

Untuk mengingat fungsi yang dimaksudkan untuk lapisan, berikan nama deskriptif.

Lapisan built-in

Ketika Anda membuka menu layer, Anda dapat melihat bahwa beberapa lapisan sudah dinamakan. Kecuali untuk Raycast Default dan Ignore, Unity tidak lagi menggunakan lapisan bawaan ini untuk banyak tujuan khusus; mereka ada terutama untuk kompatibilitas mundur:

Layer number Layer name Description
1 TransparentFX Unity menggunakan lapisan ini di sistem flare.
2 Ignore Raycast Fisika ray cast API mengabaikan lapisan ini secara default.
3 Water Penggunaan unity layer ini
4 UI(User Interface) Memungkinkan pengguna untuk berinteraksi dengan aplikasi Anda. Unity saat ini mendukung tiga sistem UI. More info
Lihat di Glossary		 Penggunaan ini sebagai lapisan default untuk elemen UI.

Anda masih dapat menggunakan layer-layer ini, tetapi Anda tidak bisa mengubah nama atau menghapusnya.

Tambahkan GameObjects ke lapisan

Untuk menetapkan GameObject ke lapisan, pilih GameObject, buka jendela Inspektur dan pilih dropdown di sebelah Layer. Menu ini menunjukkan lapisan yang memiliki nama sehingga Anda dapat memilih yang satu karya terbaik untuk GameObject Anda.

Anda hanya dapat menetapkan setiap GameObject ke satu lapisan, tetapi Anda dapat menambahkan beberapa GameObjects ke satu lapisan.
