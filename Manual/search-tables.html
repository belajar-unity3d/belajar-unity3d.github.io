Tabel pencarian

Anda dapat menggunakan tabel pencarian untuk melihat, membandingkan, dan memilah item pencarian dengan properti. Dari tabel Anda juga dapat memodifikasi data dan data ekspor ke file .csv atau JSON.

Lihat tabel Pencarian

Untuk melihat pencarian Anda dalam format tabel, masukkan pertanyaan pencarian dan klik ikon tabel di kanan bawah jendela Pencarian Unity.

Secara default, tabel akan menunjukkan kolom Label dan Deskripsi kecuali Anda telah menambahkan kolom baru atau Anda menggunakan pernyataan pilih{} yang membuat kolom baru dari pemilih.

Klik header kolom untuk memilah hasil pencarian dalam urutan naik atau turun oleh properti kolom itu.

Klik ikon ulang () jika tidak ada kolom muncul, atau jika kolom muncul dari pencarian sebelumnya dan Anda ingin mengatur ulang ke tata letak kolom default.

Tambahkan kolom

Untuk menambahkan kolom baru di meja pencarian, klik ikon plus (+) dan pilih properti atau selector.

Sesuaikan kolom

Anda dapat menyesuaikan nama, ikon, penyelarasan dan apakah kolom dapat diurutkan.

Untuk mengedit kolom, klik kanan header kolom dan pilih Edit <column name>.

Field Description
Format Mengubah data tentang hasil ke format lain. Jika hasilnya tidak memiliki data untuk format baru (misalnya meminta informasi warna dari data posisi), sel akan kosong.
Icon Mengubah ikon.
Name Mengubah nama kolom
Alignment Mengubah keselarasan kolom
Sortable Pilih untuk membuat kolom diurutkan di meja
Path Mengindikasikan jalan properti atau sub properti yang berguna jika Anda perlu memahami di mana properti terletak.
Selector Menunjukkan properti mana yang diakses dalam permintaan pencarian. Ini berguna jika Anda ingin membuat pertanyaan untuk sifat spesifik, misalnya jika Anda memiliki kolom untuk properti Tinggi, Anda dapat memeriksa bidang Selector dan membuat query pencarian tinggi untuk Aset menggunakan #m_Height.

Mengatur kolom

Klik dan tahan header kolom dan seret ke mengatur ulang kolom.

Hapus kolom

Klik kanan header kolom dan pilih Remove <column name>.

Tampilkan / sembunyikan kolom

Setelah tabel Anda diatur, Anda mungkin ingin menyembunyikan kolom tertentu sementara tanpa menghapusnya.

Klik kanan di area header kolom dan pilih Show Columns > <name of column>. Kolom dengan tanda centang muncul di meja.

Reset tables

Set ulang tabel kembali ke default, dengan Label dan Deskripsi kolom atau akan menyertakan kolom yang ditentukan dalam pernyataan pilih{}.

Klik ikon ulang () untuk menghapus semua kolom tambahan atau jika kolom muncul dari pencarian sebelumnya dan Anda ingin mengatur ulang ke tata letak kolom default.

Sitemap Jika Anda ingin menjaga tata letak kolom meja pencarian saat ini, simpan meja Anda sebelum mengatur ulang.Note: If you want to keep your current search table column layout, save your table before resetting.

Simpan tabel

Untuk menyimpan tata letak kolom meja pencarian Anda:

  • Klik ikon Simpan di area panel Pengguna atau Proyek Saved Searches, atau ikon Simpan ke kanan lapangan Pencarian.

    Pencarian Unity mencakup tata letak meja dengan query pencarian yang tersimpan. Ketika sebuah query disimpan dengan tabel perubahan ikon ke ikon tabel.

Data tabel ekspor

Anda dapat mengekspor data tabel di format JSON atau .csv. Ketika Anda menyimpan menggunakan JSON, Anda dapat membuka data tabel yang tersimpan di Unity. Ini adalah data statis dari waktu yang Anda simpan itu yang tidak segar.

Untuk mengekspor data tabel:

  1. Dari menu Save drop-down, pilih Export Report (untuk JSON) atau Export CSV.
  2. Nama file Anda, pilih lokasi untuk menyimpan file Anda, lalu klik Save

Modifikasi data tabel

Anda dapat memodifikasi data di tabel menggunakan format Experimental.

Sitemap Ketika Anda mengubah data di tabel Pencarian Unity menggunakan format eksperimental, Pencarian tidak menyadari ketergantungan pada data. Tidak ada perubahan pada kode Note: khusus yang terkait dengan pengeditan properti dipicu. Untuk memastikan bahwa perubahan Anda termasuk ketergantungan atau perubahan terkait yang sering tertanam dalam kode kustom untuk sifat umum, membuat perubahan Anda dalam Inspektur bukan.InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai.
