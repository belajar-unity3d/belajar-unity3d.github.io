Penerbitan ke Toko Aset

Ini adalah alur kerja dasar untuk menerbitkan aset Anda ke Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
Lihat di Glossary:

  1. Buat Akun Penerbit Anda.
  2. Buat draft paket baru. Anda juga dapat menghapus draft Anda setiap saat dan membuat draft baru.
  3. Upload aset ke dalam paket.
  4. Memutar rincian paket.
  5. Kirim paket untuk persetujuan.
  6. Lihat status pengajuan Toko aset Anda.

Meskipun Anda menunggu mendengar apakah Unity telah menerima paket Anda ke Asset Store, Anda dapat mengatur profil pembayaran Anda sehingga Anda dapat mengumpulkan pendapatan dari paket Anda. Jika Anda belum melakukan ini, Anda juga bisa mendirikan dukungan pelanggan untuk akun penerbit Anda.

Setelah paket Anda berada di Aset Store, Anda dapat mengambil tindakan ini untuk meningkatkan penemuan dan mendorong pelanggan untuk membeli paket Anda:

  • Tambahkan tag ke paket Anda untuk memudahkan pelanggan untuk menemukannya.
  • Menyambungkan akun Anda ke Google Analytics untuk melacak lalu lintas situs web ke halaman paket Anda di Aset Store.
  • Unity mungkin menawarkan kesempatan untuk menyertakan paket Anda dalam promosi penjualan Asset Store. berpartisipasi dalam promosi penjualan Unity atau mendirikan salah satu promosi Anda sendiri.
  • Authorize Unity untuk mengembalikan pelanggan Anda untuk paket.

Jika Anda ingin memperbarui atau meningkatkan paket yang ada, Anda dapat membuat versi upgrade utama. Setelah upgrade Anda dipublikasikan ke Aset Store, Anda dapat menentukan paket lama Anda sehingga pengguna baru hanya menemukan versi terbaru. Anda juga dapat membuat versi upgrade "lite", yang dapat berguna untuk memungkinkan pelanggan untuk menguji paket Anda tanpa menghabiskan jumlah penuh.

Anda dapat menggunakan tab di halaman Akun Penerbit Anda untuk:

  • Baca Reviews tentang paket Anda.
  • Periksa Sales dan Revenue.
  • Lihat statistik pada paket Anda Downloads dan setiap Vouchers yang Anda berikan kepada pelanggan.
  • Kelola Users pada tim Anda.
  • Kelola akun Anda Info.

Untuk bantuan tambahan dengan prosedur ini, lihat online dan . Penerbit yang memiliki aset yang diterima di Aset Store juga dapat berlaku untuk bergabung.

Concepts

Berikut adalah beberapa konsep penting yang perlu Anda ketahui tentang penerbitan Toko Aset Unity:

Concept: Definition:
Unity ID (account) Akun pengguna Anda untuk digunakan dalam Unity. ID Unity Anda memungkinkan Anda untuk membeli langganan, mengelola Proyek Anda dan berkolaborasi dengan tim, dan memperoleh alat dan sampel di Toko Aset.

Anda hanya dapat mengakses dan mengelola pembelian Aset Store Anda ketika Anda login dengan akun Unity Anda. Jika Anda memutuskan untuk membuat akun penerbit, Anda juga perlu akun Unity Anda.
Publisher account Akun ini diikat ke akun Unity Anda, tetapi juga memberi Anda akses untuk membuat dan menerbitkan paket, mengatur pembayaran, melihat penjualan dan pendapatan, dan mengelola semua aspek profil Anda di Toko Aset.
Payout profiles Profil Payout memungkinkan Anda menentukan bagaimana (dan bagaimana sering) Unity membayar Anda untuk penjualan Aset Anda, dan apakah Unity perlu menahan pajak (seperti beberapa otoritas pajak memerlukan). Profil Payout dikaitkan dengan Organizations. Anda harus menjadi Owner Organisasi untuk melihat, membuat atau memperbarui profil pembayarannya. Untuk informasi lebih lanjut, lihat Mengelola Organisasi Anda.
Asset Store Tools Anda harus dapat membuat paket untuk menjual di Asset Store dari dalam Unity. Anda bisa mendapatkan mereka dari Asset Store sendiri: mereka seperti Aset lain yang tersedia di toko. Unduh alat ke komputer Anda dan impornya ke komputer Anda Proyek.

Paket Asset Store Tools juga mencakup Alat Validator, yang dapat Anda gunakan untuk memeriksa paket Aset Anda sebelum mengunggah dan mengirimkannya untuk ulasan. Mengvalidasi paket Anda dengan alat ini adalah bagian dari proses peninjauan yang dilakukan oleh tim curation Asset Store pada Aset Anda ketika Anda mengirimkan paket ke Asset Store, sehingga menjalankannya pada paket Anda dapat mengurangi waktu peninjauan. Namun, perhatikan bahwa alat Validator tidak dapat sepenuhnya mensimulasikan proses persetujuan, dan mungkin kadang-kadang memberikan positif palsu.
Package status Jelaskan bagian dari siklus hidup paket setiap paket. Untuk informasi lebih lanjut, lihat Status paket.
