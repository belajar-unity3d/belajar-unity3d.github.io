perspektif permainan untuk game 2D

Tentukan perspektif permainan atau sudut pandang Anda ingin permainan Anda untuk memiliki. Tabel berikut menjelaskan jenis utama perspektif permainan 2D:

Game perspective Description
Top-down Sebuah permainan 2D yang menyediakan tampilan mata overhead atau bird.
Side-on Sebuah permainan 2D di mana karakter bergerak kiri atau kanan (atau naik dan turun) dan gulir layar dengan mereka. perspektif adalah dari sisi.
Isometric dan 2.5D (pertemuan tiga lantai) 2D permainan dengan tampilan isometrik simulasi geometri 3D dan sumbu kedalaman, tetapi gunakan kamera ortoografi bukan kamera perspektif. Ini memberi pemain tampilan unggas dari aksi. Untuk informasi tentang Kamera ortografi dan perspektif, lihat KameraKomponen yang menciptakan gambar sudut pandang tertentu di tempat kejadian Anda. Output ditarik ke layar atau ditangkap sebagai tekstur. More info
Lihat di Glossary.
2.5D (juga dikenal sebagai tampilan tiga lantai) permainan adalah game 2D yang menggunakan geometri 3D untuk lingkungan dan karakter, tetapi membatasi gameplay ke dua dimensi. Efek 3D memiliki visual daripada tujuan fungsional.

Banyak bagian dari panduan ini relevan untuk semua perspektif permainan. Untuk perspektif top-down dan side-on, alat TilemapGameObject yang memungkinkan Anda untuk dengan cepat membuat tingkat 2D menggunakan ubin dan overlay grid. More info
Lihat di Glossary dan 9-pengirisan sangat berguna untuk merancang tingkat Anda. Untuk isometric, tampilan tiga lantai atau perspektif 2.5D, alat Isometric Tilemaps sangat berguna.
