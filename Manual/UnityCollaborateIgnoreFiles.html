Mengekspos Aset dari penerbitan ke Collaborate

Dalam beberapa kasus, Anda mungkin ingin mengecualikan Aset tertentu dalam Proyek Anda dari penerbit ke cloud. Collaborate menggunakan file gitignore, bernama .collabignore, untuk mengecualikan file.

Untuk menambahkan pengecualian Anda sendiri ke file .collabignore:

  1. Temukan file .collabignore dalam akar file Proyek Anda. Jika Anda tidak melihat file ini, lihat petunjuk di bawah untuk menampilkan file tersembunyi.
  2. Gunakan file Dokumentasi GitIgnore untuk mengedit file .collabignore dan tambahkan aturan baru Anda. Anda dapat mengedit file di editor teks.
  3. perubahan file .collabignore Anda di Collaborate untuk berbagi pengecualian Anda dengan sisa tim Anda.

Menampilkan file tersembunyi

Jika Anda tidak dapat melihat file .collabignore, atau file Aset yang ingin Anda masukkan, sistem Anda dapat dikonfigurasi untuk menyembunyikan file sistem. Windows dan Macs menyembunyikan file sistem secara default.

Windows

Untuk menampilkan file tersembunyi di Windows 10 dan di atas:

  1. Buka File Explorer dari taskbar.
  2. Pilih View > Options > Change folder and search options.
  3. Pilih tab View, kemudian Advanced settings.
  4. Pilih Show hidden files, folders, and drives, kemudian pilih OK.

Untuk rincian, lihat Dukungan Windows.

MacOS

Untuk menampilkan file tersembunyi di MacOS:

  1. Navigasi ke direktori akar Proyek Anda.
  2. Menampilkan file sebagai daftar
  3. Tekan tombol Shift+Command+Period untuk beralih file sistem antara tersembunyi dan terlihat.

Untuk rincian, lihat Dukungan Macintosh

File yang tidak dapat dikecualikan

Ada file proyek dan folder yang tidak dapat Anda ecualikan dari Collaborate menggunakan file .collabignore. Ini adalah:

  • File .collabignore.
  • Folder AssetsSetiap media atau data yang dapat digunakan dalam permainan atau proyek Anda. Aset mungkin berasal dari file yang dibuat di luar Unity, seperti model 3D, file audio atau gambar. Anda juga dapat membuat beberapa jenis aset di Unity, seperti Pengontrol Animator, Audio Mixer atau Tekstur Render. More info
    Lihat di Glossary     (meskipun Anda dapat mengecualikan file atau folder tertentu dalam folder Assets.)
  • Folder Project SettingsPengumpulan yang luas dari pengaturan yang memungkinkan Anda untuk mengkonfigurasi bagaimana Fisika, Audio, Jaringan, Grafik, Input dan banyak bidang lain dari jangkauan proyek Anda. More info
    Lihat di Glossary    .
  • Setiap file .asset di dalam folder Pengaturan Proyek.
  • ProjectVersion.txt dalam folder Pengaturan Proyek.
