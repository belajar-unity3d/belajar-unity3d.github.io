Jendela Animator

Animator Window memungkinkan Anda untuk membuat, melihat dan memodifikasi aset Animator Controller.

Jendela Animator memiliki dua bagian utama: area tata letak gridded utama, dan lapisan kiri & pane Parameter.

Bagian utama dengan grid abu-abu gelap adalah area tata letak. Anda dapat menggunakan daerah ini untuk membuat, mengatur dan menghubungkan negara di Pengontrol AnimatorKontrol animasi melalui Lapisan Animasi dengan Mesin Negara Animasi dan Pohon Animasi Blend, dikendalikan oleh Parameter Animasi. Pengontrol Animator yang sama dapat direferensikan oleh beberapa model dengan komponen Animator. More info
Lihat di Glossary Anda.

Anda dapat klik kanan pada grid untuk membuat node negara baru. Gunakan tombol mouse tengah atau seret Alt/Option untuk memanci tampilan di sekitar. Klik untuk memilih node negara untuk mengedit mereka, dan klik & drag node state untuk mengatur ulang tata letak state machineSatu set negara di Animator Controller yang karakter atau animasi GameObject dapat masuk, bersama dengan satu set transisi antara negara-negara tersebut dan variabel untuk mengingat keadaan saat ini. Negara-negara yang tersedia akan tergantung pada jenis gameplay, tetapi keadaan khas termasuk hal-hal seperti idling, berjalan, berjalan dan melompat. More info
Lihat di Glossary Anda.

Pane tangan kiri dapat dialihkan tampilan dan tampilan Layers. Tampilan parameter memungkinkan Anda untuk membuat, melihat dan mengedit Parameter Pengontrol Animator. Ini adalah variabel yang Anda mendefinisikan bahwa bertindak sebagai masukan ke mesin negara. Untuk menambahkan parameter, klik ikon Plus dan pilih jenis parameter dari menu pop up. Untuk menghapus parameter, pilih parameter dalam daftar dan tekan tombol hapus (pada macOS menggunakan fn-Delete untuk menghapus parameter yang dipilih).

Ketika pane tangan kiri beralih ke tampilan lapisan, Anda dapat membuat, melihat dan mengedit layersLapisan dalam Unity dapat digunakan untuk memilih kelompok GameObjects secara selektif atau keluar dari proses atau perhitungan tertentu. Ini termasuk rendering kamera, pencahayaan, tabrakan fisika, atau perhitungan kustom dalam kode Anda sendiri. More info
Lihat di Glossary dalam Pengontrol Animator Anda. Ini memungkinkan Anda untuk memiliki beberapa lapisan animasi dalam satu pengontrol animasi bekerja pada saat yang sama, setiap dikendalikan oleh mesin state terpisah. Penggunaan umum ini adalah untuk memiliki lapisan terpisah bermain animasi atas-tubuh di atas lapisan dasar yang mengontrol animasi gerakan umum untuk karakter.

Untuk menambahkan lapisan, klik ikon plus. Untuk menghapus lapisan, pilih layer dan tekan tombol hapus.

Mengklik ikon "mata" pada atau off akan menunjukkan atau menyembunyikan Parameter & Lapisan sisi-pane, memungkinkan Anda lebih banyak ruang untuk melihat dan mengedit mesin negara Anda.

Lokasi hirarkis “breadcrumb” dalam mesin state saat ini. Negara dapat mengandung sub-states dan treesKomponen GameObject dan terkait yang memungkinkan Anda untuk menambahkan aset pohon ke Adegan Anda. Anda dapat menambahkan tingkat cabang dan daun ke pohon di jendela Inspektur Pohon. More info
Lihat di Glossary dan struktur ini dapat diulang. Ketika mengebor ke dalam keadaan sub, hirarki negara induk dan keadaan saat ini sedang dilihat di sini. Mengklik pada negara induk memungkinkan Anda untuk melompat kembali ke negara induk atau kembali lurus ke lapisan dasar mesin negara.

Mengaktifkan ikon kunci akan menjaga Jendela Animator difokuskan pada mesin state saat ini. Ketika ikon kunci mati, klik aset animator baru atau Objek Permainan dengan animator componentKomponen pada model yang merupakan model yang menggunakan sistem Animasi. Komponen ini memiliki referensi aset Animator Controller yang mengontrol animasi. More info
Lihat di Glossary akan beralih Jendela Animator untuk menunjukkan bahwa mesin state item. Mengunci jendela memungkinkan Anda untuk menjaga jendela Animator menunjukkan mesin state yang sama, terlepas dari aset lain atau objek Game dipilih.
