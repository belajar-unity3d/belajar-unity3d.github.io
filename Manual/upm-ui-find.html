Menemukan paket dan set fitur

Jendela Paket Manager menyediakan beberapa cara untuk membantu Anda menemukan paket khusus atau feature setSebuah fitur set adalah koleksi paket terkait yang dapat Anda gunakan untuk mencapai hasil tertentu dalam Editor Unity. Anda dapat mengelola set fitur langsung di Manajer Paket Unity. More info
Lihat di Glossary:
(A) Packages Pilih "context" yang menentukan apa yang muncul dalam daftar. konteks mungkin menjadi sumber paket, seperti server registry, Asset Store, atau Editor Unity sendiri (untuk paket bawaan). Namun, konteks In Project hanya menampilkan paket dan set fitur yang sudah dipasang di proyek saat ini, terlepas dari asal mereka.

Misalnya, Anda dapat memilih konteks My Assets untuk menampilkan hanya paket Aset Store yang tersedia untuk Anda dalam daftar, atau memilih konteks In Project untuk menampilkan hanya paket Unity dan set fitur yang sudah dipasang di proyek Anda.
(B) Sort Urutkan daftar dalam urutan naik atau turun dengan nama, tanggal diterbitkan (Unity Registry, Pendaftaran Saya, Dalam Proyek hanya), tanggal yang dibeli (paket TokoAsset hanya), atau baru-baru ini diperbarui (In Project and My Assets only).

Misalnya, jika Anda ingin menemukan paket Unity yang memiliki pembaruan baru-baru ini tetapi Anda tidak dapat mengingat nama itu, mencoba menyortir oleh Recently updated dalam urutan turun dan menelusuri daftar dari bagian atas untuk menemukannya.

Sorting mempengaruhi item di bawah setiap expander, tetapi meninggalkan expander di tempat. Jika Anda memilah dari Z-A, Manajer Paket mengubah semua set fitur dari Z- A di dalam expander, dan semua paket di bawah setiap expander tetapi tidak mencampur konten dalam daftar.
(C) Filters Pilih opsi untuk mempersempit paket yang terdaftar. Paket yang tercantum dalam Aset Saya telah meningkatkan pilihan penyaringan:

- Status (Downloaded, Update tersedia, Unlabeled, Tersembunyi, Deprecated)
- Categories (2D, 3D, Audio, Template, Alat, dll)
-Labels (Custom labels anda mendefinisikan di Asset Store)
(D) Search Gunakan kotak pencarian untuk mencari paket Unity atau paket Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info

Kontrol ini membantu mempersempit paket dan set fitur yang muncul dalam tampilan daftar dan dalam urutan apa. Ini memudahkan untuk menemukan apa yang Anda cari atau membantu Anda menjelajah ketika Anda tidak tahu persis apa yang Anda inginkan.

Ketika Anda menggunakan beberapa kontrol ini pada saat yang sama, Anda mempersempit set pertandingan yang muncul dalam daftar.

Setelah Anda menemukan paket Unity yang Anda butuhkan, maka Anda dapat menemukan versi tertentu dari daftar. Untuk set fitur, selalu ada satu versi yang tersedia, jadi Anda bisa baik menginstalnya atau menghapusnya Meme it.

Mencari versi tertentu

Untuk menemukan versi paket tertentu:

  1. Dalam Unity, buka jendela Paket Manager (navigasi ke menu utama Unity dan pergi ke Window > Package Manager).

  2. Jika Anda mencari paket pra-release, ikuti langkah-langkah ini:

    • Pilih Project SettingsPengumpulan yang luas dari pengaturan yang memungkinkan Anda untuk mengkonfigurasi bagaimana Fisika, Audio, Jaringan, Grafik, Input dan banyak bidang lain dari jangkauan proyek Anda. More info
      Lihat di Glossary       di bawah ikon Pengaturan canggih .

      Jendela Pengaturan Proyek Package Manager muncul.

    • Di bawah kelompok Advanced Settings, aktifkan opsi Enable Pre-release Packages.

    • Tutup jendela Pengaturan Proyek. Setiap paket pra-release sekarang muncul dalam daftar dengan indikator .

  3. Anda dapat menelusuri Daftar paket atau menemukan paket tertentu menggunakan salah satu metode ini:

    • Cari paket dengan nama atau kata kunci.
    • Daftar (misalnya, dengan terakhir diperbarui dalam urutan turun untuk menemukan paket yang paling baru diperbarui).
    • Terapkan filter ke daftar.

  4. Pada Daftar paket, klik ikon expander ke kiri nama paket.

    Note: Jika Anda melihat tautan See other versions setelah memperluas paket, Anda dapat mengkliknya untuk melihat daftar semua versi yang tersedia untuk paket tersebut.

  5. Pilih versi apa pun untuk melihat spesifikasi details untuk versi itu.

  6. Anda dapat melakukan berbagai tindakan ketika Anda memilih versi tertentu:

    • Anda dapat mengklik tautan View documentation ke Buka dokumentasi yang sesuai dengan versi ini.
    • Anda dapat mengklik tautan View changelog untuk melihat daftar perubahan dari pembuatan paket hingga versi ini.
    • Anda dapat mengklik tautan View licenses untuk melihat informasi lisensi untuk paket ini secara khusus untuk versi ini.
    • Anda dapat melihat apakah paket ini tergantung paket lain, atau apakah paket lain tergantung pada satu ini.
    • Jika paket ini sudah dipasang dan berisi aset sampel, Anda dapat mengklik tombol Import untuk menyimpan salinan dari mereka di bawah folder Assets proyek Anda.
    • Jika Anda belum memiliki paket ini diinstal, Anda dapat install versi ini.
    • Jika versi lain dari paket ini sudah diinstal, Anda dapat update paket ke versi ini.
    • Jika Anda menginstal paket ini tetapi tidak ingin lagi, Anda dapat remove paket.
