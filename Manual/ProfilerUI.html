Modul UI(User Interface) Memungkinkan pengguna untuk berinteraksi dengan aplikasi Anda. Unity saat ini mendukung tiga sistem UI. More info
Lihat di Glossary dan UI Details ProfilerJendela yang membantu Anda untuk mengoptimalkan permainan Anda. Ini menunjukkan berapa banyak waktu yang dihabiskan di berbagai bidang permainan Anda. Sebagai contoh, dapat melaporkan persentase waktu yang dihabiskan rendering, aimating, atau dalam logika permainan Anda. More info
Lihat di Glossary menyediakan informasi tentang berapa banyak waktu dan sumber daya Unity menghabiskan peletakan keluar dan rendering antarmuka pengguna dalam aplikasi Anda. Anda dapat menggunakan modul ini untuk memahami bagaimana Unity menangani UI batching untuk aplikasi Anda, termasuk mengapa dan bagaimana itu batch objek. Anda juga dapat menggunakan modul ini untuk mengetahui bagian mana UI bertanggung jawab untuk kinerja lambat, atau untuk melihat pratinjau UI saat Anda menggosok timeline.

Untuk informasi lebih lanjut tentang cara menggunakan jendela Profiler, lihat dokumentasi di Memulai dengan jendela Profiler.

Kategori grafik

UI dan UI Details Diagram modul profiler dibagi menjadi lima kategori. Untuk mengubah urutan kategori di grafik, Anda dapat menyeret dan menjatuhkannya di legenda grafik. Anda juga dapat mengklik legenda berwarna kategori untuk memotong tampilannya.

Batches Menampilkan jumlah total panggilan penarikan yang di batch bersama.

Detail modul pane

Ketika Anda memilih UI atau UI DetailsProfiler modul, rincian modul pane di bagian bawah jendela Profiler menampilkan lebih detail di UI dalam aplikasi Anda. Anda dapat menggunakannya untuk memeriksa informasi profil tentang objek UI dalam aplikasi Anda. Pane dibagi menjadi kolom berikut:

Column Description
Object Daftar UI kanvas aplikasi Anda digunakan selama periode diprofilkan. Klik ganda pada baris untuk menyoroti objek yang cocok di SceneAdegan berisi lingkungan dan menu permainan Anda. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info

Batch Breaking Reason Mengapa Satuan membagi batch. Kadang-kadang Unity mungkin tidak mampu batch benda bersama. Alasan umum termasuk:

Not Coplanar With Canvas, di mana batching membutuhkan transformasi rect objek untuk disalin (unrotated) dengan kanvas.
CanvasInjectionIndex, di mana komponen CanvasGroup hadir dan memaksa batch baru, seperti ketika menampilkan daftar drop down kotak combo di atas sisanya.
Different Material Instance, Rect clipping, Texture, or A8TextureUsage, di mana Unity hanya dapat batch bersama benda dengan bahan identik, masking, tekstur, dan tekstur penggunaan saluran alfa.
