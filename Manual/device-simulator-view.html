Tampilan Simulator

Tampilan Simulator menampilkan aplikasi Anda pada perangkat seluler simulasi. Gunakan untuk melihat bagaimana aplikasi Anda muncul dengan bentuk layar, resolusi, dan orientasi perangkat itu.

Menggunakan tampilan Simulator

Untuk membuka tampilan Simulator, lakukan salah satu berikut:

  • Pada sudut kiri atas, gunakan menu drop-down untuk beralih antara tampilan Game dan tampilan Simulator.Tampilan permainan, in the top left corner, use the drop-down menu to switch between the Game view and the Simulator view.
  • Di menu: Window > General > Device Simulator

Unity mensimulasikan perangkat dalam tampilan Simulator. Untuk mengontrol simulasi, gunakan toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
Lihat di Glossary dan Panel kontrol.

Toolbar

Toolbar adalah di bagian atas tampilan Simulator dan berisi opsi yang, bersama dengan Panel kontrol, mengontrol simulasi.

Control Description
Game/Simulator view Gunakan menu drop-down ini untuk beralih antara tampilan Simulator dan tampilan Game.
Device selection drop-down menu Pilih perangkat untuk mensimulasikan dari definisi perangkat yang tersedia.
Scale Zoom atau keluar pada layar simulasi.
Fit to Screen Skala tampilan agar pas di dalam jendela.
Rotate Simulasi rotasi fisik perangkat. Gambar pada layar perangkat berputar bersama dengan perangkat jika Anda mengaktifkan auto-rotasi dan jika perangkat mendukung rotasi. Jika Anda memutar perangkat itu menyebabkan gambar di samping atau terbalik. Sitemap Simulator Perangkat tidak mendukung simulasi gyroskop.Note: The Device Simulator doesn’t support gyroscope simulation.
Safe Area Menunjukkan apakah pandangan menarik garis di sekitar SafeArea dari layar perangkat saat ini.
Play Mode behavior Bagian ini menjelaskan perilaku mode bermain berdasarkan pilihan Anda di bawah ini.
Focused Aktifkan fokus pada tampilan Game yang dipilih saat Editor berada dalam mode Play. Login Hanya satu tampilan permainan yang bisa difokuskan ketika Anda memasukkan mode Play. Menggunakan Maximized atau Fullscreen pada mode Tampilan menyiratkan fokus pada tampilan Game Maksimal. Mengaktifkan Fokus pada pandangan Game menonaktifkannya pada pandangan Game lain.
Only one game view can be in focus when you enter the Play mode. Using Maximized or Fullscreen on Display mode implies focus on the Maximized Game view. Enabling Focussed on a Game view disables it on other Game views.
VSync (Game view only) Aktifkan VSync (Game view only) untuk memungkinkan sinkronisasi, yang berguna ketika merekam video, misalnya. Bersatu berusaha untuk membuat pandangan Game pada tingkat refresh monitor, meskipun ini tidak dijamin. Ketika opsi ini diaktifkan, masih berguna untuk memaksimalkan tampilan Game dalam mode Bermain untuk menyembunyikan pandangan lain dan mengurangi jumlah pandangan yang disatukan render.
Enter Play Mode: Pilih dari opsi di bawah ini untuk menentukan pengaturan untuk Editor ketika masuk ke mode Play.
Normally Pilih ini untuk melihat tampilan Game tanpa memaksa fokus atau memaksimalkan pandangan apa pun ke layar penuh.
Maximized Ketika opsi ini dipilih, Unity menjalankan mode Play dengan tampilan Simulator memaksimalkan 100% dari jendela Editor.
Note: Pilihan ini tidak membuat tampilan Simulator jika Anda menonaktifkan preferensi Editor Create Game View On Play.

Panel kontrol

Untuk membuka Panel Kontrol, klik Control Panel di sudut kanan atas Toolbar. Secara default, Panel Kontrol mengandung pengaturan untuk Aplikasi simulasi kelas.

Property Description
System Language Menentukan nilai untuk menerima dari Perangkat.Aplikasi.systemBahasa.
Internet Reachability Menentukan nilai untuk menerima dari Perangkat.Aplikasi.internetReachability.
On Low Memory Panggilan acara Login Login.

Plugin Simulator perangkat dapat mengubah tampilan panel kontrol dan menambahkan konten dan kontrol. Beberapa paket, seperti paket, mengandung plugin Simulator Perangkat.
