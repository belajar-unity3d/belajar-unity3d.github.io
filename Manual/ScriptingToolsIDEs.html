Lingkungan pengembangan terintegrasi (IDE) dukungan

Lingkungan pengembangan terintegrasi (IDE) adalah perangkat lunak komputer yang menyediakan alat dan fasilitas untuk memudahkan pengembangan perangkat lunak lainnya. Unity mendukung IDE berikut:

  • Visual Studio
  • Visual Studio Kode
  • JetBrains Rider

Visual Studio (default IDE di Windows dan macOS)

Ketika Anda menginstal Unity di Windows dan macOS, secara default Unity juga menginstal Visual Studio atau Visual Studio untuk Mac, masing-masing. Anda dapat memilih untuk mengecualikan ketika Anda memilih komponen mana untuk mengunduh dan menginstal. Secara default, External Script Editor (menu: Unity > Preferences > External Tools > External Script Editor) diatur ke Visual Studio. Ketika Anda mengaktifkan opsi ini, peluncuran Unity Visual Studio dan menggunakannya sebagai editor default untuk semua file skrip. Untuk informasi lebih lanjut, lihat Visual Studio C# API.

Di macOS, Unity menginstal Visual Studio untuk Mac sebagai C# IDE. Visual Studio Tools untuk Unity (VSTU) menyediakan integrasi Unity untuk Visual Studio untuk Mac (VS4M). Untuk informasi tentang mengatur dan menggunakan Visual Studio untuk Mac, lihat halaman dokumentasi Microsoft berikut:

  • Alat Studio Visual untuk Unity
  • Setup Visual Studio untuk Mac Tools untuk Unity
  • Menggunakan Visual Studio untuk Mac Tools untuk Unity

Note: Pada Windows, Unity juga mencakup Visual Studio 2019 Login secara default dalam paket instalasi.

Visual Studio Code (Windows, macOS, Linux)

Unity mendukung script pembukaan di Visual Studio Kode (VS Code). Untuk membuka skrip di VS Code, pergi ke Unity > Preferences > External Tools > External Script Editor dan pilih Visual Studio Code. Untuk informasi menggunakan VS Code dengan Unity, lihat dokumentasi Visual Studio pada Pengembangan Unity dengan Kode VS.

Prerequisites

Untuk menggunakan Kode Studio Visual untuk pengeditan kode C# dan dukungan debugging Unity C #, Anda perlu menginstal:

  • MonoBackend scripting digunakan dalam Unity. More info
    Lihat di Glossary     (hanya diperlukan pada macOS dan Linux)
  • Visual Studio Kode C# Ekstensi
  • Sitemap (Tidak didukung secara resmi oleh Unity)Visual Studio Code Debugger untuk Ekstensi Unity (Not officially supported by Unity)

JetBrains Rider (Windows, macOS, Linux)

Unity mendukung script pembukaan di JetBrains Rider. Untuk membuka skrip di Rider, pergi ke Unity > Preferences > External Tools > External Script Editor dan pilih Rider.

Rider didasarkan pada ReSharper dan mencakup sebagian besar fiturnya. Ini mendukung semua fitur C # 7.2 serta debugging C # pada runtime .NET 4.6 scripting di Unity. Untuk informasi lebih lanjut, lihat dokumentasi JetBrains pada Rider untuk Unity.
