Kunci Inspektur

Biasanya, Inspektur menunjukkan sifat dari GameObject yang dipilih saat ini, skrip, atau Aset. Tapi kadang-kadang, berguna untuk menyimpan satu item di Inspektur sementara Anda bekerja dengan item lain. Dalam hal itu, Anda dapat mengunci jendela Inspektur ke item tertentu.

Untuk mengunci jendela Inspektur ke item, klik ikon kunci di jendela Inspektur. Ikon kunci berubah untuk menunjukkan bahwa Inspektur sekarang terkunci untuk seleksi.

Ini berguna ketika Anda mengedit satu properti item sering. Misalnya, jika Anda menyempurnakan posisi Kamera, Anda dapat mengunci jendela Inspektur ke Kamera, dan membuka jendela Inspektur kedua (More Items (⋮) menu: Add Tab > Inspector) untuk terus menampilkan sifat pilihan saat ini.

Note:
Inspektur Fokus adalah alternatif untuk mengunci jendela Inspektur. Inspektur yang terfokus hanya pernah menampilkan sifat item yang Anda buka.

Reorder GameObject komponen

Anda dapat memesan komponen GameObject di jendela Inspector. Perintah komponen yang berlaku di Inspektur adalah urutan yang sama yang perlu Anda gunakan ketika Anda komponen query di skrip Anda.

Transform componentKomponen Transform menentukan Posisi, Rotasi, dan Skala setiap objek di tempat kejadian. Setiap GameObject memiliki Transform. More info
Lihat di Glossary GameObject selalu merupakan komponen terpenting. Anda tidak dapat memindahkannya, atau menempatkan komponen lain di atasnya.

Untuk memesan komponen GameObject di jendela Inspektur, lakukan salah satu berikut:

  • Seret-and-drop header komponen dari satu posisi ke posisi lain. Ketika Anda menyeret header komponen, penanda penyisipan biru muncul di mana Unity akan menempatkan komponen.
  • Klik kanan komponen untuk membuka menu konteks, dan pilih Move Up atau Move Down.

Reordering komponen untuk beberapa GameObjects

Ketika Anda memilih beberapa GameObjects, Inspektur menampilkan semua komponen yang dipilih GameObjects memiliki umum. Jika Anda memesan komponen umum ini, perubahan berlaku untuk semua GameObjects yang dipilih.

Menetapkan ikon kustom untuk GameObjects, script, dan Prefab

Untuk menetapkan ikon kustom ke GameObject atau script:

  1. Buka item di jendela Inspektur.
  2. Klik tombol Select icon.
  3. Pilih ikon dari menu Select icon (lihat di bawah).

Untuk menetapkan ikon kustom untuk Prefab, Anda harus membuka Prefab di Mode Prefab.

Pilih ikon dari menu ikon Pilih

Anda menetapkan ikon kustom dari menu Select Icon.

Jenis ikon: Deskripsi:
1 Ikon label berwarna kapsul. Dalam pandangan adegan, mereka menampilkan nama item yang mereka ditugaskan untuk.

Klik ikon label untuk menetapkannya.
2 Gambar-only ikon berwarna panggilan. Mereka berguna ketika Anda bekerja dengan objek yang mungkin tidak memiliki representasi visual. Misalnya, Anda dapat menetapkan ikon gambar-hanya ke titik-titik navigasi sehingga Anda dapat melihat dan memilihnya dalam tampilan Adegan.

Klik ikon gambar-hanya untuk menetapkannya.
3 Aset ikon adalah Aset gambar di Adegan Anda yang Anda gunakan sebagai ikon kustom. Misalnya, Anda dapat menggunakan ikon tengkorak dan tulang untuk menunjukkan area bahaya di tingkat permainan.

Klik tombol Other untuk membuka jendela Object Picker dan pilih Aset gambar untuk menetapkan ikon.

Biasanya, jendela Inspektur dikonfigurasi sebagai editor untuk properti seleksi. Tapi kadang-kadang berguna untuk hanya melihat sifat dan nilai-nilai mereka. Ketika Anda mengaktifkan mode Debug, Inspektur hanya menunjukkan sifat dan nilai-nilai mereka. Jika pemilihan memiliki komponen skrip, mode Debug juga menampilkan variabel pribadi, meskipun Anda tidak dapat mengedit nilai-nilai mereka.

Anda dapat beralih mode Debug untuk setiap jendela Inspektur secara individual.

  • Untuk mengaktifkan mode Debug, klik tombol More Items (⋮) untuk membuka menu konteks, dan pilih Debug.
  • Untuk kembali ke mode normal, klik tombol More Items (⋮) untuk membuka menu konteks, dan pilih Normal.
