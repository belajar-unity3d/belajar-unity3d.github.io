Mengaktifkan Streaming Virtual Texturing di proyek Anda

Untuk mengaktifkan Streaming Virtual Texturing, Anda perlu mengaktifkan Virtual Texturing dalam proyek Anda. Untuk melakukan ini, pergi ke Edit > Project Settings > Player dan mengaktifkan pengaturan Virtual Texturing.

Virtual Texturing adalah pengaturan luas proyek yang dibagikan untuk semua platform. Anda tidak dapat membangun pemain untuk platform dan API grafis yang tidak mendukung Virtual Texturing. Teksur virtual dapat mengalokasikan sumber daya seperti penyangga, bahkan jika Anda tidak menggunakan fitur di proyek Anda, jadi jangan mengaktifkan Virtual Texturing jika Anda tidak berencana menggunakannya.

Ketika Anda mengaktifkan Virtual Texturing di proyek Anda, Unity menambahkan petunjuk kompiler berikut:

  • ENABLE_VIRTUALTEXTURES: C# mendefinisikan bahwa mengevaluasi ke True jika proyek Anda mendukung Virtual Texturing.
  • ENABLE_VIRTUALTEXTURING: C++ dan C# editor mendefinisikan bahwa mengevaluasi ke True jika Virtual Texturing dimungkinkan pada target build saat ini.
  • UNITY_VIRTUAL_TEXTURING: ShaderProgram yang berjalan di GPU. More info
    Lihat di Glossary     versi ENABLE_VIRTUALTEXTURES.

Perhatikan bahwa ini mungkin berganti nama dalam versi Unity di masa depan.
