Variabel AS dalam Pembuat UI

Login Variabel adalah sifat gaya AS biasa dengan nama yang ditentukan pengguna kustom yang dapat menahan jenis nilai gaya. Mereka kemudian dapat digunakan di tempat nilai properti gaya ASS lainnya. Variabel AS terutama digunakan untuk tema, dengan tema Unity default mengekspos daftar panjang variabel standar untuk membuat UI(User Interface) Memungkinkan pengguna untuk berinteraksi dengan aplikasi Anda. Unity saat ini mendukung tiga sistem UI. More info
Lihat di Glossary Anda lebih konsisten dengan kontrol standar. Contoh di bawah ini menunjukkan definisi variabel dan penggunaan di USS:

* {
    --my-var: 20px;
}

.my-button {
    width: var(--my-var);
}

Notes

Beberapa poin tambahan untuk diingat ketika menggunakan Variabel ASS di UI Builder:

  1. Login Variabel tidak didukung dalam gaya inline per-element di inti UI Toolkit. Ketika memilih elemen yang akan Anda lihat sifat gaya mana yang didorong oleh variabel dengan highlight label lapangan, tetapi hanya memiliki opsi untuk View Variable. Anda dapat menimpa variabel dengan nilai nyata dengan menginjak properti gaya inline.
  2. Login Variabel berguna untuk Editor Extension Authoring, ketika menggunakan tema Editor (Dark atau Cahaya). Seperti itu, Anda tidak akan menemukan banyak variabel ketika mencari melalui dropdown Inspector secara default. Pastikan untuk mengaktifkan pengaturan Editor Extension Authoring melalui pengaturan Canvas untuk melihat semua variabel Editor yang tersedia.
  3. UI Builder tidak saat ini mendukung penciptaan variabel USS baru. Untuk membuat variabel USS baru, Anda harus menggunakan editor teks sambil mengedit Lembar Style secara langsung.
