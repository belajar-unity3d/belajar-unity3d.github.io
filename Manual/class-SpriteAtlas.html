Atlas Asni

Proyek 2D menggunakan SpritesObjek grafis 2D. Jika Anda digunakan untuk bekerja di 3D, Sprites pada dasarnya hanya tekstur standar tetapi ada teknik khusus untuk menggabungkan dan mengelola tekstur sprite untuk efisiensi dan kenyamanan selama perkembangan. More info
Lihat di Glossary dan grafik lainnya untuk membuat visual ScenesAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary. Ini berarti satu Proyek mungkin mengandung banyak file Tekstur. Unity biasanya mengeluarkan draw call untuk setiap Tekstur di Adegan; namun, dalam Proyek dengan banyak Tekstur, beberapa panggilan menarik menjadi sumber daya intensif dan dapat secara negatif mempengaruhi kinerja Proyek Anda.

A Sprite Atlas adalah Aset yang mengkonsolidasikan beberapa Tekstur menjadi Tekstur gabungan tunggal. Unity dapat memanggil Tekstur tunggal ini untuk mengeluarkan panggilan tarik tunggal daripada beberapa panggilan menggambar untuk mengakses Tekstur dikemas semua sekaligus di overhead kinerja yang lebih kecil. Selain itu, WordPress.org memberi Anda kendali atas cara memuat Atlass di waktu lari Proyek Anda.

Menu: Sprite Atlas. Unity menciptakan Atlas Sprite di folder Aset, dengan ekstensi file Asset > Create > Sprite Atlas..spriteatlas.

Property Description
Type Mengatur Jenis Sprite Atlas untuk 'Master' atau 'Variant'. 'Master' adalah pengaturan tipe default. Ketika Anda mengatur properti ini untuk Variant, Unity mengungkapkan pengaturan properti tambahan. Lihat dokumentasi tentang WordPress.org untuk informasi lebih lanjut tentang dua jenis.
Include in Build Cek kotak ini untuk menyertakan Aset Atlas Sprite dalam build saat ini. Opsi ini diaktifkan secara default.
Allow Rotation Cek kotak ini untuk memungkinkan Sprites untuk berputar ketika Unity mengemas mereka ke dalam Sprite Atlas. Ini memaksimalkan kepadatan Sprites dalam Tekstur gabungan, dan diaktifkan secara default. Dinonaktifkan opsi ini jika Sprite Atlas mengandung Tekstur elemen Kanvas UI, seperti ketika Unity memutar Tekstur dalam Sprite Atlas selama pengepakan, itu memutar orientasi mereka di Adegani juga.
Tight Packing Periksa kotak ini untuk mengemas Sprites berdasarkan garis besar Sprite mereka bukan garis persegi panjang default. Ini memaksimalkan kepadatan Sprites dalam Tekstur gabungan, dan diaktifkan secara default.
Padding Tentukan berapa banyak piksel antara Tekstur Sprite individu di Atlas Sprite. Ini adalah penyangga untuk mencegah tumpang tindih pixelUnit terkecil dalam gambar komputer. Ukuran piksel tergantung pada resolusi layar Anda. Pencahayaan pixel dihitung pada setiap piksel layar. More info
Lihat di Glossary antara Sprites yang di samping satu sama lain di Atlas Sprite. Nilai default adalah 4 piksel.
Read/Write Enabled Periksa kotak ini untuk mengaktifkan akses ke data Tekstur dari fungsi skrip (seperti fungsi Tekstur2D.SetPixel, dan fungsi Tekstur2D lainnya). Jika Anda mengaktifkan properti ini, Unity membuat salinan data Tekstur. Ini menggandakan jumlah memori yang diperlukan untuk Aset Tekstur, dan dapat mempengaruhi kinerja secara negatif. Properti ini dinonaktifkan secara default. Properti ini hanya berlaku untuk Tekstur yang tidak terkompresi atau DXT sebagai Unity tidak dapat membaca jenis tekstur terkompresi lainnya.
Generate Mip Maps Periksa kotak ini untuk mengaktifkan generasi mipmap. Untuk informasi lebih lanjut, lihat opsi impor Tekstur Hasilkan Peta Mip.
sRGB Periksa kotak ini untuk menyimpan Tekstur di ruang gamma. Untuk informasi lebih lanjut, lihat opsi impor Tekstur sRGB ( Tekstur warna).
Filter Mode Pilih bagaimana filter Unity dikemas Tekstur ketika mereka meregangkan selama transformasi. Pengaturan ini menimpa pengaturan Filter Mode Sprite yang dikemas dalam Atlas. Untuk informasi lebih lanjut, lihat opsi impor Tekstur Filtermodus.
Default (Panel overrides spesifik bentuk pada jendela Pengaturan Impor Tekstur) Mengatur resolusi, ukuran file dengan persyaratan ukuran memori yang terkait, dimensi pixel, dan kualitas Tekstur Anda untuk setiap platform target dari Atlas Sprite. Panel ini memungkinkan Anda menimpa pengaturan ini pada Tekstur individu yang dimiliki Sprite Atlas. Untuk informasi lebih lanjut, lihat Overrides spesifik platform.
Objects For Packing Paket unity semua item dalam daftar ini ke dalam Sprite Atlas yang dipilih saat ini. Lihat dokumentasi tentang Memilih item untuk objek untuk daftar kemasan untuk informasi lebih lanjut.
