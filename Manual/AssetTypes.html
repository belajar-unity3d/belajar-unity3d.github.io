Jenis Aset yang Didukung

Unity mendukung berbagai jenis aset dan jenis file gambar yang paling umum, termasuk BMP, TIFF, TGA, JPG, dan PSD.

Jika Anda menyimpan file Photoshop berlapis (.psd) di folder Assets Anda, Unity mengimpor mereka sebagai gambar rata. Anda dapat mengetahui lebih dari mengimpor gambar sebagai tekstur, atau mengimpor gambar sebagai kecambah.

Untuk daftar lengkap importir bawaan Unity, jenis file yang didukung dan ekstensi nama file yang didukung, lihat Importir internal.

Terdaftar di bawah ini adalah beberapa jenis aset yang lebih umum yang mungkin ingin Anda gunakan ketika memulai dengan Unity, dan tautan untuk mengetahui lebih lanjut tentang cara bekerja dengan mereka.

Aset di Manajer Paket Unity

Anda dapat menginstal berbagai aset, termasuk plug-ins, alat, dan perpustakaan langsung ke Unity melalui Unity Package Manager (UPM). Ini adalah jenis paket baru, dan tersedia melalui Paket jendela Manager. Untuk informasi lebih lanjut tentang paket secara umum, lihat dokumentasi PaketPaket adalah koleksi aset untuk dibagikan dan digunakan kembali dalam Unity. Manajer Paket Unity (UPM) dapat menampilkan, menambahkan, dan menghapus paket dari proyek Anda. Paket-paket ini berasal dari Unity Package Manager dan memberikan metode mendasar untuk memberikan fungsi Unity. Namun, Unity Package Manager juga dapat menampilkan Paket Asset Store yang Anda unduh dari Asset Store. More info
Lihat di Glossary.

Memanfaatkan aset antara proyek

Ketika Anda membangun permainan, Unity menyimpan banyak metadata tentang aset Anda, seperti pengaturan impor dan tautan ke aset lain, di antara informasi lainnya. Jika Anda ingin mentransfer aset Anda ke proyek yang berbeda dan melestarikan semua informasi ini, Anda dapat mengekspor aset Anda menjadi salah satu wadah ini:

  • A paket kustom, yang dapat Anda akses dan mengelola dalam Manajer Paket Unity.
  • Sebuah Paket asetKoleksi file dan data dari proyek Unity, atau elemen proyek, yang dikompresi dan disimpan dalam satu file, mirip dengan file Zip, dengan ekstensi .unitypackage. Paket aset adalah cara praktis berbagi dan menggunakan proyek Unity dan koleksi aset. More info
