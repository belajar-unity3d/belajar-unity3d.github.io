Unity Peluncur Manifest

Manifest Peluncur Unity mengkonfigurasi bagaimana aplikasi terlihat dan berperilaku sebelum peluncuran aplikasi. Misalnya, mengandung ikon aplikasi, nama, dan lokasi instalasi. Manifest Peluncur Unity adalah konsep khusus untuk pengembangan Android dan Anda dapat menuliskannya untuk mengintegrasikan Unity sebagai komponen ke proyek yang ada. Untuk informasi lebih lanjut, lihat Mengintegrasikan Unity ke aplikasi Android.

Settings

Anda dapat mengkonfigurasi semua pengaturan di Unity Launcher Manifest dari Pengaturan Pemain Unity. Ini berarti, kecuali Anda ingin mengintegrasikan Unity sebagai komponen ke proyek yang ada, Anda tidak perlu untuk menulis Unity Launcher Manifest atau secara manual mengedit pengaturan apa pun langsung dalam file.

File Manifest Peluncur Unity menyatakan sebagai berikut:

  • Nama paket.
  • Ikon aplikasi.
  • Nama aplikasi.
  • aktivitas awal aplikasi dan maksudnya.
  • Lokasi instalasi aplikasi.
  • Ukuran layar yang didukung aplikasi.
  • Lisensi isGame aplikasi.
