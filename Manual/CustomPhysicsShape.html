Menggunakan Fisika Khusus Editor bentuk

Fisika Khusus Editor bentuk menyediakan berbagai fungsi untuk memungkinkan Anda untuk membuat atau menghasilkan bentuk fisika Sprite. Ada dua cara untuk membuat bentuk fisika kustom: memiliki Unity secara otomatis generate bentuk, atau membuat dan mengedit secara manual di jendela editor.

Menghasilkan bentuk fisika

Untuk memiliki Unity secara otomatis menghasilkan bentuk fisika yang mengikuti bentuk tekstur Sprite asli, dan itu juga memperhitungkan area transparan dalam tekstur, klik tombol Generate. Namun, Anda dapat menyesuaikan bagaimana bentuk fisika yang dihasilkan mengikuti tekstur Sprite dengan menyesuaikan Outline Tolerance slider.

Sesuaikan slider Outline Tolerance untuk memperbaiki garis besar bentuk fisika yang menghasilkan Unity. Meningkatkan nilai meningkatkan seberapa era garis besar mengikuti bentuk tekstur Sprite. Leaving slider di 0 menghasilkan bentuk fisika yang lebih longgar mengikuti tekstur Sprite.

Setelah menyesuaikan nilai Toleransi Garis Besar, untuk memiliki Unity secara otomatis menghasilkan bentuk fisika berdasarkan pengaturan slider, klik Generate. Jika Anda menyesuaikan nilai slider setelah menghasilkan garis besar, untuk menggenerasi garis besar berdasarkan nilai slider yang diperbarui, klik Generate again.

Secara manual mengedit bentuk fisika

Anda dapat membuat bentuk fisika kustom Anda sendiri dan mengeditnya dalam dua cara. Cara pertama adalah dengan secara otomatis generating garis besar, dan kemudian memindahkan titik kontrol garis yang dihasilkan untuk memperbaikinya lebih lanjut. Atau, klik dan seret di ruang apa pun di jendela editor Bentuk Fisika Kustom untuk membuat garis besar persegi panjang dengan empat titik kontrol dan mulai penyulingan garis dengan memindahkan titik kontrol atau menambahkan dan menghapusnya.

Pindahkan titik kontrol

Untuk menyesuaikan bentuk garis keluar Mesh, klik dan seret titik kontrol. Ketika Anda menggigit titik kontrol, ternyata biru untuk menunjukkan bahwa Anda dapat memilihnya. Klik dan seret titik kontrol ke posisi yang berbeda untuk menyesuaikan bentuk garis bentuk fisika. Anda dapat memilih beberapa titik kontrol sekaligus dengan mengklik dan menyeret kursor di daerah untuk memilih semua titik kontrol di daerah itu. Anda kemudian dapat kembali posisi atau remove semua titik kontrol yang dipilih.

Menambahkan / memindahkan titik kontrol

Untuk menambahkan titik kontrol, letakkan kursor di tepi garis besar. Pratinjau titik kontrol muncul di sepanjang garis besar. Klik untuk menambahkan titik kontrol baru di lokasi tersebut. Untuk menghapus titik kontrol, pilih dan tekan Del/Command+Del keys.

Tepi bergerak

Untuk memilih tepi bentuk fisika bukan titik kontrolnya, tahan kunci Control/Ctrl. Klik dan seret tepi yang disorot untuk memindahkannya ke posisi baru untuk mempengaruhi bentuk fisika.
