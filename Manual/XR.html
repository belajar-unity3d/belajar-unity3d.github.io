XR

Pengembangan XR baru? Sitemap

XR adalah istilah payung yang mencakup jenis aplikasi berikut:

  • Realitas Virtual (VR): Aplikasi ini mensimulasikan lingkungan yang sama sekali berbeda di sekitar pengguna.
  • Realitas campuran (MR): Aplikasi ini menggabungkan lingkungannya sendiri dengan lingkungan dunia nyata pengguna dan memungkinkan mereka untuk berinteraksi satu sama lain.
  • Augmented Reality (AR): Lapisan aplikasi konten melalui tampilan digital dunia nyata.

Platform yang didukung

Bersatu bekerja sama dengan mitra strategis untuk memastikan bahwa para pembuat memiliki semua dukungan yang mereka butuhkan ketika mengembangkan konten untuk platform yang didukung Unity. Ini melibatkan integrasi platform dalam, peningkatan mesin Unity, dan optimasi tumpukan teknologi XR untuk setiap platform.

Unity mendukung platform ini untuk XR:

  • ARKit
  • ARCore
  • Mozilla Firefox
  • Windows Server
  • Magic Leap
  • Oculus
  • OpenXR
  • Mozilla Firefox

Unity tidak mendukung XR pada WebGLLogin API yang membuat grafis 2D dan 3D di browser web. Opsi build Unity WebGL memungkinkan Unity untuk mempublikasikan konten sebagai program JavaScript yang menggunakan teknologi HTML5 dan WebGL rendering API untuk menjalankan konten Unity di browser web. Unity XR kerangka plug-in

Unity telah mengembangkan kerangka plug-inSatu set kode yang dibuat di luar Unity yang menciptakan fungsi dalam Unity. Ada dua jenis plug-ins yang dapat Anda gunakan di Unity: Managed plug-ins (diproduksi. Rakitan NET dibuat dengan alat-alat seperti Studio Visual) dan plug-ins asli (pustaka kode asli yang spesifik platform). More info
Lihat di Glossary baru yang disebut XR SDK yang memungkinkan penyedia XR untuk mengintegrasikan dengan mesin Unity dan memanfaatkan fitur-fiturnya. Pendekatan berbasis plug-in ini meningkatkan kemampuan Unity untuk membuat perbaikan bug cepat, mendistribusikan pembaruan SDK dari mitra platform, dan untuk mendukung perangkat XR baru dan runtimes tanpa harus memodifikasi mesin inti. Untuk mempelajari lebih lanjut tentang:

  • Cara meningkatkan framework baru, lihat halaman Mengkonfigurasi Proyek Unity Anda untuk XR.
  • Bagaimana fungsi framework baru, lihat halaman XR plug-in kerangka.
