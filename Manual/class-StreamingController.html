Jika Kamera dinonaktifkan, Unity tidak menghitung Mipmap Streaming untuk itu, kecuali Pengontrol Streaming diaktifkan dan dalam keadaan preloading. Property Description
Mip Map Bias Gunakan pengaturan ini untuk memaksa Unity untuk memuat tingkat mipmap yang lebih kecil atau lebih besar daripada sistem Mipmap Streaming telah secara otomatis dipilih. Masukkan nilai di bidang ini untuk mengatur offset yang Unity berlaku untuk tingkat mipmap ini. Unity menambahkan offset ini ke semua mipmap yang ditugaskan untuk Tekstur yang berada dalam tampilan Kamera.

Misalnya, jika Unity memuat tekstur di mip 1 dan Mip Map Bias diatur ke 1, ini berarti akan memuat mip 2 sebagai gantinya.

Untuk mengontrol pengaturan ini menggunakan API, gunakan StreamingController.streamingMipmapBias.
