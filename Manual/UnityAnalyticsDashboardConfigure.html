Configure page

Lihat dan edit pengaturan untuk proyek individual Unity Analytics di halaman Konfigurasi. Cari halaman Konfigurasi di bawah Settings > Analytics Settings di dashboard.

Halaman Konfigurasi berisi bagian ini:

  • Pengaturan Layanan Proyek
  • Pengaturan Fitur
  • Pengaturan Admin

Pengaturan Layanan Proyek

Pengaturan Layanan Proyek aspek kontrol proyek untuk layanan AnalyticsUnity Analytics
Lihat di Glossary. Seperti dicatat di tabel di bawah ini, beberapa pengaturan juga mempengaruhi layanan Ads dan IAP.

Name Nama proyek. Tampilkan di Dasbor Layanan Unity dan jendela Layanan Editor Unity.
Project ID Pengidentifikasi unik untuk proyek. Digunakan untuk mengidentifikasi proyek antara Layanan Unity dan Editor.
Engine Version Tentukan Editor Unity dan versi Engine dalam penggunaan. Nilai yang ditetapkan di sini mengubah informasi yang ditampilkan di halaman Integrasi.
Google License Key Login Kunci lisensi Google jika Anda ingin memvalidasi pembelian dalam aplikasi (IAP) yang dibuat melalui toko Google Play.
Organization Nama organisasi Unity Anda.
COPPA Compliance Tentukan apakah proyek Anda diarahkan pada anak-anak dan jatuh di bawah domain Undang-Undang Perlindungan Privasi Online Anak (COPPA).

Ketika Anda menentukan bahwa aplikasi Anda tunduk pada COPPA( Undang-Undang Perlindungan Privasi Online Anak) COPPA adalah hukum AS yang berlaku untuk aplikasi yang mengumpulkan informasi pribadi dan ditargetkan kepada anak-anak di bawah usia 14. More info
Lihat di Glossary, layanan Unity Ads mengubah cara itu melayani iklan kepada pengguna sesuai dengan aturan. Data yang dikumpulkan oleh layanan Analytics tetap sama.

Ini adalah tanggung jawab Anda untuk memastikan bahwa aplikasi Anda dan penggunaan data Analytics Anda sesuai dengan hukum yang berlaku (Unity tidak dapat memberikan saran hukum pada masalah ini.) Lihat Kepatuhan COPPA untuk informasi lebih lanjut.

Pengaturan Fitur

Bagian pengaturan fitur menyediakan informasi dan pengaturan untuk fitur terkait Analytics tertentu.

Project Secret Key Kunci rahasia proyek memberikan akses ke data Analytics Anda. Gunakan kunci rahasia proyek dengan API REST Ekspor Data Baku, serta dukungan Pengaturan Jarak Jauh dan Heatmaps di Editor Unity.

Perhatikan bahwa Anda harus menjaga rahasia Project Secret Key. Siapa pun yang memiliki nilai ini dapat berpotensi mengakses data Analytics Anda.

Juga dicatat bahwa Project Secret Key sebelumnya dikenal sebagai Raw Data Export API Key. Nilai utama adalah sama; gunakan Project Secret Key di tempat mana Anda sebelumnya menggunakan Raw Data Export API Key.
HeatmapsHeatmaps adalah visualisasi spasial data analitik. More info

Pengaturan Admin

Bagian Pengaturan Admin mengandung kontrol yang memungkinkan Anda menghapus data Analytics yang ada atau disable Analytics altogether.

Data Analisis Ulang

Mengatur ulang data Analytics Anda secara permanen menghapus semua data diproses yang ada sehingga tidak lagi muncul dalam laporan Dashboard, segmen atau corong. Namun, data mentah Anda tidak dihapus dan Anda masih dapat mengunduhnya menggunakan Ekspor Data Baku.

Data diproses dihapus pada waktu yang tercantum di halaman Konfigurasi, di mana titik pengumpulan data Analytics, pemrosesan, dan agregasi dimulai baru.

Klik tombol Begin Process dan ikuti petunjuk untuk menyelesaikan ulang data.
