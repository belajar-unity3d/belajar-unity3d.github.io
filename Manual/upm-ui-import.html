Mengimpor paket Aset Store

Anda dapat mengimpor paket Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
Lihat di Glossary di jendela Paket Manager:

  1. Buka jendela Paket Manager dan pilih My Assets dari menu PaketPaket adalah koleksi aset untuk dibagikan dan digunakan kembali dalam Unity. Manajer Paket Unity (UPM) dapat menampilkan, menambahkan, dan menghapus paket dari proyek Anda. Paket-paket ini berasal dari Unity Package Manager dan memberikan metode mendasar untuk memberikan fungsi Unity. Namun, Unity Package Manager juga dapat menampilkan Paket Asset Store yang Anda unduh dari Asset Store. More info
    Lihat di Glossary    .

    Tampilan daftar menampilkan paket Asset Store yang telah Anda beli.

  2. Jika banyak paket Asset Store muncul dalam daftar My Assets, Anda dapat Cari paket Aset tertentu dengan nama atau nomor versi.

  3. Pilih paket Asset Store yang ingin Anda impor dari Daftar paket. Informasi paket Asset Store muncul di details view.

    Jika Anda belum mengunduh paket Asset Store ke komputer atau perangkat ini sebelum, tombol Download muncul. Jika tidak, tombol Import muncul dan Anda dapat melewatkan langkah 6 untuk mengimpor paket Asset Store segera.

  4. Klik tombol Download untuk mengunduh paket Asset Store jika muncul dalam tampilan detail.

    Sementara paket Asset Store mengunduh ke komputer Anda, bar kemajuan muncul dan tombol Cancel dan Pause menggantikan tombol Download yang Anda klik.

    Tombol ini memberi Anda kendali atas kemajuan unduhan:

    • Klik tombol Cancel untuk menghentikan unduhan sepenuhnya.
    • Klik tombol Pause untuk menghentikan mengunduh. Anda dapat mengklik tombol Resume setiap kali Anda ingin melanjutkan.

    Ketika selesai download, Manajer Paket mengganti Download dengan Re-Download, dan tombol Import muncul di sebelah Re-Download.

  5. Klik tombol Import untuk mengimpor paket Aset Store yang dipilih.

    Jika paket Asset Store adalah proyek lengkap, dialog konfirmasi muncul. SitemapMengimpor proyek lengkap dari Toko Aset.

    Untuk semua paket Aset Store lainnya, jendela Paket Unity Import menampilkan semua item yang sudah dipilih dalam paket, siap dipasang.

  6. Jelas setiap item yang Anda tidak ingin mengimpor dan klik Import.

    The Package Manager menempatkan isi paket Asset Store yang diimpor ke folder Assets, sehingga Anda dapat mengaksesnya dari project windowJendela yang menunjukkan isi folder Assets (Project tab) More info
    Lihat di Glossary     Anda.

Note: Anda dapat mengunduh beberapa paket dengan satu klik dengan menggunakan beberapa fitur pilih. Untuk informasi lebih lanjut, lihat Melakukan tindakan pada beberapa paket atau set fitur.

Mengimpor proyek lengkap dari Toko Aset

Proyek lengkap berisi aset dan project settingsPengumpulan yang luas dari pengaturan yang memungkinkan Anda untuk mengkonfigurasi bagaimana Fisika, Audio, Jaringan, Grafik, Input dan banyak bidang lain dari jangkauan proyek Anda. More info
Lihat di Glossary. Proses untuk mengimpor proyek lengkap mirip dengan paket Aset Store lainnya, kecuali untuk langkah tambahan untuk pengaturan proyek.

Pengaturan proyek dalam proyek lengkap mungkin bertentangan dengan pengaturan dalam proyek Anda, sehingga ketika Anda mengimpor proyek lengkap dari Asset Store, Paket Manager menawarkan pilihan.

Pilih Import jika Anda yakin bahwa Anda ingin menambahkan aset paket ke proyek Anda. Kotak dialog Paket Unity Import muncul, seperti yang digambar di bawah ini. Jelaskan konten aset apa pun yang Anda tidak ingin mengimpor dan klik Next. Langkah kedua dari kotak dialog daftar pengaturan proyek paket. Jika Anda memiliki pengaturan proyek di proyek Anda, Manajer Paket mengidentifikasi mereka dengan bendera Override; mereka akan ditimpakan dengan pengaturan proyek dari paket kecuali Anda menghapus pilihan. Hapus pengaturan proyek apa pun yang Anda tidak ingin mengimpor dan klik Import.

Pilih Switch Project jika Anda ingin menjelajahi aset paket dan pengaturan proyek di lingkungan yang aman. Memilih Switch Project menciptakan proyek sementara baru yang hanya mengandung aset paket dan pengaturan proyek. Setelah menjelajahi proyek sementara, Anda dapat menutupnya tanpa menyimpan. Jika Anda ingin terus menggunakan paket di proyek utama Anda, kembali ke proyek utama Anda dan mengimpor paket lagi. Pilih Import pada kotak dialog peringatan yang muncul.
