Lukisan di Tilemaps

Rincian berikut langkah dan alat untuk lukisan Tiles pada Tilemap. Namun, jika Anda melukis Isometric Tilemap, merujuk ke Membuat palet ubin untuk Isometric Tilemap untuk langkah dan informasi tertentu.

Untuk memulai lukisan ubin, pilih Tilemap Anda ingin melukis dari menu drop-down Active Tilemap di jendela Tilemap Palette. Tilemaps dibuat dalam hierarki SceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary secara otomatis ditambahkan ke menu drop-down. Hanya Active Tilemap yang dipilih saat ini dipengaruhi oleh alat lukisan Tile Palette.

Alat lukisan Tilemap ditemukan di baris di bagian atas jendela Tile Palette. Alat-alat ini juga dapat digunakan untuk mengedit Palet keramik itu sendiri.

Pilih Tool

Gunakan Select Tool untuk memilih ubin di Active Tilemap atau tahan dan seret lebih banyak ubin untuk memilih lebih banyak sekaligus. Rincian properti dari ubin yang dipilih ditunjukkan di jendela InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info
Lihat di Glossary. Lihat halaman Pilih Alat untuk informasi lebih lanjut tentang dan contoh pilihan yang berbeda tersedia.

Setelah memilih ubin dengan alat pilih, Anda dapat menggunakan Move Tool untuk memindahkan ubin yang dipilih ke posisi yang berbeda pada ubinmap.

Alat Pindah

Gunakan Move Tool untuk memindahkan pilihan Select Tool's Tile (The Move Tool tidak dapat memilih Tiles sendiri). Tahan dan seret ubin yang dipilih ke posisi lain di Tilemap.

Cat brush Tool

Gunakan Paintbrush Tool untuk memilih ubin di palet ubin, atau tahan dan tarik lebih dari beberapa ubin untuk memilih lebih banyak sekaligus. Kemudian klik di lokasi apa pun di Tilemap di Adegani untuk memulai melukis dengan ubin yang dipilih Anda. Tahan Pergeseran saat menggunakan Paintbrush Tool untuk beralih ke Erase Tool.

Untuk memilih dan melukis dengan Ubin yang sudah ada di Tilemap bukan Tilemap Palette, tahan Ctrl saat Anda memilih Ubin di Tilemap dengan Paintbrush Tool. Ini secara singkat beralih ke Picker Tool untuk memilih ubin untuk melukis.

Box Isi Alat

Gunakan Box Fill Tool untuk memilih ubin di palet ubin, atau tahan dan tarik lebih dari beberapa ubin untuk memilih lebih banyak sekaligus. Pemilihan yang dibuat dengan alat lukisan lainnya juga valid. Tahan dan seret Box Fill Tool atas Tilemap untuk menggambar bentuk persegi panjang yang mengisi dengan ubin yang dipilih.)

Alat Picker

Gunakan Picker Tool untuk memilih ubin dari palet Tilemap atau Tilemap; tahan dan seret lebih banyak ubin untuk memilih lebih banyak sekaligus. Alat aktif beralih ke Paintbrush Tool setelah seleksi dibuat, untuk memungkinkan Anda untuk melukis segera di Tilemap dengan ubin yang dipilih.

Alat Pembersih

Gunakan Eraser Tool untuk menghapus ubin dari ubinmap dengan memilih mereka sementara alat diaktifkan. Untuk menghapus area yang lebih besar sekaligus, tahan dan seret Alat Eraser ke ukuran yang diinginkan di Palet Tilemap, kemudian cat di atas Ubin yang ingin Anda hapus di Tilemap.

Alat Isi Banjir

Flood Fill Tool digunakan untuk mengisi area sel kosong atau ubin identik dengan ubin yang dipilih. Pilih Tile untuk digunakan sebagai Isi Banjir dengan memilih dari palet Tilemap. Alat ini tidak dapat digunakan dengan beberapa ubin.

Mode Fokus Tilemap

Gunakan mode Fokus Tilemap untuk fokus pada Tilemap atau Grid GameObjectObjek mendasar dalam adegan Unity, yang dapat mewakili karakter, props, pemandangan, kamera, waypoints, dan banyak lagi. Fungsi GameObject didefinisikan oleh Komponen yang melekat padanya. More info
Lihat di Glossary tertentu, dan memudar GameObjects lain di Adegan. Ini berguna ketika bekerja dengan lapisan mutilple Tilemap untuk menghindari kebingungan dan kekacauan.

Pertama pilih target Tilemap dari dropdown Active Target di jendela Tilemap Palette atau dari jendela Hierarchy. Fokus Tilmap Pada menu dropdown ditemukan di sebelah kanan bawah Scene ViewTampilan interaktif ke dunia yang Anda buat. Anda menggunakan Adegan Lihat untuk memilih dan posisi pemandangan, karakter, kamera, lampu, dan semua jenis lain dari Game Object. More info

Inspektur Sikat Aktif

Inspektur Brush Aktif berada di bagian bawah jendela Palet keramik. Ini memungkinkan Anda untuk mengubah Sikat aktif saat ini dan sifatnya. Ini diminimalkan secara default. Untuk memperluasnya, seret bagian bawah toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
Lihat di Glossary ke atas.

Sikat pratinjau dan pintas lainnya

Ketika melukis Tilemap di Tampilan Adegan dengan Sikat aktif, pratinjau dari ubin yang dipilih (s) ditunjukkan di lokasi kursor. Ini adalah Brush preview yang membantu Anda untuk memvisualisasikan penempatan ubin sebelum melukis. Sikat pratinjau dengan beberapa ubin mungkin terlihat berbeda ketika kursor hovers atas baris atau kolom tetangga. Ini lebih jelas pada Hexagonal Tilemaps.

Potongan-potongan berikut memungkinkan Anda untuk mengubah orientasi atau z-posisi ubin yang dipilih, yang secara visual tercermin dalam preview sikat.
