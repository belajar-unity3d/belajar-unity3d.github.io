Menambahkan filter

Secara default, Preset berlaku untuk semua komponen atau importir aset dari jenis Preset setelah dibuat. Jika Anda ingin hanya menerapkannya ke komponen atau jenis aset tertentu, Anda dapat menggunakan field Filter untuk menentukan kapan untuk menerapkan Preset.

Filter memungkinkan Anda untuk membuat beberapa Presets untuk tipe tertentu, dan menerapkannya ke komponen atau importir tertentu berdasarkan apakah komponen dan importir memenuhi kriteria filter. Anda dapat menyaring dengan nama file, direktori, dan ekstensi file.

Anda juga dapat menerapkan Preset parsial yang mempengaruhi hanya properti yang Anda tentukan dalam komponen atau aset Anda.

Sitemap Preset default dalam Preset Jenis diterapkan dalam urutan dari atas ke bawah. Ini berarti bahwa filter di bagian bawah daftar dapat override filter sebelumnya jika tidak secara logis dipesan.Note: Default Presets in a Preset Type are applied in order from top to bottom. This means that filters at the bottom of the list can override previous filters if not logically ordered.

  1. Buka Preset Manager dengan memilih Edit > Project Settings, lalu pilih kategori Preset Manager.
  2. Cari default Preset yang ingin Anda filter, dan masukkan string pencarian di bidang Filter.

Pencarian filter canggih

Presets dapat menggunakan pencarian glob, yang merupakan cara untuk menggunakan simbol dalam filter Anda untuk memungkinkan filter impor yang lebih inklusif. Mengedit daftar Presets default

Bagian ini menjelaskan cara beralih, bergerak dan menghapus Presets dalam daftar Preset default Preset Manager untuk setiap tipe Preset.

  • Sitemap Pilih pegangan (Reorder a Preset list:) dari Preset default yang ingin Anda pindahkan, dan tarik ke posisi baru.=) of the default Preset that you want to move, and drag it to the new position.
  • Sitemap Pilih pegangan (Remove a Preset:) dari Preset default yang ingin Anda hapus, lalu klik tombol Hapus (=) untuk menghapus Preset yang dipilih.-) button to remove the selected Preset.
  • Sitemap Hapus semua Presets ditugaskan untuk jenis Preset default.Remove a Preset type: Remove all Presets assigned to that default Preset type.

Mengaktifkan dan mengaktifkan Presets default

Anda dapat sementara menonaktifkan / menonaktifkan Preset default tanpa kehilangan pengaturan filter.

  1. Buka Preset Manager dengan memilih Edit > Project Settings, lalu pilih kategori Preset Manager.

  2. Jelaskan kotak cek ke kiri Preset default untuk menonaktifkan Preset. Pilih kotak cek untuk mengaktifkan.
