Penggunaan pencarian

Untuk menggunakan Pencarian, lakukan berikut:

  1. Luncurkan Pencarian
  2. Search
  3. Melakukan tindakan pada item dalam hasil

Luncurkan Pencarian

Ada beberapa cara untuk membuka jendela Pencarian:

  • Dari menu Unity, pilih Edit > Search All.
  • Klik ikon kaca pembesar di kanan atas di jendela Editor.

Jaga pembukaan jendela Pencarian

Untuk menjaga jendela terbuka setelah memilih item pencarian, pilih Keep Open dari menu More Options (:).

Perform searches

Untuk melakukan pencarian, ketik pertanyaan ke bidang pencarian.

Untuk sebagian besar pertanyaan pencarian, gunakan jendela Search All dan indeks Aset default yang dibuat dengan proyek Anda akan menemukan konten Anda secara efektif.

Untuk pencarian yang lebih halus, gunakan Penyedia Pencarian spesifik dan Search Expressions.

Untuk menyertakan paket dalam pencarian Anda, pilih Show package results dari menu More Options (:).

Lihat hasil pencarian

Gunakan siklus Alt + ↑ (up panah) dan ↓ (down panah) melalui riwayat pencarian, atau memilih pencarian yang disimpan dari pane kiri.

Mengubah tampilan hasil pencarian

Gunakan slider sizing untuk mengubah dari daftar ke tampilan grid dengan thumbnails, atau gunakan ikon tampilan Daftar, ikon tampilan Grid, atau ikon tampilan Meja di bagian bawah kanan jendela Search untuk mengubah tampilan pencarian.

Tambahkan item hasil pencarian favorit

Klik bintang di sebelah item hasil pencarian untuk menandai item yang muncul di bagian atas daftar pencarian waktu berikutnya Anda melakukan pencarian yang sama untuk menemukannya lebih mudah.

Status pencarian

Untuk melihat waktu pencarian dan jumlah hasil di bagian bawah jendela pencarian, pilih Show Status dari menu More Options (:).

Simpan pencarian

Menyimpan pencarian memungkinkan Anda untuk menjaga pencarian yang berguna untuk digunakan kembali sesuai kebutuhan. Simpan pencarian ke Unity pribadi Anda untuk penggunaan Anda sendiri (User), atau untuk proyek untuk menyimpannya sebagai Aset untuk membuat pencarian yang tersedia bagi siapa pun yang bekerja di proyek.

Untuk menyimpan pencarian ke versi pribadi Anda Unity:

  1. Buat pencarian Anda.
  2. Klik menu drop-down di sebelah ikon Simpan di kanan atas jendela Pencarian dan pilih Save User. Login Pencarian Anda ditambahkan ke panel
    di jendela Saved Searches.Search window.
  3. Untuk menyimpan kembali setelah membuat perubahan, menyoroti pencarian di jendela Saved Searches dan dari Save drop-down menu di kanan atas jendela Pencarian, pilih Save <search name>

Anda juga dapat mengklik ikon Simpan di sebelah Pengguna di panel Saved Searches.

Untuk menyimpan pencarian proyek:

  1. Buat pencarian Anda.
  2. Klik menu drop-down di sebelah ikon Simpan di kanan atas jendela Search dan pilih Save Project.
  3. Masukkan nama file di jendela Save search query, dan pilih Save. Login Pencarian Anda ditambahkan ke panel
    di jendela Pencarian.Saved Searches panel in the Search window.
  4. Untuk menyimpan kembali setelah membuat perubahan, menyoroti pencarian di jendela Saved Searches dan dari Save drop-down menu di kanan atas jendela Pencarian, pilih Save <search name>

Anda juga dapat mengklik ikon Simpan di sebelah Proyek di panel Saved Searches.

Cari pencarian yang tersimpan

Untuk menemukan pencarian yang tersimpan:

  1. Klik ikon kaca pembesaran di panel Saved Searches.
  2. Masukkan nama pencarian yang tersimpan, atau bagian dari nama. Jika ditemukan, itu akan muncul di panel.

Actions

Setelah Anda mencari, Anda dapat melakukan tindakan pada item Cari kembali di pane Preview InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info
Lihat di Glossary atau melalui More Options (⋮) . Tindakan yang dapat Anda lakukan tergantung pada jenis item.

Misalnya, jika Cari kembali paket, Anda dapat menginstal / menginstalnya. Jika Pencarian mengembalikan Aset, Anda dapat memilih, membuka, atau menyorotinya di jendela Hierarchy.

  • Setiap jenis item memiliki tindakan default.
  • Beberapa item mendukung tindakan tambahan melalui menu konteks.
  • Beberapa item juga mendukung drag dan drop tindakan.

Untuk mengetahui tindakan yang dapat Anda lakukan pada berbagai jenis item, lihat halaman Penyedia Pencarian untuk filter pencarian individu.

Tindakan default

Setiap jenis item memiliki aksi default.

Untuk melakukan tindakan default untuk item melakukan salah satu berikut:

  • Klik dua kali item.
  • Pilih item dan gunakan Enter.

Sitemap Anda dapat mengedit tindakan default di bagian Note: dari halaman Search.Preferences page.

Tindakan tambahan

Beberapa item mendukung tindakan tambahan yang Anda akses dari menu Preview Inspector.

Untuk mengakses menu konteks tindakan tambahan untuk item, lakukan salah satu berikut:

  • Klik kanan item.
  • Pada entri item, pilih More Options (⋮).

Anda juga dapat menggunakan shortcut berikut untuk melakukan tindakan tambahan pada item yang dipilih tanpa membuka menu kontekstual:

Shortcut: Fungsi:
Alt + Enter Tindakan kedua
Alt + Ctrl + Enter Tindakan ketiga
Alt + Ctrl + Shift + Enter Tindakan keempat
