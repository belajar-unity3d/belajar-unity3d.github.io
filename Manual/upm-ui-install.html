Menginstal dari registry

Note: Ketika Anda menginstal paket melalui jendela Manajer Paket, Manajer Paket mengevaluasi paket lain dan ketergantungan mereka untuk melihat apakah ada konflik versi dengan versi yang Anda pilih. Jika mendeteksi konflik atau pembatasan, menginstal versi apa pun menyelesaikan masalah ini. Untuk informasi lebih lanjut, lihat Ketergantungan dan resolusi.

Gunakan proses yang sama untuk menginstal paket dari registry Unity atau setiap registry lingkup yang didefinisikan dalam proyek Anda. Satu-satunya perbedaan adalah apa yang Anda pilih dari menu drop-down Packages:

  1. Buka jendela Paket Manager dan pilih Unity Registry dari menu drop-down PaketPaket adalah koleksi aset untuk dibagikan dan digunakan kembali dalam Unity. Manajer Paket Unity (UPM) dapat menampilkan, menambahkan, dan menghapus paket dari proyek Anda. Paket-paket ini berasal dari Unity Package Manager dan memberikan metode mendasar untuk memberikan fungsi Unity. Namun, Unity Package Manager juga dapat menampilkan Paket Asset Store yang Anda unduh dari Asset Store. More info
    Lihat di Glossary    . Jika Anda ingin menginstal paket dari registry lingkup, pilih My Registries.

    Note: Jika Anda belum mendefinisikan registries lingkup untuk proyek ini, My Registries tidak muncul di menu drop-down.

  2. Pilih paket yang ingin Anda instal dari Daftar paket. Informasi paket muncul di details view.

    Sitemap Jika Anda tidak melihat paket yang ingin Anda instal dalam daftar paket, itu mungkin paket yang tidak dapat dipulihkan, seperti paket Note: dan experimental. Manajer Paket tidak menampilkan paket eksperimental kecuali mereka sudah diinstal, tetapi paket pra-release muncul di Manajer Paket ketika Anda mengaktifkan pengaturan proyek pre-release.Paket Pre-release yang Dapat Diaktifkan project setting.

  3. Pilih versi untuk menginstal. Untuk informasi lebih lanjut tentang versi yang tersedia, lihat Mencari versi tertentu.

  4. Klik tombol Install.

Ketika selesai bar kemajuan, paket baru siap digunakan.

Note: Anda dapat menginstal beberapa paket dengan satu klik dengan menggunakan beberapa fitur pilih. Untuk informasi lebih lanjut, lihat Melakukan tindakan pada beberapa paket atau set fitur.
