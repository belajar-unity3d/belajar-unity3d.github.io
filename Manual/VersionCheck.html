Cek Untuk Pembaruan

Cek Unity apakah pembaruan tersedia. Pemeriksaan ini terjadi baik ketika Unity dimulai, atau ketika Anda memilih Bantuan->Check untuk Update item menu. Pemeriksaan pembaruan mengirimkan nomor revisi Unity saat ini (jumlah lima digit yang muncul dalam kurung setelah nama versi di dialog Tentang Unity) ke server pembaruan di mana dibandingkan dengan versi yang paling terbaru. Jika versi Unity yang lebih baru tersedia dialog berikut ditampilkan:

Jika versi dalam penggunaan adalah yang paling up-to-date maka dialog berikut ditunjukkan:

Klik tombol Download versi baru yang akan diambil ke situs web di mana Anda dapat mengunduh versi baru.

Memperbarui Periksa Frekuensi

Tanggapan dari server juga berisi interval waktu yang menyarankan ketika pemeriksaan pembaruan berikutnya harus dilakukan. Ini memungkinkan pemeriksaan pembaruan untuk dilakukan kurang sering ketika Unity tidak mengharapkan pembaruan yang tersedia.

Le Meridien

Jika Anda berada di tengah proyek Anda mungkin tidak ingin memperbarui ke versi baru Unity. Ticking tombol versi ini di dialog Unity Editor Update Check akan mencegah Unity dari memberitahu Anda tentang pembaruan ini.

Mengaktifkan Pemeriksaan Pembaruan

Tidak mungkin untuk menonaktifkan cek untuk pembaruan. Cek Untuk Memperbarui kotak centang pada kontrol dialog apakah Anda diberitahukan pembaruan (jika mereka tersedia) ketika Awal Unity. Bahkan jika Anda telah menunticked Check for Updates pilihan Anda masih dapat memeriksa pembaruan dengan menggunakan Help->Check untuk Update item menu.
