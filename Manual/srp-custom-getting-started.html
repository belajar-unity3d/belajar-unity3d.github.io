Membuat proyek baru dan menginstal paket yang diperlukan untuk pipa render kustom

Instruksi ini menunjukkan cara membuat pipa render kustom menggunakan paket SRP Core. Login Inti adalah paket yang dibuat oleh Unity yang berisi kode yang dapat digunakan kembali untuk membantu Anda membuat pipa render Anda sendiri, termasuk kode boilerplate untuk bekerja dengan API grafis spesifik platform, fungsi utilitas untuk operasi rendering umum, dan perpustakaan shaderProgram yang berjalan di GPU. More info
Lihat di Glossary yang digunakan URP dan HDRP. Untuk informasi lebih lanjut tentang SRP Core, lihat .

  1. Buat Proyek Unity baru.
  2. Gunakan Git untuk membuat klon dari SRP kode sumber repositori. Anda dapat menempatkan kode sumber SRP di lokasi apa pun di disk Anda, selama tidak dalam salah satu Login Sub-folder proyek.
  3. Gunakan Git untuk memperbarui salinan kode sumber SRP ke cabang yang kompatibel dengan versi Editor Unity Anda. Baca Menggunakan versi terbaru dalam dokumentasi repositori SRP untuk informasi di cabang dan versi.
  4. Buka Proyek Anda di Unity, dan instal paket berikut dari folder kode sumber SRP di disk Anda, dalam urutan berikut. Untuk informasi tentang menginstal paket dari disk, lihat Menginstal paket dari folder lokal.
    • com.unity.render-pipelines.core.
    • Opsional: com.unity.render-pipelines.shadergraph. Instal paket ini jika Anda ingin menggunakan Shader Graph atau memodifikasi kode sumber Shader Graph sebagai bagian dari SRP kustom Anda.
    • Opsional: com.unity.render-pipelines.visual. Instal paket ini jika Anda ingin menggunakan Visual Effect Graph atau memodifikasi kode sumber Visual Effect Graph sebagai bagian dari SRP kustom Anda.

Membuat versi kustom dari URP atau HDRP

Pipa Render Universal (URP) dan Pipa Render Definisi Tinggi (HDRP) menawarkan opsi kustomisasi yang luas untuk membantu Anda mencapai grafik dan kinerja yang Anda butuhkan. Namun, jika Anda ingin lebih banyak kontrol, Anda dapat membuat versi kustom dari salah satu pipa render ini, dan memodifikasi kode sumber.

Ikuti langkah 1–3 di bagian di atas, Creating a new Project and installing the packages needed for a custom SRP. Ketika Anda mencapai langkah 4, pasang paket berikut dalam urutan berikut:

URP:

  • com.unity.render-pipelines.core
  • com.unity.render-pipelines.shadergraph
  • com.unity.render-pipelines.universal

HDRP:

  • com.unity.render-pipelines.core
  • com.unity.render-pipelines.shadergraph
  • com.unity.render-pipelines.defintion tinggi
