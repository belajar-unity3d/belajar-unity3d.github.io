Pemetaan Avatar menunjukkan bahwa tulang diperlukan (lingkar padat) dan yang opsional ( lingkaran yang tertidur). Unity dapat mengganggu gerakan tulang opsional secara otomatis.

Menyimpan dan menggunakan ulang data Avatar (file Template Manusia)

Anda dapat menyimpan pemetaan tulang di kerangka Anda ke Avatar pada disk sebagai Human Template file (extention *.ht). Anda dapat menggunakan pemetaan ini untuk karakter apa pun. Misalnya, Anda ingin menempatkan pemetaan Avatar di bawah kendali sumber dan Anda lebih suka melakukan file berbasis teks; atau mungkin Anda ingin membuat file dengan alat kustom Anda sendiri.

Untuk menyimpan data Avatar dalam file Human TemplateTulang yang telah ditentukan sebelumnya. Digunakan untuk mencocokkan tulang dari file FBX ke Avatar. More info
Lihat di Glossary, pilih Save dari menu drop-down Mapping di bagian bawah jendela Avatar.

Unity menampilkan kotak dialog untuk Anda memilih nama dan lokasi file untuk menyimpan.

Untuk memuat file Template Manusia yang sebelumnya diciptakan, pilih Mapping > Load dan pilih file yang ingin Anda beban.

Menggunakan Masker Avatar

Kadang-kadang berguna untuk membatasi animasi ke bagian tubuh tertentu. Sebagai contoh, animasi berjalan mungkin melibatkan karakter yang menggiurkan lengan mereka, tetapi jika mereka mengambil obor, mereka harus memegangnya ke lampu cor. Anda dapat menggunakan Avatar Body Mask untuk menentukan bagian mana dari sebuah karakter animasi harus dibatasi.
