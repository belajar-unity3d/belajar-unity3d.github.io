Sprite Editor

Kadang-kadang Tekstur Sprite hanya mengandung elemen grafis tunggal tetapi seringkali lebih nyaman untuk menggabungkan beberapa grafis terkait bersama menjadi gambar tunggal. Misalnya, gambar dapat mengandung bagian komponen dari satu karakter, seperti dengan mobil yang bergerak secara independen dari tubuh. Unity memudahkan untuk mengekstrak elemen dari gambar komposit dengan menyediakan Sprite Editor untuk tujuan.

Note: Pastikan grafik yang ingin Anda edit memiliki set Texture Type untuk Sprite (2D and UI). Lihat dokumentasi SpritesObjek grafis 2D. Jika Anda digunakan untuk bekerja di 3D, Sprites pada dasarnya hanya tekstur standar tetapi ada teknik khusus untuk menggabungkan dan mengelola tekstur sprite untuk efisiensi dan kenyamanan selama perkembangan. More info
Lihat di Glossary untuk informasi tentang impor dan pengaturan Sprites.

Tekstur Sprite dengan beberapa elemen membutuhkan Sprite Mode untuk diatur ke Multiple di InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info

Membuka Editor Sprite

Untuk membuka Sprite Editor:

  1. Pilih gambar 2D yang ingin Anda edit dari Project View (Gambar 1: Tampilan Proyek).

    Perhatikan bahwa Anda tidak dapat mengedit Sprite dengan memilih di SceneA Scene contains the environments and menus of your game. Think of each unique Scene file as a unique level. In each Scene, you place your environments, obstacles, and decorations, essentially designing and building your game in pieces. More info
    See in Glossary     View    .

  2. Klik pada tombol Sprite Editor di Texture Import InspectorAn Inspector that allows you to define how your images are imported from your project’s Assets folder into the Unity Editor. More info
    See in Glossary     (Gambar 2: Tekstur Impor Inspektur) dan tampilan Sprite Editor (Gambar 3: Editor Sprite).

Sitemap Anda hanya dapat melihat tombol Note: jika Sprite Editor pada gambar yang telah Anda pilih diatur ke Texture Type.Sprite (2D and UI).

Sitemap Mengatur Note: ke Sprite Mode dalam Multiple jika gambar Anda memiliki beberapa elemen.Texture Import Inspector if your image has several elements.

Seiring dengan gambar komposit, Anda akan melihat sejumlah kontrol di bar di bagian atas jendela. Slider di kanan atas mengontrol zoom, sementara tombol bar warna ke kiri memilih apakah Anda melihat gambar itu sendiri atau tingkat alfa. Slider paling tepat mengontrol perpipaan (mipmap) dari Tekstur. Pindahkan slider ke kiri mengurangi resolusi Tekstur Sprite. Kontrol yang paling penting adalah menu Slice di kiri atas, yang memberi Anda pilihan untuk memisahkan elemen gambar secara otomatis.

Pilih tombol Apply dan Revert untuk menjaga atau membuang setiap perubahan yang Anda buat di jendela Sprite Editor masing-masing. Anda mengatur tombol ini untuk menunjukkan kotak dialog konfirmasi sebelum menerapkan efek mereka di jendela Unity Preferences (menu: Edit > Preferences…). Di jendela Preferensi, pergi ke 2D > Sprite Editor Window untuk membuka opsi jendela Sprite Editor.

Pilih opsi masing-masing untuk memiliki kotak dialog muncul setelah memilih Apply atau Revert untuk mengkonfirmasi pilihan Anda, mencegah penghematan kecelakaan atau membuang perubahan yang dilakukan.

Menggunakan Editor

Cara paling langsung untuk menggunakan editor adalah untuk mengidentifikasi elemen secara manual. Jika Anda mengklik gambar, Anda akan melihat area seleksi persegi panjang muncul dengan pegangan di sudut. Anda dapat menyeret pegangan atau tepi SpriteRect untuk mengubah ukurannya di sekitar elemen tertentu. Memiliki elemen yang terisolasi, Anda dapat menambahkan yang lain dengan menyeret SpriteRect baru di bagian terpisah dari gambar. Anda akan melihat bahwa ketika Anda memiliki SpriteRect dipilih, panel muncul di bagian bawah kanan jendela:

Kontrol di panel memungkinkan Anda memilih nama untuk grafik Sprite dan mengatur posisi dan ukuran SpriteRect oleh koordinatnya. Lebar perbatasan, untuk kiri, atas, kanan dan bawah dapat ditentukan dalam pixelsUnit terkecil dalam gambar komputer. Ukuran piksel tergantung pada resolusi layar Anda. Pencahayaan pixel dihitung pada setiap piksel layar. More info
Lihat di Glossary. Batas berguna ketika Sprites. Ada juga pengaturan untuk pivot Sprite, yang digunakan Unity sebagai asal koordinat dan utama “tichor point” grafis. Anda dapat memilih dari sejumlah posisi SpriteRect-relatif default (misalnya, Center, Top Right, dll) atau menggunakan koordinat kustom.

Tombol Trim di sebelah item menu Slice akan mengubah ukuran SpriteRect sehingga cocok erat di sekitar tepi grafis berdasarkan transparansi.

Slicing otomatis

Pilih alat Slice dan atur Type ke Automatic untuk memiliki Unity secara otomatis iriskan Tekstur menjadi beberapa Sprit individu dengan menghasilkan batas berdasarkan transparansi sekitar masing-masing potensi Sprit. Misalnya, Anda dapat menggunakan elemen grafis secara teratur di Tekstur, seperti Automatic Slicing individual dalam lembaran ubin.Sebuah kelas sederhana yang memungkinkan sprite untuk diberikan di sebuah Tilemap. More info
Lihat di Glossary in a tile sheet.

Setelah memilih alat Slice dari Sprite Editor toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
Lihat di Glossary dan mengatur Type untuk Automatic, panel editor berikut muncul:

Dalam panel ini, Anda dapat mengatur pivot default untuk setiap Sprite irisan dengan opsi Pivot dan Custom. Jika ada pilihan yang ada di jendela editor sebelum menerapkan lisensi otomatis, pilih dari opsi berikut dalam menu Metode untuk memilih bagaimana Unity menangani pilihan yang ada:

Method Function
Delete Existing Menghapus semua SpriteRects dalam Tekstur dan menambahkan SpriteRect baru.
Smart Jika baru diiris SpriteRect tumpang tindih dengan satu atau lebih SpriteRects yang ada, SpriteRect asli yang terbaik akan memiliki posisi dan ukurannya diperbarui ke SpriteRect irisan baru. SpriteRect baru ini kemudian akan dibuang.
Safe Mempertahankan semua SpriteRects asli. Jika tumpang tindih SpriteRect baru dengan SpriteRect yang ada, itu akan dibuang.

Grid by Cell Size atau opsi Grid by Cell Count juga tersedia untuk jenis lisensi. Ini sangat berguna ketika Sprites telah diletakkan dalam pola biasa selama pembuatan:

Nilai Pixel Size menentukan ketinggian dan lebar ubin dalam piksel. Jika Anda memilih grid dengan jumlah sel, Column & Row menentukan jumlah kolom dan baris yang digunakan untuk lisensi. Anda juga dapat menggunakan nilai Offset untuk mengubah posisi grid dari kiri atas gambar dan nilai Padding untuk menetapkan SpriteRects sedikit dari grid. Opsi Keep Empty Rects memungkinkan Anda menjaga SpriteRects yang tidak memiliki piksel dalam mereka. Ini berguna untuk mengatur Sprites yang diciptakan dari tekstur berdasarkan posisi mereka dalam tekstur. Pivot dapat diatur dengan salah satu lokasi preset sembilan atau lokasi Custom Pivot dapat diatur.

Opsi Isometric Grid mencoba untuk mengiris lembar Sprite yang memiliki Spritesnya diletakkan di baris bergantian dari setengah berlian tinggi. Ketika diiris, SpriteRects yang dihasilkan akan memiliki garis besar kustom dan bentuk fisika yang sesuai dengan berlian ketinggian setengah.

Opsi Is Alternate menentukan apakah berlian isometric terhuyungi di baris alternatif. Jika set, baris paling atas diasumsikan untuk memiliki berlian pertama mulai setengah ukuran pixel dari sisi kiri. Jika tidak, ia menganggap bahwa berlian pertama dimulai dari sisi kiri langsung.

Ketika mengajukan pilihan Grid, pratinjau SpriteRects ditunjukkan pada tekstur untuk membantu membimbing lisensi. Menyesuaikan pengaturan lisensi akan memperbarui pratinjau sesuai. Pratinjau SpriteRects yang ditunjukkan mungkin tidak dihasilkan setelah lisensi tergantung pada metode yang digunakan untuk menangani SpriteRects yang ada atau konten pixel dari SpriteRects Pratinjau.

Perhatikan bahwa Anda dapat secara manual mengedit SpriteRect yang dihasilkan oleh metode Automatic Slicing ini untuk menyempurnakan bentuk SpriteRects.

Polygon Resizing

Buka Sprite Editor untuk poligon dan Anda memiliki opsi untuk mengubah bentuk, ukuran, dan posisi pivot.

Bentuk poligon

Masukkan jumlah sisi yang Anda inginkan poligon untuk memiliki bidang Sides dan klik Change.

Ukuran poligon dan pivot

Ukuran poligon

Untuk mengubah ukuran poligon, klik Sprite untuk menampilkan garis perbatasan hijau dan kotak informasi Sprite. Klik dan tarik di garis hijau untuk membuat perbatasan yang Anda inginkan, dan nilai-nilai dalam perubahan bidang Border.

Poligon-Keter

Untuk mengubah titik pivot poligon (yaitu sumbu titik pergerakan poligon di sekitar), klik pada gambar untuk menampilkan kotak informasi Sprite. Klik pada menu drop down Pivot dan pilih opsi. Ini menampilkan lingkaran pivot biru di poligon; lokasinya tergantung pada opsi pivot untuk Anda telah dipilih. Jika Anda ingin mengubahnya lebih lanjut, pilih Custom Pivot dan klik dan tarik pada lingkaran pivot biru untuk memposisikannya.
