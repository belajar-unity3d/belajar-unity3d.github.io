Mengatur Kanvas

Dalam pane ViewportThe user’s visible area of an app on their screen.
See in Glossary, Anda dapat menemukan Canvas, wadah edit-time re-sizable mengambang yang berisi versi live dari dokumen UI(User Interface) Memungkinkan pengguna untuk berinteraksi dengan aplikasi Anda. Unity saat ini mendukung tiga sistem UI. More info
Lihat di Glossary (UXML). Jika Anda tidak bisa melihatnya, pilih tombol Fit Canvas di Viewport toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
Lihat di Glossary untuk membawanya ke dalam pandangan. Setiap pengaturan yang terkait dengan Canvas, seperti ukurannya, tidak disimpan sebagai bagian dari Dokumen UI (UXML) tetapi disimpan dalam file pengaturan terpisah untuk waktu berikutnya Anda membuka dokumen UI yang sama (UXML).

Ukuran Kanvas

Untuk mengubah ukuran Canvas di dalam Viewport, tarik tepi atau sudut. Untuk ukuran yang tepat, klik pada header Canvas melalui pane InspectorA Unity window that displays information about the currently selected GameObject, asset or project settings, allowing you to inspect and edit the values. Ubah latar belakang Canvas

Anda dapat mengubah latar belakang Canvas untuk membuat mengedit UI dalam konteks lebih mudah. Untuk mengubah latar belakang, pilih Canvas di pane Inspector. Anda dapat mengaturnya menjadi warna yang solid, tekstur khusus (mencukupi dari UI Designer), atau pandangan langsung dari CameraA component which creates an image of a particular viewpoint in your scene. The output is either drawn to the screen or captured as a texture. More info
