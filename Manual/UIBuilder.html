Pembuat UI

UI(User Interface) Allows a user to interact with your application. Unity currently supports three UI systems. More info
See in Glossary Builder memungkinkan Anda secara visual membuat dan mengedit aset UI, seperti Dokumen UI (.uxml), dan Lembar Style (.uss), yang Anda gunakan dengan UI Toolkit.

Compatibility

Sebagai 2021.1, UI Builder sekarang merupakan fitur inti dari Unity.

Accessing UI Builder

Anda dapat membuka jendela UI Builder dalam dua cara:

  1. Di bawah jendela Editor, pilih Window > UI Toolkit > UI Builder
  2. Dari Project windowJendela yang menunjukkan isi folder Assets (Project tab) More info
    Lihat di Glossary    , klik dua kali aset .uxml.

Ketergantungan opsional

UI Builder memiliki paket opsional yang menawarkan fungsionalitas tambahan untuk membuat UI untuk permainan Anda.

Grafik vektor

Anda dapat menginstal paket com.unity.vectorgraphics untuk memiliki UI Builder mengekspos kemampuan untuk menetapkan VectorImage sebagai gaya latar belakang pada elemen.

Dukungan Sprit 2D

Anda dapat menginstal paket com.unity.2d.sprite untuk memiliki UI Builder mengekspos kemampuan untuk menetapkan aset 2D SpriteObjek grafis 2D. Jika Anda digunakan untuk bekerja di 3D, Sprites pada dasarnya hanya tekstur standar tetapi ada teknik khusus untuk menggabungkan dan mengelola tekstur sprite untuk efisiensi dan kenyamanan selama perkembangan.
