Ragdoll Wizard

Unity memiliki wizard sederhana yang memungkinkan Anda dengan cepat membuat ragdoll Anda sendiri. Anda hanya perlu menyeret tungkai yang berbeda pada properti masing-masing di jenggot. Kemudian pilih buat dan Unity akan secara otomatis menghasilkan semua CollidersBentuk tak terlihat yang digunakan untuk menangani tabrakan fisik untuk objek. Seorang sari tidak perlu menjadi bentuk yang sama dengan jala objek - perkiraan kasar sering lebih efisien dan tidak dapat dibedakan dalam gameplay. More info

Membuat Karakter

Ragdolls menggunakan Skinned Meshes, yaitu karakter meshGrafik utama primitif Unity. Mesh membuat sebagian besar dunia 3D Anda. Unity mendukung mesh poligon triangulat atau Quadrangulasi. Nurbs, Nurms, permukaan Subdiv harus dikonversi ke poligon. More info
Lihat di Glossary yang dihilang dengan tulang dalam aplikasi pemodelan 3D. Untuk alasan ini, Anda harus membangun karakter ragdoll dalam paket 3D seperti Autodesk® Maya®.

Ketika Anda telah membuat karakter Anda dan merobeknya, simpan aset biasanya di Project Folder Anda. Ketika Anda beralih ke Unity, Anda akan melihat file aset karakter. Pilih file dan dialog Import Settings akan muncul di dalam inspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. Menggunakan Wizard

Tidak mungkin untuk membuat aset sumber yang sebenarnya menjadi ragdoll. Ini akan membutuhkan memodifikasi file aset sumber, dan karenanya tidak mungkin. Anda akan membuat contoh aset karakter menjadi ragdoll, yang kemudian dapat disimpan sebagai PrefabJenis aset yang memungkinkan Anda untuk menyimpan GameObject lengkap dengan komponen dan properti. Prefab bertindak sebagai template dari mana Anda dapat membuat instance objek baru di tempat kejadian. More info
Lihat di Glossary untuk digunakan kembali.

Buat contoh karakter dengan menyeretnya dari Project View ke Hierarchy View. Memperluas Transform Hierarchy dengan mengklik panah kecil ke kiri nama instance di Hierarchy. Sekarang Anda siap untuk memulai menetapkan bagian ragdoll Anda.

Buka Ragdoll Wizard dengan memilih GameObject > 3D Object > Ragdoll… dari bilah menu. Anda sekarang akan melihat Wizard itu sendiri. Meme it

Menetapkan bagian ke wizard harus bersifat mandiri. Seret Transformasi yang berbeda dari instance karakter Anda ke properti yang tepat di jenggot. Ini harus sangat mudah jika Anda membuat aset karakter sendiri.

Ketika Anda selesai, klik Create Button. Sekarang ketika Anda memasukkan Play Mode, Anda akan melihat karakter Anda pergi limp sebagai ragdoll.

Langkah terakhir adalah untuk menyimpan ragdoll setup sebagai Prefab. Pilih Assets -> Create -> Prefab dari bilah menu. Anda akan melihat Prefab Baru muncul di Tampilan Proyek. Rename itu untuk "Ragdoll Prefab". Seret instance karakter ragdoll dari Hierarchy di atas "Ragdoll Prefab". Sekarang Anda memiliki karakter ragdoll yang benar-benar diatur untuk digunakan sebanyak yang Anda suka dalam permainan Anda.
