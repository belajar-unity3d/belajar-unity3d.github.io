Paket jendela Manager

Gunakan Manajer Paket Unity (dalam menu atas Unity: Window > Package Manager) untuk melihat packages dan fitur set tersedia untuk instalasi atau sudah dipasang di proyek Anda. Anda juga dapat menggunakan jendela ini untuk melihat versi paket yang tersedia, dan install, remove, disable, atau paket update untuk setiap proyek.fitur setSebuah fitur set adalah koleksi paket terkait yang dapat Anda gunakan untuk mencapai hasil tertentu dalam Editor Unity. Anda dapat mengelola set fitur langsung di Manajer Paket Unity. More info
Lihat di Glossary for each project.

Tampilan jendela Manajer Paket:

Sitemap Indikator paket eksperimental, yang memperingatkan Anda jika proyek Anda mengandung (A).paket eksperimental.

Sitemap Tombol (B) The add, yang dapat Anda klik ke langsung ke proyek Anda dengan memasukkan URL git, jalur lokal, atau nama paket.instal paket directly into your project by entering a git URL, a local path, or a package name.

Sitemap Menu (C), yang dapat Anda gunakan untuk mengubah apa yang muncul dalam daftar (context).Paket menu, which you can use to change what appears in the list (context).

Sitemap Menu (D), yang dapat Anda gunakan untuk mengurutkan daftar paket dan set fitur berdasarkan nama atau tanggal.Sort menu, which you can use to sort the list of packages and feature sets by name or date.

Sitemap Menu (E) dan tampilan tombol Filters pada semua daftar kecuali daftar Built-in. Clear Filters memungkinkan Anda untuk mempersempit paket yang muncul dalam daftar.Filtering allows you to narrow down which packages appear in the list.

Sitemap Menu (F) Advanced, yang dapat Anda gunakan untuk mengakses untuk Manajer Paket dan banyak lagi. Lihat project settingsPengumpulan yang luas dari pengaturan yang memungkinkan Anda untuk mengkonfigurasi bagaimana Fisika, Audio, Jaringan, Grafik, Input dan banyak bidang lain dari jangkauan proyek Anda. More info
Lihat di Glossary untuk rincian.Pengaturan lanjutan for details.

Sitemap (G), yang dapat Anda gunakan untuk mencari paket dan set fitur berdasarkan nama.kotak pencarian, which you can use to look for packages and feature sets by name.

Sitemap (H), yang menampilkan semua paket yang sesuai dengan filter dan parameter pencarian yang Anda tentukan.list view, which displays all packages that match the filter and search parameters you specify.

Sitemap Tampilan detail, yang menampilkan informasi spesifik ke (I) atau fitur set yang dipilih dalam daftar. Misalnya, jika Anda memilih set fitur, informasi spesifik ke set fitur muncul, termasuk paket yang mengandung.packageWadah yang menyimpan berbagai jenis fitur dan aset untuk Unity, termasuk alat Editor atau Runtime dan perpustakaan, koleksi Aset, dan template proyek. Paket adalah unit mandiri yang dapat dibagikan di proyek Unity. Sebagian besar waktu ini disebut packages, tetapi kadang-kadang mereka disebut Unity Package Manager (UPM) packages. More info
Lihat di Glossary selected in the list. For example, if you select a feature set, the information specific to the feature set appears, including which packages it contains.

Sitemap (J), yang menampilkan informasi ketika paket paket Manajer paket dan set fitur. Informasi ini termasuk kesalahan dan pesan peringatan, jumlah paket status bar tersedia, dan tautan untuk memuat lebih banyak paket Aset Store.Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
Lihat di Glossary packages available, and a link to load more Asset Store packages.

Sitemap (K) The Refresh list memungkinkan Anda menyegarkan daftar paket yang ditampilkan. Dalam konteks My Assets, adalah menu, yang berisi opsi Refresh list, sehingga Anda dapat memeriksa pembaruan ke semua paket di komputer Anda; bukan hanya yang terlihat dalam konteks My Assets.Check for updates option, so you can check for updates to all packages on your computer; not just the ones that are visible in the My Assets context.

Sitemap Tombol untuk melakukan tindakan berikut:(L) Buttons to perform any of the following actions:

  • Menginstal atau menghapus set fitur
  • Install, update, atau paket remove packages
  • Unduh dan impor, atau paket Aset update Asset Store packages
  • menonaktifkan atau mengaktifkan paket built-in

Status bar

Manajer Paket menampilkan pesan di bilah status di sebelah kiri bawah jendela Manajer Paket.

Biasanya ada empat pesan status yang mungkin Anda lihat:

  • Pertama kali Anda membuka jendela Paket Manager dalam proyek baru, pesan Refreshing packages and features muncul secara singkat:

    Pesan ini juga muncul ketika Anda klik Refresh list

  • Ketika konteks My Assets aktif, batang beban muncul di atas tanggal. Ini menampilkan jumlah paket Asset Store dan tautan Load:

  • Sebagian besar waktu bar status menampilkan tanggal dan waktu ketika jendela Manajer Paket terakhir menyegarkan informasinya. Namun, jika Manajer Paket, seperti masalah jaringan, Manajer Paket menampilkan kesalahan di bilah status:

  • Jika koneksi jaringan Anda bekerja, tetapi Anda tidak masuk ke dalam Anda, Manajer Paket tidak menampilkan paket Aset Store. Ketika Anda mencoba menggunakan konteks My Assets, Manajer Paket menampilkan kesalahan di bilah status:

    Dalam tampilan daftar, klik Sign in untuk masuk ke akun Unity Anda melalui .
