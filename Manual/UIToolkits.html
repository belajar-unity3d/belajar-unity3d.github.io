Paket Unity UI (uGUI)

Paket Antarmuka Pengguna Unity ( UI Unity) (juga disebut uGUI) adalah sistem UI berbasis GameObject yang dapat Anda gunakan untuk mengembangkan UI runtime untuk permainan dan aplikasi. Di Unity UI, Anda menggunakan komponen dan tampilan Game untuk mengatur, posisi, dan gaya antarmuka pengguna. Ini mendukung fitur rendering dan teks canggih.

Memilih sistem UI untuk proyek Anda

Unity berniat untuk UI Toolkit untuk menjadi sistem UI yang direkomendasikan untuk proyek pengembangan UI baru, tetapi masih hilang beberapa fitur yang ditemukan di Unity UI (uGUI) dan IMGUI. Sistem lama ini lebih baik dalam kasus penggunaan tertentu, dan diperlukan untuk mendukung proyek legacy.

Pilihan sistem UI Anda untuk proyek yang diberikan tergantung pada jenis UI yang Anda rencanakan untuk mengembangkan, dan fitur yang Anda butuhkan dukungan.
