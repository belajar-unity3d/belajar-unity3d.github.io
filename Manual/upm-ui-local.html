Menginstal paket dari folder lokal

Manajer Paket dapat memuat paket dari mana saja di komputer Anda bahkan jika Anda menyimpannya di luar folder proyek Unity Anda (misalnya, jika Anda memiliki paket yang disebut com.unity.my-local-package dan Anda menyimpannya di Desktop tetapi proyek Unity Anda berada di bawah folder Documents).

Anda juga dapat menggunakan folder di dalam folder proyek Anda, asalkan itu bukan salah satu reserved project sub-folders.

Untuk memuat paket dari disk lokal Anda:

  1. Klik tombol add di bilah status.

  2. Pilihan untuk menambahkan paket muncul.

  3. Pilih Add package from disk dari menu tambahan untuk membawa browser file.

  4. Navigasi ke akar folder dari local package Anda.

  5. Klik dua file package.json di browser file.

Browser file menutup, dan paket sekarang muncul di Daftar paket dengan label .

Note: Jika Anda memiliki paket dipasang secara lokal yang cocok dengan versi di registry, tombol Update masih muncul dan daftar mungkin menampilkan versi yang sama dua kali sebagai Currently Installed dan Recommended:

Ingat bahwa jika Anda diperbarui ke versi registry tetapi Anda membuat perubahan secara lokal ke proyek Anda, versi registry akan menimpa perubahan lokal Anda.

Paket lokal di dalam proyek Anda

Anda dapat menempatkan paket lokal di mana saja di dalam proyek Anda kecuali di bawah folder ini:

Project folder: Reason:
Assets Jika Anda menempatkan paket di dalam folder ini, Aset Database mengimpor aset di bawah folder ini dua kali: setelah aset dan sekali sebagai konten paket.
Library Jangan memodifikasi isi folder ini.
ProjectSettings Folder ini hanya untuk aset settings.
