Curves

Anda dapat melampirkan animation curves untuk mengimpor animation clipsAnimasi data yang dapat digunakan untuk karakter animasi atau animasi sederhana. Ini adalah bagian "unit" sederhana dari gerakan, seperti (salah satu contoh spesifik) "Idle", "Walk" atau "Run". More info
Lihat di Glossary di Animasi tab.

Anda dapat menggunakan kurva ini untuk menambahkan data animasi tambahan ke klip impor. Anda dapat menggunakan data untuk menghidupkan waktu item lain berdasarkan keadaan animator. Misalnya, dalam set permainan dalam kondisi icy, kurva animasi tambahan dapat digunakan untuk mengontrol tingkat emisi particle systemKomponen yang mensimulasikan entitas fluida seperti cairan, awan dan nyala dengan menghasilkan dan memikat sejumlah besar gambar 2D kecil di tempat kejadian. More info
Lihat di Glossary untuk menunjukkan napas kondensor pemain di udara dingin.

Untuk menambahkan kurva ke animasi impor, memperluas bagian Curves di bagian bawah Animasi tab, dan klik ikon plus untuk menambahkan kurva baru ke klip animasi saat ini:

Jika file animasi impor Anda dibagi menjadi beberapa klip animasi, setiap klip dapat memiliki kurva kustom sendiri.

X-axis kurva mewakili waktu normal dan selalu berkisar antara 0,0 dan 1.0 (korrespon pada awal dan ujung klip animasi masing-masing, terlepas dari durasinya).

Double-klik kurva animasi membawa editor kurva Unity standar yang dapat Anda gunakan untuk menambahkan keys ke kurva. Kunci adalah poin di sepanjang garis waktu kurva di mana memiliki nilai secara eksplisit diatur oleh aimator daripada hanya menggunakan nilai yang diserbuki. Kunci sangat berguna untuk menandai poin penting sepanjang garis waktu animasi. Misalnya, dengan animasi berjalan, Anda mungkin menggunakan kunci untuk menandai poin di mana kaki kiri ada di tanah, kemudian kedua kaki di tanah, kaki kanan di tanah, dan sebagainya. Setelah kunci diatur, Anda dapat bergerak dengan mudah antara bingkai kunci dengan menekan tombol Previous Key Frame dan Next Key Frame. Ini memindahkan garis merah vertikal dan menunjukkan waktu normal pada keyframeBingkai yang menandai titik awal atau akhir transisi dalam animasi. Bingkai di antara bingkai kunci disebut inbetweens.
Lihat di Glossary. Nilai yang Anda masukkan dalam kotak teks menggerakkan nilai kurva pada saat itu.

Kurva animasi dan parameter pengontrol animator

Jika Anda memiliki kurva dengan nama yang sama sebagai salah satu parameters di Pengontrol AnimatorKontrol animasi melalui Lapisan Animasi dengan Mesin Negara Animasi dan Pohon Animasi Blend, dikendalikan oleh Parameter Animasi. Pengontrol Animator yang sama dapat direferensikan oleh beberapa model dengan komponen Animator. More info
Lihat di Glossary, maka parameter itu mengambil nilai dari nilai kurva pada setiap titik dalam timeline. Misalnya, jika Anda membuat panggilan ke GetFloat dari script, nilai pengembalian sama dengan nilai kurva pada saat panggilan dibuat. Perhatikan bahwa setiap titik tertentu dalam waktu, mungkin ada beberapa klip animasi mencoba untuk mengatur parameter yang sama dari pengontrol yang sama. Dalam hal itu, Unity memadukan nilai kurva dari beberapa klip animasi. Jika animasi tidak memiliki kurva untuk parameter tertentu, maka campuran Unity dengan nilai default untuk parameter itu.
