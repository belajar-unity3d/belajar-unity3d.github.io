Jika Anda digunakan untuk bekerja di 3D, Sprites pada dasarnya hanya tekstur standar tetapi ada teknik khusus untuk menggabungkan dan mengelola tekstur sprite untuk efisiensi dan kenyamanan selama perkembangan. Mengakses Pencipta Sprite

Pilih Assets>Create>Sprites dan kemudian pilih sprite placeholder yang ingin Anda buat (persegi, segitiga, berlian, segi enam, atau poligon).

Menggunakan Sprite

Tangkap placeholder baru Anda muncul sebagai bentuk putih di folder aset yang saat ini Anda buka. Nama sprite baru secara default dengan nama bentuknya tetapi Anda memiliki opsi untuk mengubah nama sprite Anda ketika pertama kali diciptakan. Jika Anda tidak yakin apa yang ingin Anda sebut, biarkan itu sebagai default; Anda dapat mengubahnya nanti dengan mengkliknya.

Anda dapat menyeret dan menjatuhkan sprite placeholder Anda ke Scene ViewTampilan interaktif ke dunia yang Anda buat. Anda menggunakan Adegan Lihat untuk memilih dan posisi pemandangan, karakter, kamera, lampu, dan semua jenis lain dari Game Object. More info
Lihat di Glossary atau Hierarchy untuk mulai menggunakannya dalam proyek Anda.

Mengganti Tempat pemegang Sprite

Untuk mengubah sprite placeholder Anda, klik di dalamnya di Scene View dan kemudian edit melalui Sprite Renderer Component di InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. Lihat di Glossary.

Edit kolom Sprite: Anda dapat mengklik lingkaran kecil ke kanan kolom input untuk mengambil Sprite Selector di mana Anda dapat menelusuri dan memilih dari menu aset grafis 2D yang tersedia.

Details

  • Pencipta Sprite membuat tekstur outline PNG putih 4x4.
  • Kecambah placeholder adalah poligon primitif sempurna (misalnya segitiga, segi enam, poligon n-sided), yang dihasilkan oleh algoritma.
  • Sitemap Kecambah placeholder tidak seperti primitif 3D: Srite adalah aset, dan poligon banyak sisi, dapat mewakili banyak bentuk yang berbeda, sehingga kecambah placeholder tidak dibangun seperti primitif 3D.NOTE: Placeholder sprites are not like 3D primitives: A sprite is an asset, and as a many-sided polygon, may represent many different shapes, so placeholder sprites are not built like 3D primitives.
