Namun, Anda juga dapat menggunakan format asset packageKoleksi file dan data dari proyek Unity, atau elemen proyek, yang dikompresi dan disimpan dalam satu file, mirip dengan file Zip, dengan ekstensi .unitypackage. Ikuti petunjuk di bagian ini untuk mengekspor paket aset dari menu Assets di Editor Unity.

Note: Jika aset Anda berkualitas tinggi dan Anda percaya pengguna lain akan menemukan mereka berguna, ikuti petunjuk dalam dokumentasi untuk Penerbitan ke Toko Aset untuk membuat draft paket dan mengunggahnya ke Toko Aset

Untuk membuat paket aset kustom Anda sendiri:

  1. Buka proyek yang ingin Anda ekspor aset.

  2. Pilih AssetsSetiap media atau data yang dapat digunakan dalam permainan atau proyek Anda. Aset mungkin berasal dari file yang dibuat di luar Unity, seperti model 3D, file audio atau gambar. Anda juga dapat membuat beberapa jenis aset di Unity, seperti Pengontrol Animator, Audio Mixer atau Tekstur Render. More info
    Lihat di Glossary     > Export Package dari menu untuk membuka kotak dialog Exporting package.

  3. Dalam kotak dialog, pilih aset yang ingin Anda masukkan dalam paket dengan mengklik kotak sehingga mereka diperiksa.

  4. Tinggalkan Include dependencies diaktifkan untuk memilih aset apa pun yang digunakan oleh aset yang telah Anda pilih.

  5. Klik Export untuk membawa penjelajah file, dan pilih di mana Anda ingin menyimpan file paket Anda.

  6. Nama dan simpan paket di mana pun Anda suka.

Sitemap Ketika mengekspor paket, Unity dapat mengekspor semua dependensi juga. Jadi, misalnya, jika Anda memilih Adegan dan mengekspor paket dengan semua dependensi, maka Ekspor Unity semua Model, Tekstur dan aset lain yang muncul di Adegan juga. Ini bisa menjadi cara yang cepat mengekspor beberapa aset tanpa memilih secara manual.Tip: When exporting a package, Unity can export all dependencies as well. So, for example, if you select a Scene and export a package with all dependencies, then Unity exports all Models, Textures and other assets that appear in the Scene as well. This can be a quick way of exporting several assets without manually locating them all.

Re-exporting paket aset

Jika Anda ingin mengubah konten paket aset dan membuat versi terbaru dari paket aset Anda, pilih file aset yang Anda inginkan dalam paket Anda (baik yang tidak berubah dan yang baru). Kemudian ikuti petunjuk di atas untuk mengekspor file.

Strategi

Nama ulang paket terbaru Anda menggunakan nama inkremental: misalnya, MyAssetPackageVer1, MyAssetPackageVer2. Unity mengenalinya sebagai pembaruan, jadi Anda harus menggunakan konvensi naming yang jelas bagi Anda dan siapa pun yang Anda bagikan.

Sitemap Jangan menghapus file dari paket aset dan kemudian tambahkan file yang berbeda dengan nama yang sama. Unity Warning: untuk melacak file, sehingga mengenali mereka sebagai file konflik yang berbeda dan mungkin. Dalam kasus ini, Unity menampilkan simbol peringatan saat mengimpornya. Jika Anda telah menghapus file dan kemudian memutuskan untuk menggantinya, mengubah nama itu sesuatu yang lain, bahkan jika itu dekat dengan nama asli.menggunakan ID unik to track files, so it recognizes them as different and possibly conflicting files. In these cases, Unity displays a warning symbol when importing them. If you have removed a file and then decide to replace it, rename it something else, even if it is close to the original name.
