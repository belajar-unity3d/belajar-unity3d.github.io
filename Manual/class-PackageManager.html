Package Manager

Pengaturan Package Manager (menu: Edit > Project Settings, maka kategori Package Manager) memungkinkan Anda mengelola registries lingkup Anda dan memodifikasi pengaturan canggih untuk proyek Anda saat ini.

Sitemap Di bawah bagian (A), toggle apakah atau tidak Advanced Settings dan paket pra-release terlihat di jendela Paket Manager.paket are visible in the Package Manager window.

(B) Add, modify, dan remove scoped registries untuk proyek Anda di bawah kelompok Scoped Registries, yang mengandung area berikut:

Login Daftar registries lingkup saat ini didefinisikan untuk proyek ini(1) The list of scoped registries currently defined for this project

Login Rincian untuk registry lingkup yang dipilih(2) The details for the selected scoped registry


Pengaturan Lanjutan

Gunakan grup Pengaturan Lanjutan untuk beralih pengaturan Paket Pre-release yang Dapat Diaktifkan dan Show Dependencies untuk proyek Anda saat ini.

Paket Pre-release yang Dapat Diaktifkan

Aktifkan pengaturan Enable Pre-release Packages untuk menampilkan paket pra-release di jendela Manajer Paket. Paket pra-release muncul dengan tag dalam tampilan daftar dan tag dalam tampilan detail, sebagai pengingat paket mana yang akan dirilis dengan versi LTS dari Editor.


Warning:

Secara default pengaturan ini dinonaktifkan karena paket pra-release masih diverifikasi cukup stabil untuk produksi, tetapi tidak sepenuhnya dirilis. Untuk informasi lebih lanjut tentang siklus hidup paket, lihat Paket negara dan siklus hidup.


Show Dependencies

Aktifkan pengaturan Show Dependencies untuk:

  • Tampilkan paket apa pun yang dipasang sebagai indirect dependenciesKetergantungan indirect, atau transitif terjadi ketika proyek Anda meminta paket yang sendiri “tergantung pada” paket lain. Misalnya, jika proyek Anda tergantung pada paket alembic@1.0.7 yang pada gilirannya tergantung pada paket timeline@1.0.0, maka proyek Anda memiliki ketergantungan langsung pada Alembic dan ketergantungan tidak langsung pada Timeline.   • Menampilkan informasi ketergantungan paket di Detail paket di jendela Manajer Paket. Ini termasuk berikut:
    • Is using daftar paket yang dibutuhkan paket ini, termasuk versi tertentu.
    • Used by mencantumkan nama dan versi paket lain yang tergantung pada yang ini.


Note:

Secara default, pengaturan ini dinonaktifkan. Untuk informasi lebih lanjut tentang bagaimana ketergantungan bekerja dengan paket, lihat Ketergantungan dan resolusi.


Pendaftaran Lingkup

Kelompok Scoped Registries menampilkan daftar registries lingkup telah ditambahkan ke proyek saat ini di sebelah kiri, dan rincian registry yang dipilih di sebelah kanan.

Untuk informasi rinci tentang registries lingkup, termasuk cara menggunakan dan mengkonfigurasinya dengan Manajer Paket Unity, lihat dokumentasi pada Pendaftaran Lingkup.

Note: Jika Anda menambahkan registry lingkup tetapi Anda tidak melihatnya dalam konteks My Registries atau konteks My Registries tidak tersedia sama sekali, itu mungkin karena server registry paket yang ditambahkan tidak menerapkan endpoint /-/v1/search atau /-/all, yang berarti bahwa tidak kompatibel dengan Manajer Paket Unity.

Menambahkan registry baru

Untuk menambahkan registry lingkup baru ke proyek Anda:

  1. Pilih tombol + di bagian bawah daftar. Masuk baru muncul sebagai New Scoped Registry dalam daftar dan nilai kosong untuk rincian di kanan dan tombol Revert dan Apply berubah menjadi tombol Cancel dan Save.
  2. Masukkan nilai untuk sifat Name, URL, dan Lingkup(s).
  3. Jika Anda perlu menentukan lebih dari satu ruang lingkup, pilih tombol + di bawah nilai Scope(s) terakhir. Kotak teks lain muncul.
  4. Ketika Anda selesai memasukkan informasi untuk registry lingkup yang dipilih, klik Save. Untuk membatalkan menambahkan registry lingkup baru sepenuhnya, klik Cancel.

Modifikasi registry yang ada

Untuk memodifikasi registry lingkup yang ada:

  1. Pilih registry yang ingin Anda modifikasi dari daftar di sebelah kiri. Informasi yang ada muncul di sebelah kanan.
  2. Modifikasi sifat Name, URL, dan Lingkup(s).
  3. Ketika Anda selesai memperbarui informasi untuk registry lingkup yang dipilih, klik Apply. Untuk membatalkan menambahkan registry lingkup baru sepenuhnya, klik Revert.

Menghapus registry

Untuk menghapus registry lingkup yang ada:

  1. Pilih registry yang ingin dihapus dari daftar
  2. Klik tombol - di bawah daftar. Kotak dialog meminta Anda untuk mengkonfirmasi penghapusan.
  3. Klik Ok untuk melanjutkan atau Cancel untuk meninggalkannya utuh.
