Beralih antara versi Unity dalam dokumentasi

Panduan Unity dan Scripting API memegang dokumentasi untuk beberapa versi Unity. Ada beberapa cara untuk beralih antara versi.

Beralih antara halaman rumah dari dua versi terbaru

Anda dapat beralih antara dua versi terbaru dengan mengklik tautan di bagian atas setiap halaman, di atas tabel konten. Tautan ini selalu membawa Anda ke halaman rumah.

Beralih versi dan memelihara halaman

Versi 5.6 dan kemudian

Dalam versi dokumentasi Unity dari 5.6 seterusnya, Anda dapat melihat halaman dokumentasi Unity individu dan beralih antara versi halaman menggunakan tautan Other Versions di sebelah kanan setiap halaman.

Kadang-kadang, Anda mungkin mendapatkan kesalahan 404 ketika Anda beralih. Ini terjadi jika halaman yang Anda cari tidak ada dalam versi Manual atau Scripting API. Ini sering terjadi jika fitur tidak hadir dalam versi Unity yang sesuai.

Versi 5.5 dan sebelumnya

Versi dokumentasi sebelumnya dari 5.6 tidak memiliki tautan Other Versions.

Untuk mempelajari cara menggunakan workaround untuk dokumentasi pra-5.6, ikuti langkah-langkah ini:

Dari halaman rumah Manual atau Scripting API, beralih versi yang Anda lihat menggunakan tautan halaman rumah di sudut kiri atas halaman. Perhatikan bahwa struktur perubahan URL. Setiap versi memiliki URL tertentu:
