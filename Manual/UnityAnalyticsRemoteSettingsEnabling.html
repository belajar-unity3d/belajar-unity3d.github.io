Mengaktifkan Pengaturan Jarak Jauh

Note: sekarang tersedia sebagai preview packageA preview* paket dalam pengembangan dan belum siap untuk produksi. Paket dalam pratinjau mungkin ada tahap perkembangan, dari tahap awal hingga penyelesaian dekat.
Lihat di Glossary. Remote Config adalah penawaran yang ditingkatkan yang memungkinkan Anda untuk mengubah perilaku dan penampilan permainan Anda tanpa memerlukan biner baru. Anda dapat mengaktifkan atau menonaktifkan fitur, mengubah spesifikasi permainan Anda untuk menargetkan audiens tertentu, atau menjalankan acara khusus dengan menjadwalkan rilis konten baru. Sitemap

Untuk mengaktifkan Pengaturan Jarak Jauh dalam proyek:

  1. Jika Anda belum, di Unity Analytics untuk proyek.

  2. Jika diperlukan, impor paket Pengaturan Jarak Jauh dari .

  3. Buka jendela Pengaturan Jarak Jauh dengan Window > Unity AnalyticsSebuah platform data yang menyediakan analitik untuk permainan Unity Anda. More info
    Lihat di Glossary     > SitemapRemote SettingsPengaturan jarak jauh adalah variabel game yang dapat Anda set jarak jauh di Dasbor Analisis Anda. More info
    Lihat di Glossary    :

  4. Masukkan proyek Project Secret Key untuk proyek AnalyticsUnity Analytics
    Lihat di Glossary     Anda dan klik Next. Anda dapat menemukan halaman Project Secret Key Anda di bawah halaman Configure di dashboard Unity Analytics

  5. Pada jendela Pengaturan Jarak Jauh, klik tombol Refresh. Jendela menampilkan pasangan nilai kunci untuk pengaturan yang telah Anda buat.

Jika Anda belum membuat Pengaturan Jarak Jauh, lihat Membuat dan mengubah Pengaturan Jarak Jauh untuk memulai.
