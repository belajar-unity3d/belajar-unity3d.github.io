Build di Unity Cloud

Unity Cloud Build adalah bagian dari dan menyediakan layanan integrasi berkelanjutan untuk proyek Unity. Ini mengotomatiskan proses build pada server Unity untuk membantu menghemat waktu dan sumber daya komputer. Ini membantu Anda menangkap masalah lebih cepat, berbagi membangun dengan kolaborator Anda, dan versi iterate perkembangan Anda lebih cepat.

Build Unity Cloud dapat membangun proyek Anda untuk beberapa platform, dan mendukung banyak sistem kontrol versi populer. Lihat Platform yang didukung untuk mempelajari lebih lanjut.

Temukan informasi tambahan tentang Build Unity Cloud di halaman berikut:

Otomatis membangun generasi

Cloud Build menyediakan integrasi berkelanjutan dengan secara otomatis menerbitkan perubahan dan membangun Proyek Anda setiap kali anggota tim melakukan perubahan ke repositori bersama.

Dengan mengkompensasi proyek Anda setiap kali perubahan dilakukan, Build Unity Cloud memberi Anda ide yang paling akurat ketika dan di mana kesalahan terjadi dan memastikan Anda selalu memiliki membangun permainan Anda berdasarkan komitmen yang baik terakhir.

Untuk mengaktifkan generasi build otomatis, atur toggle Auto-build ke yes di Cloud Build > Config > Basic Info settings.

Bagaimana cara kerja generasi build otomatis

Cloud Build terhubung ke sistem kontrol sumber Unity Collaborate atau sumber Anda dan memantau bahwa sistem untuk perubahan pada proyek Anda. Ketika mendeteksi perubahan proyek Anda, itu mengunduh proyek Anda dan membangunnya untuk platform target Anda. Ketika build selesai, Cloud Build memberi tahu Anda hasil, bersama dengan tautan untuk mengunduh dan menginstal build. Jika ada kesalahan, Cloud Build memberitahu Anda segera, biarkan Anda dengan cepat memperbaikinya, melakukan perubahan, dan memicu build baru.

Tautan berbagi

Anda dapat berbagi build dengan pengguna yang tidak memiliki akun yang terkait dengan proyek Anda. Cloud Build dapat menghasilkan tautan secara otomatis dan mengirimkannya melalui integrasi email atau aplikasi, misalnya, dengan Slack. Anda juga dapat menghasilkan tautan secara manual dari menu dropdown dari build yang diberikan.

Berbagi tautan berakhir setelah dua minggu. Anda dapat menyesuaikan ini di menu link share build yang ingin Anda bagikan.

Platform yang didukung

Build Unity Cloud dapat membangun Proyek Unity Anda untuk platform dan sistem operasi berikut:

  • iOS

  • Android

  • Windows Server

  • Mac OS desktop

  • Linux desktop

  • WebGL

Versi yang didukung Unity

Cloud Build dapat membangun Proyek yang dibuat dengan sebagian besar versi Editor Unity. Cloud Build biasanya tersedia untuk versi baru dari Editor Unity dalam waktu 48 jam rilis versi terbaru.

Pada halaman "Pengaturan", Anda dapat memilih versi Unity untuk digunakan untuk membangun aplikasi Anda.

Sistem kontrol versi

Untuk Cloud Bangun untuk membangun proyek Anda, Anda harus menghosting proyek Anda dalam sistem kontrol sumber (juga dikenal sebagai sistem kontrol versi). Dukungan Build Cloud:

  • Unity Collaborate

  • Plastik SCM

  • Git

  • Apache Subversion (SVN)

  • Mercurial

  • PerforceSistem kontrol versi untuk manajemen perubahan file. More info
