Unity Remote

Unity Remote adalah aplikasi yang dapat diunduh yang membantu pengembangan Android, iOS, dan TVOS. Editor Unity masih melakukan pemrosesan aktual aplikasi pada mesin desktop. Ini berarti bahwa kinerja bukanlah refleksi yang akurat tentang bagaimana aplikasi yang dibangun melakukan pada perangkat target. Untuk mendapatkan penilaian kinerja aplikasi yang lebih akurat, pastikan untuk membangun aplikasi dan menguji build pada perangkat target.

Persyaratan dan kompatibilitas

Unity Remote mendukung perangkat Android, iOS, dan tvOS.

Note: Unity Remote menggantikan aplikasi iOS Remote dan Android Remote dari versi sebelumnya Unity. Unity tidak lagi mendukung aplikasi lama ini.

Mengatur Jarak Unity

Untuk mengatur Jarak Jauh Unity:

  1. Instal aplikasi pada perangkat target.
  2. Hubungkan perangkat target ke komputer Anda.
  3. Hubungkan Editor Unity ke Remote Unity.

Instalasi Remote Unity

Unduh aplikasi Unity Remote rom Appstore.

Hubungkan perangkat target ke komputer Anda

Setelah Anda menginstal Unity Remote pada perangkat target, jalankan Unity Remote kemudian menghubungkan perangkat target ke komputer Anda. Untuk melakukan ini untuk iOS atau tvOS, gunakan iTunes di Windows atau koneksi USB di OS X.

Hubungkan Unity Remote ke Editor Unity

Dengan perangkat target yang terhubung ke komputer Anda, Editor Unity harus dapat melihatnya. Untuk menghubungkan Unity Remote pada perangkat target ke Editor Unity:

  1. Pengaturan editor (menu: Edit > Project Settings > Editor).
  2. Pada bagian Unity Remote, set Device ke perangkat yang ingin Anda gunakan.

Pratinjau aplikasi di Unity Remote

Di Editor, klik tombol Play untuk melihat aplikasi muncul di perangkat target dan di Game View. Performance

Ketika Anda menggunakan Unity Remote, aplikasi sebenarnya berjalan di Editor, dan Unity merampingkan konten visual ke perangkat target. Bandwidth antara Editor dan perangkat terbatas sehingga Unity mengompreskan aliran untuk transmisi. Metode compressionMetode menyimpan data yang mengurangi jumlah ruang penyimpanan yang dibutuhkan. Kompresi Tekstur, Kompresi Animasi, Kompresi Audio, Membangun Kompresi.
Lihat di Glossary default adalah JPEG yang merupakan metode kompresi lossy yang mengurangi kualitas gambar.

Mengubah metode kompresi

Satu cara untuk meningkatkan kualitas gambar pada Unity Remote adalah untuk mengubah metode kompresi ke PNG. PNG adalah metode kompresi tanpa kehilangan yang menggunakan lebih banyak bandwidth tetapi tidak mengurangi kualitas gambar. Untuk mengubah metode kompresi ke PNG:

  1. Pengaturan editor (menu: Edit > Project Settings > Editor).
  2. Pada bagian Unity Remote, set Compression ke PNG.

Mengubah resolusi

Secara default, ketika Unity merampingkan aplikasi ke Unity Remote, itu menuruni resolusi aplikasi. Hasil ini dalam gambar yang lebih kecil yang mengurangi kualitas gambar tetapi memiliki persyaratan bandwidth yang lebih rendah dan menghasilkan kinerja runtime yang lebih baik. Untuk melihat pratinjau aplikasi Anda pada resolusi penuh aplikasi:

  1. Pengaturan editor (menu: Edit > Project Settings > Editor).

  2. Pada bagian Unity Remote, set Resolution ke Normal.
