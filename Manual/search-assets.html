Aset Proyek Pencarian

Gunakan Penyedia Pencarian Aset untuk mencari semua Aset di Proyek saat ini. Anda dapat mencari menggunakan kata kunci atau GUID (Instance ID).

Anda juga dapat mencari Asset Database atau sistem file dari jendela Pencarian.

Search token: p: (untuk "proyek")

Sitemap Buka Aset, baik di Unity atau dalam editor eksternal.Default action: Open the Asset, either in Unity or in an external editor.

Penyedia Pencarian Aset

Penyedia Pencarian Aset vs. Pencarian Basis Data

Cari menggunakan indeks Aset sendiri, yang lebih cepat dan lebih fleksibel daripada Basis Data Aset atau Browser Proyek.

Contoh:

  • Anda dapat mengetik ekstensi file dan akan menemukan semua aset dengan ekstensi itu.
  • Ini cocok dengan nama direktori
  • Ini adalah pencarian parsial

Cari sistem file

Jika Anda menggunakan asterisk (*) dalam pertanyaan, Cari melakukan pencarian normal dan pencarian wildcard terhadap sistem file. Ini memungkinkan Anda untuk menyertakan file yang tidak diindeks oleh database Aset dalam pencarian Anda.
