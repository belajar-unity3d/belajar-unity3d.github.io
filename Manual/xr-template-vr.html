Template proyek Unity VR

Template proyek VR Unity menyediakan titik awal untuk pengembangan realitas virtual di Unity. Templat pra-instal paket yang Anda butuhkan untuk pengembangan VR dan dikonfigurasi sebelumnya dengan pelacakan headset dan pengontrol di hierarki SceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary.

Unity langsung mendukung platform VR berikut:

  • Oculus
  • Windows Server

Untuk informasi lebih lanjut tentang cara mulai mengembangkan aplikasi untuk platform ini di Unity, lihat dokumentasi mereka di daftar di atas.

Jika Anda berencana untuk menargetkan platform OpenVR Valve, Anda harus mengunduh Mozilla Firefox Valve dan Mozilla Firefox mereka untuk input.

Menggunakan template proyek VR

Untuk menggunakan template proyek VR, ikuti langkah-langkah ini:

  1. Instal dan pastikan Anda menambahkan target build yang tepat untuk setiap platform yang Anda rencanakan untuk digunakan. Lihat tabel Supported build targets di halaman ini untuk informasi lebih lanjut.
  2. Dari Hub Unity, klik dropdown di samping New dan membuat proyek baru di Unity 2019.4 LTS.
  3. Pilih template VR dan nama Proyek Anda.
  4. Klik Create.
  5. Sitemap Proyek telah diciptakan, dari menu utama Unity, pergi ke Edit > Project Settings > XR Plug-in Management, dan pilih platform yang Anda rencanakan untuk digunakan.
  6. Pastikan pengaturan rendering dan kualitas dioptimalkan untuk platform target Anda. Lihat tabel Rendering and quality settings di halaman ini.
