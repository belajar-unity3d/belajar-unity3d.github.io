Tautan dalam di iOS

Tautan dalam adalah tautan URL di luar aplikasi Anda yang mengarahkan pengguna ke lokasi di aplikasi Anda. Ketika pengguna mengklik tautan dalam untuk aplikasi, sistem operasi membuka aplikasi Unity di tempat tertentu (misalnya, adegan tertentu). Untuk informasi lebih lanjut tentang tautan mendalam dan cara menggunakannya, lihat Deep links.

Mengaktifkan tautan mendalam untuk aplikasi iOS

Sebelum Anda dapat memproses tautan dalam, Anda perlu mengkonfigurasi aplikasi Anda untuk bereaksi terhadap mereka. Bagian ini berisi instruksi tentang bagaimana mengatur tautan dalam untuk iOS.

Ada dua cara untuk mengaktifkan tautan dalam untuk aplikasi iOS: skema URL dan Tautan universal.

URL

Skema URL menentukan struktur tautan yang bereaksi terhadap aplikasi iOS Anda. Jika perangkat memproses tautan dalam yang sesuai dengan struktur ini, perangkat membuka aplikasi Anda. Untuk menambahkan skema URL:

  1. Buka jendela Login Pengaturan Pemain (menu: Edit > Project Settings > Player Settings, kemudian pilih iOS).
  2. Pilih Other, lalu gulir ke Configuration.
  3. Memperluas bagian Supported URL schemes dan, di bidang Element 0, masukkan skema URL untuk mengasosiasikan aplikasi Anda. Sebagai contoh, unitydl.

Aplikasi iOS Anda sekarang buka ketika perangkat memproses tautan apa pun yang dimulai dengan unitydl://.

Tautan universal

Untuk informasi tentang cara mengaktifkan tautan universal, lihat dokumentasi Apple pada Memungkinkan Aplikasi dan Situs Web untuk Tautan Konten Anda.

Menggunakan tautan mendalam di iOS

Setelah Anda mengaktifkan tautan dalam untuk iOS, cara yang Anda gunakan adalah platform-agnostic. Untuk informasi tentang cara menangani tautan dalam ketika aplikasi Anda terbuka, lihat Menggunakan tautan dalam.
