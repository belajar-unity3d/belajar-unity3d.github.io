Tampilan Game

Tampilan Game diberikan dari Kamera (s) dalam aplikasi Anda. Ini menunjukkan bagaimana tampilan aplikasi akhir, dibangun. Anda perlu menggunakan satu atau lebih CamerasKomponen yang menciptakan gambar sudut pandang tertentu di tempat kejadian Anda. Output ditarik ke layar atau ditangkap sebagai tekstur. More info
Lihat di Glossary untuk mengontrol apa yang pemain melihat ketika mereka menggunakan aplikasi Anda. Untuk informasi lebih lanjut tentang Kamera, lihat Camera Component page.

Unity memungkinkan Anda beralih antara tampilan Game dan tampilan Simulator. Tampilan Simulator menunjukkan bagaimana aplikasi bawaan Anda terlihat pada perangkat seluler.

Modus

Mode Play memungkinkan Anda menjalankan proyek Anda dan melihat bagaimana bekerja seperti itu akan dalam aplikasi yang dibangun. Gunakan tombol di ToolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
Lihat di Glossary untuk mengontrol mode Play. Dalam mode Play, setiap perubahan yang Anda buat sementara, dan reset ketika Anda keluar Mode Play. Ketika Anda memasukkan mode Play, bagian-bagian gelap Unity dari antarmuka di luar tampilan Game.

Menggunakan tampilan Simulator

Gunakan tampilan Simulator untuk Pratinjau bagaimana aplikasi Anda dibangun terlihat pada perangkat seluler. Untuk beralih antara tampilan Game dan Simulator, di tab Game/Simulator, pilih opsi yang tepat dari menu Game/Simulator.

Atau, Anda dapat pergi ke Window > General, dan pilih Game atau Device Simulator dari menu untuk membuka jendela yang diperlukan. Jika tidak ada kasus permainan atau jendela Simulator terbuka, secara default mereka akan terbuka sebagai jendela mengambang. Namun, jika salah satu jendela sudah terbuka di suatu tempat di antarmuka, kemudian mencoba membukanya dari menu akan membawanya menjadi fokus.

Play Mode behavior Bagian ini menjelaskan perilaku mode bermain berdasarkan pilihan Anda di bawah ini.
Focused Aktifkan fokus pada tampilan Game yang dipilih saat Editor berada dalam mode Play. Login Hanya satu tampilan permainan yang bisa difokuskan ketika Anda memasukkan mode Play. Menggunakan Maximized atau Fullscreen pada mode Tampilan menyiratkan fokus pada tampilan Game Maksimal. Mengaktifkan Fokus pada pandangan Game menonaktifkannya pada pandangan Game lain.
Only one game view can be in focus when you enter the Play mode. Using Maximized or Fullscreen on Display mode implies focus on the Maximized Game view. Enabling Focussed on a Game view disables it on other Game views.
VSync (Game view only) Aktifkan VSync (Game view only) untuk memungkinkan sinkronisasi, yang berguna ketika merekam video, misalnya. Bersatu berusaha untuk membuat pandangan Game pada tingkat refresh monitor, meskipun ini tidak dijamin. Ketika opsi ini diaktifkan, masih berguna untuk memaksimalkan tampilan Game dalam mode Bermain untuk menyembunyikan pandangan lain dan mengurangi jumlah pandangan yang disatukan render.
Enter Play Mode: Pilih dari opsi di bawah ini untuk menentukan pengaturan untuk Editor ketika masuk ke mode Play.
Normally Pilih ini untuk melihat tampilan Game tanpa memaksa fokus atau memaksimalkan pandangan apa pun ke layar penuh.
Maximized Ketika opsi ini dipilih, Unity menjalankan mode Play dengan tampilan Game memaksimalkan 100% dari jendela Editor.
Fullscreen On Display X Pilih ini untuk memasukkan mode Play di jendela layar penuh tanpa batas pada tampilan yang dipilih. Jumlah pilihan layar penuh yang muncul di menu tergantung pada jumlah tampilan yang terhubung terdeteksi oleh Unity pada start-up. Biasanya, sebagian besar opsi layar penuh terdaftar sebagai Monitor PnP Generik di menu kecuali bila driver tampilan khusus dipasang.
Mute audio Klik untuk mengaktifkan: gunakan audio in-application ini ketika Anda memasukkan mode Play.
Stats Klik ini untuk menunjukkan atau menyembunyikan overlay Statistik, yang berisi Rendering Statistics tentang audio dan grafis aplikasi Anda. Overlay memungkinkan Anda memantau kinerja aplikasi Anda sementara dalam mode Bermain.
GizmosOverlay grafis yang terkait dengan GameObject di Adegani, dan ditampilkan di Tampilan Adegan. Alat adegan internal seperti alat bergerak adalah Gizmos, dan Anda dapat membuat Gizmos kustom menggunakan tekstur atau scripting. Beberapa Gizmos hanya ditarik ketika GameObject dipilih, sementara Gizmos lainnya ditarik oleh Editor terlepas dari GameObjects mana yang dipilih. More info

Opsi lanjutan

Klik kanan tab Game untuk menampilkan opsi tampilan Game canggih.

Warn if No Cameras Rendering: Opsi ini diaktifkan secara default. Hal ini menyebabkan Unity untuk menampilkan peringatan jika tidak ada Kamera rendering ke layar; misalnya, untuk mendiagnosis masalah seperti sengaja menghapus atau menonaktifkan Kamera. Tinggalkan diaktifkan ini kecuali Anda sengaja tidak menggunakan Kamera untuk membuat aplikasi Anda.

Clear Every Frame in Edit Mode: Opsi ini diaktifkan secara default. Hal ini menyebabkan Unity untuk membersihkan pandangan Game dari setiap bingkai ketika aplikasi Anda tidak bermain. Ini mencegah efek smearing saat Anda mengonfigurasi aplikasi Anda. Tinggalkan diaktifkan ini kecuali Anda tergantung pada konten bingkai sebelumnya ketika tidak dalam mode Play.
