Aim Constraints

Kontratraint Aim memutar GameObjectObjek mendasar dalam adegan Unity, yang dapat mewakili karakter, props, pemandangan, kamera, waypoints, dan banyak lagi. Fungsi GameObject didefinisikan oleh Komponen yang melekat padanya. More info
Lihat di Glossary untuk menghadapi sumbernya GameObjects. Ini juga dapat mempertahankan orientasi yang konsisten untuk sumbu lain. Misalnya, Anda dapat menambahkan Aim Constraint ke CameraKomponen yang menciptakan gambar sudut pandang tertentu di tempat kejadian Anda. Output ditarik ke layar atau ditangkap sebagai tekstur. More info
Lihat di Glossary. Untuk menjaga Kamera tegak sementara Kontratraint bertujuan, tentukan sumbu kamera dan arahkan untuk menyelaraskannya.

Gunakan Up Vector untuk menentukan sumbu up dari GameObject yang terlatih. Gunakan World Up Vector untuk menentukan arah ke atas. Sebagai Kontratraint Aim memutar GameObject untuk menghadapi sumbernya GameObjects, Constraint juga menyelaraskan sumbu atas GameObject yang terlatih dengan arah ke atas.

Properties

Property: Function:
Activate Setelah Anda memutar GameObject yang terlatih dan memindahkan GameObjects sumbernya, klik Activate untuk menyimpan informasi ini. Activate menyimpan offset saat ini dari sumber GameObjects dalam Rotation At Rest dan Rotation Offset, kemudian periksa Is Active dan Lock.
Zero Menetapkan rotasi GameObject yang terlatih untuk sumber GameObjects. Nol mengatur kolom Rotation At Rest dan Rotation Offset, kemudian periksa Is Active dan Lock.
Is Active Apakah atau tidak mengevaluasi Kontras. Untuk juga menerapkan Kontras, pastikan Lock diperiksa.
Weight Kekuatan Kontras. Berat 1 menyebabkan Kontras untuk memutar GameObject pada tingkat yang sama bahwa sumbernya GameObjects bergerak. Berat 0 menghilangkan efek dari Kontras sepenuhnya. Berat ini mempengaruhi semua sumber GameObjects. Permainan Objek dalam daftar Sources juga memiliki berat badan.
Aim Vector Menentukan sumbu yang menghadapi arah GameObjects sumbernya. Misalnya, untuk menentukan bahwa GameObject harus orient hanya sumbu Z positif untuk menghadapi sumber GameObjects, masukkan Aim Vector 0, 0, 1 untuk X, Y, dan Z axes, masing-masing.
Up Vector Menentukan sumbu atas GameObject ini. Misalnya, untuk menentukan bahwa GameObject harus selalu menjaga sumbu Y positif menunjuk ke atas, masukkan Up Vector 0, 1, 0 untuk X, Y, dan Z sumbu, masing-masing.
World Up Type Menentukan sumbu untuk arah ke atas. Kontras Aim menggunakan vektor ini untuk menyelaraskan sumbu atas GameObject arah ke atas.

Constraint Settings
Lock Aktifkan pengaturan ini untuk membiarkan Constraint memutar GameObject. Periksa properti ini untuk mengedit rotasi GameObject ini. Anda juga dapat mengedit Rotasi Di Rest dan Rotation Offset properti. Jika Aktif diperiksa, Kontras memperbarui Rotasi Di Rest atau Rotasi Offset properti untuk Anda saat Anda memutar GameObject atau sumber GameObjects. Ketika Anda puas dengan perubahan, periksa Kunci untuk membiarkan kontrol Kontras GameObject ini. Properti ini tidak berpengaruh dalam Mode Bermain.
Rotasi Di Rest X, Y, dan nilai-nilai Z untuk digunakan ketika Berat 0 atau ketika yang sesuai Beku Rotasi Axes tidak diperiksa. Untuk mengedit bidang ini, uncheck Lock.
Rotasi Offset X, Y, dan Z offset dari rotasi yang dihitung oleh Kontras. Untuk mengedit bidang ini, uncheck Lock.
Beku Rotasi Axes Periksa X, Y, atau Z untuk memungkinkan Kontras untuk mengontrol sumbu yang sesuai. Memeriksa sumbu untuk menghentikan Kontras dari mengendalikannya. Ini memungkinkan Anda untuk mengedit, menghidupkan, atau skrip sumbu yang tidak beku.
Sources Daftar GameObjects yang membatasi GameObject ini. Unity mengevaluasi sumber GameObjects dalam urutan bahwa mereka muncul dalam daftar ini. Order ini mempengaruhi bagaimana Kontratraint ini berputar GameObject yang terlatih. Untuk mendapatkan hasil yang Anda inginkan, tarik dan drop item dalam daftar ini. Setiap sumber memiliki berat dari 0 hingga 1.
