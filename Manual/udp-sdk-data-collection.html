UDP SDK

Untuk berbagai toko aplikasi, penerbit aplikasi harus menentukan data pengumpulan aplikasi mereka, termasuk data yang dikumpulkan oleh SDK pihak ketiga terintegrasi seperti UDP. Untuk kenyamanan Anda, UDP menyediakan informasi tentang praktik pengumpulan data di bawah ini.

Important: Pengungkapan data di bawah ini hanya untuk SDK UDP. Anda juga bertanggung jawab untuk memberikan pengungkapan tambahan untuk aplikasi Anda, termasuk SDK pihak ketiga lainnya yang digunakan dalam aplikasi Anda.
