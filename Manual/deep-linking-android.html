Tautan dalam di Android

Tautan dalam adalah tautan URL di luar aplikasi Anda yang mengarahkan pengguna ke lokasi di aplikasi Anda. Ketika pengguna mengklik tautan dalam untuk aplikasi, sistem operasi membuka aplikasi Unity di tempat tertentu (misalnya, adegan tertentu). Untuk informasi lebih lanjut tentang tautan mendalam dan cara menggunakannya, lihat Deep links.

Mengaktifkan tautan mendalam untuk aplikasi Android

Sebelum Anda dapat memproses tautan dalam, Anda perlu mengkonfigurasi aplikasi Anda untuk bereaksi terhadap mereka. Bagian ini berisi instruksi tentang bagaimana mengatur tautan dalam untuk Android.

Untuk mengaktifkan tautan dalam untuk aplikasi Android, gunakan intent filter. Filter niat menimpa standar Android SDK untuk menyertakan bagian filter niat khusus untuk Activity. Untuk mengatur filter niat:

  1. Di jendela Proyek, pergi ke Assets > Plugins > Android.
  2. Buat file baru dan sebut AndroidManifest.xml. Unity secara otomatis memproses file ini ketika Anda membangun aplikasi Anda.
  3. Salin sampel kode berikut ke file baru dan menyimpannya.
<?xml version="1.0" encoding="utf-8"?>
<manifest xmlns:android="http://schemas.android.com/apk/res/android" xmlns:tools="http://schemas.android.com/tools">
  <application>
    <activity android:name="com.unity3d.player.UnityPlayerActivity" android:theme="@style/UnityThemeSelector" >
      <intent-filter>
        <action android:name="android.intent.action.MAIN" />
        <category android:name="android.intent.category.LAUNCHER" />
      </intent-filter>
      <intent-filter>
        <action android:name="android.intent.action.VIEW" />
        <category android:name="android.intent.category.DEFAULT" />
        <category android:name="android.intent.category.BROWSABLE" />
        <data android:scheme="unitydl" android:host="mylink" />
      </intent-filter>
    </activity>
  </application>
</manifest>

Aplikasi Android Anda sekarang buka ketika perangkat memproses tautan apa pun yang dimulai dengan unitydl://.

Menggunakan tautan mendalam di Android

Setelah Anda mengaktifkan tautan dalam untuk Android, cara yang Anda gunakan adalah platform-agnostic. Untuk informasi tentang cara menangani tautan dalam ketika aplikasi Anda terbuka, lihat Menggunakan tautan dalam.
