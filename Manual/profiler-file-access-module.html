File Access Profiler module

File Access ProfilerJendela yang membantu Anda untuk mengoptimalkan permainan Anda. Ini menunjukkan berapa banyak waktu yang dihabiskan di berbagai bidang permainan Anda. Sebagai contoh, dapat melaporkan persentase waktu yang dihabiskan rendering, aimating, atau dalam logika permainan Anda. More info
Lihat di Glossary modul menampilkan informasi tentang aktivitas file dalam aplikasi Anda, seperti jumlah membaca atau menulis operasi Unity melakukan atau jumlah pegangan file terbuka, baik untuk bingkai tertentu atau di semua bingkai dalam tangkapan. Anda dapat menggunakan informasi ini untuk membantu menentukan bagaimana efisien aplikasi Anda melakukan operasi file. Modul ini dapat menangkap informasi tentang operasi file pada setiap file dalam struktur folder aplikasi Anda yang dibangun, atau dalam folder proyek Unity Anda jika Anda menjalankan Profiler di Editor Unity. Modul ini tidak diaktifkan secara default. Untuk mengaktifkan modul File Access Profileer, buka jendela Profiler, pilih menu dropdown Profiler Modules, dan toggle kotak cemen.File Access checkbox.

Kategori grafik

File Access Profiler modul grafik dibagi menjadi lima kategori. Untuk mengubah urutan kategori dalam grafik, seret dan menjatuhkannya di legenda grafik. Anda juga dapat mengklik legenda berwarna kategori untuk memotong tampilannya di atau mati. Untuk informasi lebih lanjut tentang cara menggunakan jendela Profiler, lihat Memulai dengan jendela Profiler. Modul ini tidak mengumpulkan informasi tentang operasi file yang tidak berhasil. Jika upaya operasi file tidak berhasil, Unity menampilkan kesalahan.

Chart Description
Files Opened Jumlah total file yang berhasil dibuka dalam sistem file lokal bingkai ini.
Files Closed Jumlah total file yang berhasil ditutup dalam sistem file lokal bingkai ini.
File Seeks Jumlah operasi pencarian file yang dilakukan dalam sistem file lokal bingkai ini. Sebuah file mencari operasi melibatkan pencarian melalui konten file.
Reads in Flight Jumlah total operasi membaca yang sedang berlangsung selama bingkai ini.
File Handles Open Jumlah total pegangan file yang diadakan terbuka setiap saat selama bingkai ini. Ini termasuk file yang buka Unity dan penutupan dalam bingkai yang sama.

Detail modul pane

Ketika Anda memilih bingkai dalam modul File Access Profileer, detail modul pane menampilkan informasi rinci tentang file yang diakses Unity selama tangkapan. File Access Profiler modul memiliki dua pandangan:

  • Tampilan Ringkasan File menyediakan ringkasan semua aktivitas file untuk setiap akses Unity file selama pengaturan bingkai yang dipilih.
  • Tampilan akses menampilkan setiap operasi akses file dalam set data dalam urutan yang terjadi.

Gunakan menu dropdown di bagian atas dari detail pane toggle yang terlihat aktif. Secara default, tampilan File Summary aktif.

Kedua pandangan dapat menampilkan informasi untuk bingkai yang dipilih tunggal, atau untuk semua bingkai dalam tangkapan. Gunakan menu dropdown kedua di bagian atas dari detail pane untuk memilih antara satu bingkai atau semua bingkai. Modul ini tidak mengisi detail pane secara otomatis. Sebaliknya, Anda perlu memilih tombol Analyze Markers di bagian atas detail pane untuk mengisi tampilan dengan data.

Tampilan Ringkasan File

Tampilan File Summary menampilkan ringkasan semua operasi akses file yang terjadi pada setiap file selama tangkapan. Satu entri di tabel mewakili ringkasan penuh untuk satu file.

Tabel di bawah ini menjelaskan semua kolom yang memungkinkan tabel tampilan File Summary dapat ditampilkan. Tidak semua kolom diaktifkan secara default; untuk mengaktifkan atau menonaktifkan kolom, klik kanan di mana saja di baris yang berisi header kolom untuk membuka menu. Gunakan menu ini untuk beralih kolom apa pun di atau mati.

Tampilan akses

Tampilan Accesses memberikan informasi rinci tentang setiap operasi akses file dalam tangkapan. Setiap entri di tabel di pane rincian Profiler mewakili operasi akses file tunggal ke satu file. Tabel di bawah ini menjelaskan semua kolom yang mungkin tampilan Accesses.

Detail Description Enabled by default
Index Nomor ID yang mewakili urutan dalam Unity melakukan operasi akses. Yes
Filename Nama file yang diakses selama operasi ini. Untuk melihat path file secara penuh, hover kursor Anda di atas entri nama file. Yes
Type Jenis operasi akses dilakukan. Ini bisa berupa Open, Close, Seek, Read, atau Write. Yes
Access Size Jumlah byte ditransfer selama akses, di mana akses dibaca atau menulis. Dimana akses adalah pencarian file, ukurannya adalah tujuan yang diimbangi. Yes
Offset The (starting) mengimbangi akses file. Yes
Duration (ms) Waktu yang diambil untuk melakukan akses ini, dalam mili detik. Yes
Average Bandwidth (MBps) Bandwidth rata-rata untuk akses ini, di mana jenis akses dibaca. Yes

Go to the last frame of this marker Pilih bingkai di mana penanda terakhir ini muncul di grafik. Pilihan ini dinonaktifkan jika penanda hanya muncul pada satu bingkai.
