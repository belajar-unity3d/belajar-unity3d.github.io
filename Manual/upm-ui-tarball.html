Menginstal paket dari file tarball lokal

Manajer Paket dapat memuat paket dari file tarball yang disimpan secara lokal. Ini adalah untuk skenario canggih di mana Anda memecahkan alur kerja penerbitan paket Anda menjadi bagian dan pengguna Anda dapat menggunakan produk perantara dari salah satu bagian tersebut.

Misalnya, jika Anda telah mengatur integrasi terus menerus (CI) pada repositori paket kustom Anda, Anda dapat menggunakan API Paket npm atau Unity Package Manager untuk membuat file tarball Pack dari folder paket. Dalam hal itu, Anda mungkin ingin menguji file tarball sebelum Anda mempublikasikannya ke registry kustom.Gzip tarball file from a package folder. In that case, you likely want to test the tarball file before you publish it to a custom registry.

Untuk memuat paket dari file tarball lokal:

  1. Klik tombol add di bilah status.

  2. Pilihan untuk menambahkan paket muncul.

  3. Pilih Add package from tarball dari menu tambahan untuk membawa browser file.

  4. Navigasi ke folder di mana Anda menyimpan tarball Anda.

    Note: Manajer Paket hanya mengenali tarball dengan ekstensi .tgz.

  5. Klik dua file tarball di browser file.

Browser file menutup, dan paket sekarang muncul di Daftar paket dengan label .
