Instalasi Unity offline tanpa Hub

The Unity Download Asisten mendukung penyebaran offline. Ini memungkinkan Anda untuk mengunduh semua file yang diperlukan untuk menginstal Unity, dan untuk menghasilkan skrip untuk mengulang instalasi yang sama pada komputer lain tanpa akses internet.

Preparation

Jalankan Asisten Unduh, dan pasang Unity seperti normal pada satu komputer. Komputer ini harus memiliki ruang disk gratis yang cukup untuk mengunduh semua file. Klik dropdown dan pilih Custom, lalu pilih lokasi yang ingin Anda unduh file ke.

Cek semua yang Anda butuhkan

Buka manajer file PC Anda, arahkan ke folder lokasi kustom yang Anda tentukan sebelumnya, dan cari file .sh atau .bat di dalam folder tersebut. Periksa isi file ini. Perlu terlihat mirip dengan contoh berikut:

Menyebarkan Unity ke komputer lain

Windows

  1. Salin seluruh folder ke PC Windows target, dan menjalankan file .bat yang disediakan.
  2. Untuk menghindari prompt Windows UAC, menjalankan install.bat dari shell Administrator. Pada menu Start, cari cmd.exe, klik kanan, dan pilih Run as administrator.

Mac

  1. Salin seluruh folder ke mesin Mac OS X target, dan menjalankan file .sh yang disediakan. Sitemapsudo install.sh.
  2. Navigasi ke folder dengan skrip. Ini biasanya akan berada di folder Unduhan Anda (cd ~/Downloads/UnityPackages).
  3. Anda dapat mengulangi instruksi ini sebanyak yang Anda butuhkan untuk setiap komputer Anda ingin menginstal Unity.
