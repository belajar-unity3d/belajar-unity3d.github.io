Dukungan untuk format file model eksklusif

Unity mendukung sejumlah format eksklusif model fileFile yang berisi data 3D, yang dapat mencakup definisi untuk jaring, tulang, animasi, bahan dan tekstur. More info
Lihat di Glossary. Anda tidak boleh menggunakan format file ini dalam produksi; bukan, ekspor ke format .fbx di mana pun mungkin. Untuk informasi lebih lanjut, lihat Format file model.

Sitemap Halaman ini berisi panduan untuk menggunakan format file eksklusif yang menggunakan konversi FBX. Namun, ada dua format file yang tidak menggunakan FBX sebagai perantara: Note: dan SketchUp. Untuk informasi lebih lanjut tentang batasan dengan format file ini, lihat SpeedTree dan Pengaturan SketchUp.SpeedTree.

Ketika Unity mengimpor file kepemilikan, itu meluncurkan perangkat lunak pemodelan 3D di latar belakang. Unity kemudian berkomunikasi dengan perangkat lunak eksklusif untuk mengkonversi file asli ke format Unity dapat membaca.

Pertama kali Anda mengimpor file eksklusif ke Unity, perangkat lunak pemodelan 3D harus meluncurkan dalam proses garis perintah. Ini dapat mengambil sementara, tetapi impor berikutnya sangat cepat.

Requirements

Anda perlu memiliki perangkat lunak model 3D yang dipasang untuk mengimpor file eksklusif langsung ke Unity. Jika Anda tidak memiliki perangkat lunak yang diinstal, gunakan format FBX.

Untuk informasi lebih lanjut tentang mengimpor file FBX, lihat Model Jendela Pengaturan impor.

Objek Penting Dari Autodesk® 3ds Max®

Unity mengimpor file Autodeskd 3ds Max® (.max) melalui format FBX, mendukung berikut:

  • Semua node dengan posisi, rotasi dan skala; poin pivot dan nama juga diimpor
  • Mesh dengan warna simpul, normal dan satu atau lebih set UV
  • Bahan dengan tekstur dan warna yang diffuse. Beberapa bahan per mesh
  • Animations
  • Animasi berbasis tulang
  • Target Morph (bentuk halus)
  • Visibility

Mengimpor objek dari Cheetah3D

Unity mengimpor file Cheetah3D (.jas) melalui format FBX, mendukung berikut:

  • Semua node dengan posisi, rotasi dan skala; poin pivot dan nama juga diimpor
  • Mesh dengan simpul, poligon, segitiga, UV, dan normal
  • Animations
  • Bahan dengan warna dan tekstur yang diffuse

Mengimpor objek dari Modo

Impor Unity File Modo (.lxo) melalui format FBX, mendukung berikut:

  • Semua node dengan posisi, rotasi dan skala; poin pivot dan nama juga diimpor
  • Mesh dengan vertices, normal dan UV.
  • Bahan dengan Tekstur dan warna diffuse; beberapa Bahan per mesh
  • Animations

Untuk memulai, simpan file .lxo Anda di folder Aset Proyek Anda. Dalam Unity, file muncul di Tampilan Proyek.

Unity re-imports Asset ketika mendeteksi perubahan dalam file .lxo.

Mengimpor objek dari Lightwave

Impor Unity file Lightwave melalui format FBX, mendukung berikut:

  • Semua node dengan posisi, rotasi dan skala; poin pivot dan nama juga diimpor
  • Mesh dengan hingga 2 saluran UV
  • Normals
  • Bahan dengan Tekstur dan warna diffuse; beberapa bahan per mesh
  • Animations
  • Animasi berbasis tulang

Anda juga dapat mengkonfigurasi Lightwave AppLink plug-inSatu set kode yang dibuat di luar Unity yang menciptakan fungsi dalam Unity. Ada dua jenis plug-ins yang dapat Anda gunakan di Unity: Managed plug-ins (diproduksi. Rakitan NET dibuat dengan alat-alat seperti Studio Visual) dan plug-ins asli (pustaka kode asli yang spesifik platform). More info
Lihat di Glossary yang secara otomatis menyimpan pengaturan ekspor FBX Anda menggunakan pertama kalinya Anda mengimpor file adegan Lightwave Anda menjadi Unity. Untuk informasi lebih lanjut, lihat Sitemap Dokumentasi pertukaran.

Limitations

Bake bahan spesifik Lightwave Anda sebagai tekstur sehingga Unity dapat membacanya. Untuk informasi tentang melakukan ini menggunakan pipa yang tidak merusak, lihat Sistem node di Lightwave.

Unity tidak mendukung splines atau patch. Mengkonversi semua splines dan patch ke poligon sebelum menyimpan dan mengekspor ke Unity. Untuk informasi lebih lanjut, lihat Antivirus.

Mengimpor objek dari Blender

Unity mengimpor file Blender (.blend) melalui format FBX, mendukung berikut:

  • Semua node dengan posisi, rotasi dan skala; poin pivot dan nama juga diimpor
  • Mesh dengan simpul, poligon, segitiga, UV, dan normal
  • Bones
  • Skinned Meshes
  • Animations

Limitations

Tekstur dan warna diffuse tidak ditugaskan secara otomatis. Anda dapat secara manual menetapkan mereka dengan menyeret tekstur ke meshGrafik utama primitif Unity. Mesh membuat sebagian besar dunia 3D Anda. Unity mendukung mesh poligon triangulat atau Quadrangulasi. Nurbs, Nurms, permukaan Subdiv harus dikonversi ke poligon. More info
