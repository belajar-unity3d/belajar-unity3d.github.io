Scenes

Adegan di mana Anda bekerja dengan konten di Unity. Mereka adalah aset yang mengandung semua atau bagian dari permainan atau aplikasi. Misalnya, Anda mungkin membangun permainan sederhana dalam satu scene, sementara untuk permainan yang lebih kompleks, Anda mungkin menggunakan satu adegan per tingkat, masing-masing dengan lingkungan, karakter, hambatan, dekorasi, dan UI(User Interface) Memungkinkan pengguna untuk berinteraksi dengan aplikasi Anda. Unity saat ini mendukung tiga sistem UI. More info
Lihat di Glossary. Anda dapat membuat sejumlah adegan dalam proyek.

Ketika Anda membuat proyek baru dan membukanya untuk pertama kalinya, Unity membuka adegan sampel yang hanya mengandung CameraKomponen yang menciptakan gambar sudut pandang tertentu di tempat kejadian Anda. Output ditarik ke layar atau ditangkap sebagai tekstur. More info
Lihat di Glossary dan Cahaya.

Untuk informasi tentang bekerja dengan adegan, lihat Membuat, memuat, dan menyimpan adegan.

Template Adegan

Unity menggunakan template adegan untuk membuat adegan baru. Templat adegan adalah aset yang disimpan dalam proyek. Mereka mirip dengan adegan, tetapi dirancang untuk disalin daripada digunakan secara langsung.

Untuk informasi tentang membuat dan menggunakan template adegan, lihat Adegan template.

Dialog Adegan Baru

Dialog Adegan Baru dibuka ketika Anda membuat adegan baru dari menu File: (File > New Scene) atau shortcut Ctrl/Cmd + n. Gunakan untuk membuat adegan baru dari template adegan tertentu di proyek Anda, dan mendapatkan informasi tentang template yang ada.


Dialog Adegan Baru

  1. Search field: menemukan template adegan yang tersedia dengan nama.
  2. Templates: daftar semua template yang tersedia dalam proyek.
  3. Template details: menampilkan informasi tentang templat yang dipilih saat ini.
  4. Command bar: menyediakan perintah dan pilihan untuk membuat adegan baru dari template yang dipilih.

Membuat adegan baru

Untuk membuat adegan baru dari dialog Adegan Baru, pilih template dari daftar template, dan klik Create. Untuk deskripsi rinci tentang menciptakan adegan cara ini, lihat Membuat adegan baru dari baru adegan dialog.

Pinning templates

Templat yang dipinjam muncul sebelum template lain di daftar template dialog Adegan Baru. Template terakhir yang dipinned muncul di bagian atas daftar.

Klik ikon pin template untuk pin itu.

Anda juga dapat mencubit template ketika Anda edit sifatnya. Dalam template adegan InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info
Lihat di Glossary, mengaktifkan opsi Pin in New Scene Dialog.

Locating and editing templates

Ketika Anda memilih template di dialog Adegan Baru, pane rincian menampilkan path ke template (1 di gambar di bawah).

Untuk menyoroti template di Project windowJendela yang menunjukkan isi folder Assets (Project tab) More info
Lihat di Glossary, pilih tautan Locate (2 di gambar di bawah).

Untuk membuka template di jendela Inspektur dan edit sifatnya, pilih tautan Edit (3 di gambar di bawah).

Multi-scene pengeditan

Anda dapat membuka beberapa adegan untuk mengedit pada saat yang sama. Untuk rincian, lihat Multi-scene pengeditan.
