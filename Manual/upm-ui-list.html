Jendela Paket Manager menampilkan daftar set fitur, paket, atau paket Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
Lihat di Glossary sesuai dengan kriteria yang Anda pilih dengan menyaring, termasuk, atau mencari:

(A) Fitur setSebuah fitur set adalah koleksi paket terkait yang dapat Anda gunakan untuk mencapai hasil tertentu dalam Editor Unity. Anda dapat mengelola set fitur langsung di Manajer Paket Unity. More info
Lihat di Glossary muncul di bagian atas tampilan daftar, ditunjukkan dengan ikon set fitur (). Toggle ikon expander untuk menunjukkan atau menyembunyikan daftar set fitur. Ketika Anda memilih set fitur dari daftar, rincian tentang itu muncul di kanan Meme it, termasuk deskripsi singkat, daftar paket disertakan, dan tautan ke panduan QuickStart untuk set fitur itu.

Sitemap Kelompok Registry mengatur daftar paket yang diinstal dalam proyek Anda (atau tersedia untuk instalasi ketika Anda berada dalam konteks (B)). Pilih expander di sebelah kiri untuk menunjukkan atau menyembunyikan daftar paket untuk setiap registry.My Registries context). Select the expander on the left to show or hide the list of packages for each registry.

Sitemap Pilih ikon expander ini untuk menunjukkan dan menyembunyikan daftar versi yang diinstal atau tersedia.(C) Select these expander icons to show and hide the list of versions installed or available.

Sitemap Versi paket menampilkan versi paket yang diinstal. Jika paket belum diinstal, versi yang muncul adalah versi yang disarankan. Ikon khusus sering muncul di samping nomor versi yang menunjukkan (D)nya:state:

  • Released
  • Release Candidate
  • Pre-release
  • Experimental
  • Custom (dirakit)

Untuk paket Asset Store, versi yang muncul adalah versi yang telah Anda unduh atau versi yang tersedia untuk diunduh dari Asset Store.

Sitemap Ikon ini menampilkan status paket atau paket Aset Store:(E) These icons display the status of the package or Asset Store package:

Icon: Description
Tanda cek menunjukkan bahwa paket atau set fitur sudah installed atau enabled.

Note: Menginstal paket Asset Store menampilkan ikon impor () jika mereka up to date.
Ikon pembaruan menunjukkan bahwa paket memiliki pembaruan yang tersedia. Untuk memperbarui paket Anda, ikuti petunjuk untuk jenis paket:

- Untuk paket Unity, lihat Beralih ke versi paket lain.
- Untuk paket Asset Store, lihat Memperbarui paket Aset Store Anda.
- Versi set fitur tetap ke versi Editor dan tidak dapat diperbarui.
Ikon impor menunjukkan bahwa ada paket Asset Store yang tersedia untuk impor. Hal ini juga muncul ketika Anda telah mengimpor paket Asset Store di proyek.
Ikon unduhan menunjukkan bahwa ada paket Asset Store yang tersedia untuk diunduh.
Ikon kesalahan menunjukkan bahwa sesuatu yang salah selama instalasi atau pemuatan. Untuk lebih banyak saran tentang menyelesaikan kesalahan, lihat Pesan kesalahan.

Sitemap konteks (F) menampilkan tautan yang menampilkan jumlah paket Aset Store tersedia tetapi tidak ditampilkan dalam daftar. Untuk memuat sisa paket Aset Store, klik tautan bernomor.My Assets context displays a link showing the number of Asset Store packages available but not shown in the list. To load the rest of the Asset Store packages, click the numbered link.

Note: Jika Anda memilih konteks My Assets tetapi Anda tidak melihat paket apa pun, lihat Pesan kesalahan di jendela Paket Manager untuk penjelasan dan solusi.

Sitemap (G) menampilkan pesan tentang status beban paket dan peringatan jaringan.status bar displays messages about the package load status and network warnings.

Sitemap Klik (H) Click the reloadbutton untuk memaksa Paket Manager untuk memuat ulang paket dan set fitur Anda.button to force Package Manager to reload your packages and feature sets.

Secara default, jendela Paket Manager menampilkan daftar semua paket dan set fitur yang tersedia pada Unity Registry, tetapi Anda dapat filter daftar untuk menampilkan paket dan set fitur yang sesuai dengan konteks tertentu.

Anda juga dapat termasuk paket pra-release dalam daftar dan search untuk paket atau fitur tertentu yang ditetapkan oleh nama (ID) atau nama tampilan.
