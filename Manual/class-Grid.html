Grid

Switch to Scripting

Komponen Grid adalah panduan yang membantu menyelaraskan GameObjectsObjek mendasar dalam adegan Unity, yang dapat mewakili karakter, props, pemandangan, kamera, waypoints, dan banyak lagi. Fungsi GameObject didefinisikan oleh Komponen yang melekat padanya. More info
Lihat di Glossary, seperti TilesSebuah kelas sederhana yang memungkinkan sprite untuk diberikan di sebuah Tilemap. More info
Lihat di Glossary, berdasarkan tata letak yang dipilih. Komponen mengubah posisi sel Grid ke local coordinates yang sesuai dari GameObject. Komponen Transform kemudian mengkonversi koordinat lokal ini ke ruang dunia atau global coordinates.

Property Function
Cell Size Ukuran sel di Grid ini.
Cell Gap Masukkan ukuran (dalam unit Unity) celah antara sel di Grid ini. Jika nomor negatif dengan nilai absolut lebih tinggi dari Cell Size dimasukkan, maka Unity akan secara otomatis mengubah nilai absolut untuk mencocokkan Cell Size bukan.
Contohnya, jika Cell Size adalah (1, 1, 0) dan Cell Gap diatur ke (–2, -2, 0), Editor akan secara otomatis mengubah nilai Cell Gap menjadi (–1, 0) bukan.
Cell Layout Pilih opsi dari menu drop-down untuk menentukan bentuk dan pengaturan sel di Grid ini.
Rectangle Sel persegi panjang.
Hexagon Sel heksagonal.
Isometric Sel berbentuk rhombus untuk tata letak isometrik.
Isometric Z as Y Mirip dengan tata letak Isometric Grid, tetapi Unity mengubah posisi Z sel menjadi koordinat Y lokal mereka.
Cell Swizzle Pilih urutan yang disatukan ulang koordinat sel XYZ untuk mengubah konversi. Lihat artikel Wikipedia di Swizzling untuk informasi lebih lanjut.
    XYZ Komponen Grid menggunakan koordinat sel XYZ default.
    XZY Komponen Grid membalikkan koordinat XYZ ke XZY.
    YXZ Komponen Grid mendamaikan koordinat XYZ ke YXZ.
    YZX Komponen Grid mengubah urutan XYZ berkoordinasi ke YZX.
    ZXY Komponen Grid mengubah urutan koordinasi XYZ ke ZXY.
    ZYX Komponen Grid mendamaikan koordinat XYZ ke ZYX.
