Memperbarui paket Aset Store Anda

Anda dapat memperbarui paket Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
Lihat di Glossary di jendela Paket Manager:

  1. Buka jendela Paket Manager dan pilih My Assets dari menu PaketPaket adalah koleksi aset untuk dibagikan dan digunakan kembali dalam Unity. Manajer Paket Unity (UPM) dapat menampilkan, menambahkan, dan menghapus paket dari proyek Anda. Paket-paket ini berasal dari Unity Package Manager dan memberikan metode mendasar untuk memberikan fungsi Unity. Namun, Unity Package Manager juga dapat menampilkan Paket Asset Store yang Anda unduh dari Asset Store. More info
    Lihat di Glossary    .

    Indikator panah muncul di samping paket Asset Store yang memiliki pembaruan yang tersedia.

  2. Untuk melihat pembaruan terbaru untuk paket Aset Store, sort the list oleh Recently updated.

  3. Pilih paket Asset Store yang ingin Anda perbarui dari Daftar paket. Informasi paket Asset Store muncul di details view.

  4. Pada Daftar paket, klik ikon expander ke kiri nama paket Asset Store.

    Versi terbaru menampilkan Recommended dan versi diinstal menampilkan Currently downloaded.

  5. Pilih versi baru untuk paket Asset Store saat ini dan lihat rincian spesifik untuk versi itu di details view.

  6. Klik tombol Update.

    Note: Jika versi yang diunduh dari paket Anda tidak kompatibel dengan versi Editor Unity Anda, pesan muncul dalam tampilan detail dan tampilan tombol Update alternatif, yang memungkinkan Anda mengunduh versi paket yang kompatibel.

    Ketika selesai bar kemajuan, informasi versi paket Asset Store baru muncul di details view, dan segera tersedia.
