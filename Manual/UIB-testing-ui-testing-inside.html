Pengujian di dalam Pembuat UI

Untuk menguji UI(User Interface) Memungkinkan pengguna untuk berinteraksi dengan aplikasi Anda. Unity saat ini mendukung tiga sistem UI. More info
Lihat di Glossary Anda di dalam UI Builder, langsung di dalam Canvas, Anda dapat mengaktifkan mode Preview, ditemukan di ViewportThe user’s visible area of an app on their screen.
See in Glossary toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
Lihat di Glossary. Modus Preview menghapus overlay dan pegangan penjemputan yang spesifik dari Canvas. Anda dapat memberi tahu apakah Anda memiliki mode Preview diaktifkan dengan mencari batas oranye di seluruh Viewport:

Dengan mode Preview diaktifkan, elemen Dokumen UI Anda (UXML) pada Canvas akan mulai menerima acara input, memungkinkan Anda menguji:

  • Kontrol berbasis negara seperti Foldout, yang dapat Anda klik untuk memperluas dan melihat bagaimana ulang UI bereaksi.
  • Bidang masukan seperti IntegerField, di mana Anda dapat menguji validasi input.
  • Wadah besar seperti ScrollView, di mana Anda dapat menguji gulir ke atas dan ke bawah.
  • :hover pseudo menyatakan untuk memeriksa gaya hover-only.

Sementara dalam mode Preview, Anda masih dapat terus memodifikasi Dokumen UI Anda (UXML) dan Lembar Style menggunakan pane StyleSheets, pane Hierarchy, dan InspectorA Unity window that displays information about the currently selected GameObject, asset or project settings, allowing you to inspect and edit the values. More info
See in Glossary. Canvas picking dan manipulator dimatikan dalam mode Preview.
