Filter Prefab

Filter prefab dapat digunakan dengan SceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary dan Object Search Providers.

Cari semua GameObjects yang merupakan bagian dari Prefab yang bukan dalam aset Prefab.

Cari semua GameObjects yang merupakan bagian dari instance atau aset prefab biasa.

File Filters

Filter file dapat digunakan dengan Penyedia Pencarian Aset dan Objek.

Filter Search token Description
Default Search <search term> Cari term mencoba untuk mencocokkan nama aset, jenis, atau path.

texture

Searches all Assets that have the word texture in their name, path, or type.
Name name: name:laser

Cari semua Aset yang mengandung kata laser.

name=laserbeam

Searches all Assets where the name is exactly laserbeam.
Directory dir:<directory exact name> dir:Scripts

Cari semua Aset yang terkandung dalam direktori dengan nama yang tepat Scripts.
Packages a:packages a:packages texture

Cari semua tekstur dalam setiap paket.
Project a:assets a:assets texture

Cari semua tekstur di folder Aset proyek saat ini.
Index file a:<index name> a:psd_textures texture

Assuming ada file indeks bernama psd_textures.index dalam proyek, mencari semua tekstur dalam indeks itu.
Size size:<number of bytes> size:4000 texture

Cari semua tekstur lebih dari 4000 byte (4KB).
Extension ext:<file extension without period> ext:png texture

Cari semua tekstur dengan ekstensi png.
Age age:<number of days since last modification> age<3 texture

Cari semua tekstur yang dimodifikasi dalam 3 hari terakhir.

Pencarian Properti yang Diindeks

Sifat pencarian tersedia jika indeks telah ditentukan dengan opsi Indeks Properti (lihat Index Manager). Untuk melihat daftar semua properti yang diindeks memeriksa tab Kata Kunci Manajer Indeks. Semua nilai properti dikonversi menjadi string atau nomor. Nama properti harus lengkap dan tidak sebagian (kasus tidak masalah meskipun). Properti indeks persatuan pada objek tingkat atas Aset PrefabJenis aset yang memungkinkan Anda untuk menyimpan GameObject lengkap dengan komponen dan properti. Prefab bertindak sebagai template dari mana Anda dapat membuat instance objek baru di tempat kejadian. More info
Lihat di Glossary. Jika Anda ingin semua hirarkis Prefab untuk diindeks, buat Indeks Prefab (lihat Index Manager). Untuk .unity file Unity mengindeks sifat AdeganAsset dan bukan konten Adegan. Jika Anda ingin semua konten Adegan yang akan diindeks, buat Indeks Adegan (lihat Index Manager).

Filter ketergantungan

Jika Anda menggunakan opsi Indexing Dependencies (Lihat indeks Unity) Index Manager dari semua Aset menggunakan direct dependenciesKetergantungan direct terjadi ketika proyek Anda “permintaan” versi paket tertentu. Untuk membuat ketergantungan langsung, Anda menambahkan paket dan versi ke properti dependencies dalam manifestasi proyek Anda (diekspresikan dalam bentuk package_name@package_version). More info
Lihat di Glossary.AssetDatabase.GetDependencies.
