Build di Penerbit

Jendela Pengaturan Build berisi semua pengaturan dan opsi yang Anda butuhkan untuk mempublikasikan build Anda ke berbagai platform. Dari jendela ini Anda dapat membuat Development BuildMembangun pengembangan termasuk simbol debug dan memungkinkan Profiler. More info
Lihat di Glossary untuk menguji aplikasi Anda, serta menerbitkan build akhir. Untuk menyesuaikan pengaturan penerbitan untuk build aplikasi Anda menjadi File > Build Settings.

Gunakan panel ScenesA Scene contains the environments and menus of your game. Think of each unique Scene file as a unique level. In each Scene, you place your environments, obstacles, and decorations, essentially designing and building your game in pieces. More info
See in Glossary in Build untuk mengelola tempat kejadian Unity termasuk dalam membangun. Anda dapat menggunakan bagian Platform dari jendela untuk memilih platform mana yang ingin Anda bangun, dan menyesuaikan pengaturan spesifik seperti CompressionA method of storing data that reduces the amount of storage space it requires. See Texture Compression, Animation Compression, Audio Compression, Build Compression.
See in Glossary Method. Pilihan ini bervariasi tergantung pada Platform yang Anda pilih. Untuk informasi lebih lanjut, lihat dokumentasi di Membangun Pengaturan

Pilih tombol Build atau Build And Run untuk memulai proses build.

Anda dapat memilih nama dan menyimpan lokasi untuk aplikasi Anda melalui Save dialog yang muncul. Note: tergantung pada platform yang Anda buat, Unity mungkin hanya meminta Anda untuk memilih folder.

Ketika Anda memilih tombol Save, Unity membangun aplikasi Anda. Jika Anda tidak yakin di mana untuk menyimpan build Anda, pertimbangkan membuat subfolder di dalam folder akar Anda untuk memegang build Anda. Anda tidak dapat menyimpan build ke folder Aset.
