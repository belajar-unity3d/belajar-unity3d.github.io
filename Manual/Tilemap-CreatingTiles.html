Membuat ubin

Ada dua cara untuk membuat Aset Tile di Unity; metode pertama adalah untuk langsung membuat Aset Tile dari menu Assets. Metode kedua adalah untuk secara otomatis menghasilkan aset ubin dari pemilihan SpritesObjek grafis 2D. Jika Anda digunakan untuk bekerja di 3D, Sprites pada dasarnya hanya tekstur standar tetapi ada teknik khusus untuk menggabungkan dan mengelola tekstur sprite untuk efisiensi dan kenyamanan selama perkembangan. Sprite Pengaturan impor.

Ketika mengimpor Sprites untuk digunakan di Tilemap, gunakan pengaturan yang disarankan berikut. Untuk informasi lebih lanjut tentang setiap pengaturan, lihat dokumentasi pada Jenis Tekstur: Sprite (2D dan UI).

  1. Texture Type - Mengatur ini menjadi Sprite (2D and UI). Jenis Tekstur lain tidak didukung untuk Tilemaps.
  2. Sprite Mode - Mengatur ini ke Single jika Tekstur hanya mengandung Sprite tunggal. Set ke Multiple jika mengandung beberapa elemen, misalnya lembaran ubin yang mengandung beberapa Tekstur ubin.
    Set Sprite Mode ke Multiple ketika mengimpor lembar Sprite.
  3. Pixels Per Unit (PPU) - Nilai ini adalah jumlah piksel yang membuat satu unit Unity untuk Sprite yang dipilih. Ini menentukan ukuran Sprite Tile ketika dirender di Tilemap. Disarankan untuk mengatur nilai ini ke lebar (dalam pixel) dari Sprite ubin tunggal sehingga ubin sama dengan satu unit Unity lebar. Untuk ubin Hexagonal dan Isometric, ukur lebar Sprites di ujungnya. Ukuran Sprit ubin juga dipengaruhi oleh properti Cell Size dari orang tua Grid dari Tilemap juga mempengaruhi ukuran Sprites ubin sebagai Ukuran Sel menentukan berapa banyak unit Unity yang sama dengan Sel tunggal.
    Dalam contoh di bawah ini, Sprites impor adalah 64x64. WordPress.org memiliki ukuran sel (XYZ: 1, 1, 0) Unity unit. Untuk membuat Sprite ubin pas persis pada satu Sel dari Tilemap, set nilai PPU untuk 64 untuk mencocokkan lebar piksel. Seluruh lebar Sprite kemudian sesuai dengan satu unit Unity, yang sama dengan lebar (nilai X: 1) dari satu Sel di Tilemap.

    Left: Sprit set ke 64 PPU. Middle: Same Sprites set ke 128 PPU. Right: Sprites set ke 32 PPU.
    Dengan menggandakan nilai PPU asli 64 ke 128, skala Sprite menjadi 0,5 (64px / 128) Unity unit lebar. Halving nilai PPU ke 32 bukan timbangan Sprites ke 2 (64px/32) Unity lebar. Sprit tampak lebih kecil atau lebih besar, namun posisi Sel mereka di Tilemap tetap tidak berubah.
  4. Sprite Editor Outlines - Setelah Sprites diimpor, perbaiki garis besar Sprit dengan membuka masing-masing dari mereka di Sprite Editor dan editingtheir outlines. Jika Tekstur diimpor dengan set Sprite Mode ke Multiple dan mengandung beberapa Sprites, kemudian edit garis besar masing-masing Sprites di editor.

Menghasilkan aset ubin secara otomatis dalam palet ubin

Ubin dapat secara otomatis dihasilkan dengan membawa Sprites individu atau Sprite lembar ubin langsung ke jendela Tile Palette (jika jendela tidak terbuka, pergi ke Window > 2D > Tile Palette).

Pertama persiapan dan impor Sprites ubin, kemudian buka menu drop-down 'New Palette' di jendela palet ubin. Pilih palet ubin yang ada dari daftar, atau pilih 'Create New Palette' untuk membuka kotak dialog Create New Palette.


Menu drop-down Palet Baru.

Setelah memilih sifat dari palet ubin (lihat dokumentasi Palet ubin untuk informasi lebih lanjut tentang pilihan yang berbeda), pilih Create untuk membuka palet baru di jendela.


Pilih 'Create' untuk membuat palet ubin baru.

Setelah membuat atau memuat palet, drag dan drop Textures atau Sprites dari folder Assets langsung ke jendela palet ubin. Anda akan diminta untuk lokasi simpan untuk Aset Ubin baru. Pilih lokasi simpan, dan aset ubin baru yang dihasilkan di folder yang dipilih. Ubin yang dihasilkan juga ditempatkan secara otomatis ke palet.

Aset Tile Deleting

Ketika Anda menghapus Aset Tile, mereka digantikan oleh ubin placeholder terbuat dari kaleng persegi putih dengan berbagai nuansa pink. Ubin placeholder ditempatkan di Tilemaps pada posisi asli Aset Tile yang dihapus. Ini membantu untuk mengidentifikasi Aset Tile yang telah dihapus dalam proyek, apakah sengaja atau tidak sengaja.

Setiap dihapus Aset Tile memiliki ubin placeholder yang sesuai dengan warna pink yang unik yang membedakan placeholders dari satu sama lain. Anda dapat mengganti ubin placeholder ini dengan ubin lain dengan menggunakan Alat Isi Banjir atau menghapusnya.


Left: Tile Palette dengan tiga aset ubin berwarna yang berbeda (dinamikan setelah warna masing-masing - Putih, Merah dan Biru). Right: Tile Palette setelah menghapus Aset Ubin Putih dan Biru. Aset Ubin Merah belum dihapus.
