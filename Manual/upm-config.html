Configuration

Bagian ini menunjukkan cara mengkonfigurasi berikut untuk Manajer Paket:

  • Konfigurasi jaringan: mendapatkan sekitar firewall dan mengatur server proxy.
  • Otentikasi registry tersedak: membuat dan menyediakan Manajer Paket dengan token otentikasi untuk setiap server registry paket yang membutuhkannya.
  • Sesuaikan lokasi cache global: menimpa lokasi folder cache global.
  • Sesuaikan lokasi cache Aset Store: menimpa lokasi folder cache paket Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
    Lihat di Glossary    .

Selain itu, Anda dapat menemukan lokasi file konfigurasi Paket Manager di bawah File konfigurasi.

File konfigurasi

Paket Manager mendukung dua file konfigurasi: file konfigurasi global dan file konfigurasi pengguna. Kedua file ini menggunakan format TOML dan mereka muncul di lokasi yang berbeda:

  • file konfigurasi GlobalSertifikat SSL berlaku untuk semua pengguna di mesin. Misalnya, Anda dapat mendefinisikan tambahan ketika menyiapkan server proxy untuk seluruh mesin.
  • File konfigurasi Usertoken otentikasi berlaku untuk satu pengguna. Misalnya, Anda dapat mengatur untuk digunakan untuk paket kustom server registry yang Anda akses dengan registries ruang lingkup. Token ini otentikasi akun pengguna tertentu.

Lokasi file konfigurasi global

Paket Manager menggunakan file konfigurasi global bernama upmconfig.toml, yang dapat Anda temukan di lokasi berikut:

Environment: Location:
Windows %ALLUSERSPROFILE%\Unity\config\upmconfig.toml
macOS dan Linux /etc/upmconfig.toml

Lokasi file konfigurasi pengguna

Paket Manager menggunakan file konfigurasi pengguna bernama .upmconfig.toml, yang dapat Anda temukan di lokasi berikut:

Environment: Location:
Windows (user account) %USERPROFILE%\.upmconfig.toml (misalnya, C:\Users\myusername\.upmconfig.toml)
Windows (akun pengguna sistem) %ALLUSERSPROFILE%\Unity\config\ServiceAccounts\.upmconfig.toml (misalnya, C:\Users\Public\Unity\config\ServiceAccounts\.upmconfig.toml)
macOS dan Linux ~/.upmconfig.toml (misalnya, /Users/myusername/.upmconfig.toml)
