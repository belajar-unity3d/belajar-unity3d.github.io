Sitemap Pengaturan ini untuk mengimpor Model yang dibuat dalam SpeedTree. Untuk informasi tentang Model dan animasi yang dibuat dalam aplikasi pemodelan 3D lainnya, lihat jendela Note:.Model Pengaturan impor window.

Ketika Anda menempatkan file SpeedTree di folder Unity Project Assets Anda, Unity secara otomatis mengimpor dan menyimpannya sebagai Aset Unity. Untuk melihat pengaturan impor di InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. Lihat di Glossary, klik pada file di jendela ProjectDalam Unity, Anda menggunakan proyek untuk merancang dan mengembangkan permainan. Proyek menyimpan semua file yang terkait dengan permainan, seperti file aset dan Adegan. More info
Lihat di Glossary. Untuk menyesuaikan bagaimana Unity mengimpor file yang dipilih, gunakan sifat pada tab Model dan Materials di jendela ini:


Unity mengenali dan mengimpor Aset Model SpeedTree dengan cara yang sama yang menangani Aset lain. Jika Anda menggunakan SpeedTree Modeler 7, pastikan untuk memasukkan kembali file .spm Anda menggunakan versi Unity dari Modeler. Jika Anda menggunakan SpeedTree Modeler 8, simpan file .st Anda langsung ke folder Proyek Unity. Importer SpeedTree menghasilkan Prefab dengan komponen LODGroup dikonfigurasi. Anda dapat meluruskan Prefab dalam Adegan sebagai instance Prefab umum, atau memilih Prefab sebagai prototipe pohon dan melukis di Terrain.


Anda dapat mengekstrak MaterialsAset yang menentukan bagaimana permukaan harus diberikan. More info
