Property **Function
Animator Gunakan bidang ini untuk mendefinisikan komponen Animator yang Anda inginkan Jaringan Animator untuk disinkronkan.

Details

Animator Jaringan memastikan sinkronisasi animasi GameObject di jaringan - berarti bahwa semua pemain melihat animasi terjadi sama. Ada dua jenis wewenang untuk animasi berjaring (lihat dokumentasi pada Konsep sistem jaringan untuk informasi lebih lanjut tentang wewenang):

  • Jika GameObject memiliki wewenang pada klien, Anda harus menghidupkannya secara lokal pada klien yang memiliki GameObject. Klien itu mengirimkan informasi state animasi ke server, yang menyiarkannya ke semua klien lainnya. Misalnya, ini akan cocok untuk karakter pemain.

  • Jika GameObject memiliki wewenang pada server, maka Anda harus menghidupkannya di server. Server kemudian mengirimkan informasi state ke semua klien. Ini umum untuk animasi GameObjects yang tidak terkait dengan klien tertentu, seperti karakter non-pemain.

Jaringan Animator menyinkronkan animation parametersDigunakan untuk berkomunikasi antara scripting dan Animator Controller. Beberapa parameter dapat diatur dalam skrip dan digunakan oleh pengontrol, sementara parameter lain didasarkan pada Kurva Khusus dalam Klip Animasi dan dapat sampel menggunakan API scripting. More info
Lihat di Glossary yang diperiksa di jendela InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info
Lihat di Glossary. Ini tidak secara otomatis menyinkronkan pemicu animasi. Sebuah GameObject dengan wewenang dapat menggunakan fungsi SetTrigger untuk menembakkan pemicu animasi pada klien lain.

Fungsi GetParameterAutoSend dan SetParameterAutoSend dapat digunakan untuk mengontrol parameter animator individu mana harus disinkronkan secara otomatis.
