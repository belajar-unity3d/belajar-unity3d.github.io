Pengaturan Gambar Splash

Bagian Splash Image dari jendela pengaturan Player (menu: Edit > Project Settings > Player > Splash Image) memungkinkan Anda untuk mengkonfigurasi layar percikan untuk proyek Anda. Tampilan layar percikan ketika aplikasi Anda dimulai.

Anda dapat menggunakan pengaturan di jendela Splash Image untuk menyesuaikan bagaimana layar percikan muncul di aplikasi Anda, termasuk menonaktifkan logo Unity, menambahkan logo Anda sendiri, atau menonaktifkan layar percikan sepenuhnya. Namun, kustomisasi penuh dari layar percikan hanya tersedia bagi mereka di. Jika Anda berada di Rencana Pribadi, keterbatasan berikut berlaku untuk pengaturan Gambar Splash:

  • Anda tidak dapat menonaktifkan pengaturan Show Splash Screen untuk menonaktifkan layar percikan
  • Anda tidak dapat menonaktifkan pengaturan Show Unity Logo untuk menonaktifkan logo Unity
  • Pengaturan Overlay Opacity memiliki nilai minimum 0,5

Layar Splash Unity adalah sama di semua platform. Ini menampilkan segera sementara SceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary pertama dari aplikasi Anda memuat secara serentak di latar belakang. Namun, jika Anda menggunakan layar pengantar atau animasi Anda sendiri, ini mungkin lebih lama muncul karena Unity harus memuat seluruh mesin dan adegan pertama sebelum dapat menampilkan layar percikan kustom Anda sendiri.

Sitemap Unity menyalin opsi ini langsung ke file Note:.Package.appxmanifest file.

Login Pengaturan gambar

Untuk mengkonfigurasi opsi untuk layar percikan aplikasi Anda, buka jendela Pengaturan Pemain (menu: Edit > Project Settings > Player > Splash Image). Anda juga dapat menggunakan API documentation untuk mengontrol pengaturan layar percikan.

Durasi Layar Splash Menampilkan total durasi urutan layar percikan, dalam detik. Ini adalah total dari semua logo ditambah 0,5 detik untuk memudar keluar. Login Layar percikan aplikasi Anda mungkin ditampilkan lebih lama jika adegan pertama tidak siap bermain. Dalam hal ini, layar percikan hanya menunjukkan gambar latar belakang atau warna dan kemudian memudar ketika adegan pertama siap dimainkan.Note: Your application’s splash screen might display for longer if the first Scene isn’t ready to play. In this case, the splash screen shows only the background image or color and then fades out when the first Scene is ready to play.
Background Pengaturan untuk menyesuaikan latar belakang layar percikan aplikasi Anda.
Overlay Opacity Sesuaikan nilai pengaturan Overlay Opacity untuk membuat logo menonjol. Ini mempengaruhi warna latar belakang dan warna gambar logo, berdasarkan bagaimana Anda mengatur Splash Style (cahaya pada Gelap, atau Gelap pada Cahaya).

Mengatur opacity menjadi nilai yang lebih rendah untuk mengurangi efek ini. Anda juga dapat menonaktifkan efek dengan mengaturnya ke 0. Misalnya, jika Gaya Splash adalah Light on Dark, dengan latar belakang putih, latar belakang menjadi abu-abu jika Anda mengatur Opacity Overlay ke 1, dan putih jika diatur ke 0.

Important: Jika Anda memiliki rencana Pribadi Unity, pengaturan ini memiliki nilai minimum 0,5.
Warna Latar Belakang Mengatur warna untuk layar percikan. Unity menggunakan warna ini jika Anda tidak mengatur Background Image.

Catatan: Pengaturan Opacity Overlay memiliki efek pada tampilan Warna Latar Belakang, dan mungkin tidak cocok dengan warna yang ditugaskan.
Gambar Latar Belakang Blur Aktifkan pengaturan ini untuk mengaburkan Gambar Latar Belakang yang Anda set. Jika Anda menonaktifkan pengaturan ini, itu menampilkan Gambar Latar Belakang tanpa efek kabur.
Gambar Latar Belakang Mengatur referensi ke gambar Sprite untuk digunakan sebagai latar belakang. Unity menyesuaikan gambar latar belakang sehingga mengisi layar. Ini seragam mengukur gambar sampai cocok dengan lebar dan tinggi layar. Ini berarti bahwa bagian dari gambar mungkin meluas di luar tepi layar dalam beberapa rasio aspek. Untuk menyesuaikan respons gambar latar belakang rasio aspek, mengubah nilai Position Sprite di Sprite Editor.
Gambar Potret Alternate Mengatur gambar alternatif untuk digunakan dengan rasio aspek potret (misalnya, perangkat seluler dalam mode potret). Jika Anda tidak menetapkan Sprite Gambar Potret Alternate, Editor Unity menggunakan Sprite yang ditugaskan sebagai Background Image untuk mode potret dan lansekap.

Anda dapat menyesuaikan Posisi dan dimensi Sprite di Editor Sprite untuk mengontrol rasio aspek dan posisi gambar latar belakang pada Layar Splash.
