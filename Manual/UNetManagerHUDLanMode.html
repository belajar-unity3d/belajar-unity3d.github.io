Ketika Anda mengklik tombol ini, HUD beralih ke tampilan sederhana dari rincian jaringan, dan tombol Stop (X) yang memungkinkan Anda untuk menghentikan hosting permainan dan kembali ke menu LAN utama.

Ketika Anda telah memulai permainan sebagai tuan rumah, pemain lain dari permainan kemudian dapat terhubung ke tuan rumah untuk bergabung dengan permainan.

Klik tombol Stop (X) untuk memutuskan setiap pemain yang terhubung ke pemain host. Mengklik Stop (X) juga mengembalikan HUD ke menu LAN.

LAN Client

Untuk terhubung ke host di jaringan lokal, gunakan kolom teks ke kanan tombol LAN Client untuk menentukan alamat host. Alamat host default adalah "localhost", yang berarti klien terlihat pada komputernya sendiri untuk host game. Klik LAN Client (C) untuk mencoba untuk terhubung ke alamat host yang telah Anda tentukan.

Gunakan default "localhost" di bidang ini jika Anda menjalankan beberapa kasus permainan Anda pada satu komputer, untuk menguji berinteraksiivitas multiplayer. Untuk melakukan ini, Anda dapat membuat membangun permainan Anda yang mandiri, dan kemudian meluncurkan beberapa kali di komputer Anda. Ini adalah cara umum untuk dengan cepat menguji bahwa interaksi permainan jaringan Anda berfungsi seperti yang Anda harapkan, tanpa Anda perlu menyebarkan permainan Anda ke beberapa komputer atau perangkat.

When you want to test your game on multiple machines within the same network (that is, on a LAN), you need to put the address of the person acting as host into the "localhost" text field.

Orang yang bertindak sebagai tuan rumah perlu memberitahu alamat IP mereka kepada semua orang yang menjalankan klien LAN, sehingga Anda dapat mengetik ini ke dalam kotak.

Masukkan alamat IP (atau biarkan sebagai "localhost" jika Anda mengujinya pada mesin Anda sendiri), kemudian klik LAN Client untuk mencoba terhubung ke host.

Ketika klien mencoba untuk menghubungkan, HUD menampilkan tombol Cancel Connection Attempt. Klik ini jika Anda ingin berhenti mencoba untuk terhubung ke host.

LAN Server Only

Klik LAN Server Only untuk memulai permainan yang bertindak sebagai server yang klien lain dapat terhubung ke, tetapi yang tidak bertindak sebagai klien untuk permainan itu sendiri. Jenis permainan ini sering disebut "server yang didedikasikan". Pengguna tidak dapat bermain game pada contoh ini dari permainan Anda. Semua pemain harus terhubung sebagai klien, dan tidak ada yang bermain pada kasus yang berjalan sebagai server.

Server khusus pada hasil LAN dalam kinerja yang lebih baik untuk semua pemain yang terhubung, karena server tidak perlu memproses gameplay pemain lokal selain bertindak sebagai server.

Anda juga mungkin memilih opsi ini jika Anda ingin menghosting permainan yang dapat dimainkan di internet (hanya dari hanya dalam jaringan lokal), tetapi ingin mempertahankan kontrol server sendiri - misalnya, untuk mencegah kecurangan oleh salah satu klien, karena hanya server memiliki wewenang atas permainan. Untuk melakukan ini, Anda harus menjalankan game dalam mode Server Only pada komputer dengan alamat IP publik.

Enable Match Maker

Klik Enable Match Maker (M) untuk mengubah HUD ke mode matchmaker. Anda perlu menggunakan mode Matchmaker jika Anda ingin membuat atau terhubung ke game yang dihosting di internet menggunakan layanan multiplayer Unity. Klik Enable Match Maker (M) untuk menampilkan kontrol Pencocokan di Manajer Jaringan HUD.

Note: Ingat bahwa fitur Jaringan Manajer HUD adalah bantuan sementara untuk pengembangan. Ini memungkinkan Anda untuk mendapatkan permainan multiplayer Anda berjalan dengan cepat, tetapi Anda harus menggantinya dengan kontrol UI Anda sendiri ketika Anda siap.
