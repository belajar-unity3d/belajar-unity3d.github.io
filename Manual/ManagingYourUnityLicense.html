Kelola Anda Lisensi melalui baris perintah

Untuk menggunakan Unity, Anda perlu lisensi aktif. Untuk lisensi Unity Plus dan Pro, Anda perlu mendapatkan nomor serial; untuk lisensi Pribadi Unity, Anda tidak perlu nomor seri.

Anda dapat mengelola lisensi Anda dengan menggunakan .

Halaman ini mencakup cara mengaktifkan lisensi dan kembali lisensi dengan menggunakan baris perintah.

Jika Anda tidak tahu informasi lisensi Unity Anda, berbicara dengan Owner lisensi Anda. Pemilik dapat menetapkan kursi kepada Anda melalui Organization, atau Anda dapat menghubungi Layanan Pelanggan Unity.

Mengaktifkan lisensi dari baris perintah

Sebelum mengaktifkan lisensi Unity Anda di baris perintah, pastikan bahwa ada folder file lisensi dan Anda telah menulis izin.

Note: Prosedur baris perintah hanya bekerja untuk lisensi Plus dan Pro karena lisensi Pribadi tidak memiliki nomor seri yang terkait dengan mereka. Gunakan Hub Unity untuk mengaktifkan lisensi Pribadi. Jika Anda memiliki OS lain, Anda dapat mengelola lisensi Anda melalui Hub.

Di macOS, masukkan berikut ke Terminal untuk meluncurkan Unity dan mengaktifkan lisensi Anda:

<editor-installation-location> -quit -batchmode -serial SB-XXXX-XXXX-XXXX-XXXX-XXXX -username 'name@example.com' -password 'XXXXXXXXXXXXX'

Ganti <editor-installation-location> dengan jalur aktual untuk installer editor Anda. Biasanya, ini biasanya sesuatu seperti /Applications/Unity/Unity.app/Contents/MacOS/Unity pada macOS.

Tip: Jika Anda menggunakan alat integrasi berkelanjutan (CI) seperti Jenkins untuk mengaktifkan melalui baris perintah, tambahkan bendera -nographics untuk mencegah kesalahan WindowServer.

Pada Windows, masukkan berikut ke Command Prompt untuk meluncurkan Unity dan mengaktifkan lisensi Anda:

"<editor-installation-location>" -batchmode -username name@example.com -password XXXXXXXXXXXXX -serial E3-XXXX-XXXX-XXXX-XXXX-XXXX â€“quit

Ganti <editor-installation-location> dengan jalur aktual untuk installer editor Anda. Biasanya, ini biasanya sesuatu seperti C:\Program Files\Unity\Editor\Unity.exe di Windows.

Tunggu beberapa detik setelah menjalankan perintah ini untuk memberikan waktu yang cukup Unity untuk berkomunikasi dengan server lisensi. Jika aktivasi gagal, Anda dapat membuka Editor.log untuk informasi. Untuk kesalahan aktivasi, periksa Masalah aktivasi untuk menemukan solusi untuk masalah Anda.

Mengembalikan lisensi melalui baris perintah

Di macOS, ketikkan berikut ke Terminal untuk mengembalikan lisensi (mengganti <editor-installation-location> dengan lokasi instalasi aktual, seperti /Applications/Unity/Unity.app/Contents/MacOS/Unity):

<editor-installation-location> -quit -batchmode -returnlicense -username 'name@example.com' -password 'XXXXXXXXXXXXX'

Pada Windows, ketikkan berikut ke Command Prompt untuk mengembalikan lisensi (mengganti <editor-installation-location> dengan lokasi instalasi aktual, seperti C:\Program Files\Unity\Editor\Unity.exe):

"<editor-installation-location>" -quit -batchmode -returnlicense -username name@example.com -password XXXXXXXXXXXXX
