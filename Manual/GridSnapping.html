Grid snapping

Unity menyediakan grid visual di jendela SceneA Scene contains the environments and menus of your game. Think of each unique Scene file as a unique level. In each Scene, you place your environments, obstacles, and decorations, essentially designing and building your game in pieces. More info
See in Glossary view Editor, yang dapat membantu secara tepat menyelaraskan GameObjectsObjek mendasar dalam adegan Unity, yang dapat mewakili karakter, props, pemandangan, kamera, waypoints, dan banyak lagi. Fungsi GameObject didefinisikan oleh Komponen yang melekat padanya. More info
Lihat di Glossary dengan snapping (moving) mereka ke lokasi grid terdekat.

Anda dapat mengambil GameObject ke grid yang diproyeksikan di sepanjang X, Y, atau Z kapak dalam beberapa cara:

  • Align dipilih GameObjects ke titik grid terdekat.
  • Nyalakan jepit grid saat Anda bergerak, berputar, atau skala GameObject.

Anda juga bisa mengubah GameObject secara bertahap tanpa menyelaraskannya dengan gridlines. Jika Anda perlu mengubah berapa banyak Anda bergerak, berputar, atau scaling, Anda dapat mengubah nilai snap yang masuk.

Secara default, grid tersembunyi, tetapi Anda dapat membuatnya terlihat dan saklar yang sumbu muncul di sepanjang. Anda dapat menyesuaikan grid dalam penampilan dan perilaku dengan mengubah berikut:

  • size grid (memungkinkan jarak antara garis).
  • color dari garis grid sendiri.
  • Bagaimana transparan atau buram garis grid muncul.
  • position grid di jendela Scene viewAn interactive view into the world you are creating. You use the Scene View to select and position scenery, characters, cameras, lights, and all other types of Game Object. More info
    See in Glossary     (menghilangkan lebih dekat atau lebih jauh).

Banyak dari tindakan mendukung pintasan keyboard ini. Anda dapat melihat kombinasi kunci yang ditugaskan untuk tindakan ini dan menggunakan Shortcuts Manager untuk menyesuaikannya.

Align, snap, dan gerakan inkremental

Bagian ini memberikan informasi tentang cara melakukan tindakan ini:

  • Aligning (pushing) ke grid
  • Mengaktifkan auto-snapping
  • Bergerak, berputar, dan scaling increments

Align (push) ke grid

Anda dapat menyelaraskan GameObject ke titik grid terdekat pada sumbu tunggal, atau pada semua sumbu sekaligus.

Untuk menyelaraskan GameObject ke titik terdekat pada grid untuk sumbu tertentu:

  1. Pada Grid and Snap Overlays toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
    Lihat di Glossary    , buka menu drop-down grid snapping ().

  2. Pada bagian Align Selected, klik tombol X, Y, atau Z yang sesuai dengan sumbu yang ingin Anda dorong.

Untuk menyelaraskan GameObject pada semua sumbu sekaligus:

  1. Pilih GameObject (s) Anda ingin menyelaraskan ke grid.

  2. Gunakan shortcut Ctrl +\ (Windows) atau Command +\ (macOS) untuk mendorong GameObject (s) ke grid pada semua sumbu.

    Atau, dari Grid and Snap Overlays toolbar, buka menu drop-down visibilitas grid () dan di bagian Align Selected, klik All Axes.

Mengaktifkan auto-snapping

Untuk mengaktifkan snapping otomatis ke grid:

  1. Mengaktifkan alat Move dan pastikan orientasi pegangan dalam Overlay Pengaturan Alat diatur ke Global ().
  2. Klik ikon grid snapping () di Overlay toolbar Grid and Snap. Ikon berwarna biru saat diaktifkan.

Ketika Anda mengaktifkan grid snapping otomatis, Move, Rotate, dan alat transformasi Scale snap GameObject yang dipilih (s) ke grid sepanjang sumbu gizmoOverlay grafis yang terkait dengan GameObject di Adegani, dan ditampilkan di Tampilan Adegan. Alat adegan internal seperti alat bergerak adalah Gizmos, dan Anda dapat membuat Gizmos kustom menggunakan tekstur atau scripting. Beberapa Gizmos hanya ditarik ketika GameObject dipilih, sementara Gizmos lainnya ditarik oleh Editor terlepas dari GameObjects mana yang dipilih. More info
Lihat di Glossary aktif. Jika Anda membutuhkan GameObject untuk bergerak dalam perbaikan yang lebih kecil, Anda juga dapat mengubahnya secara bertahap.

Pindahkan, putar, dan peningkatan skala

Untuk bergerak, memutar, atau skala dengan nilai snap yang masuk:

  • Tahan tombol Control (Windows) atau kunci Command (macOS) saat menggunakan salah satu transformasi gizmos.

Untuk mengubah nilai snap increment default:

  1. Dari Grid and Snap Overlays toolbar, buka menu drop-down snap ().

  2. Bagian Increment Snap mengandung beberapa properti yang memungkinkan Anda mengatur nilai yang sama untuk semua kapak atau nilai yang berbeda:

    Move: Dengan ikon tautan yang dipilih, masukkan nilai snap yang seragam untuk semua sumbu, atau unlink axes dan mengatur nilai snap increment yang berbeda dalam sifat sumbu X, Y, dan sumbu Z.

    Rotate: Masukkan nilai kenaikan rotasi dalam derajat.

    Scale: Masukkan nilai kenaikan skala sebagai faktor skala. Misalnya, jika nilai Skala adalah 2, GameObject yang dipilih akan berskala dua kali ukuran aslinya.

Tampilkan dan menyembunyikan garis grid

visibilitas grid Toggle dengan mengklik ikon visibilitas grid () di Grid dan Snap Overlay toolbar untuk menampilkan atau menyembunyikan grid pada setiap sumbu (X, Y, Z). Jika Anda berada di Mode ortopografi (Iso), Unity memilih tampilan.

Mengubah sumbu di mana grid muncul

Untuk mengubah sumbu mana grid muncul di:

  1. Dari Grid and Snap Overlays toolbar, buka menu drop-down visibilitas grid ().

  2. Dari bagian Grid Plane, pilih sumbu yang ingin Anda tampilkan.

Ubah ukuran grid

Anda dapat mengatur ukuran gridlines karena muncul di jendela Scene view. Ini mempengaruhi penampilan grid dan bagaimana GameObjects Anda secara otomatis terkunci ke grid, tetapi tidak mempengaruhi berapa banyak GameObject Anda (s) bergerak, memutar, atau skala increments.

Jika Anda mengatur ukuran untuk semua kapak sekaligus, seragam (dasar persegi) grid muncul. Namun, Anda juga dapat menggunakan nilai yang berbeda pada salah satu dari tiga sumbu untuk melihat grid non-uniform (dasar persegi). Secara default, grid diatur ke jarak seragam (1 pada semua sumbu).

Untuk mengubah ukuran grid:

  1. Dari Grid and Snap Overlays toolbar, buka menu drop-down snap ().

  2. Gunakan properti Size dengan ikon link yang dipilih untuk memasukkan nilai seragam untuk grid visual (semua garis grid panjang yang sama), atau tidak menghubungkan sumbu dan menentukan jarak non-uniform antara garis grid untuk X, Y, dan Z.

    Misalnya, untuk membuat grid persegi panjang pada sumbu x dan y, unlink axes dan set X dan Y untuk 1, dan set Z ke 2.

  3. Jika Anda ingin beralih kembali ke nilai seragam antara garis grid:

    • Pilih ikon tautan.
    • Properti Size sekarang menampilkan nilai dari semua kapak dengan nilai yang dimasukkan dalam X.

Note: Anda juga dapat menggunakan pintas keyboard untuk meningkatkan dan menurunkan ukuran grid:

  • Untuk meningkatkan ukuran grid, gunakan Ctrl +] (Windows) atau Command +] (macOS).
  • Untuk menurunkan ukuran grid, gunakan Ctrl +[ (Windows) atau Command +[ (macOS).

Mengubah warna default dari garis grid

Untuk mengubah warna garis grid yang terlihat di jendela Scene view:

  1. Buka halaman Unity Preferences (go ke Edit > Preferences pada Windows atau Unity > Preferences pada macOS).

  2. Klik pada kategori Warna dari daftar untuk melihat halaman Colors.

  3. Pilih warna baru menggunakan pemilih warna properti Grid.

Mengubah opacity grid

Jika garis grid terlalu ringan atau terlalu gelap, Anda dapat menyesuaikannya:

  1. Dari Grid and Snap Overlays toolbar, buka menu drop-down visibilitas grid ().

  2. Sesuaikan Opacity menggunakan slider.

Sesuaikan lokasi grid

Anda dapat menggunakan shortcut berikut untuk memindahkan grid mundur dan maju sepanjang sumbunya:

  • Untuk mendebarkan grid mundur, gunakan Shift +[.
  • Untuk mendebarkan grid ke depan, gunakan Shift +].

Nudging grid membawanya lebih dekat (misalnya, jika sulit untuk melihat GameObject Anda terhadap grid jauh) atau mengirimkannya jauh.

Pindahkan grid ke pegangan objek permainan

Dari menu drop-down visibilitas grid () di bilah alat Grid dan Snap Overlay, gunakan bagian Move To untuk memindahkan grid ke pegangan GameObject yang dipilih (Handle) atau kembali ke posisi default (Origin).

Atur ulang nilai dan pengaturan secara default

Untuk mengatur ulang sumbu grid dan pengaturan ke default mereka:

  1. Dari Grid and Snap Overlays toolbar, buka menu drop-down visibilitas grid ().

  2. Klik menu More.Reset.

Untuk mengatur ulang ukuran grid ke default:

  1. Dari Grid and Snap Overlays toolbar, buka menu drop-down grid ().

  2. Klik menu More.Reset.

Untuk mengatur ulang nilai snap increment ke default:

  1. Dari Grid and Snap Overlays toolbar, buka menu drop-down snap ().

  2. Klik menu More.Reset.

    Hanya nilai di bawah bagian Increment Snap kembali ke nilai default mereka.
