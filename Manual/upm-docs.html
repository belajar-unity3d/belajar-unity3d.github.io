Menemukan dokumentasi paket

Manual Unity menyediakan dokumentasi untuk versi Unity tertentu. Setiap paket memiliki versi sendiri, sehingga setiap paket menyediakan dokumentasi untuk versi paket tertentu. Untuk alasan ini, dokumentasi paket bukan bagian dari dokumentasi Manual Unity utama; bukan, dokumentasi untuk setiap paket hidup pada server dokumentasi Unity.micro-site on Unityâ€™s documentation server.

Untuk mengakses dokumentasi untuk paket tertentu, Anda memiliki dua pilihan:

  • Dapatkan dokumentasi untuk versi terbaru (dari Manual Unity).
  • Dapatkan dokumentasi untuk versi paket tertentu (dari jendela Manajer Paket).

Ketika halaman paket terbuka, Anda dapat melihat empat tautan di bagian atas halaman (A), dan menu drop-down (B) di mana Anda dapat memilih versi yang berbeda:

Untuk beralih kembali dan ke depan antara dokumentasi Manual, dokumentasi Scripting API, Changelog, dan informasi License untuk paket ini, klik tautan yang sesuai di bagian atas halaman.

Dapatkan dokumentasi untuk versi terbaru

Setiap versi dokumentasi Manual Unity menyediakan daftar paket verified dan preview, daftar paket built-inBuilt-in
Lihat di Glossary, dan Daftar kata kunci dengan tautan ke dokumentasi paket yang cocok yang dapat Anda gunakan dengan versi Unity:

Note: Hanya paket yang kompatibel dengan versi Unity yang sesuai dengan dokumentasi yang muncul dalam daftar ini.

Anda dapat memperluas entri di navigasi sidebar (A) untuk mengakses informasi spesifik tentang setiap paket, termasuk daftar versi yang kompatibel, deskripsi singkat dari paket, dan kata kunci yang terkait. Atau Anda dapat mengklik tautan di halaman daftar (B) untuk secara langsung membuka versi dokumentasi paket yang paling kompatibel yang sesuai dengan versi dokumentasi.

Jika Anda tidak dapat menemukan paket yang Anda inginkan dalam daftar ini, mungkin ada beberapa alasan:

  • Tidak kompatibel dengan versi Unity ini.
  • Ini adalah paket eksperimental atau pribadi. Beberapa paket tidak tersedia untuk semua orang karena seseorang di luar Unity sedang mengembangkan mereka atau karena mereka berada di bawah lisensi khusus.

Anda mungkin dapat mengakses dokumentasi melalui jendela Paket Manager untuk paket yang dirilis dalam versi Unity lain. Dalam kasus paket pribadi, coba hubungi pengembang langsung untuk meminta akses.

Mendapatkan dokumentasi untuk versi paket tertentu

Anda dapat menemukan lebih banyak tentang paket dengan melihat dokumentasi, perubahan, atau informasi lisensinya.

Untuk mengakses halaman ini, Anda dapat mengklik tautan View documentation, View changelog, atau View licenses dari dalam jendela Paket Manager.

Untuk mengakses dokumentasi untuk versi paket tertentu:

  1. Buka Paket jendela Manager di Unity.

  2. Ikuti panduan di Cari paket untuk menemukan paket dan nomor versi tertentu yang Anda inginkan.

  3. Klik tautan View documentation untuk membuka dokumentasi untuk versi paket yang dipilih.

    Note: Jika Anda offline ketika Anda mengklik tautan View documentation dan paket sudah diinstal, Manajer Paket membuka folder <package-root-folder>/Documentation~ di browser file Anda. Jika Manajer Paket tidak diinstal, pesan muncul untuk memberi tahu Anda bahwa paket tidak tersedia secara offline.

Anda dapat mengikuti prosedur ini untuk setiap versi paket. Perhatikan bahwa dokumentasi tidak tentu berbeda untuk setiap rilis versi paket, karena beberapa pembaruan versi (patches) hanya melibatkan perbaikan bug atau perubahan sepele.
