Python untuk Unity

Description

Integrasi Python untuk . NET dikirim ke proyek Anda melalui Manajer Paket, Python untuk Unity menyediakan proyek Unity Anda berarti untuk mengimpor Python Runtime Environment sebagai namespace dan mengeksekusi skrip Python dari kedua dalam editor Unity dan aplikasi yang dibangun.
