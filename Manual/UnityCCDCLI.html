Sitemap Antarmuka baris perintah (CLI)
Important: Dokumentasi Pengiriman Konten Cloud Unity telah pindah ke . Silakan kunjungi situs baru untuk informasi terbaru tentang CCD Unity.

Antarmuka baris perintah (CLI) adalah metode yang disarankan untuk berinteraksi dengan Pengiriman Konten Cloud (CCD). Dengan alat ini, Anda dapat menggunakan jalur perintah untuk mengontrol setiap aspek proyek Anda, dari manajemen file untuk mengatur pipa rilis Anda.

Anda dapat melakukan banyak fungsi serupa di Kembangkan Dasbor, tetapi Anda perlu menggunakan CLI untuk menyalin file ke bucket Anda, atau menyinkronkan seluruh folder.

Mulai

Untuk mengatur CCD untuk CLI:

  1. Jika Anda menggunakan layanan Unity untuk pertama kalinya:
    1. Buat akun (jika Anda tidak memiliki satu).
    2. Masuk ke tab Develop.
    3. Klik Create New Project, dan masukkan rincian dalam Buat Project windowJendela yang menunjukkan isi folder Assets (Project tab) More info
      Lihat di Glossary       Baru.
  2. Di Kembangkan Dasbor, pilih proyek Anda.
  3. Pilih nama proyek, atau klik View.
  4. Di halaman Kembangkan halaman Overview Dasbor, di bilah navigasi sisi kiri, pilih Content Delivery.
  5. Di bilah navigasi sisi kiri, pilih Download CLI.
  6. Untuk mengunduh CLI, cari versi sistem operasi Anda, lalu pilih ikon unduh.
  7. Di bar navigasi sisi kiri, pergi ke bagian API Key.
  8. Salin kunci API Anda.
  9. Jalankan CLI di mesin dari mana Anda mengunggah konten.
  10. Masuk ke CLI menggunakan tombol API Anda melalui perintah auth.

CCD sekarang siap untuk Anda gunakan.

Penggunaan CLI

Bentuk dasar dari perintah CCD di CLI adalah sebagai berikut:

$ ucd [command]

Perintah yang tersedia adalah:

  • auth: Otentikasi dengan Pengiriman Konten.
  • badges: Mengelola lencana untuk rilis.
  • buckets: Kelola ember untuk proyek.
  • config: Mengelola opsi untuk mengatur dan mengidentifikasi ember saat ini.
  • entries: Mengelola entri untuk ember saat ini.
  • releases: Mengatur rilis untuk bucket saat ini.

Untuk melihat versi CLI saat ini Anda menggunakan, tambahkan bendera --version:

$ ucd --version

Anda dapat menambahkan bendera global berikut setelah perintah apa pun:

  • --apikey string: Kunci API yang ditentukan pengguna untuk mengakses Pengiriman Konten.
  • -h, --help: Menampilkan parameter dan bendera lebih lanjut untuk perintah yang diberikan.
  • -q, --quiet: Memungkinkan tampilan operasi yang dilakukan oleh perintah yang ditentukan.
  • -v, --verbose: Menampilkan informasi kesalahan.

Contoh:

  • Untuk membuat bucket:
    $ ucd buckets create [project_id] [bucket_name]
  • Untuk daftar semua ember untuk sebuah proyek:
    $ ucd buckets list [project_id]
  • Untuk mengatur bucket saat ini sebagai bucket aktif:
    $ ucd config set bucket [bucket_id]
  • Untuk menyinkronkan folder lokal dengan ember saat ini:
    $ ucd entries sync [local_dir_path]
  • Untuk melihat daftar semua entri di ember saat ini:
    $ ucd entries list

Nama ember dan entri sensitif kasus.
