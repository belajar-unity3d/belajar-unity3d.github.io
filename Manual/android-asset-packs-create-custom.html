Buat paket aset kustom

Untuk membuat paket aset kustom, buat direktori dengan nama yang berakhir dengan .androidpack. Anda dapat menempatkan direktori ini di mana saja di direktori Aset proyek Anda, atau subdirectory.

Important: Unity tidak mengimpor aset dari direktori .androidpack, sehingga Anda tidak dapat menggunakan aset dalam paket aset kustom langsung dalam Unity scenesAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary. Untuk menggunakan aset dari paket aset kustom, Anda harus secara manual mengakses dan memuatnya secara dinamis pada runtime. Untuk informasi tentang cara melakukan ini, lihat Kelola paket aset pada runtime.

Langkah-langkah berikut menjelaskan cara membuat paket aset kustom bernama MyAssets1:

  1. Pergi ke direktori yang ingin Anda buat paket aset. Ini bisa langsung di Assets atau subdirectory seperti Assets/CustomAssetPacks.
  2. Buat direktori baru dan sebutkan MyAssets1.androidpack. Untuk menambahkan aset apa pun ke paket aset, letakkan di dalam folder ini.
    Note: Unity tidak termasuk paket aset kosong dalam membangun. Juga, nama paket aset harus dimulai dengan surat dan terdiri dari karakter alfanumerik Inggris atau underscore.
  3. Secara default, mode pengiriman adalah on-demand, yang berarti bahwa jika Anda tidak mengubah mode pengiriman, Anda perlu mengunduh paket aset secara manual pada runtime. Untuk informasi tentang cara melakukan ini, lihat Kelola paket aset pada runtime.
  4. Untuk menggunakan mode pengiriman yang berbeda, buat file yang disebut build.gradle di dalam direktori paket aset kustom. Tempel berikut ke file:
apply plugin: 'com.android.asset-pack'
assetPack {
    packName = "MyAssets1"
    dynamicDelivery {
        deliveryType = "fast-follow"
    }
}

Ini mengatur mode pengiriman ke fast-follow, yang berarti Google Play secara otomatis mengunduh paket aset setelah menginstal aplikasi. Untuk informasi tentang format file ini, lihat Mengintegrasikan pengiriman aset.

Note: packName Anda menentukan dalam file build.gradle harus sesuai dengan nama direktori paket aset yang Anda set tanpa ekstensi .androidpack.
