Pengaturan Audio (menu utama: Edit > Project Settings, kemudian pilih kategori Audio) memungkinkan Anda untuk mengubah volume maksimum semua suara bermain di Adegan.

Property Function
Global Volume
Volume Rolloff Scale Sumber berbasis rolloff logarithmic. The higher the value, the faster the volume attenuates. Conversely, the lower the value, the slower it attenuates.
Tip: A value of 1 simulates the “real world”.
Doppler Factor Mengatur bagaimana audible efek Doppler. Gunakan 0 untuk menonaktifkannya. Gunakan 1 membuatnya terdengar untuk objek bergerak cepat.
Tip: Setelah mengatur Doppler Factor ke 1, Anda dapat mengubah kedua Speed of Sound dan Doppler Factor sampai Anda puas.
Default Speaker Mode AudioSpeakerMode API reference.
Note: You can also change the speaker mode at runtime through scripting. See Pengaturan Audio for details.
System Sample Rate
Note: This only serves as a reference only, since certain platforms allow you to change the sample rate, such as iOS or Android.

Max Real Voices Mengatur jumlah suara nyata yang dapat bermain pada saat yang sama. Pada setiap bingkai, suara paling keras dipilih.
