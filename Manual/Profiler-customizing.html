Sesuaikan Profiler

Anda dapat menyesuaikan Unity ProfilerJendela yang membantu Anda untuk mengoptimalkan permainan Anda. Ini menunjukkan berapa banyak waktu yang dihabiskan di berbagai bidang permainan Anda. Sebagai contoh, dapat melaporkan persentase waktu yang dihabiskan rendering, aimating, atau dalam logika permainan Anda. More info
Lihat di Glossary untuk menangkap dan memvisualisasikan data kinerja tertentu dalam aplikasi Anda. Data kinerja ini memberikan informasi tentang karakteristik kinerja utama, yang dapat Anda gunakan untuk optimasi atau untuk membuat keputusan perkakas.

Anda dapat menggunakan Editor Modul Profiler untuk menambahkan modul kustom tanpa menulis kode apa pun. Editor Modul Profiler adalah Unity built-in UI(User Interface) Memungkinkan pengguna untuk berinteraksi dengan aplikasi Anda. Unity saat ini mendukung tiga sistem UI. More info
Lihat di Glossary untuk kustomisasi Profiler. Untuk informasi lebih lanjut, lihat Profiler Modul Editor.

Untuk kontrol lebih lanjut, Anda juga dapat menggunakan kode untuk menyesuaikan Profiler dengan cara berikut:

  • Anda dapat tambahkan Profiler counter ke kode Anda untuk memberitahu Profiler untuk mengumpulkan data kustom tertentu.
  • Anda dapat menggunakan Profiler countersDitempatkan dalam kode dengan ProfilerCounter API untuk melacak metrik, seperti jumlah musuh yang ditidur dalam permainan Anda.
    Lihat di Glossary     untuk membuat modul Profiler kustom di Jendela Profiler.
  • Anda dapat Sesuaikan jendela modul lebih lanjut dengan visualisasi di panel detail modul.

Untuk menyesuaikan profiler menggunakan kode, Anda perlu menginstal paket.

Menggunakan penanda profil untuk memvisualisasikan kode

Anda dapat menggunakan profiler markersDitempatkan dalam kode untuk menggambarkan acara CPU atau GPU yang kemudian ditampilkan di jendela Unity Profileer. Ditambahkan ke kode Unity secara default, atau Anda dapat menggunakan untuk menambahkan penanda kustom Anda sendiri. More info
Lihat di Glossary untuk memvisualisasikan blok kode intensif sumber daya di jendela Profiler.

Penanda profil memungkinkan Anda untuk melihat metode panggilan yang Unity tidak profil secara default. Anda kemudian dapat menampilkan metode ini panggilan dalam modul Profiler CPU tanpa perlu menggunakan Deep Profileing. Untuk informasi lebih lanjut, lihat .
