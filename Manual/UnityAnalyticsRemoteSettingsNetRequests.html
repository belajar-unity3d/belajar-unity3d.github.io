Lihat di Glossary, toko Layanan Unity AnalyticsSebuah platform data yang menyediakan analitik untuk permainan Unity Anda. More info
Lihat di Glossary yang mengatur di Configuration untuk proyek Anda yang telah ditentukan (baik konfigurasi Release atau Development). Setiap kali pemain memulai sesi baru aplikasi Anda, Unity membuat permintaan jaringan untuk konfigurasi terbaru dari Layanan Analytics. Unity menganggap sesi baru untuk memulai ketika pemain meluncurkan aplikasi, atau kembali ke aplikasi yang telah ada di latar belakang setidaknya 30 menit. Permintaan unity konfigurasi Release ketika menjalankan build yang teratur, non-development dari aplikasi Anda, dan meminta konfigurasi Development ketika menjalankan pembangunan build. Mode bermain dalam jumlah Editor Unity sebagai pembangunan build.

Ketika permintaan jaringan untuk konfigurasi Pengaturan Jarak Jauh selesai, objek RemoteSettings mengirimkan acara Updated ke setiap pemegang acara terdaftar, termasuk yang terdaftar oleh Komponen Pengaturan jarak jauh.

Jika komputer atau perangkat tidak memiliki koneksi Internet dan tidak dapat berkomunikasi dengan Layanan Analytics, Unity menggunakan konfigurasi terakhir yang diterima dan disimpan. Objek RemoteSettings masih mengirimkan acara Updated ketika menggunakan konfigurasi yang tersimpan. Namun, jika Unity belum menyimpan pengaturan belum (seperti ketika pemain tidak memiliki koneksi jaringan pertama kali mereka menjalankan permainan Anda), maka objek RemoteSettings tidak mengirimkan acara Updated, dan tidak memperbarui variabel game Anda. Meminta konfigurasi Pengaturan Jarak Jauh atas jaringan adalah proses asinkron yang mungkin tidak lengkap sebelum awal SceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary telah selesai memuat, atau mungkin tidak lengkap sama sekali, sehingga Anda harus selalu menginisialisasi variabel permainan Anda ke default yang wajar.

Sitemap Layanan web dari mana Unity mengunduh konfigurasi Pengaturan Jarak Jauh dibaca, tetapi tidak aman. Ini berarti bahwa konfigurasi bisa dibaca oleh pihak ketiga. Anda tidak boleh memasukkan informasi sensitif atau rahasia ke Pengaturan Jarak Jauh Anda. Demikian pula, file pengaturan yang disimpan bisa dibaca dan dimodifikasi oleh pengguna akhir (meskipun modifikasi apa pun yang mengalami waktu berikutnya sesi dimulai dengan koneksi Internet yang tersedia).Note: The web service from which Unity downloads the Remote Settings configuration is read-only, but is not secured. This means that the configuration could be read by third-parties. You should not put sensitive or secret information into your Remote Settings. Similarly, the saved settings file could be read and modified by end-users (although any modifications are overwritten the next time a session starts with an available Internet connection).
