Adegan melihat navigasi

Tampilan SceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary memiliki set kontrol navigasi untuk membantu Anda bergerak secara efisien:

  • Adegan Gizmo
  • Alat Pindah, Orbit dan Zoom tools
  • Alat Center tool

Adegan Gizmo

Adegan GizmoOverlay grafis yang terkait dengan GameObject di Adegani, dan ditampilkan di Tampilan Adegan. Alat adegan internal seperti alat bergerak adalah Gizmos, dan Anda dapat membuat Gizmos kustom menggunakan tekstur atau scripting. Beberapa Gizmos hanya ditarik ketika GameObject dipilih, sementara Gizmos lainnya ditarik oleh Editor terlepas dari GameObjects mana yang dipilih. More info
Lihat di Glossary muncul di Scene viewTampilan interaktif ke dunia yang Anda buat. Anda menggunakan Adegan Lihat untuk memilih dan posisi pemandangan, karakter, kamera, lampu, dan semua jenis lain dari Game Object. More info
Lihat di Glossary. Tampilan ini menampilkan orientasi arus CameraKomponen yang menciptakan gambar sudut pandang tertentu di tempat kejadian Anda. Output ditarik ke layar atau ditangkap sebagai tekstur. More info
Lihat di Glossary, dan memungkinkan Anda untuk memodifikasi sudut pandang dan mode proyeksi.

Adegan Gizmo memiliki lengan kerucut di setiap sisi kubus. Lengan di garis depan label X, Y, dan Z. Klik pada salah satu lengan sumbu kerucut untuk snap Kamera Pemandangan ke sumbu itu mewakili (misalnya: pandangan atas, pandangan kiri dan pandangan depan). Anda juga dapat klik kanan kubus untuk membawa menu dengan daftar sudut pandang. Untuk kembali ke sudut pandang default, klik kanan Gizmo Adegan dan klik Free.

Anda juga dapat beralih Perspective di dan mati. Perubahan mode proyeksi dari tampilan Adegan antara Perspective dan Orthographic (kadang kali disebut “isometric”). Untuk melakukan ini, klik kubus di pusat Adegan Gizmo, atau teks di bawahnya. Tampilan orografi tidak memiliki perspektif, dan berguna dalam kombinasi dengan mengklik salah satu lengan sumbu kerucut untuk mendapatkan ketinggian depan, atas atau sisi.

Sitemap Tampilan adegan adalah di sudut pandang canggung (di bawah, atau hanya sudut yang Anda temukan membingungkan), Shift-klik kubus di pusat Adegan Gizmo untuk kembali ke tampilan Perspective dengan sudut yang melihat Adegan dari sisi dan sedikit dari atas.

Klik pada gembok di kanan atas adegan gizmo untuk mengaktifkan atau menonaktifkan rotasi adegan. Setelah adegan rotasi dinonaktifkan, klik kanan mouse pans tampilan bukan berputar. Ini sama dengan Lihat alat.

Perhatikan bahwa pada 2D Mode Adegan Gizmo tidak muncul, karena satu-satunya pilihan adalah untuk memiliki pandangan yang terlihat tegak di pesawat XY.

gestur trackpad Mac

Di Mac dengan trackpad, Anda dapat menyeret dengan dua jari untuk memperbesar tampilan.

Anda juga dapat menggunakan tiga jari untuk mensimulasikan efek mengeklik lengan Scene Gizmo: menyeret, kiri, kanan atau turun untuk menangkap Kamera pandangan adegan ke arah yang sesuai.

Pindahkan, orbit dan zoom dalam tampilan adegan

Bergerak, mengorbit, dan memperbesar adalah operasi utama dalam navigasi tampilan adegan. Unity menyediakan beberapa cara untuk melakukan aksesibilitas maksimum:

  • Gerakan panah
  • Alat View
  • Modus
  • Kecepatan kamera
  • Gerakan shortcuts

Gerakan panah

Anda dapat menggunakan Arrow Keys untuk bergerak di sekitar Adegan seolah-olah "perjalanan" melaluinya. panah atas dan turun memindahkan kamera ke depan dan mundur ke arah itu menghadap. panah kiri dan kanan pan saluran pandang. Tahan tombol Shift dengan panah untuk bergerak lebih cepat.

Alat View

Ketika alat View dipilih (shortcut: Q), kontrol mouse berikut tersedia:

Control: Description:
Move

Klik dan seret untuk memindahkan Kamera di sekitar.
Orbit

Tahan Alt (Windows) atau Option (macOS), dan klik kiri dan tarik untuk mengorbit Kamera di sekitar titik pivot saat ini.

Pilihan ini tidak tersedia dalam mode 2D, karena tampilannya ortografi.
Zoom

Tahan Alt (Windows) atau Option (macOS), dan klik kanan dan tarik untuk memperbesar tampilan adegan.

Di macOS, Anda juga dapat menahan Control, dan klik kiri dan drag.

TahanShift untuk meningkatkan tingkat pergerakan dan zoom.

Modus

Gunakan Flythrough modeLogin Mode navigasi tampilan adegan yang memungkinkan Anda untuk terbang di sekitar adegan di orang pertama, mirip dengan bagaimana Anda akan menavigasi dalam banyak permainan. More info
Lihat di Glossary untuk menavigasi pemandangan dengan terbang di sekitar orang pertama, mirip dengan bagaimana Anda akan menavigasi dalam banyak permainan:

  • Klik dan tahan tombol mouse yang tepat.
  • Pindahkan pandangan di sekitar menggunakan mouse, tombol WASD untuk memindahkan kiri / kanan / ke atas / belakang, dan tombol Q untuk bergerak naik dan turun.E keys to move up and down.
  • TahanShift untuk bergerak lebih cepat.

Mode terbang dirancang untuk Perspective Mode. Pada Orthographic Mode, menahan tombol mouse kanan dan memindahkan mouse mengorbit Kamera.

Perhatikan bahwa mode Flythrough tidak tersedia dalam mode 2D. Alih-alih, memegang tombol mouse kanan turun sambil memindahkan panci mouse di sekitar pemandangan.

Kecepatan kamera

Untuk mengubah kecepatan saat ini Kamera dalam pandangan Adegan, klik ikon Kamera di toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). More info
Lihat di Glossary. Pada Flythrough mode Anda dapat mengubah kecepatan Kamera saat bergerak di sekitar Adegan. Untuk melakukan ini, gunakan roda gulir mouse atau tarik dua jari di trackpad.

Untuk informasi lebih lanjut, lihat dokumentasi Pengaturan kamera.

Pusat pandangan pada GameObject

Untuk pusat pandangan Adegan pada GameObject, pilih GameObject di Hierarchy, kemudian memindahkan mouse di atas pandangan Adegan dan tekan F. Jika GameObject sudah dipilih, zoom F ke titik pivot. Fitur ini juga dapat ditemukan di bilah menu di bawah Edit > Frame Selected.

Untuk mengunci pandangan ke GameObject bahkan ketika GameObject bergerak, tekan Shift+F. Fitur ini juga dapat ditemukan di bilah menu di bawah Edit > Lock View to Selected.
