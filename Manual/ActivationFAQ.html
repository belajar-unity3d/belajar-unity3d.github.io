Masalah aktivasi

Jika Anda menemukan kode kesalahan atau masalah lain selama proses aktivasi, mereka mungkin disebabkan oleh masalah berikut:

Issue: Solution:
No internet access Cobalah untuk mengaktifkan lisensi Anda secara manual. Untuk informasi lebih lanjut, lihat aktivasi lisensi Offline (manual).
Security settings block Unity Ulasan firewall, proxy, atau pengaturan keamanan internet. Mereka mungkin memblokir Unity dari mengirim dan menerima data tentang file lisensi Anda.
No write privileges for saving the license file Memberikan atau mendapatkan izin menulis ke akun pengguna yang Anda gunakan untuk mengaktifkan lisensi. Ini sering terjadi di jaringan tertutup, seperti di tempat kerja dan sekolah. Hubungi administrator TI Anda untuk mengetahui apakah ada batasan di tempat.
License fails to recognize your machine Perubahan atau pembaruan sistem operasi utama mungkin mempengaruhi ID mesin Anda, yang mungkin menyebabkan lisensi Anda untuk menghentikan mengenali mesin Anda. Lihat artikel Pangkalan Pengetahuan Unity ini:
Unity’s license servers are down Lihat pembaruan status pada semua server Unity.
Too many license activations Kembalikan lisensi Anda jika Anda tidak perlu lagi untuk mesin tertentu.

Anda harus kembali lisensi Anda sebelum Anda beralih mesin. Anda juga harus mengembalikan lisensi Anda sebelum membuat perubahan hardware atau perubahan OS ke mesin Anda. Jika Anda gagal mengembalikan lisensi, server lisensi Unity melihat permintaan dari mesin lain dan memberi tahu Anda bahwa Anda telah mencapai batas aktivasi Anda untuk lisensi.

Jika Anda mendapatkan pesan ini tetapi Anda berpikir itu kesalahan, Anda selalu dapat mengembalikan semua lisensi sekaligus, dan kemudian mengaktifkan setiap mesin sesuai kebutuhan.
Account credentials not recognized Pastikan Anda memasukkan detail Anda dengan benar. Kata sandi sensitif, jadi pastikan Anda memasukkannya persis seperti yang terdaftar. Untuk mengatur ulang kata sandi Anda, buka .

Jika Anda masih memiliki masalah penebangan, hubungi.

Masalah dengan lisensi pengembalian

Jika Anda menemukan kode kesalahan atau masalah lain saat mengembalikan lisensi Anda, mereka mungkin disebabkan oleh berikut:

Issue: Solution:
Machine not accessible for returning licenses Kunjungi Toko Unity, dan pilih tombol Disable all activations untuk kembali semua aktivasi sekaligus. Jika Anda memiliki lisensi diaktifkan pada mesin lain, Anda perlu mengaktifkan kembali mereka setelah itu.

Jika Anda masih tidak dapat mengembalikan lisensi, email, dan menjelaskan situasi Anda. Termasuk rincian ini:

- Nomor serial yang Anda gunakan pada mesin. Login Nama (jaringan lokal) mesin yang mati. Login Pesanan atau nomor faktur yang digunakan untuk melakukan pembelian.

Tim Dukungan dapat mengembalikan lisensi Anda secara manual jika Anda memberikan informasi di atas; namun, dapat memakan waktu beberapa waktu. Proses ini hanya tersedia untuk lisensi yang dibeli.

The Support Team can return your license manually if you provide the above information; however, it can take some time. This process is only available for purchased licenses.

Jawaban atas pertanyaan lain

Question: Answer:
How many machines can I install my copy of Unity on? Setiap lisensi Unity komersial berbayar memungkinkan seseorang untuk menggunakan Unity pada dua mesin yang memiliki penggunaan eksklusif. Tidak masalah di mana mesin berada secara fisik atau sistem operasi apa yang mereka jalankan.

Educational lisensi yang dijual melalui Unity atau salah satu reseller kami hanya bagus untuk satu aktivasi. Yang sama berlaku untuk lisensi Percobaan (Unity 4.x saja), kecuali dinyatakan sebaliknya.

Versi gratis Unity tidak dapat dilisensikan oleh entitas komersial dengan pendapatan kotor tahunan (berdasarkan tahun fiskal) lebih dari US$100,000, atau oleh entitas pendidikan, nirlaba atau pemerintah dengan anggaran tahunan lebih dari US$100,000.

Jika Anda adalah Entitas Hukum, Anda tidak dapat menggabungkan file yang dikembangkan dengan versi Unity gratis dengan file apa pun yang dikembangkan oleh Anda (atau oleh pihak ketiga) melalui penggunaan Unity Pro.

Lihat informasi lebih lanjut mengenai penggunaan lisensi.
Can I use the current version of Unity with a Serial number from a previous version? Jika nomor serial yang ingin Anda gunakan adalah untuk versi Unity 2.x, 3.x, atau 4.x, Anda perlu membeli lisensi Unity Plus atau Pro baru. Pergi ke atau .

Jika nomor serial yang ingin Anda gunakan adalah untuk versi Unity 5.x dan kemudian, Anda dapat menggunakan nomor serial yang sama.
I have two licenses, each with an add-on I require, how do I activate them in unison on my machine? Anda dapat menggunakan lisensi tunggal hanya pada satu mesin sekaligus. Anda tidak dapat mengaktifkan dua atau lebih lisensi pada saat yang sama pada satu mesin.
How can I use different Unity versions? Unity menganggap bahwa hanya satu versi Unity dapat dijalankan pada mesin. Anda dapat, namun, memiliki beberapa versi Unity yang diinstal dan berjalan di mesin Anda. Versi ini semua membutuhkan nomor serial yang sama. Jika Anda memiliki versi Unity yang berbeda yang membutuhkan lisensi yang berbeda, Anda perlu cara untuk menyalin lisensi di sekitar. Misalnya, Anda bisa menyimpan lisensi di Desktop Anda. Sebelum menjalankan versi tertentu, menyalin lisensi Unity (.ulf) file ke lokasi di mana lisensi harus disimpan.

Untuk bantuan lebih lanjut, silakan hubungi .
I am planning to replace an item of hardware and/or my OS. What should I do? Anda harus kembali lisensi Anda sebelum Anda membuat perubahan hardware atau perubahan OS ke mesin Anda. Anda juga harus kembali lisensi Anda sebelum Anda beralih mesin. Jika Anda gagal mengembalikan lisensi, server lisensi Unity melihat permintaan dari mesin lain dan memberi tahu Anda bahwa Anda telah mencapai batas aktivasi Anda untuk lisensi.
Where is my Unity license file stored? - /Library/Application Support/Unity (OS X)

- C:\ProgramData\Unity\ (Windows)

- C:\Users\(username)\AppData\Local\VirtualStore\ProgramData\Unity, jika Kontrol Akun Pengguna Windows (UAC) telah membatasi akses Anda ke C:\ProgramData\Unity. (Ini dapat terjadi jika folder dihapus atau Unity dimulai dengan izin administrasi pertama kali.)
