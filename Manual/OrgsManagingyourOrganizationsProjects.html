Mengelola Proyek Organisasi Anda

Ketika Anda membuat Proyek baru, ditugaskan ke Organisasi. Jika Anda tidak membuat organisasi tambahan, secara otomatis ditugaskan ke organisasi default Anda. Jika beberapa organisasi tersedia untuk akun Anda, Anda dapat memilih mana yang harus digunakan. Seperti itu, Proyek mewarisi pengaturan peran organisasi mereka. Namun, jika pengguna memerlukan akses ke alat atau informasi tertentu, Anda dapat memberi mereka peran yang berbeda yang unik untuk setiap Proyek, tanpa memberikan visibilitas di seluruh portofolio Proyek Organisasi. Untuk informasi lebih lanjut, lihat Members & Groups di Managing your Organization.

Sebagai Pemilik Organisasi, Anda juga memiliki alat berikut untuk mengelola Proyek Anda:

  • Archiving and restoring Projects
  • Mengubah nama Proyek
  • Proyek Transfer ke Organisasi baru

Archiving and restoring Projects

Proyek tidak dapat dihapus yang dapat menyebabkan kekacauan di dasbor Anda. Untuk mengurangi kekacauan, Anda dapat mengarsipkan Proyek yang Anda tidak lagi gunakan. Meneruskan Proyek mencegah tampilan dalam daftar ProjectsDalam Unity, Anda menggunakan proyek untuk merancang dan mengembangkan permainan. Proyek menyimpan semua file yang terkait dengan permainan, seperti file aset dan Adegan. More info
Lihat di Glossary Anda.

Sitemap Untuk menyaring Proyek: To archive a Project:

  1. Masuk ke .

  2. Pilih Proyek ke arsip.

  3. Dalam pane navigasi kiri, klik Settings > General.

  4. Dalam pane Proyek, klik tombol Archive my project for ALL Unity services.

  5. Di jendela pop-up arsip Proyek, masukkan nama Proyek untuk mengkonfirmasi bahwa Anda ingin menarsipkan Proyek, kemudian klik tombol Yes, archive it.

Untuk melihat Proyek arsip Anda dan mengaktifkannya:

  1. Masuk ke .

  2. Klik Lihat Proyek Arsip di kanan atas halaman.

  3. Temukan Proyek untuk mengaktifkan kembali dan klik tombol Unarchive.

Mengubah nama Proyek

Sitemap Untuk mengubah nama Proyek: To change the name of a Project:

  1. Masuk ke .
  2. Pilih Proyek untuk mengubah nama.
  3. Dalam pane navigasi kiri, klik Settings > General.
  4. Dalam kotak teks Rename this project, masukkan nama baru untuk Proyek, kemudian klik tombol Rename.
