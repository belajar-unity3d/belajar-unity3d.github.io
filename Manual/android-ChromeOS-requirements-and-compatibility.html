Persyaratan dan kompatibilitas

Sitemap Chrome OS bukan platform Editor Unity yang didukung.Important: Chrome OS isn’t a supported Unity Editor platform.

Walaupun Chrome Pengembangan OS sangat mirip dengan Android, ada perbedaan spesifik yang perlu Anda ketahui. Sebelum Anda mulai mengembangkan aplikasi Anda untuk OS Chrome, meninjau persyaratan, kompatibilitas fitur, dan masalah yang diketahui khusus untuk OS Chrome.

Requirements

Persyaratan sistem untuk OS Chrome mirip dengan Pengembangan Android. Satu-satunya perbedaan adalah bahwa ada masalah koneksi USB. Untuk menghindari ini, Anda dapat menghubungkan Editor Unity ke perangkat Chrome OS melalui Wi-Fi / Ethernet.

Unity mendukung semua merek laptop OS Chrome, tablet, netbooks dan desktop. Perangkat Chromecast tidak didukung. Unity mendukung versi Chrome OS R89 dan kemudian rilis pada saluran rilis yang stabil. Seperti platform lain, Anda dapat menggunakan beta dan pengembangan OS membangun kebijaksanaan Anda sendiri.

Known issues

  • Aplikasi yang mendukung auto-rotasi di semua orientasi menderita beberapa bug, termasuk berputar ketika mereka tidak harus dan mendapatkan set ke resolusi yang salah.
  • Jendela yang tidak tahan lama tidak berperilaku dengan benar pada perangkat OS Chrome yang memiliki kerang dan mode tablet. Ketika dalam mode tablet, aplikasi berjalan layar penuh dan tidak pernah kembali ke ukuran aslinya, bahkan ketika kembali ke mode clamshell.
