Membuat palet ubin untuk Isometric Tilemap

Untuk membuat Tile Palette untuk melukis Isometric TilemapGameObject yang memungkinkan Anda untuk dengan cepat membuat tingkat 2D menggunakan ubin dan overlay grid. More info
Lihat di Glossary:

  1. Buka jendela palet ubin (menu: Window > 2D > Tile Palette):

  2. Pilih Create New Palette untuk membuka menu drop-down.

  3. Mengatur Grid type ke tata letak yang sama dengan Isometric atau Isometric Z As Y Tilemap Anda melukis.

  4. Set Cell Size adalah untuk Manual. Tinggalkan X dan Z pada nilai default mereka, tetapi set Y ke nilai yang sama seperti nilai Y dari Ukuran Sel Tilemap. Nilai ini tergantung pada jenis proyeksi dari Tilemap. Lihat halaman Membuat Isometric Tilemap untuk rincian lebih lanjut.

  5. Pilih Create untuk menyelesaikan pengaturan Anda dan buat Tile Palette Asset baru.

  6. Untuk membuat perubahan pada palet ubin, klik dua kali Aset di Folder Aset untuk membukanya sebagai PrefabJenis aset yang memungkinkan Anda untuk menyimpan GameObject lengkap dengan komponen dan properti. Menyesuaikan tinggi ubin dalam palet

Ketika lukisan ubin pada Isometric Z as Y Tilemap, menentukan posisi Z dari Grid Anda melukis pada dengan mengatur nilai Z Position. Untuk jenis Tilemap ini, nilai posisi Z diterjemahkan ke dalam offset sepanjang sumbu Y, dan ubin dicat dengan posisi Z yang berbeda tampaknya memiliki ketinggian yang berbeda pada Tilemap.

Editor Z Position adalah di bagian bawah palet ubin:

Sesuaikan Posisi Z dengan memasukkan nilai yang diinginkan (hanya seluruh nomor). Ubin dicat pada Grid dengan posisi Z yang ditetapkan sampai nilainya diubah atau direset. Untuk mengubah nilai kembali ke default 0, pilih Reset.

Dalam pandangan SceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info
Lihat di Glossary, pratinjau sikat menampilkan kedua ubin pada posisi asli Sel dengan nilai Z 0 sebagai garis keluar biru, dan posisi dicatnya dengan offset Z-as-Y diterapkan ditampilkan dengan garis putih.

Dengan sikat ubin yang dipilih, gunakan tombol pintas berikut untuk dengan cepat beralih ke posisi Z yang berbeda.

Shortcut key: Command:
- Meningkatkan posisi Z oleh 1
= Mengurangi posisi Z oleh 1

Untuk menghapus ubin, atur Erase Brush ke posisi Z yang sama sebagai ubin target untuk dihapus. Sikat Erase tidak menghapus ubin dicat pada posisi Z yang berbeda untuk itu.
