Menginstal Unity dari baris perintah

Jika Anda ingin mengotomatiskan penyebaran Unity dalam organisasi, Anda dapat menginstal Editor dan komponen lain dari baris perintah. Komponen adalah program dan paket executable installer normal yang dapat Anda gunakan untuk mengotomatiskan penyebaran Unity.

Untuk mengunduh komponen, Anda menggunakan installer Unity untuk mengunduh komponen Unity ke folder di komputer Anda.

  1. Unduh dan jalankan installer. Versi rilis dari installer tersedia dari . Untuk versi pra-release dari installer, lihat halaman.

  2. Pada Choose Components, pilih semua komponen yang Anda otomatiskan deployment.

  3. Pada Choose Download and Install locations, klik tombol radio Download to.

  4. Masukkan lokasi di mana untuk menyimpan installer komponen.

  5. Selesai instalasi.

Navigasi ke lokasi unduh yang Anda tentukan. Dalam folder, Anda akan menemukan file installer untuk komponen yang Anda pilih dan file skrip (install.bat pada Windows atau install.sh pada Linux atau OS X) yang dapat Anda jalankan untuk menginstal komponen.

Untuk menginstal Unity pada komputer lain menggunakan komponen yang diunduh, menyalin isi folder ke komputer tujuan dan mengeksekusi skrip instalasi.

Menginstal individu Komponen unity pada Windows dari baris perintah

Gunakan opsi berikut ketika menginstal Editor dan komponen lain dari baris perintah di Windows.

Note: Menginstal argumen baris perintah adalah case-sensitif.

Instalasi editor Unity

Command Details
/S Melakukan instalasi diam (tidak ada pertanyaan yang diminta).
/D=PATH Mengatur direktori instalasi default. Berguna ketika dikombinasikan dengan opsi instalasi diam. Folder default adalah C:\Program Files (x86)\Unity (32-bit) atau C:\Program Files\Unity (64-bit).

Example:

UnitySetup64.exe /S /D=E:\Development\Unity

Contoh ini menginstal Unity diam-diam ke folder E:\Development\Unity, yang menjadi akar instalasi Unity. Dalam kasus ini, executable Editor dipasang di E:\Development\Unity\Editor\Unity.exe. Direktori instalasi default harus menjadi argumen terakhir pada baris perintah. Jalur yang disediakan tidak boleh dikelilingi oleh kutipan, bahkan jika jalan mengandung ruang.

WordPress.org

Untuk melakukan uninstall diam, menjalankan Uninstall.exe /S dari baris perintah atau script.

Note: Meskipun proses selesai segera, ada penundaan sebelum file benar-benar dihapus. Ini karena uninstaller disalin ke lokasi sementara untuk mengaktifkannya untuk menghapusnya sendiri. Pastikan direktori kerja tidak berada di dalam lokasi instalasi Unity. Jika itu, uninstaller tidak bisa menghapus folder.

Aset Standar menginstal

Untuk memasang Aset Standar secara diam-diam:

UnityStandardAssetsSetup.exe /S /D=E:\Development\Unity

Note: Jika menentukan folder, gunakan folder akar Unity (yaitu, folder yang berisi folder Editor, dan bukan folder di mana Unity.exe dipasang.)

Contoh Instalasi proyek

Untuk memasang proyek contoh secara diam-diam, gunakan:

UnityExampleProjectSetup.exe /S /D=E:\Development\Unity

Note: Folder default adalah C:\Users\Public\Documentation\Unity Projects\Standard Assets Example Project.

Menginstal Unity di OS X dari baris perintah

Penginstal individu Unity disediakan sebagai file .pkg, yang dapat diinstal menggunakan perintah installer, seperti yang dijelaskan di bawah ini.

Instalasi editor Unity

Untuk menginstal editor ke folder /Applications/Unity pada volume target yang ditentukan, masukkan:

sudo installer [-dumplog] -package Unity.pkg -target /

Aset Standar menginstal

Untuk menginstal Aset Standar ke dalam folder /Applications/Unity/Standard Assets pada volume yang ditentukan, masukkan:

sudo installer [-dumplog] -package StandardAssets.pkg -target /

Contoh Instalasi proyek

Untuk menginstal Contoh Proyek ke folder /Users/Shared/Unity/Standard-Assets pada volume yang ditentukan, masukkan:

sudo installer [-dumplog] -package Examples.pkg -target /

Unduhan Torrent

Jika Anda lebih suka mengunduh Unity melalui klien BitTorrent, Anda dapat mengunduh tautan torrent dari . Tidak semua versi memiliki unduhan torrent. Jika versi tersedia untuk diunduh sebagai torrent, opsi disajikan sebagai menu dropdown Torrent download (Win+Mac) di menu dropdown Downloads.

Menginstal beberapa versi sekaligus

Anda dapat menginstal beberapa versi Unity pada komputer yang sama.

Pada Mac, installer membuat folder yang disebut Unity, dan menulis folder yang ada dengan nama ini. Untuk menginstal lebih dari satu versi Unity di Mac Anda, mengubah nama folder Unity yang ada sebelum menginstal versi lain.

Pada PC, folder instalasi selalu dinamai Unity X.Y.Z[fp]W, di mana f adalah untuk rilis resmi, dan p digunakan untuk menandai rilis patch.

Kami sangat menyarankan bahwa jika Anda mengubah nama folder Unity, Anda nama folder baru secara logis (misalnya, tambahkan nomor versi ke akhir nama). Setiap shortcut yang ada, alias dan link ke docs offline mungkin tidak lagi menunjuk ke versi lama Unity. Ini dapat sangat membingungkan dengan docs offline; jika Anda tiba-tiba menemukan bahwa bookmark browser ke docs offline tidak lagi bekerja, kemudian periksa bahwa mereka memiliki nama folder yang tepat di URL.
