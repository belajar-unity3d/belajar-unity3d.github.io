Sprite Atlas V2 kriteria pengembangan

Sistem Sprite Atlas V2 dikembangkan dengan kriteria berikut dalam pikiran:

  1. Gunakan alur kerja Importer yang mendukung Cache Server dan fitur ADBV2 lainnya.

  2. Tidak ada perubahan yang dilakukan pada objek Sprite Atlas runtime.

  3. Pastikan bahwa ada jalur migrasi antara Atlas Sprite saat ini ke Sprite Atlas V2.

  4. Tidak ada perubahan besar ke alur kerja UX.

Enabling Sprite Atlas V2

Untuk mengaktifkan mode Sprite Atlas V2 Sprite Packer, pergi ke Edit > Project Settings > Sprite Packer > Mode, dan pilih Sprite Atlas V2 - Enabled dari menu dropdown.

Ketika Anda mengaktifkan Sprite Atlas V2, ada efek berikut:

  1. Ini menjadi mode default ketika Anda membuat Atlas Sprite melalui menu Aset (Assets > Create > Sprite Atlas).

  2. Unity secara otomatis meniru semua Aset Sprite Atlas V1 yang ada ke Sprite Atlas V2.

Karena migrasi otomatis, disarankan untuk membuat cadangan V1 Sprite Anda Atlases sebelum Anda mengaktifkan Sprite Atlas V2. Aset dikonversi ke V2 tidak kompatibel dengan V1, dan tidak dapat dikonversi kembali.
