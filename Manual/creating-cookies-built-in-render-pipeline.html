Membuat cookie untuk Pipa Render Built-in

Cara yang paling nyaman untuk membuat cookie untuk digunakan dengan Built-in Render PipelineRangkaian operasi yang mengambil isi dari Adegan, dan menampilkannya di layar. Unity memungkinkan Anda memilih dari pipa render yang dibangun sebelumnya, atau menulis sendiri. More info
Lihat di Glossary adalah untuk membuat tekstur abu-abu, impor tekstur itu ke Unity, dan kemudian biarkan Unity mengubah kecerahan tekstur menjadi alfa.

Perhatikan bahwa di Pipeline Render Built-in, cookie hanya menggunakan data dari saluran alfa. Ini berarti bahwa Anda dapat mendefinisikan bentuk untuk cookie, tetapi bukan warna.

Untuk melakukan ini:

  1. Buat tekstur abu-abu dalam editor gambar pilihan Anda. Jika Anda membuat cookie digunakan dengan Point Light, letakkan tekstur Anda seperti cubemapKoleksi enam tekstur persegi yang dapat mewakili refleksi di lingkungan atau skybox yang digambar di balik geometri Anda. Enam kotak membentuk wajah kubus imajiner yang mengelilingi objek; setiap wajah mewakili pandangan di sepanjang arah sumbu dunia (hingga, kiri, kanan, ke depan dan belakang). More info
    Lihat di Glossary    . Jika Anda membuat cookie digunakan dengan Spot Light atau Directional Light, meletakkan tekstur Anda sebagai tekstur 2D biasa.
  2. Tempatkan tekstur di folder Aset Proyek Anda untuk mengimpor tekstur menjadi Unity.
  3. Dalam pandangan Proyek Anda, pilih Aset Tekstur yang mewakili tekstur. Unity menampilkan Pengaturan impor tekstur di InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info
    Lihat di Glossary    .
  4. Dalam Inspektur, set nilai-nilai berikut:
    • Set Texture Type untuk Cookie
    • Set Light Type untuk mencocokkan Jenis Cahaya yang Anda buat cookie
    • Set Alpha Source untuk From Gray Scale
  5. Di bagian bawah Inspektur, klik Apply. Unity menerapkan pengaturan impor terbaru ke Aset Tekstur.

Perhatikan bahwa pixelsUnit terkecil dalam gambar komputer. Ukuran piksel tergantung pada resolusi layar Anda. Pencahayaan pixel dihitung pada setiap piksel layar. More info
Lihat di Glossary cookie tidak perlu benar-benar transparan atau buram, tetapi juga dapat menggabungkan nilai apa pun di antara. Anda dapat menggunakan nilai in-between untuk mensimulasikan debu atau kotoran di jalan cahaya, atau untuk mensimulasikan efek kaustik seperti yang diproduksi oleh lemari di lampu mobil.

Untuk informasi lebih lanjut tentang mengkonfigurasi pengaturan impor untuk cookie di Pipa Render Built-in, lihat Jenis Tekstur: Cookie.
