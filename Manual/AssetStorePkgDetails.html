Mengisi detail paket

Ketika Anda membuat draft paket, jendela Create new package muncul, mengandung beberapa bagian di mana Anda dapat mengisi informasi tentang paket Anda. Bagian-bagian ini juga tersedia setiap kali Anda mengedit draft paket Anda, tetapi Anda harus menyelesaikan semua bagian yang diperlukan sebelum mengirimkan paket Anda untuk persetujuan.

Topik di halaman ini memberikan instruksi tentang cara mengisi bagian ini:

  • Detail Paket
  • Metadata & Artwork
  • Paket
  • Review
  • Kirim paket untuk persetujuan

Detail Paket

Bagian Package Detail memungkinkan Anda mengatur versi (dengan deskripsi), kategori, dan harga untuk paket Anda.

Note: Meskipun judul paket muncul di bagian Package Detail, Anda hanya dapat mengubahnya di bagian Metadata.

Information: Description:
Status Status saat ini paket (termasuk Draft, Published, Declined, dan Pending).
Version Jumlah versi untuk paket Anda (seperti 1,5, 1.6, 1.7). Paket Aset Baru harus menggunakan versi default 1.0.

Ketika Anda melepaskan pembaruan untuk Anda Aset, Anda dapat meningkatkan jumlah versi di sini. Misalnya, jika Anda perlu memperbaiki bug, mengubah nomor versi minor (1.3 hingga 1.4). Jika Anda memperkenalkan perubahan besar, ubah versi utama (1.3 hingga 2.0).
Version Changes Jelaskan pembaruan apa pun yang memicu perubahan versi, seperti perbaikan bug, Aset baru, atau perubahan untuk mencocokkan versi Unity baru.
Category Pilih kategori yang paling cocok dengan paket Anda, karena ketika pelanggan mencari Aset pada Asset StoreSebuah perpustakaan yang berkembang dari aset gratis dan komersial yang diciptakan oleh Unity dan anggota komunitas. Menawarkan berbagai aset, dari tekstur, model dan animasi ke seluruh contoh proyek, tutorial dan ekstensi Editor. More info
Lihat di Glossary, mereka dapat menyaring daftar paket berdasarkan kategori yang Anda pilih. Jika Anda tidak yakin bahwa kategori untuk memilih, pergi ke dan melihat jenis aset apa yang tersedia dalam kategori serupa.
Price USD Anda dapat menetapkan harga $ 4.99 atau jumlah yang lebih tinggi dalam dolar AS. Untuk harga yang lebih rendah, aktifkan Free.

Ketika Anda selesai, klik Save.

Metadata & Artwork

Bagian Metadata & Artwork memungkinkan Anda mengatur nama paket, dan menyediakan gambar pemasaran, audio, dan video untuk menambahkan merek untuk paket Anda.

Karya seni (gambar kunci, audio/video, dan screenshots) dibagikan dalam semua bahasa.

Anda harus mengatur nama dan deskripsi Inggris, tetapi Anda juga dapat menentukan nama dan deskripsi di Jepang, Korea, dan Cina; untuk melakukan ini, klik tautan di bawah kolom Language. Ketika Anda menambahkan bahasa, tautan Edit baru muncul di bawah kolom Metadata untuk bahasa itu.

Anda hanya dapat mengedit Tags setelah paket Anda diterima untuk penerbitan.

Metadata

Metadata termasuk nama, deskripsi, dan kata kunci untuk paket Anda: itu, apa pun untuk membantu pelanggan menemukan paket Anda sementara mereka browsing atau mencari Toko Aset.

Untuk mengedit nama dan deskripsi paket Anda:

  1. Pada bagian Metadata & Artwork, di bawah Metadata, klik tombol Edit.

    Halaman Metadata untuk bahasa tertentu muncul:

  2. Masukkan Title paket Anda (diperlukan). Judul tidak boleh dimulai dengan kata “Unity” jika mungkin. Misalnya, Anda tidak dapat menggunakan "Unity Sculpting Tools", tetapi Anda dapat menggunakan "Sculpting Tools for Unity".

  3. Masukkan paket Description:

    • Jika paket Anda mengandung 3D Asset, termasuk informasi tentang jumlah poligon, ukuran Tekstur, dan yang membuat pipa kompatibel.
    • Jika paket Anda mengandung scriptsSepotong kode yang memungkinkan Anda untuk membuat Komponen Anda sendiri, memicu peristiwa permainan, memodifikasi sifat komponen dari waktu ke waktu dan menanggapi input pengguna dengan cara apa pun yang Anda sukai. More info
      Lihat di Glossary       dan alat, menyebutkan dependensi yang diperlukan untuk Aset (misalnya, Aset lain atau perangkat lunak pihak ketiga), karena Anda tidak dapat menambahkannya ke paket Aset Store dan pengguna harus mengimpornya secara terpisah.

    Anda dapat menghias teks ini dengan salah satu tag HTML berikut:

    • EM (untuk italics). Contohnya: This is for <EM>italicized</EM> text.
    • STRONG (untuk berani). Contohnya: This is for <STRONG>bold</STRONG> text.
    • A (untuk hyperlink). Contohnya: This links <A HREF="http://www.example.com">here</A>.
    • P (untuk mendefinisikan paragraf). Contohnya: <P>This is a new paragraph</P>.
    • BR (untuk istirahat garis). Contohnya: This is the first line.</BR>This text appears on the next line.

  4. Anda dapat secara opsional menambahkan daftar Keywords yang dipisahkan oleh whitespace untuk mempengaruhi hasil Aset Anda dalam pencarian Aset Store. Ada batas karakter 255.

  5. Klik Save ketika Anda selesai dan kemudian klik OK ketika kotak dialog muncul.

  6. Untuk kembali ke jendela Package, klik Go Back.

Gambar kunci

Gambar utama harus menunjukkan paket Aset Store Anda dan membantu menonjol. Anda dapat mengatur empat resolusi yang berbeda sehingga gambar kunci Anda muncul sebagai ikon atau thumbnails ketika pelanggan menelusuri atau menyaring daftar paket yang tersedia di Asset Store, dan sebagai gambar yang lebih besar di halaman produk utama atau di media sosial. Ini juga mungkin muncul dalam materi promosi.

Berikut adalah beberapa persyaratan umum untuk gambar yang Anda unggah di bagian ini:

  • Gambar tidak dapat mengandung berikut:
    • logo Unity
    • Photogallery
    • Default Unity SkyboxJenis khusus Bahan yang digunakan untuk mewakili langit. Biasanya enam sisi. More info
      Lihat di Glossary       (misalnya, sebagai latar belakang)
  • Gambar tidak boleh buram, diregangkan, atau ditumpuk.
  • Gambar tidak hanya bisa screenshot dari Editor Unity.
  • Anda harus membuat desain yang berbeda untuk gambar ukuran yang berbeda sehingga objek atau visual dalam ikon dan gambar kartu yang legible dan terpisah.

Audio/Video

Anda dapat meng-upload file audio atau video yang menunjukkan paket Asset Store Anda atau mengandung. Ini tersedia di halaman produk utama Anda (seperti thumbnails Anda dapat mengklik untuk bermain).

Sitemap Jika paket Anda berisi file audio atau video, Anda harus mengunggah sampel Aset media Anda di sini. Jika paket Anda berisi alat, scripting, atau aset teknis lainnya, termasuk tutorial video menjelaskan proses pengaturan, dan memberikan gambaran tentang fungsionalitas Aset Anda. Jika paket Anda berisi animasi, termasuk video dari semua animasi yang tersedia.Note: If your package contains audio or video files, you must upload samples of your media Assets here. If your package contains tools, scripting, or other technical Assets, include a video tutorial explaining the setup process, and give an overview of your Asset’s functionality. If your package contains animations, include a video of all the available animations.

Anda dapat menanamkan file yang diunggah di reel acara dengan screenshot paket Anda. File audio dan video dapat berasal dari komputer atau perangkat lokal Anda, atau Anda dapat menentukan tautan ke layanan ini:

  • YouTube
  • Vimeo
  • SoundCloud
  • MixCloud
  • SketchFab

Untuk mengatur satu atau lebih gambar kunci:

  1. Pada bagian Metadata & Artwork, di bawah kolom Audio/Video, klik tombol Edit.

    Halaman Audio/Video muncul:

  2. Anda dapat meng-upload file audio atau video menggunakan metode ini:

    • Seret-dan-jatuhkan file ke kotak abu-abu.
    • Klik Choose File untuk memilih file dari browser file.
    • Masukkan tautan ke file online di kotak teks dan klik Upload link.

    Gambar yang diunggah muncul dengan tombol Remove di bawahnya.

  3. Untuk kembali ke jendela Package, klik Go Back.

Screenshots

Anda dapat meng-upload screenshot dari Aset paket Anda dalam tindakan. Ini tersedia di halaman produk utama Anda (seperti thumbnails Anda dapat mengklik untuk memperbesar).

Screenshot sama pentingnya dengan gambar kunci saat mencoba menjual paket Anda. Screenshot menampilkan paket Anda, menunjukkan pengguna bagaimana bekerja dan apa yang bisa dilakukan. Pengguna sering menggunakan screenshot sebagai jenis pratinjau paket sebelum mereka benar-benar membelinya. Kurator Toko Aset lebih suka memiliki screenshot untuk sebagian besar Aset. Misalnya, jika itu Model, tambahkan gambar yang menampilkan Aset dari perspektif yang berbeda, baik sepenuhnya bertekstur dan dalam bingkai kawat.

Untuk mengatur satu atau lebih screenshot:

  1. Klik tombol Edit di bawah bagian Screenshots.

    Halaman Screenshot muncul:

  2. Anda dapat meng-upload satu atau lebih screenshot menggunakan salah satu metode ini:

    • Seret-dan-jatuhkan file ke kotak abu-abu.
    • Klik Choose File untuk memilih file dari browser file.

    Gambar yang diunggah muncul dengan ikon X merah (untuk menghapusnya) di bawahnya.

  3. Untuk kembali ke jendela Package, klik Go Back.

Paket

Bagian Package upload memungkinkan Anda meluncurkan Unity secara langsung setelah membuat paket draft atau mengedit rincian paket.

Ketika Anda mengklik tombol Launch Unity, Proyek Anda muncul di Editor Unity jika Unity sudah terbuka. Jika tidak, Hub membuka, memungkinkan Anda untuk memilih Proyek mana yang terbuka.

Jika Anda telah mengunggah Aset ke draft paket Anda, mereka muncul di bagian ini dengan tombol Remove dan Edit:

Jika Anda ingin menghapus Aset dari paket Anda, klik tombol Remove.

Sebelum mengirimkan paket untuk persetujuan, klik tombol Edit.

Untuk setiap bagian, pilih Yes untuk menampilkan pertanyaan dan satu set kotak centang. Kemudian periksa atau periksa opsi yang menjawab pertanyaan tentang paket Anda. Ketika Anda selesai, klik Save untuk kembali ke bagian Package upload.

Untuk informasi lebih lanjut tentang menambahkan dan mengunggah Aset ke draft paket Anda, lihat Unggah Aset ke paket Anda.

Review

Di bawah bagian Review, klik Preview in Asset Store untuk melihat bagaimana paket Anda akan terlihat di Aset Store setelah menerbitkannya. Anda juga dapat melihat pratinjau URL di bawah Public links:

Kirim paket untuk persetujuan

Di bawah bagian Submit package for approval, klik Auto publish untuk menunjukkan apakah Anda ingin paket muncul di Aset Store langsung setelah persetujuan Unity.

Anda juga harus mengkonfirmasi apakah Anda memiliki hak untuk menjual Aset.

Untuk instruksi rinci, lihat Mengirimkan paket Anda untuk persetujuan.
