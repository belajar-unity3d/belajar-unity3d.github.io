Alat Terrain

Ini menyediakan fungsi tambahan di atas alat Terrain built-in Unity. Ini termasuk fitur yang lebih canggih seperti Erosion, alat pemupuk, dan filter masker sikat. Ini juga mengandung TerrainPemandangan di tempat kejadian Anda. Terrain GameObject menambahkan pesawat datar besar untuk adegan Anda dan Anda dapat menggunakan jendela Inspektur Terrain untuk menciptakan lanskap rinci. More info
Lihat di Glossary Toolbox, yang memungkinkan Anda membuat Terrain baru dari file preset atau impor heightmapsTekstur abu-abu yang menyimpan data tinggi untuk objek. Setiap pixel menyimpan perbedaan tinggi tegak lurus dengan wajah yang pixel mewakili.
Lihat di Glossary, pengaturan perubahan batch pada beberapa ubin Terrain, dan splatmap impor / ekspor dan peta tinggi.

Alat add-on ini cocok jika Anda memerlukan lebih banyak kontrol atas penampilan Terrain Anda, dan ingin merampingkan alur kerja Terrain. Ini juga berguna ketika Anda ingin membuat Terrain yang lebih kompleks, atau penulis Terrain Tekstur data dalam alat eksternal seperti Houdini dan Mesin Dunia.
