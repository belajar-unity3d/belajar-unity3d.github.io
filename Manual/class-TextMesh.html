Sitemap Ini adalah komponen legacy yang memiliki fungsi terbatas. Untuk informasi tentang cara-cara yang lebih baru-baru ini, lengkap untuk menampilkan teks, lihat Note:.Membuat antarmuka pengguna (UI).

Ikuti langkah-langkah ini untuk membuat Mesh Teks dengan Font kustom:

  1. Mengimpor font dengan menyeret Font TrueType - sebuah file .ttf - dari Explorer (Windows) atau Finder (OS X) ke dalam Project View.
  2. Pilih font impor di Tampilan Proyek.
  3. Pilih GameObject > Create Other > 3D Text. Sekarang Anda telah membuat jala teks dengan Font TrueType kustom Anda. Anda dapat meningkatkan teks dan memindahkannya di sekitar menggunakan kontrol Scene ViewAn interactive view into the world you are creating. Sitemap Jika Anda ingin mengubah font untuk Teks Mesh, perlu mengatur properti font komponen dan juga mengatur tekstur bahan font ke tekstur font yang benar. Tekstur ini dapat terletak menggunakan lipatan aset font. Jika Anda lupa untuk mengatur tekstur maka teks di mesh akan muncul blocky dan misaligned.Note: If you want to change the font for a Text Mesh, need to set the component’s font property and also set the texture of the font material to the correct font texture. This texture can be located using the font asset’s foldout. If you forget to set the texture then the text in the mesh will appear blocky and misaligned.

Praktikum

  • Anda dapat mengunduh Font TrueType gratis dari 1001font gratis.com (download font Windows karena mengandung Font TrueType).
  • Jika Anda skrip properti Text, Anda dapat menambahkan garis istirahat dengan memasukkan karakter melarikan diri "\n" dalam string Anda.
  • Anda dapat menggunakan markup sederhana untuk style text meshes. Lihat halaman Catalog untuk rincian lebih lanjut.
  • Font di Unity render font glyphs ke peta tekstur sebelum rendering lebih lanjut. Jika ukuran font diatur terlalu kecil, tekstur font ini akan muncul blocky. Karena aset TeksMesh dirender menggunakan quadsObjek primitif yang menyerupai pesawat tetapi tepinya hanya satu unit panjang, hanya menggunakan 4 simpul, dan permukaan berorientasi pada pesawat XY ruang koordinat lokal.
