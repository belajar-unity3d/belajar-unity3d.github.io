Menerapkan produk IAP

Unity merekomendasikan menambahkan dan mengelola produk IAP Anda di konsol UDP. Konsol UDP adalah sumber kebenaran untuk apa yang diajukan ke sistem back-end toko. Selain itu, konsol UDP memungkinkan Anda:Catalog in the UDP console is the source of truth for what is submitted to the storeâ€™s back-end systems. Additionally, the UDP console lets you:

  • Kelola beberapa harga dan mata uang IAP
  • Tambahkan deskripsi dalam beberapa bahasa
  • Unggah item dalam jumlah besar melalui file CSV (multiple currency)

Note: Jika Anda menggunakan paket UDP hanya, Anda masih harus membuat setidaknya satu item IAP di Editor Unity untuk menguji bahwa itu dapat menyinkronkan dengan konsol UDP dengan benar. Jika Anda menggunakan paket Unity IAPUnity In App Purchase
Lihat di Glossary (implementasikan codeless), Anda harus menambahkan setiap item IAP di Editor Unity dan mengunggah semuanya ke konsol UDP. Jika Anda menggunakan paket Unity IAP dan telah menerapkan IAP dalam kode, buat item IAP di konsol UDP.

Jika Anda mempertahankan Katalog IAP di klien game Anda, UDP tetap disinkronkan antara Editor Unity dan konsol UDP.

Mengelola pembelian dalam aplikasi di konsol UDP

Dalam, arahkan ke permainan Anda melalui halaman My Games. Di halaman Game Info, pilih tombol EDIT INFO untuk memasukkan mode edit. Untuk menghemat perubahan pilih SAVE. Untuk membuang perubahan, pilih CANCEL. Bagian ini mencakup cara mengedit, membuat dan menghapus produk IAP dari konsol UDP.

Membuat item IAP baru

Untuk membuat item IAP baru di konsol UDP:

  1. Pilih EDIT INFO di halaman Informasi Game.
  2. Pilih Add Item untuk membuat item baru.
  3. Tentukan dan simpan Informasi produk Anda.
    1. Ikuti persyaratan untuk ID Produk untuk memastikan mereka berlaku untuk toko aplikasi.
    2. Pastikan produk IAP yang Anda tentukan dalam permainan Anda menggunakan Product ID yang sama yang ditetapkan dalam Katalog IAP.
  4. Pilih Manage amounts and currencies ke:
    1. Mengkonversi harga IAP dari USD ke mata uang asing.
    2. Sesuaikan harga dalam mata uang ini.
  5. Pilih SAVE untuk menyimpan perubahan Anda.

Note: Jika Anda memiliki banyak item IAP untuk membuat, mempertimbangkan menggunakan fitur Impor IAP Massal untuk membuat semua item IAP Anda sekaligus, termasuk deskripsi dan harga lokal dalam beberapa mata uang.

Jika menggunakan paket UDP, item IAP baru yang Anda tambahkan di Konsol disinkronkan ke proyek Unity Anda. Anda tidak perlu membangun kembali permainan Anda untuk memasukkannya.

Mengedit Deskripsi IAP

Untuk mengedit deskripsi IAP, lanjutkan sebagai berikut.

  1. Pilih EDIT INFO di halaman Informasi Game.
  2. Pilih bahasa yang ingin Anda edit deskripsi Anda melalui pemilih bahasa drop-down.
  3. Di bagian Pembelian In-App, pilih ikon pensil untuk mengedit item IAP yang ada.
  4. Edit informasi item IAP Anda:
    1. Product Name, nama item IAP
    2. Description, untuk menjelaskan item IAP
  5. Pilih SAVE untuk menyimpan perubahan Anda.

Note: Langkah di atas hanya mengedit informasi item IAP Anda untuk bahasa yang dipilih. Jika Anda memiliki sejumlah besar item IAP dan bahasa untuk mendukung, pertimbangkan menggunakan Impor IAP Massal untuk mengimpor informasi ini.

Mengedit harga IAP dan mata uang

  1. Pilih EDIT INFO di halaman Informasi Game.
  2. Pilih Manage amounts and currencies untuk item IAP untuk:
    1. Mengubah harga IAP.
    2. Konversikan dari USD ke mata uang asing.
    3. Sesuaikan harga dalam mata uang ini.
  3. Pilih SAVE untuk menyimpan perubahan Anda.

Hint: Untuk mengatur mata uang sebagai mata uang favorit, pilih ikon bintang di sebelah kode mata uang mereka. Ini mencubit mata uang ke bagian atas daftar. Pilih bintang ke pin.

Note: Jika Anda memiliki banyak item IAP dan mata uang untuk mendukung, Anda dapat menggunakan fungsi Convert untuk secara otomatis mengubah semua harga IAP Anda menjadi mata uang global, berdasarkan jumlah USD yang ditetapkan untuk setiap item IAP. Ini menimpa harga lokal yang telah Anda set sebelumnya.

Atau, gunakan Impor IAP Massal untuk mengimpor semua informasi ini sekaligus.

Menghapus item IAP

  1. Pilih EDIT INFO di halaman Informasi Game.
  2. Pilih trash dapat ikon untuk menghapus item IAP.
  3. Pilih SAVE untuk menyimpan perubahan Anda.

Mengimpor item IAP dalam jumlah besar

  1. Pilih EDIT INFO di halaman Informasi Game.
  2. Pilih IMPORT CATALOG di sudut kanan atas panel Pembelian In-App.
    Lihat IAP Massal Impor untuk cakupan penuh fungsi ini.
  3. Pilih SAVE untuk menyimpan perubahan Anda.

Mengimpor informasi permainan dari Google Play

Jika Anda telah menerbitkan game Anda di Google Play, Anda dapat mengimpor informasi dasar menggunakan URL Google Play game Anda:

  1. Pilih Import from Google Play.
  2. Tempel Google Play URL permainan Anda ke kotak input.
  3. Untuk menentukan bahasa mana yang secara khusus impor, gunakan URL Google Play yang berisi suffix bahasa, misalnya untuk penggunaan Italia &hl=it. Login Sitemap Login Play URL tidak mengandung suffix bahasa, UDP mengambil informasi untuk bahasa yang saat ini Anda mengedit di konsol UDP (jika diatur di Google Play).
    If your Google Play URL doesnâ€™t contain the language suffix, UDP fetches the information for the language that youâ€™re currently editing in the UDP console (if it is set on Google Play).
  4. Pilih Import.
  5. Double-check hasilnya dalam beberapa kasus alamat IP komputer Anda mungkin akhirnya menentukan bahasa mana yang Anda layani.

Impor IAP Massal

IAP Massal Fitur impor memudahkan mengelola Katalog IAP Anda jika Anda memiliki banyak item IAP. Ini memungkinkan Anda mengunggah file CSV dalam format UDP biasa dan dalam format Google Play ke Konsol UDP dengan semua rincian produk IAP Anda, termasuk harga dalam beberapa mata uang dan deskripsi dalam beberapa bahasa.

Unity menyarankan Anda masih menambahkan item IAP di Editor Unity untuk tujuan pengujian untuk memastikan Anda dapat membeli item IAP di lingkungan kotak pasir.

Note: Jika file CSV Anda berisi beberapa bahasa, menambahkan bahasa yang didukung sebelum Anda mengimpor file CSV.

Bagian ini menjelaskan cara mengimpor barang IAP Anda dalam jumlah besar melalui CSV.

Persyaratan implementasi

Untuk menambahkan produk IAP baru

Anda hanya boleh menggunakan fungsi Impor IAP Massal untuk menambahkan produk IAP baru jika Anda menerapkan UDP:

  • Dengan paket UDP
  • Dengan Unity IAP (pengimplementasikanan kode)

Unity menyarankan Anda masih menambahkan item IAP di Editor Unity untuk tujuan pengujian untuk memastikan Anda dapat membeli item IAP di lingkungan kotak pasir.

Untuk menambahkan deskripsi dan harga lokal

Untuk semua implementasi, Anda dapat menggunakan Impor IAP Massal untuk menetapkan deskripsi dalam lebih banyak bahasa dan harga dalam lebih banyak mata uang untuk produk IAP yang ada.

Dengan Paket UDP

Anda dapat menggunakan Bulk IAP Import untuk sepenuhnya mendefinisikan ulang katalog IAP permainan Anda tanpa harus membangun kembali permainan Anda.

Anda hanya perlu memiliki implementasi UDP dalam urutan kerja yang baik.

Dengan Unity IAP (pengimplementasikanan kode)

Dengan Unity IAP, Anda hanya dapat menyinkronkan Katalog IAP Anda dari Editor ke Konsol UDP.

Jika Anda akan mengunggah Katalog IAP permainan Anda ke Konsol UDP melalui Bulk IAP Upload, Anda harus mempersiapkan permainan Anda untuk mengambil informasi produk IAP dari Konsol UDP, daripada dari katalog IAP default klien game.

Untuk memungkinkan permainan Anda untuk mengambil Katalog IAP dari konsol UDP, jangan beri tahu metode pengambilan produk IAP dalam kode Anda.

Unduh Template CSV

UDP menyediakan template CSV yang dapat Anda gunakan untuk impor massal produk IAP. Templat dalam format UDP biasa untuk file CSV.

  1. Pilih EDIT INFO pada halaman Game Information.
  2. Pergi ke panel In-App Purchases.
  3. Pilih IMPORT CATALOG.
  4. Pilih Download Template.

Isi file CSV

File CSV menggunakan kommas (,) dan semicolons (;) untuk memisahkan nilai data. Commas memisahkan nilai data primer, dan subnilai terpisah semicolons.

Setiap item IAP harus muncul sepenuhnya pada satu baris dalam file CSV.

Anda harus memasukkan data dalam format yang tepat dari template yang disediakan. Konsol UDP menampilkan pesan kesalahan jika Anda mengunggah file yang berisi kesalahan, seperti sintaks harga tidak valid atau bahasa yang hilang.

Unggah file CSV Anda

  1. Pilih EDIT INFO pada halaman Game Information.
  2. Pergi ke bagian In-App Purchases.
  3. Pilih IMPORT CATALOG.
  4. Pilih Upload CSV atau drag dan drop file CSV Anda di kotak yang ditunjuk.
  5. Pilih IMPORT untuk mengunggah IAP Anda Katalog ke konsol UDP.

Sitemap Mengungkapkan file CSV sepenuhnya menimpa katalog IAP revisi ini. Konsol UDP hanya menggunakan informasi yang terkandung dalam file CSV Anda; informasi IAP lain yang sebelumnya Anda masukkan melalui konsol UDP hilang.Note: Uploading a CSV file entirely overwrites this revisionâ€™s IAP catalog. The UDP console only uses the information contained in your CSV file; any other IAP information you previously entered via the UDP console is lost.

Ekspor file CSV

Untuk menggunakan ulang atau memodifikasi Katalog IAP yang ada:

  1. Pilih EDIT INFO di halaman Informasi Game.
  2. Pergi ke bagian Pembelian In-App.
  3. Pilih EXPORT CATALOG.
    Ini mengekspor Katalog IAP yang ada sebagai file CSV dalam format UDP biasa.
  4. Modifikasi file CSV.
  5. Unggah file CSV Anda.

Tombol EXPORT CATALOG hanya ditampilkan jika Anda memiliki produk IAP di Katalog IAP Anda, dan hanya terlihat dalam mode edit. Gambar di bawah ini menunjukkan tombol ini di bagian Pembelian In-App.

Produk IAP impor dari format Google Play

UDP juga mendukung file CSV dalam format Google Play.

Untuk mengimpor produk IAP menggunakan file Google Play CSV:

  1. Pilih EDIT INFO pada halaman Game Information.
  2. Pergi ke bagian In-App Purchases.
  3. Klik IMPORT CATALOG.
  4. Pilih kotak centang CSV file in Google Play format.
  5. Pilih Unggah CSV atau drag dan drop file CSV Anda di kotak yang ditunjuk.
    1. Dalam bidang Google Play default currency, pilih mata uang default untuk permainan Anda.
  6. Pilih Import untuk mengunggah IAP Anda Katalog ke konsol UDP.

Pricing

Google Play menetapkan harga negara, UDP menetapkan harga dengan mata uang. Oleh karena itu negara-negara peta impor untuk mata uang. Jika produk IAP Anda telah menetapkan harga yang berbeda di seluruh negara menggunakan mata uang yang sama (misalnya, EUR di negara-negara Eropa) UDP menggunakan negara pertama dalam daftar untuk mata uang. Jika ada beberapa negara dengan USD yang terdaftar, penduduk setempat AS digunakan.

UDP tidak mendukung template harga Google Play. Pengaturan ini diabaikan. UDP hanya menggunakan harga lokal yang disediakan dalam file CSV.

Mata uang default

UDP menggunakan USD sebagai mata uang default. Jika Anda tidak memiliki harga USD di Google Play, UDP mengubah harga dan mata uang default Google Play menjadi USD dan menggunakan ini sebagai harga default UDP.

Harga lokal yang ditetapkan dalam file CSV Anda dipertahankan dalam mata uang masing-masing.

Untuk negara-negara yang tidak secara eksplisit diatur dalam file CSV, UDP mengubah harga dan mata uang default Google Play Anda menjadi mata uang masing-masing.

Produk IAP

Jenis produk IAP (mungkin / tidak dapat dikonsumsi) tidak dapat ditentukan dari file CSV Anda; Anda perlu memberitahu UDP jenis setiap produk IAP.

UDP tidak mendukung Auto-Translate. Pengaturan ini diabaikan. UDP hanya menggunakan deskripsi lokal yang disediakan dalam file CSV.
