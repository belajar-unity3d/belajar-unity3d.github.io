Fitur set

Fitur set adalah kelompok packagesPaket adalah koleksi aset untuk dibagikan dan digunakan kembali dalam Unity. Manajer Paket Unity (UPM) dapat menampilkan, menambahkan, dan menghapus paket dari proyek Anda. Paket-paket ini berasal dari Unity Package Manager dan memberikan metode mendasar untuk memberikan fungsi Unity. Namun, Unity Package Manager juga dapat menampilkan Paket Asset Store yang Anda unduh dari Asset Store. More info
Lihat di Glossary yang memungkinkan Anda untuk mencapai hasil spesifik dalam Unity. Misalnya, 2D feature set menyediakan semua fungsi yang Anda butuhkan untuk membuat game 2D untuk berbagai gaya seni.

Semua paket dalam set fitur kompatibel satu sama lain untuk versi set fitur. Anda dapat menginstal set fitur menggunakan Package Manager.

Set fitur berikut tersedia dalam rilis Unity ini:

  • 2D: Buat dunia game 2D dengan karakter animasi dan fisika 2D.
  • Gedung Dunia 3D: Gunakan alat yang ramah seniman untuk menciptakan lingkungan 3D, open-world yang dapat dipercaya dan interaktif.
  • Karakter 3D dan Animasi: Animate karakter dan membuat cutscenes, urutan atau film.
  • Gameplay dan Storytelling: Buat tembakan cameraKomponen yang menciptakan gambar sudut pandang tertentu di tempat kejadian Anda. Output ditarik ke layar atau ditangkap sebagai tekstur. More info
    Lihat di Glossary     dinamis, dan mengatur konten bioskop untuk membuat cerita untuk proyek Anda.
  • Mobil: Optimalkan aplikasi Anda untuk perangkat seluler.
  • AR: Menciptakan pengalaman nyata yang berinteraksi dengan dunia nyata.
  • VR: Tambahkan berinteraksiivitas ke aplikasi Anda untuk menciptakan pengalaman realitas virtual.
  • Photogallery: Buat film, acara TV, dan konten linear animasi lainnya.
  • Engineering: Dukungan yang dapat diaktifkan untuk lingkungan pengembangan terintegrasi, dan gunakan alat untuk menguji dan menganalisis kinerja proyek Anda.

Untuk informasi tentang cara menemukan apa set fitur tertentu mengandung, lihat Fitur (detail) pandangan.

Untuk informasi tentang cara menginstal set fitur di Manajer Paket, lihat Menginstal set fitur.
