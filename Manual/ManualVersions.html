Documentation versions

Dokumentasi Unity adalah Panduan Pengguna Unity dan Referensi API Scripting Unity.

Karena kami terus meningkatkan Unity (yaitu; Editor dan mesin), kami menambahkan fitur baru, meningkatkan fitur yang ada, dan kadang-kadang menghapus fitur lama.

Dengan setiap rilis Unity, dokumentasi Unity berubah untuk mencerminkan ini, jadi pastikan Anda menggunakan versi yang benar dari User Manual dan Scripting API Reference untuk mencocokkan versi Unity yang Anda gunakan.

Versi dokumentasi yang menyertai rilis penuh Unity saat ini selalu tersedia secara online pada .

Dokumentasi Installer (offline)

Serta menggunakan dokumentasi online, Anda dapat menginstal salinan lokal dokumentasi (offline) ketika Anda menginstal perangkat lunak Unity, menggunakan:

  • Download Asisten
  • The

Documentation updates

Kami menerbitkan dokumentasi online untuk versi rilis penuh Unity saat ini secara teratur dengan koreksi dan konten yang hilang secara teratur. (Anda dapat melihat versi penerbit dokumentasi di bagian bawah setiap halaman.)

Jika ada rilis Unity beta saat ini, kami juga memperbarui dokumentasi rilis beta publik saat ini dengan koreksi, konten yang hilang, dan pembaruan fitur secara teratur.

Namun, perhatikan bahwa kami tidak memperbarui dan menerbitkan ulang:

  • dokumentasi installer (offline).

  • dokumentasi legacy, yaitu; dokumentasi untuk versi Unity yang tidak lagi beta saat ini atau rilis penuh Unity.

Jika Anda memerlukan penerbitan dokumentasi terbaru secara offline, Anda dapat mengunduhnya di luar installer. Lihat halaman Dokumentasi Offline untuk rincian.

Versi dokumentasi mana yang akan digunakan

Kebanyakan orang menggunakan versi rilis penuh terbaru Unity dan menggunakan dokumentasi di .

Beberapa orang perlu menggunakan versi Unity yang lebih tua. Ini mungkin kasus bagi Anda jika Anda mempertahankan atau dalam pengembangan jangka panjang proyek yang menggunakan versi Unity yang lebih tua.

Jika Anda menggunakan versi Unity, installer (offline) dokumentasi cocok dengan versi Unity. Namun jika Anda memilih untuk tidak menginstal dokumentasi lokal, atau ingin memastikan Anda dapat mengakses penerbitan dokumentasi terbaru, Anda dapat mengakses versi lama dari dokumentasi Unity secara online. (Lihat daftar di bawah ini.)
