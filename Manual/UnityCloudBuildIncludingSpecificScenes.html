Termasuk adegan spesifik

Secara default, Unity Cloud Build membangun Adegan yang telah Anda tambahkan ke proyek Anda di Editor Unity ke File > Build Settings > Scenes in Build. Klik Add Open Scenes untuk menambahkan Adegan tambahan. Untuk informasi lebih lanjut, lihat Photogallery.

Untuk menginstruksikan Unity Cloud BuildLayanan integrasi berkelanjutan untuk proyek Unity yang mengotomatiskan proses pembuatan build pada server Unity. More info
Lihat di Glossary untuk mengabaikan daftar Scenes in Build di Editor Unity dan membangun daftar yang berbeda dari Adegan, Anda perlu memberikan SceneA Scene contains the environments and menus of your game. Think of each unique Scene file as a unique level. In each Scene, you place your environments, obstacles, and decorations, essentially designing and building your game in pieces. More info

Pada website Unity Developer, pergi ke build target Opsi Lanjutan.

Tambahkan daftar adegan yang ingin Anda masukkan ke Scene List. Menyediakan path file sebagai jalur relatif dari direktori Assets ke setiap file Adegan yang ingin Anda masukkan.

Ini berguna untuk pengembangan. Misalnya, Anda mungkin memiliki satu target build yang mencakup semua Adegan, tetapi membutuhkan waktu lama untuk membangun. Jika Anda hanya bekerja pada satu Adegan, itu mungkin membantu untuk menggunakan target build lain yang hanya mencakup bahwa Adegan. Ini harus membangun lebih cepat, dan memungkinkan iterasi lebih cepat dari versi perkembangan.
