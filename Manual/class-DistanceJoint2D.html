Notes

Tujuan dari Joint 2D ini adalah untuk menjaga jarak antara dua poin. Dua poin bisa menjadi dua komponen RigidbodyKomponen yang memungkinkan GameObject untuk dipengaruhi oleh gravitasi simulasi dan kekuatan lainnya. More info
Lihat di Glossary 2D atau komponen 2D yang kaku dan posisi tetap di dunia. Untuk menghubungkan komponen Rigidbody 2D ke posisi tetap di dunia, atur kolom Connected Rigidbody ke None.

Sendi ini 2D tidak menerapkan torsi, atau rotasi. Ini berlaku gaya linier untuk item yang terhubung, menggunakan pegas yang sangat kaku, simulasi untuk mempertahankan jarak. Anda tidak dapat mengkonfigurasi pegas.

Bersama 2D ini memiliki batasan yang dapat dipilih:

  • Melanjutkan A: Mempertahankan jarak tetap antara dua titik jangkar pada dua tubuh (ketika Max Distance Only tidak dicentang).
  • Constraint B: Mempertahankan jarak maksimum hanya antara dua titik jangkar pada dua tubuh (ketika Max Distance Only diperiksa).

Anda dapat menggunakan Joint 2D ini untuk membangun objek fisik yang perlu berperilaku seolah-olah terhubung dengan koneksi yang kaku yang dapat berputar.

  • Menggunakan batasan A (Max Distance Only unchecked), Anda dapat membuat koneksi panjang tetap, seperti dua roda pada sepeda.
  • Menggunakan constraint B (Max Distance Only diperiksa), Anda dapat membuat koneksi panjang yang terlatih, seperti yo-yo bergerak menuju dan jauh dari tangan.
