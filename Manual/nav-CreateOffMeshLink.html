Membuat Link Off-mesh

Tautan Off-Mesh digunakan untuk membuat persimpangan jalan di luar permukaan meshGrafik utama primitif Unity. Mesh membuat sebagian besar dunia 3D Anda. Unity mendukung mesh poligon triangulat atau Quadrangulasi. Nurbs, Nurms, permukaan Subdiv harus dikonversi ke poligon. More info
Lihat di Glossary navigasi yang dapat berjalan. Misalnya, melompati parit atau pagar, atau membuka pintu sebelum berjalan melaluinya, dapat semua digambarkan sebagai tautan Off-mesh.

Kami akan menambahkan komponen Off-Mesh Link untuk menggambarkan melompat dari platform atas ke tanah.

  1. 1 two cylinders: Game Object > 3D Object > cylinder.
  2. Anda dapat meningkatkan silinder ke (0.1, 0.5, 0,1) untuk memudahkan bekerja dengan mereka.
  3. Pindahkan first cylinder di tepi platform atas, dekat dengan permukaan NavMeshSebuah mesh yang Unity menghasilkan perkiraan daerah dan hambatan yang dapat berjalan di lingkungan Anda untuk mencari jalan dan navigasi yang dikendalikan AI. More info
    Lihat di Glossary    .
  4. Tempatkan second cylinder di tanah, dekat dengan NavMesh, di lokasi di mana tautan harus mendarat.
  5. Pilih silinder di sebelah kiri dan tambahkan komponen Off-Mesh Link ke dalamnya. Pilih Add Component dari inspektur dan pilih Navigation > Off Mesh Link.
  6. Menetapkan silinder paling kiri di bidang Start dan silinder paling tepat di bidang End.

Sekarang Anda memiliki fungsi Off-Mesh Link mengatur! Jika jalan melalui tautan off-mesh lebih pendek daripada melalui berjalan di sepanjang Navmesh, tautan off-mesh akan digunakan.

Anda dapat menggunakan objek permainan di sceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info

Lihat di Glossary bisa mengandung komponen tautan off-mesh. Demikian pula Anda dapat menggunakan objek permainan dengan Transform sebagai penanda awal dan akhir.
