Tampilan dan menyembunyikan Login

  1. Klik di mana saja di Scene view dan pilih Spacebar untuk membuka menu Overlays.
  2. Klik Overlay Anda ingin menampilkan atau menyembunyikan. Jika Overlay sudah ditampilkan, ikon mata muncul di sebelah kirinya. Ketika Anda meluncurkan opsi yang ditampilkan, sorotan Overlay di biru di Scene view.

Sembunyikan semua Overlays

Hubungi kami Overlays ke hide/show dengan menekan backtick ( ` ).

Posisi Overlays

Untuk memposisikan Overlay di Scene view:

  • Klik dan tahan pegangan (=) atau klik header Overlay dan tarik ke posisi yang diinginkan di Scene view.

Sitemap Anda dapat tumpang tindih Overlays yang mengambang di Note:. Klik di Overlay untuk membawa Overlay ke depan.Scene view. Click on the Overlay to bring an Overlay to the front.

Dock Overlays

Untuk dermaga Overlay ke bagian atas, bawah, atau sisi Scene view:

  • Klik dan tahan pegangan (=) Overlay atau klik header Overlay dan tarik di atas batas Scene view sampai batas disorot dalam biru dan melepaskannya. Overlay akan dermaga sendiri di posisi yang disorot.

Untuk memindahkan Overlay ke posisi yang ditempati oleh Overlay dermaga lain:

  • Seret Overlay sampai sorotan biru menunjukkan posisi dan rilis yang diinginkan. Overlays akan mengatur ulang diri.

Sitemap Sebuah Overlay yang bukan Note: (misalnya, Orientasi) akan runtuh ketika dermaga. Akses opsi Overlay dengan mengklik panah bawah.toolbarSe baris tombol dan kontrol dasar di bagian atas Editor Unity yang memungkinkan Anda untuk berinteraksi dengan Editor dengan berbagai cara (misalnya scaling, terjemahan). Lihat di Glossary (for example, Orientation) will collapse when docked. Access the Overlay’s options by clicking the down arrow.

Kehilangan dan memperluas Login

Untuk mengetuk Overlay, klik kanan pegangan (=) atau klik perbatasan Overlay dan pilih Collap.

Untuk memperluas, klik kanan dan pilih Expand.

Snap Overlays

Untuk snap Overlay ke sudut Scene view:

  • Klik dan tahan pegangan (=) Overlay dan seret ke sudut Scene view sampai persegi berwarna biru muncul dan melepaskannya.

Sitemap Login Menghabiskan ke sudut tidak terkunci satu sama lain. Setiap Overlay tetap terkunci ke sudut sampai Anda mengubah posisinya.Note: Snapping several Overlays into the corner does not snap them to each other. Each Overlay remains snapped to the corner until you change its position.

Mengubah orientasi Overlay

Sitemap Overlay memungkinkan Anda untuk memilih orientasi vertikal atau horizontal.

Untuk mengubah orientasi Overlay:

  • Klik kanan pegangan (=) dan pilih Horizontal atau Vertical.
  • Jika Overlay mengambang atau terkunci dalam Scene view, pilih Panel untuk mengubah Overlay ke panel bernama.

Simpan konfigurasi Overlay

Anda dapat menyimpan konfigurasi Overlay Anda dengan mudah beralih konfigurasi di proyek Anda.

Untuk menyimpan Anda Konfigurasi Overlay sebagai preset:

  1. Tekan bilah ruang untuk membuka menu Overlays.
  2. Dari menu drop-down, pilih Save Preset.
  3. Nama preset dan klik Save. Login Preset disimpan ke menu drop-down.
    The preset is saved to the drop-down menu.

Sitemap Ketika Anda menyimpan preset Overlay dalam cara ini, itu tidak membuat file yang dapat Anda beban ke proyek lain, tetapi praset disimpan tersedia untuk setiap proyek Unity di komputer Anda.Note: When you save an Overlay preset in this way, it does not create a file that you can load to another project, but saved presets are available to any Unity project on your computer.

Konfigurasi Overlay juga disimpan ketika Anda menyimpan tata letak Editor.

Konfigurasi Overlay Switch

Untuk mengganti Konfigurasi overlay:

  1. Tekan bilah ruang untuk membuka menu Overlays.

  2. Pilih preset Overlay dari daftar drop-down.

    Konfigurasi Overlay ditunjukkan pada Scene view.

Impor dan ekspor Konfigurasi Overlay

Untuk mengimpor konfigurasi Overlay ke proyek Anda:

  1. Tekan bilah ruang untuk membuka menu Overlays.
  2. Dari menu drop-down, pilih Load Preset From File.
  3. Jelajahi file .wpr dan klik Open. Preset ditambahkan ke menu drop-down.

Untuk mengekspor konfigurasi Overlay, simpan file .wpr dan kemudian impor ke proyek lain:

  1. Tekan bilah ruang untuk membuka menu Overlays.
  2. Dari menu drop-down, pilih Save Preset to File.
  3. Nama preset dan klik Save. Preset disimpan ke folder dan di menu drop-down.
