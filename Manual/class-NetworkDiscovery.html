Jaringan Discovery
Important: UNet adalah solusi yang diuraikan, dan Solusi Multiplayer baru dan NetworkingSistem Unity yang memungkinkan game multiplayer di jaringan komputer. More info
Lihat di Glossary (Netcode untuk GameObjects) sedang berkembang. Untuk informasi lebih lanjut dan langkah selanjutnya melihat informasi di .

Network Discovery adalah komponen yang memungkinkan aplikasi Unity menggunakan sistem jaringan untuk menemukan satu sama lain di jaringan lokal (LAN). Ini tidak memungkinkan penemuan untuk bermain internet. Gunakan Layanan Multiplayer (bersama dengan layanan MatchMaker dan Relay) untuk terhubung ke .

Komponen Network Discovery tidak memerlukan integrasi dengan layanan Unity, dan dimaksudkan sebagai solusi yang sepenuhnya mandiri untuk menemukan game lain di jaringan lokal Anda untuk terhubung dengan.

Jaringan Discovery komponen di jendela InspectorJendela Unity yang menampilkan informasi tentang Pengaturan GameObject yang dipilih saat ini, aset atau proyek, memungkinkan Anda untuk memeriksa dan mengedit nilai. More info
Lihat di Glossary window

Property Function
Broadcast Port Jaringan port untuk menyiarkan dan mendengarkan.
Broadcast Key Kunci untuk disiarkan. Ini harus menjadi nilai unik untuk menunjukkan kompatibilitas penemuan Anda ke instance Discovery Jaringan lainnya. Kunci siaran unik menghindari permainan berbagai jenis dari mencoba untuk terhubung satu sama lain jika mereka berjalan di jaringan lokal yang sama.
Broadcast Version Versi utama untuk termasuk dalam siaran. Gunakan ini bersama dengan SubVersi Penyiaran untuk menunjukkan kompatibilitas versi.
Broadcast SubVersion Versi minor untuk termasuk dalam siaran. Gunakan ini bersama Versi Broadcast untuk menunjukkan kompatibilitas versi.
Broadcast Interval Tentukan bagaimana sering Unity harus menyiarkan informasi penemuan, dalam hitungan detik.
Use NetworkManager Aktifkan ini untuk menggunakan pengaturan Network ManagerKomponen Jaringan yang mengelola state jaringan proyek. More info
Lihat di Glossary untuk penyiaran, dan kemudian auto-join menemukan permainan.
Broadcast Data Masukkan data kustom untuk dimasukkan dalam siaran. Jaringan Manager menimpa ini jika Anda telah mengaktifkan Gunakan NetworkManager.
Show GUI Aktifkan ini untuk menunjukkan siaran default GUI dalam mode Bermain. GUI ini hanya dimaksudkan untuk pengujian pengembang.
Offset X X-axis mengimbangi siaran GUI. Pengaturan ini hanya terlihat jika Tampilkan GUI diaktifkan.
Offset Y Pengimbangan sumbu y dari siaran GUI. Pengaturan ini hanya terlihat jika Tampilkan GUI diaktifkan.

Ketika berjalan dalam mode bermain

Ketika berjalan dalam mode bermain, informasi berikut juga terlihat dalam inspektur:

Property Function
hostId Host Id digunakan untuk disiarkan.
running Benar jika saat ini penyiaran.
isServer Benar jika penyiaran sebagai server.
isClient Benar jika mendengarkan siaran sebagai klien.
broadcastsReceived Daftar pesan siaran yang diterima.
