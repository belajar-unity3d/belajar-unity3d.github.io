Rect Transform

Switch to Scripting

Komponen Rect Transform adalah rekan tata letak 2D dari komponen Transform. Di mana Transform mewakili satu titik, Transform Rect mewakili persegi panjang bahwa elemen UI(User Interface) Memungkinkan pengguna untuk berinteraksi dengan aplikasi Anda. Unity saat ini mendukung tiga sistem UI. More info
Lihat di Glossary dapat ditempatkan di dalam. Jika orang tua dari Transformator Rect juga Transformator Rect, Transform anak dapat juga menentukan bagaimana harus diposisikan dan berukuran relatif ke persegi panjang induk.

Properties

Property: Function:
Pos (X, Y, Z) Posisi titik pivot persegi panjang relatif terhadap jangkar. Titik pivot adalah lokasi di mana putar persegi panjang.
Width/Height Lebar dan tinggi persegi panjang.
Left, Top, Right, Bottom Posisi tepi persegi panjang relatif terhadap jangkar mereka. Ini dapat dianggap sebagai bantalan di dalam persegi panjang yang ditentukan oleh jangkar. Tampilkan di tempat Pos dan Lebar / Berat ketika jangkar dipisahkan (lihat di bawah). Untuk mengakses opsi ini klik kotak Anchor Presets persegi di sebelah kiri atas komponen RectTransform.
AnchorsAlat tata letak UI yang memperbaiki elemen UI ke elemen induk. Jangkar ditampilkan sebagai empat pegangan segitiga kecil di Tampilan Adegan dan informasi jangkar juga ditampilkan dalam Inspektur. More info
Lihat di Glossary		 Titik jangkar untuk sudut kiri bawah dan sudut kanan atas persegi panjang.
Min Titik jangkar untuk sudut kiri bawah persegi panjang didefinisikan sebagai sebagian kecil dari ukuran persegi panjang induk. 0,0 sesuai dengan jangkar ke sudut kiri bawah orang tua, sementara 1,1 sesuai dengan jangkar ke sudut kanan atas orang tua.
Max Titik jangkar untuk sudut kanan atas persegi panjang didefinisikan sebagai sebagian kecil dari ukuran persegi panjang induk. 0,0 sesuai dengan jangkar ke sudut kiri bawah orang tua, sementara 1,1 sesuai dengan jangkar ke sudut kanan atas orang tua.
Pivot Lokasi titik pivot di mana sudut putar, didefinisikan sebagai sebagian kecil dari ukuran persegi panjang itu sendiri. 0,0 sesuai dengan sudut kiri bawah sementara 1,1 sesuai dengan sudut kanan atas.
Rotation Sudut rotasi (dalam derajat) dari objek di sekitar titik pivot di sepanjang sumbu X, Y dan Z.
Scale Faktor skala diterapkan pada objek dalam dimensi X, Y dan Z.
Blueprint Mode Edit RectTransforms seolah-olah mereka tidak diputar dan berskala. Ini diaktifkan terkunci juga.
Raw Edit Mode Ketika diaktifkan, mengedit pivot dan nilai jangkar tidak akan mengatur posisi dan ukuran persegi panjang untuk membuatnya tetap di satu tempat.

Details

Perhatikan bahwa beberapa perhitungan RectTransform dilakukan pada akhir bingkai, hanya sebelum menghitung vertic UI, untuk memastikan bahwa mereka up to date dengan semua perubahan terbaru di seluruh bingkai. Ini berarti bahwa mereka belum dihitung untuk pertama kalinya dalam panggilan Start dan callback Update pertama.

Anda dapat bekerja di sekitar ini dengan membuat panggilan Start() dan menambahkan metode Canvas.ForceUpdateCanvases() untuk itu. Ini akan memaksa Canvas untuk diperbarui tidak pada akhir bingkai, tetapi ketika metode itu disebut.
