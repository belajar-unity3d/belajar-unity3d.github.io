Kelas DataPrivacy mengkonfigurasi layanan Unity AnalyticsUnity Analytics
Lihat di Glossary berdasarkan pilihan manajemen privasi data pemain.

NAMASPACE: UnityEngine.Analytics 

    public class DataPrivacy

Kelas DataPrivacy secara otomatis mengambil status privasi data pemain dan mengkonfigurasi layanan Analytics sesuai.

Gunakan fungsi FetchPrivacyUrl() untuk mengambil URL halaman manajemen data pribadi pemain. Buka URL untuk memberikan pemain opsi untuk mengelola pengaturan privasi data mereka.

Halaman ini rincian fungsi berikut:

  • Initialize() - Mempersiapkan API Privasi Data untuk digunakan (untuk Unity 5.1 atau sebelumnya).
  • FetchPrivacyUrl(Action<String>, Action<String>) - Gets URL dari halaman manajemen data pribadi pemain.

Initialize()

Mempersiapkan API Privasi Data untuk digunakan.

Declaration

    public static void Initialize()

Details

Fungsi ini menciptakan GameObjectObjek mendasar dalam adegan Unity, yang dapat mewakili karakter, props, pemandangan, kamera, waypoints, dan banyak lagi. Fungsi GameObject didefinisikan oleh Komponen yang melekat padanya. More info
Lihat di Glossary tersembunyi dan menambahkan contoh kelas DataPrivacy untuk itu sebagai komponen.

Pada Unity 5.1 atau sebelumnya, hubungi Initialize() di awal startup aplikasi Anda (secara pribadi, tepat setelah Anda memanggil UnityAnalytics.StartSDK (projectId). Versi terbaru dari panggilan Unity Initialize() secara otomatis.

FetchPrivacyUrl(Action<String>, Action<String>)

Dapatkan URL halaman manajemen data pribadi pemain.

Declaration

    public static void FetchPrivacyUrl(Action<string> success, Action<string> failure = null)

Parameters

  • Action<String> success - Objek aksi untuk membentang ketika URL berhasil diambil. string yang dioper ke Tindakan berisi URL.

  • [opsional] Action<String> failure - Keberatan tindakan untuk memanggil ketika Unity tidak dapat mengambil URL. string yang dioper ke Tindakan mengandung alasan kegagalan.

Details

Buka URL yang dikirimkan ke fungsi success Anda di browser atau webview untuk memberikan pemain kesempatan untuk mengelola opsi perlindungan data mereka. Anda dapat menggunakan Application.OpenURL() untuk membuka halaman.

URL berlaku untuk jangka waktu singkat. Selalu mengambil URL segera sebelum membukanya.
