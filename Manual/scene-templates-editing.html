Note:
Ketika Anda pertama kali membuat template adegan kosong, Anda harus mengedit sifatnya untuk mengasosiasikannya dengan adegan sebelum Anda dapat menggunakannya. Templat yang Anda buat dari adegan aktif, atau aset adegan yang ada, memiliki beberapa sifat yang ditetapkan secara default.

Details

Gunakan bagian Rincian untuk menentukan adegan mana yang digunakan untuk template, dan mengontrol bagaimana template muncul di Dialog Adegan Baru.


Property: Description:
Template Scene Tentukan adegan untuk digunakan sebagai template. Ini bisa menjadi adegan dalam Proyek.
Title Nama template. Nama yang Anda masukkan di sini muncul di Dialog Adegan Baru.
Description Deskripsi template. Deskripsi yang Anda masukkan di sini muncul di Dialog Adegan Baru.
Pin in New Scene Dialog Kontrol apakah template ini dipinned dalam template Dialog Adegan Baru.

Pinned selalu muncul di bagian atas daftar Scene Templates in Project.

Dependencies

Bagian ini mencantumkan semua Dependencies adegan template. Anda dapat menentukan apakah atau tidak ke Clone setiap ketergantungan ketika Anda menciptakan adegan baru dari template.

Untuk mencari ketergantungan dengan nama, masukkan teks di bilah pencarian.

Untuk memilah daftar Dependencies:

  • Klik judul Dependencies untuk memilah nama ketergantungan dalam urutan abjad.
  • Klik judul Type untuk memilah jenis ketergantungan.
  • Klik judul Clone untuk memilah dengan ketergantungan yang dikloning dan tidak dicukur.


Untuk setiap ketergantungan dalam daftar, toggle opsi Clone untuk mengkloning ketergantungan, atau mati untuk merujuk ketergantungan. Ketika Anda menginjak ketergantungan, Anda membuat salinan. Ketika Anda merujuk ketergantungan, semua perubahan yang dilakukan pada aslinya akan mempengaruhi ketergantungan.

Ketika Anda membuat adegan baru dari templat, pemeriksaan Unity apakah adegan template memiliki ketergantungan yang rentan. Jika itu, Unity membuat folder dengan nama yang sama seperti adegan baru, dan menempatkan ketergantungan gumpalan dalam folder itu.

Untuk informasi lebih lanjut tentang ketergantungan yang dikloning dan direferensikan, lihat Template dan ketergantungan adegan.

Untuk menentukan jenis aset Unity clones secara default, edit Sitemap Pengaturan proyek.
