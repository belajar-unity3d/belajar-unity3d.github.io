The Index Manager

Manajer Indeks memungkinkan Anda untuk membuat, mengedit, dan menghapus indeks pencarian. Pembaruan indeks ketika Anda menambahkan atau mengubah konten dalam proyek Anda.

Jenis indeks

  • Asset - Indeks semua Aset dalam Proyek. Indeks ini dibuat secara otomatis ketika Anda membuat Proyek dan membangun kembali ketika Anda menambahkan atau memodifikasi konten baru. Indeks Aset mencakup semua Aset termasuk Prefab dan adegan, tetapi tidak mengindeks konten sceneAdegan berisi lingkungan dan menu permainan Anda. Pikirkan setiap file Adegan unik sebagai tingkat yang unik. Di setiap Adegan, Anda menempatkan lingkungan, hambatan, dan dekorasi, pada dasarnya merancang dan membangun permainan Anda dalam potongan-potongan. More info

    Lihat di Glossary    .
  • Package - Indeks yang mengandung Aset dalam Paket.
  • Prefab - Indeks yang mengandung hanya GameObjectsObjek mendasar dalam adegan Unity, yang dapat mewakili karakter, props, pemandangan, kamera, waypoints, dan banyak lagi. Fungsi GameObject didefinisikan oleh Komponen yang melekat padanya. More info
    Lihat di Glossary     yang merupakan bagian dari Prefab.
  • Scene - Indeks Aset di setiap adegan
  • User - Indeks yang dibuat untuk Anda dan tidak dibagikan dengan proyek.

Secara default Unity membuat indeks Aset. Untuk menggunakan Prefab, Paket, Adegan, dan indeks Pengguna, Anda harus membuat mereka di Manajer Indeks.

Untuk membuka Manajer Indeks pilih Windows > Search > Index Manager.

Hasil indeks

Tab berikut mengandung informasi tentang indeks:

  • Sitemap Menampilkan file yang diindeks.Assets: Displays the files indexed.
  • Objects: (Prefab dan Adegan indeks hanya) Menampilkan pengenal objek.
  • Sitemap Menampilkan Properti atau filter yang tersedia yang dapat Anda gunakan untuk lebih mempersempit hasil pencarian Anda.Keywords: Displays the available searchable Properties or filters you can use to further narrow your search results.

Ukuran indeks

Untuk menentukan pilihan indeks yang Anda butuhkan, lihat tabel ini.

Ukuran proyek Indeks yang diperlukan
Kurang dari 1000 aset Minimal - memungkinkan set minimal opsi pengindeksan
Antara 1000 dan 2000 aset Default - indeks aset proyek Anda dengan File dan Type
Lebih dari 2000 aset Diperpanjangkan - mengindeks proyek Anda dengan File, Type, Properties, Sub objek, dan Dependencies

Sitemap Semakin banyak pilihan yang Anda tambahkan ke indeks, semakin lama akan mengambil untuk menghasilkan indeks pertama Anda.Note: The more options you add to the index, the longer it will take to generate your first index.

Tambahkan indeks

  1. Buka Index Manager.
  2. Klik tanda plus (+) di pane kiri dari jendela Index Manager dan pilih jenis indeks yang ingin Anda buat.
  3. Sesuaikan pengaturan indeks Anda di pane kanan dan klik Create.
  4. Nama indeks Anda di jendela Save Index Settings dan klik Save. Login Indeks Anda membangun secara otomatis. Tergantung pada ukuran proyek dan pengaturan indeks Anda, ini dapat mengambil beberapa waktu.
    Your index builds automatically. Depending on the size of the project and your index settings, this may take some time.

Edit indeks

  1. Buka Index Manager.
  2. Pilih indeks dari pane kiri dari jendela Index Manager.
  3. Membuat perubahan nama indeks atau pengaturan dan klik Save.

Sitemap Jika Anda melakukan perubahan tanpa menyimpan indeks, asterisk (*) muncul di sebelah judul jendela. Jika Anda mencoba untuk menutup jendela tanpa menyimpan, meminta Unity Anda untuk menyimpan indeks Anda. Ketika Anda menyimpan, membangun kembali indeks Anda dengan perubahan Anda.Note: If you make changes without saving your index, an asterisk (*) appears next to the window title. If you try to close the window without saving, Unity prompts you to save your index. When you save, Unity rebuilds your index with your changes.

Hapus indeks

  1. Buka Index Manager.
  2. Pilih indeks dari pane kiri dari jendela Index Manager.
  3. Klik tanda minus (-).
  4. Klik Yes untuk mengkonfirmasi bahwa Anda ingin menghapus indeks.
