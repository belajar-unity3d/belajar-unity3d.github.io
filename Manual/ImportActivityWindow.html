Jendela Aktivitas impor

Overview

Jendela Aktivitas Impor memberi Anda informasi tentang apa yang terjadi ketika mengimpor aset Anda. Anda dapat menggunakan informasi ini untuk mengidentifikasi aset mana di proyek Anda diimpor baru-baru ini, berapa lama setiap aset yang diperlukan untuk impor, dan mengapa diimpor (atau re-imported).

Informasi ini memungkinkan Anda untuk menganalisis aktivitas impor proyek Anda, dan membuat keputusan tentang cara meningkatkan waktu Unity diperlukan untuk mengimpor aset Anda, atau cara menghindari impor yang tidak perlu sama sekali. Jendela Aktivitas Impor oleh karena itu bertindak seperti profilerJendela yang membantu Anda untuk mengoptimalkan permainan Anda. Ini menunjukkan berapa banyak waktu yang dihabiskan di berbagai bidang permainan Anda. Sebagai contoh, dapat melaporkan persentase waktu yang dihabiskan rendering, aimating, atau dalam logika permainan Anda. More info
Lihat di Glossary untuk proses impor.

Mengakses jendela Aktivitas Impor

Untuk membuka jendela Aktivitas Impor, buka Window > Analysis > Import Activity Window.

Anda juga dapat membuka jendela Aktivitas Impor langsung dari aset, yang menyebabkan jendela untuk segera menampilkan rincian impor untuk aset yang dipilih. Ini berguna jika Anda sudah tahu bahwa data impor aset yang ingin Anda analiyse. Ada dua cara untuk melakukan ini:

Layout

Jendela Aktivitas Impor memiliki tiga bagian: toolbarA row of buttons and basic controls at the top of the Unity Editor that allows you to interact with the Editor in various ways (e.g. scaling, translation). More info
See in Glossary at atas, asset list di sebelah kiri, dan bagian information di sebelah kanan.

Toolbar

Toolbar di bagian atas jendela memiliki tiga fitur berikut:

  1. Show overview tombol
  2. Options menu
  3. Search field

Tampilkan tombol Ikhtisar

Ketika Anda mengklik tombol Show Overview, bagian informasi pada informasi gambaran yang tepat, yang mencakup informasi umum tentang impor aset Anda. Lihat informasi Ikhtisar, di bawah ini.

menu pilihan

Dropdown Options memungkinkan Anda untuk menyesuaikan informasi yang ditunjukkan dalam daftar aset. Opsi berikut tersedia:

Gunakan kalitamps relatif

Mengontrol apakah waktu impor terakhir ditunjukkan dalam format relatif (misalnya "seeberapa detik yang lalu"), atau format absolut, dalam bentuk: Day-Month-Year Hour:Minute:Second

Tampilkan impor sebelumnya

Kontrol apakah impor sebelumnya ditampilkan, atau hanya impor terbaru. Menunjukkan impor sebelumnya memungkinkan Anda untuk melihat berapa banyak revisi aset saat ini disimpan di folder Perpustakaan. Daftar ini umumnya dibersihkan ketika Artifact Garbage Collection berjalan, ketika Anda restart Editor.

Ketika Anda mengaktifkan opsi ini, tampilan daftar vertikal kedua muncul di samping daftar aset, menunjukkan semua hasil impor saat ini disimpan untuk aset yang dipilih.

Jika Anda ingin menyimpan hasil impor dari sesi Editor sebelumnya untuk membantu dengan debugging atau analisis, Anda dapat mematikan Koleksi Sampah Artief dengan pergi ke Project Settings > Editor > Remove unused Artifacts on Restart. Anda juga dapat mengontrol pengaturan ini melalui script dengan menggunakan WordPress.org.

Sertakan PreviewImporter

Kontrol apakah termasuk artefak yang dihasilkan oleh jendela pratinjau. Ini tersembunyi secara default, karena mereka tidak biasanya berguna ketika memeriksa impor aset Anda.

Search field

Bidang pencarian di sisi kanan bilah alat memungkinkan Anda untuk dengan cepat menyaring asset list dengan nama untuk menemukan aset tertentu yang Anda minati.

Daftar aset

Bagian kiri dari jendela menunjukkan daftar semua aset dalam proyek Anda. Kolom adalah:

  • Asset: Nama aset

  • Last Import: Ketika aset terakhir diimpor

  • Duration: Jumlah waktu impor terakhir mengambil, dalam mili detik.

Untuk mengubah urutan daftar, klik pada judul kolom.

Daftar aset, dengan hasil yang ditunjukkan diurutkan oleh durasi terpanjang terlebih dahulu.

Panel informasi

Bagian kanan dari jendela adalah panel informasi, dan memiliki dua mode:

  • Informasi ikhtisar muncul ketika tidak ada aset yang dipilih.

  • Informasi aset muncul ketika Anda memilih aset dari daftar di sebelah kiri, atau jika Anda membuka jendela langsung melalui aset seperti yang dijelaskan di atas.

Informasi

Informasi Ikhtisar menampilkan daftar aset dengan ketergantungan yang paling, dan aset dengan durasi impor terpanjang. Hal ini berguna untuk mengidentifikasi aset mana yang mungkin paling lambat proses impor Anda. Aset dengan lebih banyak ketergantungan lebih cenderung secara teratur dikembangkan kembali, karena aset direportasi setiap kali setiap ketergantungannya dimodifikasi.

Ketika Anda membuka jendela Aktivitas Impor dari menu utama, menampilkan informasi Ikhtisar secara default. Jika Anda melihat informasi aset dan Anda ingin kembali ke informasi Ikhtisar, pilih tombol Show overview di toolbar.

Informasi aset

Ketika Anda memilih aset dari daftar aset, panel informasi menampilkan informasi aset. Ini termasuk rincian berikut tentang aset dan impor terbaru.

Title Description
Asset Nama aset yang dipilih saat ini. Anda dapat mengklik bidang ini untuk menemukan aset di jendela Proyek.
GUID GUID unik yang ditugaskan ke aset. Anda dapat menggunakan ini untuk melacak referensinya di proyek Anda.
Asset Size Ukuran file sumber aset.
Path Jalur file aset, relatif terhadap folder akar proyek.
Editor Revision Versi Editor yang menciptakan artefak ini.
Timestamp Waktu di mana file Artief diciptakan. Ini cocok dengan nilai timestamp dari file di folder Library.
Duration Waktu Unity perlu mengimpor aset ini dalam impor terbaru, dalam mili detik.
Reason for Import Deskripsi perubahan (atau perubahan) dalam proyek Anda yang menyebabkan impor terbaru aset ini, dan rincian tentang jenis ketergantungan terkait.

Alasan untuk bidang impor memiliki bilah pencarian sendiri, yang memungkinkan Anda untuk mencari teksnya. Ini berguna dalam situasi di mana aset memiliki banyak alasan untuk impor (misalnya, naungan compute dapat memiliki banyak ketergantungan yang dimodifikasi ketika Anda beralih platform).
Produced Files / Artifacts Daftar file yang ada di folder Library yang diproduksi Unity ketika terakhir mengimpor aset ini. Biasanya hanya ada satu artefak per aset, meskipun kadang-kadang ada banyak. Misalnya, ketika Unity menghasilkan pratinjau aset.

Selama penggunaan khas Editor, Unity mungkin menghasilkan banyak versi dari hasil impor aset ( artefak). Untuk alasan ini, setiap artefak memiliki ID artefak untuk referensi. Ini berbeda dengan GUID aset.

Revisi yang dipilih saat ini disebut sebagai revisi saat ini, yang dipilih secara default ketika aset di Daftar Aset dipilih. revisi

Generally revisions are in chronological order with the current revision as the newest. However, if you undo an operation in Unity which changes an asset’s artifact, the current revision reverts back to a previously cached artifact and therefore would not be the most recently listed artifact.
Dependencies Daftar editor dan variabel spesifik proyek yang mengontrol keadaan aset, yang, jika berubah, memicu re-impor aset.

Dependencies adalah bagaimana database aset melacak keadaan aset. Ini berarti bahwa jika perubahan ketergantungan aset, hasil impor akan berbeda dan revisi baru dari artefak akan dihasilkan.



Bidang ini memiliki bilah pencarian sendiri, yang memungkinkan Anda untuk mencari teks yang terkandung di bidang Dependencies.
