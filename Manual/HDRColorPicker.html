HDR warna picker

Penjemputan warna HDRkisaran dinamis tinggi
Lihat di Glossary terlihat mirip dengan pemilih warna biasa, tetapi mengandung kontrol tambahan untuk menyesuaikan paparan warna.

Property: Function:
Mode
(default: RGB 0–255)		 Saat menggunakan mode HSV atau RGB 0–255, pemilih warna memperlakukan penyesuaian eksposur secara independen dari data saluran warna. Namun, data saluran warna yang ditampilkan dalam mode RBG (0–1.0) mencerminkan hasil penyesuaian paparan Anda pada data warna. Login Tidak seperti pemilih warna biasa, Anda dapat langsung memasukkan nilai float lebih dari 1.0 ketika mengedit saluran warna dalam mode
. Dalam hal ini, pemilih warna derives nilai RGB 0–1.0 secara otomatis dari nilai yang Anda set.Intensity value automatically from the value you set.
RGBA Gunakan slider atau kotak teks untuk menentukan nilai RGBA. Nilai Hexadecimal secara otomatis diperbarui untuk mencerminkan nilai RGBA.
Intensity Gunakan slider Intensity untuk mengekspos atau menggarisbawahi warna. Setiap langkah positif sepanjang slider menyediakan dua kali lebih banyak cahaya sebagai posisi slider sebelumnya, dan setiap langkah negatif memberikan setengah sebanyak cahaya. Login Gunakan swatch eksposur di bawah slider
untuk meninjau apa nilai warna saat ini terlihat seperti dalam berbagai dua langkah dalam arah. Untuk dengan cepat menyesuaikan paparan warna, klik swatch pratinjau.Intensity slider to preview what the current color value looks like within a range of two steps in either direction. To quickly adjust the color’s exposure, click a preview swatch.
Swatches Gunakan bagian Swatches untuk menghemat warna menjadi swatch library. Anda dapat menggunakan kembali, menyimpan, dan berbagi warna, gradien, dan animation curvesMemungkinkan Anda untuk menambahkan data ke klip impor sehingga Anda dapat menghidupkan waktu item lain berdasarkan keadaan animator. Misalnya, untuk set permainan dalam kondisi icy, Anda dapat menggunakan kurva animasi ekstra untuk mengontrol tingkat emisi sistem partikel untuk menunjukkan napas kondensor pemain di udara dingin. Note: Kapan pun Anda menutup jendela warna HDR dan membuka kembali, jendela derives saluran warna dan nilai intensitas dari warna yang Anda edit. Karena ini, Anda mungkin melihat nilai yang sedikit berbeda untuk saluran warna dalam mode HSV dan RGB 0–255 atau untuk slider Intensity, meskipun nilai saluran warna dalam mode RGB 0–1.0 sama dengan waktu terakhir Anda mengedit warna.
