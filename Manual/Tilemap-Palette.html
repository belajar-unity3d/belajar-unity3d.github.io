Membuat palet ubin

Tempatkan pilihan ubin ke Tile Palettes sehingga Anda dapat memilih ubin dari palet ke cat di Tilemaps. Untuk membuat jendela Tile Palette, buka jendela Tile Palette dengan pergi ke Window > 2D > Tile Palette. Jika Anda tidak memiliki opsi ini, maka paket 2D Tilemap Editor tidak dapat diinstal. Jika demikian, unduh paket 2D Tilemap Editor dari Package Manager.

Jendela palet ubin kosong secara default. Pilih menu drop-down New Palette untuk membuka daftar palet ubin yang tersedia di Proyek, atau untuk opsi untuk membuat palet baru. Pilih opsi Create New Palette untuk membuat palet baru.


Pilih opsi Buat Palet Baru dari menu drop-down.

Setelah memilih opsi untuk membuat palet ubin baru, kotak dialog Create New Palette menjadi tersedia. Ini berisi pengaturan dan opsi properti yang berbeda yang tersedia ketika membuat palet baru:

Mengedit Palet Tile

Alat untuk memilih dan melukis dengan ubin juga dapat digunakan untuk mengedit palet ubin secara langsung, memungkinkan Anda untuk bergerak dan memanipulasi ubin saat ini ditempatkan pada palet ubin. Pilih palet yang ingin Anda edit dari menu drop-down 'Palette', lalu klik Edit untuk membuka palet untuk mengedit.

Lihat dokumentasi pada Lukisan di Tilemaps untuk pintasan dan fungsi alat palet yang tersedia untuk mengedit palet juga.

Membuat Aset Palet dari Prefab Grid yang ada

Anda dapat mengubah PrefabJenis aset yang memungkinkan Anda untuk menyimpan GameObject lengkap dengan komponen dan properti. Prefab bertindak sebagai template dari mana Anda dapat membuat instance objek baru di tempat kejadian. More info
Lihat di Glossary yang ada ke Aset Palet, sehingga Anda dapat menggunakannya di jendela palet ubin. Untuk melakukan ini, Prefab tidak boleh menjadi Aset Palet, dan harus memiliki komponen Grid pada GameObjectObjek mendasar dalam adegan Unity, yang dapat mewakili karakter, props, pemandangan, kamera, waypoints, dan banyak lagi. Fungsi GameObject didefinisikan oleh Komponen yang melekat padanya. More info
Lihat di Glossary paling atas.

Untuk mengubah prefab, drag dan drop ke bilah alat palet ubin (tertinggikan dalam gambar di atas). Editor secara otomatis mengubahnya menjadi Aset Palet, dan menambahkan Aset Palet Grid. Aset Palet baru memiliki nama yang sama dengan sumbernya, dan Anda dapat memilihnya dari menu drop-down palet.

Keramik palet Gizmos

Palet ubin dapat menampilkan ikon (GizmosOverlay grafis yang terkait dengan GameObject di Adegani, dan ditampilkan di Tampilan Adegan. Alat adegan internal seperti alat bergerak adalah Gizmos, dan Anda dapat membuat Gizmos kustom menggunakan tekstur atau scripting. Beberapa Gizmos hanya ditarik ketika GameObject dipilih, sementara Gizmos lainnya ditarik oleh Editor terlepas dari GameObjects mana yang dipilih. More info
Lihat di Glossary) atas Aset Palet saat ini, untuk membantu Anda memvisualisasikan kriteria spesifik. Misalnya, Anda dapat menambahkan Gizmo yang menampilkan ikon khusus untuk Ubin yang tidak mengandung Sprites.

Untuk menampilkan Unity default dan Gizmos Aset Palet pada Palet Tile, aktifkan toggle Gizmos (tertinggikan gambar di atas). Palet Tile segera menampilkan komponen apa pun dengan Aset Palet.

Menu kontekstual palet ubin

Klik kanan pada bilah alat palet ubin untuk membawa daftar menu kontekstual beberapa tindakan yang dapat Anda gunakan untuk memilih dan menemukan benda-benda yang palet ubin berinteraksi dengan. Tindakan yang tersedia tercantum di bawah ini:

Action Function
Select Paint Target Pilih GameObject di jendela Hierarchy yang merupakan peta ubin aktif saat ini di palet ubin.
Select Palette Prefab Pilih Aset Prefab Palet di jendela Proyek yang saat ini aktif di palet ubin.
Select Tile Asset Pilih Aset Tile yang dipilih saat ini di jendela Proyek.
Unlock/Lock Palette Editing Membuka kunci atau mengunci palet ubin untuk mengedit.
