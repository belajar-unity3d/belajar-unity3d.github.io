Sumber video

Web Browser dapat memainkan konten yang diimpor dari berbagai sumber.

Video Klip

Untuk membuat dan menggunakan Aset Klip Video, Anda harus mengimpor file video pertama.

Menyeret dan menjatuhkan file video ke dalam Project windowJendela yang menunjukkan isi folder Assets (Project tab) More info
Lihat di Glossary membuat Klip Video.

Cara lain untuk membuat Klip Video adalah untuk menavigasi ke AssetsSetiap media atau data yang dapat digunakan dalam permainan atau proyek Anda. Aset mungkin berasal dari file yang dibuat di luar Unity, seperti model 3D, file audio atau gambar. Anda juga dapat membuat beberapa jenis aset di Unity, seperti Pengontrol Animator, Audio Mixer atau Tekstur Render. More info
Lihat di Glossary > Import New Asset… untuk mengimpor file video.

Setelah diimpor, Klip Video yang baru dibuat dapat dipilih di komponen Video Player menggunakan jendela Select VideoClip (dapat diakses dengan mengklik lingkaran pilih tombol ke kanan kolom Klip Video) atau dengan menyeret dan menjatuhkan Aset Klip Video ke dalam bidang komponen Video Player yang sesuai.

URLs

Gunakan menu drop-down Source untuk mengatur sumber video ke URL (kaya ini diatur ke Video Clip secara default). Menyiapkan Source ke URL memungkinkan untuk langsung menggunakan file di sistem file, dengan atau tanpa prefiks file://.

Opsi sumber URL bypasses Manajemen aset, berarti Anda harus memastikan bahwa video sumber dapat ditemukan oleh Unity. Misalnya, URL web membutuhkan server web untuk menghosting video sumber, sementara file normal harus terletak di suatu tempat di mana Unity dapat menemukannya, diindikasikan dengan scripting. Namun, ini dapat berguna untuk situasi di mana konten tidak berada di bawah kontrol langsung Unity, atau jika Anda ingin menghindari menyimpan file video besar secara lokal.

Menyiapkan sumber komponen Video Player ke URL juga dapat digunakan untuk membaca video dari sumber web melalui http:// dan https://. Dalam kasus ini, Unity melakukan manajemen pra-buffering dan kesalahan yang diperlukan.

Asset Bundles

Klip video juga bisa dibaca dari Asset Bundles.

Setelah diimpor, Klip Video ini dapat digunakan dengan menetapkannya ke bidang Video Player Video Clip.

Aset Streaming

File yang ditempatkan di folder Unity StreamingAssets dapat digunakan melalui opsi URL Video Player (lihat di atas), atau dengan menggunakan jalur spesifik platform (Aplikasi.streamingAssetsPath).
